# <a name="edit-an-existing-topic-preview"></a>編輯現有的主題 (預覽) 

> [!Note] 
> 本文內容適用于 Project Cortex 私人預覽。 [如需詳細資訊，請參閱 Project Cortex](https://aka.ms/projectcortex)。

</br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4LA4n]  

</br>

在主題經驗中，您可以編輯現有的主題。 如果您想要更正或將其他資訊新增至現有的主題頁面，您可能需要執行此動作。 

## <a name="requirements"></a>需求

若要編輯現有的主題，您必須：
- 有一個主題經驗豐富授權。
- 具有 [**可建立或編輯主題的人員**](https://docs.microsoft.com/microsoft-365/knowledge/topic-experiences-user-permissions)許可權。 知識系統管理員可以在知識網路主題許可權設定中，將此許可權授與使用者。 

> [!Note] 
> 有權管理主題中心主題的使用者 (知識管理員) 已具備建立及編輯主題的許可權。

## <a name="edit-a-topic-page"></a>編輯主題頁面

如果您有編輯主題頁面的許可權，開啟主題後，您就可以在頁面右上方看到 [ **編輯** ]。

   ![編輯控制項](../media/knowledge-management/topic-page-edit.png) </br> 

1. 在 [主題] 頁面上，選取 [ **編輯**]。

2. 對頁面進行所需的變更。 這包括下欄欄位的更新：

    -  替代名稱
    -  說明
    -  固定人員
    -  檔案與頁面
    -  網站

    您也可以將靜態專案新增至頁面（如文字、圖像或連結），方法是選取 [畫布] 圖示。

3. 選取 [重新 **發佈** ] 以儲存變更。
