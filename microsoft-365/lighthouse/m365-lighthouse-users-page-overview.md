# <a name="microsoft-365-lighthouse-users-page-overview"></a>Microsoft 365 Lighthouse使用者頁面概述 

> [!NOTE]
> 本文所述的功能都是在預覽中進行變更，而且只有符合 [需求](m365-lighthouse-requirements.md)的合作夥伴才能使用。 如果您的組織沒有 Microsoft 365 Lighthouse，請參閱[註冊 Microsoft 365 Lighthouse](m365-lighthouse-sign-up.md)。

Microsoft 365 Lighthouse 可讓您在不同租使用者帳戶中，選取 [**使用者**] 以開啟 [使用者] 頁面，以管理各租使用者帳戶的使用者。 您可以從這個頁面搜尋使用者，並評估使用者帳戶的安全性狀態並採取行動。 您也可以查看深入瞭解危險的使用者，以及多重要素驗證和自助密碼重設的狀態。  
  
## <a name="search-users-tab"></a>搜尋使用者] 索引標籤  
  
您可以從 [搜尋使用者] 索引標籤，針對特定使用者快速搜尋各個承租人，並執行基本的使用者管理動作，例如重設帳戶密碼。

:::image type="content" source="../media/m365-lighthouse-users-page-overview/users-search-users-tab.png" alt-text="[搜尋使用者] 索引標籤的螢幕擷取畫面。":::

## <a name="risky-users-tab"></a>危險使用者] 索引標籤

[危險的使用者] 索引標籤會顯示您的承租人中已標示為危險行為的使用者帳戶。 選取任何使用者，以查看偵測到之風險的詳細資訊，或重設使用者的密碼或封鎖登入，以減輕風險。

:::image type="content" source="../media/m365-lighthouse-users-page-overview/users-risky-users-tab.png" alt-text="危險使用者] 索引標籤的螢幕擷取畫面。":::

## <a name="multifactor-authentication-tab"></a>多重因素驗證] 索引標籤

[多重要素驗證] 索引標籤提供有關多因素驗證狀態的詳細資訊，請 (MFA) 在您的承租人中啟用。 選取清單中的任何租使用者，以查看該租使用者的詳細資訊，包括已設定需要 MFA 的條件式存取原則，以及哪些使用者尚未為 MFA 註冊。

:::image type="content" source="../media/m365-lighthouse-users-page-overview/users-mfa-tab.png" alt-text="多重因素驗證] 索引標籤的螢幕擷取畫面。":::

## <a name="password-reset-tab"></a>重設密碼] 索引標籤

[密碼重設] 索引標籤會顯示在您的承租人內啟用自助密碼重設狀態的詳細資訊。

:::image type="content" source="../media/m365-lighthouse-users-page-overview/users-password-reset-tab.png" alt-text="[密碼重設] 索引標籤的螢幕擷取畫面。":::
