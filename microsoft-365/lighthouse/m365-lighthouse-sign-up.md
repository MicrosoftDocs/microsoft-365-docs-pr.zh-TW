# <a name="sign-up-for-microsoft-365-lighthouse"></a>註冊 Microsoft 365 Lighthouse

> [!NOTE]
> 本文所述的功能都是在預覽中進行變更，而且只有符合 [需求](m365-lighthouse-requirements.md)的合作夥伴才能使用。 如果您的組織沒有 Microsoft 365 Lighthouse，請依照本文中的指示進行註冊。

## <a name="before-you-begin"></a>開始之前

- 請確認您和您的客戶租使用者符合[Microsoft 365 Lighthouse 需求](m365-lighthouse-requirements.md)中所列的需求。

- 您必須是您要登入之協力廠商租使用者的全域系統管理員。

## <a name="steps-to-sign-up-for-microsoft-365-lighthouse"></a>註冊 Microsoft 365 Lighthouse 的步驟

1. 移至 Microsoft 365 系統管理中心 <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a> 。 

1. 移至 **帳單**  >  **購買服務**  >  **其他服務**。

    如果 [服務] 清單中沒有 Microsoft 365 Lighthouse，請選取 [**查看所有其他服務產品**]。

    如果您遇到 [購買服務錯誤] 頁面，請選取頁面底部的 [ **查看產品** ]。

1. 在 [**合作夥伴公開預覽 Microsoft 365 Lighthouse**] 底下，選取 [**詳細資料**]。 

1. 選取 [ **購買**]。

    > [!NOTE]
    > Microsoft 365 Lighthouse 需要租使用者的一個授權。 不需要其他每一使用者授權。 

    若要確認 Microsoft 365 Lighthouse 已成功新增至您的租使用者，請在 Microsoft 365 系統管理中心的 [**帳單 > 產品**] 中尋找 Microsoft 365 Lighthouse。

1. 如果您未重新導向至 Microsoft 365 Lighthouse 入口網站，請移至 <a href="https://go.microsoft.com/fwlink/p/?linkid=2168110" target="_blank">https://lighthouse.microsoft.com</a> 。

1. 選取 [ **同意 & 繼續** 完成合作夥伴合約修正。

    > [!NOTE]
    > 完成註冊後，客戶資料可能需要長達48小時才能出現 Microsoft 365 Lighthouse。
