# <a name="overview-of-microsoft-365-lighthouse"></a>Microsoft 365 Lighthouse 綜述

> [!NOTE]
> 本文所述的功能都是在預覽中進行變更，而且只有符合 [需求](m365-lighthouse-requirements.md)的合作夥伴才能使用。 如果您的組織沒有 Microsoft 365 Lighthouse，請參閱[註冊 Microsoft 365 Lighthouse](m365-lighthouse-sign-up.md)。

Microsoft 365 Lighthouse 是協助受管理的服務提供者 (MSPs) 安全且管理裝置、資料和使用者規模，以供小型和中型企業 (SMB) 客戶使用 Microsoft 365 商務進階版的系統管理員入口網站。 

Microsoft 365 Lighthouse 會建議針對中小企業客戶量身定制的安全性設定基線，並在所有客戶環境中提供多承租人視圖，以簡化 Microsoft 365 商務進階版承租人的加入。 透過 Microsoft 365 Lighthouse，MSPs 可以擴充其客戶的管理，將重點放在最重要的專案上，快速找出並調查風險，並採取行動讓客戶進入健康且安全的狀態。

使用 Microsoft 365 Lighthouse 來管理 Microsoft 365 服務和連線的裝置時，不會有其他相關的成本。 Microsoft 365 Lighthouse 目前正在預覽中，且可供 MSPs 登記雲端解決方案提供者 (CSP) 計畫，並為 SMB 客戶提供 Microsoft 365 商務進階版訂閱。

請注意，Microsoft 365 Lighthouse 目前不支援 CSP 間接提供者。 

> [!IMPORTANT] 
> 若要使用 Microsoft 365 Lighthouse，MSPs 及其客戶租使用者必須符合[Microsoft 365 Lighthouse 需求](m365-lighthouse-requirements.md)中所列的需求。     

如需 CSP 程式的詳細資訊，請參閱[雲端解決方案提供者程式概述](/partner-center/csp-overview)。

> [!NOTE]  
> 類似的產品，Azure Lighthouse 可協助服務提供者使用 Azure 平臺內建的全面且可靠的管理工具，為 Azure 服務提供受管理的服務。 若要深入瞭解，請參閱 [什麼是 Azure Lighthouse？](/azure/lighthouse/overview)   

## <a name="microsoft-365-lighthouse-benefits"></a>Microsoft 365 Lighthouse 優點

Microsoft 365 Lighthouse 會協助 MSPs 以下列方式保護和管理 Microsoft 365 服務和連接的端點：

- 提供承租人部署 journeys，讓技術人員可以遵循一組一致的步驟來保護及設定客戶承租人。 
- 使用預設的 SMB 安全性基準，以規定針對小型和中型商務承租人的最佳作法。 
- 在裝置規範中提供多租使用者見解，以清晰瞭解如何在所有組織中評估裝置、比較原則的工具，以及未滿足的最上層設定。 
- 簡化常見工作，例如重設密碼。
- 設定多重要素驗證和自助密碼重設，包括説明推動使用者採用的工具。 
- 瞭解和防範危險登入。
- 透過提供 Microsoft Defender 所偵測到之威脅的詳細資料和採取的動作來處理 Windows 10 裝置上的威脅，以解決問題，並維持最新的裝置。
- 提供深入瞭解 Microsoft 365 服務事件及建議，以影響客戶所管理的客戶承租人。

> [!NOTE] 
> 如需如何註冊的詳細資訊，請參閱[註冊 Microsoft 365 Lighthouse](m365-lighthouse-sign-up.md)。
