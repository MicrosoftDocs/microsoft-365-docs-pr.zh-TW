# <a name="configure-windows-10-devices"></a>設定 Windows 10 裝置

在安裝期間，您可以建立裝置原則套用至組織中的所有使用者的 Windows 10。 當使用者登入他們的公司或學校帳戶，您的設定與 Windows 10 裝置上的 Office 應用程式會自動套用至確定使用者正在連線安全的方式。
  
建議您在設定時接受預設設定。 在安裝之後，您可以建立更多原則，並鎖定至特定使用者群組。
  
若要新增原則來管理 Windows 10 裝置設定，請參閱[設定 Windows 10 電腦裝置設定](protection-settings-for-windows-10-pcs.md)。
