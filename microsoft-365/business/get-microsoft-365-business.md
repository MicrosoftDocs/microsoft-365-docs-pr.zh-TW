# <a name="get-microsoft-365-business-premium"></a>取得 Microsoft 365 商務進階版

## <a name="get-microsoft-365-business-premium-from-microsoft"></a>從 Microsoft 取得 Microsoft 365 商務進階版

如果您沒有合作夥伴，且想要取得 Microsoft 365 商務進階版，您可以在[這裡購買](https://www.microsoft.com/en-US/microsoft-365/business)。

如需詳細指示，請參閱[註冊 Microsoft 365 商務進階版](sign-up.md)。

您也可以使用[Microsoft Store](https://www.microsoft.com/en-us/store/locations/find-a-store?icid=en_US_Store_UH_FAS)購買 Microsoft 365 商務進階版並取得安裝程式的說明。
  
## <a name="get-microsoft-365-business-premium-from-microsoft-partner-center"></a>從 Microsoft Partner Center 取得 Microsoft 365 商務進階版

1. 前往 [Microsoft 合作夥伴中心](https://go.microsoft.com/fwlink/p/?linkid=849910)使用您在註冊雲端服務提供者 (CSP) 計劃時建立的認證登入。 
    
2. 在 [合作夥伴儀表板] 上，選擇 [**客戶**]，然後選取您的客戶，或加入新客戶，再取得 Microsoft 365 商務進階版。
    
    ![在 Microsoft Partner center 中，新增客戶。](../media/ec807d07-bbd2-411f-8fe1-c644cf9a3882.png)
  
3. 在客戶的 [**訂閱**] 頁面上，選取 [**新增訂閱**]，選擇 [目錄] 底下的 [小型商務] 選項，然後選擇 [ **Microsoft 365 商務進階版**]。
    
    選取您需要的授權數量 (最多 300 個)。 如果您的使用者超過300，請參閱[Microsoft 365 企業版](../enterprise/index.yml)。 
    
    ![在 [新增訂閱] 頁面上，選擇 [小型企業]。](../media/52d99e89-2175-4974-84bb-dd626048541b.png)
  
    完成新增客戶的其餘步驟，包括公司名稱。
