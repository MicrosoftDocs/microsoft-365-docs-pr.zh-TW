# <a name="remove-company-data-from-devices"></a>從裝置移除公司資料

## <a name="remove-company-data"></a>移除公司資料

您可以使用 Microsoft 365 商務移除公司資料您的使用者必須在其[裝置](app-protection-settings-for-android-and-ios.md)或[Windows Pc](protection-settings-for-windows-10-devices.md) Microsoft 365 來保護。**如果您移除公司資料來源裝置]，您無法將它稍後還原**。 
  
1. 使用全域系統管理員認證登入 [Microsoft 365 商務版](https://portal.office.com)。 
    
2. 在管理中心的 [**裝置動作**在卡片上，選擇 [**移除公司資料**。
    
    ![On the Devices card, choose Remove company data](media/b6fcf74b-0d7d-4e1a-894f-40f9d4a215b8.png)
  
3. 在**移除公司資料**] 窗格中，選擇 [或搜尋使用者誰具有您要移除，並選擇名稱的資料。 
    
4. 在 [下一步] 窗格中選取的裝置從**受管理的裝置**清單中移除的資料並選擇 [**下一步**。 
    
    ![On the remove comapany data pane, select the device from which you want to remove the data.](media/f3725ff9-ebdb-4c13-9523-b2df362640cf.png)
  
5. 在**您確定要移除之裝置上的公司資料**] 窗格中，選擇 [**移除** \> **關閉**。
