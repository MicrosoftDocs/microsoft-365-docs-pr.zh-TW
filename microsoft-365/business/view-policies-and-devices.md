# <a name="view-and-manage-policies-and-devices"></a>檢視及管理原則和裝置

![指向橫幅https://aka.ms/aboutM365preview。](media/m365admincenterchanging.png)

## <a name="view-and-edit-device-policies"></a>檢視及編輯裝置原則

1.  移至系統管理中心， <a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a>。
2. 在左側導覽中，選擇 [**裝置** \> **原則**。

    您可以在這個頁面上建立、編輯、變更目標群組或刪除原則。

    ![Screenshot of the Policies page](media/devicepolicies.png)
  
## <a name="view-and-manage-devices"></a>檢視及管理裝置


1. 在左側導覽中，選擇 [**裝置** \> **管理**。 
    
    您可以在這個頁面上選取一或多個裝置以及移除公司資料。針對您已設定裝置保護設定的 Windows 10 裝置，您也可以選擇將裝置重設成出廠預設值。
  
   ![管理裝置] 頁面](media/devicesmanage.png)
