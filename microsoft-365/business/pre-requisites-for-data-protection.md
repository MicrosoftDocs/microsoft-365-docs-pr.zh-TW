# <a name="prerequisites-for-protecting-data-on-devices-with-microsoft-365-for-business"></a>使用 Microsoft 365 商務裝置保護裝置資料的先決條件

本文適用于 Microsoft 365 商務進階版。

使用商務用 Microsoft 365 設定組織的第一步是，確定您可以符合必要條件。
  
## <a name="requirements-for-setting-up-your-organization-with-microsoft-365-for-business"></a>使用商務用 Microsoft 365 設定組織的需求

- Windows 裝置必須執行 Windows 7 專業版、Windows 8 專業版或 Windows 8.1 專業版。
    
    [將 Windows 裝置升級到 Windows 專業版 Creators Update](upgrade-to-windows-pro-creators-update.md)
    
    如果您正在執行 Windows 10 家用版，您必須 **購買** Windows 10 Pro。 如需相關指示，請參閱[upgrade Windows 10 家用版 to Windows 10 專業版](../business-video/upgrade.md)。 
    
- 從行動裝置管理解決方案中移除裝置 (行動鐵 AirWatch 等等) 。 您將在商務行動裝置管理的 Microsoft 365 中登記組織中的所有人員。
    
- Apple iOS 8.0 及更新版本。
    
    Google Android 4.0 及更新版本 (包括 Samsung KNOX Standard 4.0 及更新版本)。 如需詳細資訊，請參閱 [Intune 支援的裝置](/mem/intune/fundamentals/supported-devices-browsers)。
    
- 如果使用者電腦上有現有的 Office 應用程式，請閱讀[準備好 Office 用戶端安裝](prepare-for-office-client-deployment.md)，瞭解您可能需要採取的步驟，才能在使用者電腦上安裝 Office 2016 的 Microsoft 365。
