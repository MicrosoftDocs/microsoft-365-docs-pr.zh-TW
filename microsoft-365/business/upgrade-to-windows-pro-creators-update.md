# <a name="upgrade-windows-devices-to-windows-pro-creators-update"></a>將 Windows 裝置升級到 Windows 專業版 Creators Update

## <a name="upgrade-to-windows-10-pro-creators-update"></a>升級至 Windows 10 Pro 建立者更新

1. 登入[Microsoft 系統管理中心](https://portal.office.com/adminportal/home)並尋找**Windows 10 升級**卡片。 
    
    ![在管理中心的 Windows 10 升級卡片。](media/066f47bf-7b88-4fea-8fd0-82798ea66716.png)
  
2. 若要升級為 Windows 10 專業人員的建立者更新時，您有數個選項。您可以：
    
- 從 Microsoft 軟體下載網站安裝升級-選取這個選項只有在您的登入的裝置位於的一個您要更新的相同裝置。
    
  a.選取**安裝升級**。
    
  b.從軟體下載網站、 按一下 [**立即更新**開始升級至 Windows 10 專業人員的建立者更新的裝置。 
    
- 共用的下載連結-如果您想要讓他們可以升級至 Windows 10 Pro 本身並傳送給組織中的人員的電子郵件選取此選項。
 
   a select**共用下載**連結以開啟電子郵件包含連結至 Windows 軟體下載網站。 
    
 - 建立使用媒體架設工具安裝媒體-選取此選項可建立 Windows 10 專業人員的建立者更新安裝媒體 （USB 快閃磁碟機或 ISO 檔案） 從一個使用不同的電腦上安裝 Windows 10。
    
    a.選取**建立安裝媒體**。
    
    b.閱讀如何使用工具及建立安裝媒體上的指示。 
    
請參閱 ＜ [Set up Windows 裝置 Microsoft 365 商務使用者](set-up-windows-devices.md)以完成設定 Windows 10 裝置。 
  
請參閱[Set up Microsoft 365 商務使用者的行動裝置](set-up-mobile-devices.md)來完成設定 Android 和 iOS 裝置。
