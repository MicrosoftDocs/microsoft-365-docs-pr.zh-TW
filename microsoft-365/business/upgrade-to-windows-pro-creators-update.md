## <a name="upgrade-to-windows-10-pro"></a>升級到 Windows 10 專業版
  
若要升級至 Windows 10 專業版，您有數個選項。 您可以：
    
- 從 [Microsoft 軟體下載網站](https://go.microsoft.com/fwlink/?LinkID=836951 )安裝升級 &ndash; 如果您登入的裝置與您想要更新的裝置位於相同的裝置上，請選取此選項。 

    - 在軟體下載網站中，按一下 [ **立即更新** ]，開始將裝置升級到 Windows 10 Pro 建立者更新。 
    
- 使用 [媒體建立工具](https://go.microsoft.com/fwlink/?LinkID=836960)建立安裝媒體 &ndash; 選取此選項，可建立 Windows 10 專業編寫者更新安裝媒體 (USB 快閃記憶體盤案或 ISO 檔案) ，在電腦上安裝 windows 10，其與您所用的電腦有所不同。

    - 閱讀如何使用工具及建立安裝媒體的指示。 

> [!NOTE]
> 如果您有執行 Windows 7 專業版、Windows 8 專業版或 Windows 8.1 Pro 的 Windows 裝置，則您的 Microsoft 365 for business 訂閱可讓您執行 Windows Pro 10 升級。
    
請參閱為 [商務用 Microsoft 365 設定 windows 裝置](set-up-windows-devices.md) 以完成設定 windows 10 裝置。 
  
請參閱為 [商務用 Microsoft 365 使用者設定行動裝置](set-up-mobile-devices.md) ，以完成 Android 和 iOS 裝置的設定。
