## <a name="upgrade-to-windows-10-pro"></a>升級到 Windows 10 專業版
  
若要升級到 Windows 10 專業版，您有數個選項。 您可以：
    
- 從[Microsoft 軟體下載網站](https://go.microsoft.com/fwlink/?LinkID=836951 )安裝升級&ndash;選取這個選項，如果您登入該裝置是您要更新的相同裝置上。 

    - 從軟體下載網站，請按一下 [**立即更新**開始升級到 Windows 10 專業版 Creators Update 的裝置。 
    
- 建立使用[媒體建立工具](https://go.microsoft.com/fwlink/?LinkID=836960)安裝媒體&ndash;選取這個選項可建立 Windows 10 專業版 Creators Update 安裝媒體 （USB 快閃磁碟機或 ISO 檔案） 與您使用一個不同的電腦上安裝 Windows 10。

    - 閱讀如何使用此工具，並建立您的安裝媒體上的指示。 

> [!NOTE]
> 如果您有執行 Windows 7 專業版、 Windows 8 專業版或 Windows 8.1 專業版的 Windows 裝置，您的 Microsoft 365 商務版訂閱賦與您 Windows Pro 10 升級;您不需要產品金鑰。
    
請參閱[為 Microsoft 365 商務版使用者的 Windows 裝置上設定](set-up-windows-devices.md)完成 Windows 10 裝置設定。 
  
完成設定 Android 和 iOS 裝置，請參閱[為 Microsoft 365 商務版使用者的行動裝置上的設定](set-up-mobile-devices.md)。
