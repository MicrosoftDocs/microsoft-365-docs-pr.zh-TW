# <a name="install-office-on-windows-10-during-setup"></a>在設定期間安裝 Windows 10 版 Office

![指向的橫幅https://aka.ms/aboutM365preview。](../media/m365admincenterchanging.png)

根據預設，此設定為關閉狀態。 您可以將其開啟，以確保使用者在其所有 Windows 10 裝置上都擁有最新版本的 Office，但請先確認所有必要條件都符合。 若要將軟體推入裝置，除了[microsoft 365 商務版的準備 Office 用戶端部署](prepare-for-office-client-deployment.md)中所述的其他需求之外，使用者必須獲得 Microsoft 365 商務版 premium 的授權。
  
建議您在安裝期間接受預設設定。
