# <a name="install-office-on-windows-10-during-setup"></a>在設定期間安裝 Windows 10 版 Office

![指向的橫幅 https://aka.ms/aboutM365preview 。](../media/m365admincenterchanging.png)

根據預設，此設定為關閉狀態。 您可以將其開啟，以確保使用者在所有 Windows 10 裝置上都有最新版本的 Office，但必須先確認所有必要條件都符合。 使用者必須獲得 Microsoft 365 商務進階版此設定，才能將軟體推入裝置，除了[Microsoft 365 商務進階版為 Office 用戶端部署準備時](prepare-for-office-client-deployment.md)所述的其他需求。
  
建議您在安裝期間接受預設設定。
