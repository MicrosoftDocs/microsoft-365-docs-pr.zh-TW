# <a name="help-users-connect-to-microsoft-365-business"></a>協助使用者連線至 Microsoft 365 商務版

在完成設定後，您的預設裝置原則都已準備好套用。在行動裝置上，使用者會下載 Office App，然後原則會在使用者以公司或學校帳戶登入後套用。在原則套用至 Windows 電腦前，還有一些工作要做，例如就地升級。請參照和分享下列主題，以協助使用者設定和連接。
  
[為 Microsoft 365 商務版使用者設定 Windows 裝置](set-up-windows-devices.md)
  
[使用逐步指南新增 Autopilot 裝置和設定檔](add-autopilot-devices-and-profile.md)
