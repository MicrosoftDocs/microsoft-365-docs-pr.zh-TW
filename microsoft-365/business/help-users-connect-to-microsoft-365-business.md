# <a name="help-users-connect-to-microsoft-365-business-premium"></a>協助使用者連接至 Microsoft 365 商務版 Premium

完成安裝之後，您的預設裝置原則已就緒，可以套用。 在行動裝置上，使用者下載 Office 應用程式。 然後，當使用者使用他們的公司或學校帳戶登入時，便會套用原則。 在原則套用至 Windows 電腦前，還有一些工作要做，例如就地升級。 請參閱和分享下列主題，以協助使用者設定及連線：
  
[為 Microsoft 365 商務版使用者設定 Windows 裝置](set-up-windows-devices.md)
  
[使用逐步指南新增 Autopilot 裝置和設定檔](add-autopilot-devices-and-profile.md)
  
[為 Microsoft 365 商務版使用者設定行動裝置](set-up-mobile-devices.md)
