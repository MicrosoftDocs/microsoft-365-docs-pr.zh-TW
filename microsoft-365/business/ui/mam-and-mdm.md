# <a name="difference-between-mdm-and-mam"></a>MDM 和 MAM 之間的差異

Microsoft 365 商務版特優為您提供許多方式來保護您的商務資料。 如需自動設定的各種保護，以及您可以自行設定以進一步保護您的業務，請參閱[Microsoft 365 商務版 Premium 的概述](../microsoft-365-business-overview.md)。 您也可以設定原則，以保護您的 Windows 10 裝置和行動裝置中的資料。
[設定 Windows 10 裝置的應用程式保護設定](../protection-settings-for-windows-10-devices.md)。

## <a name="mobile-device-management-or-mdm"></a>行動裝置管理或 MDM

Microsoft 365 商務版功能可讓您設定保護 Windows 10 裝置資料的原則。 當裝置處於行動裝置管理下時，您可以控制整個裝置，並且可以從該裝置中清除資料，也可以將其重設為出廠設定。 如需詳細資訊，請參閱[設定 Windows 10 電腦的裝置保護設定](../protection-settings-for-windows-10-pcs.md)。

## <a name="mobile-application-management-or-mam"></a>行動應用程式管理或 MAM

行動應用程式管理可讓您控制使用者個人裝置中的商務資料，例如 Iphone 和 Androids，以及其個人勝10部電腦。 您可以使用應用程式管理原則，以防止使用者將商務資料從 Office 應用程式複製到其個人應用程式。 您也可以移除其個人裝置上 Office 應用程式中的所有資料。 如需詳細資訊，請參閱[設定適用于 Android 或 iOS 裝置的 app 保護設定](../app-protection-settings-for-android-and-ios.md)，以及[設定 Windows 10 裝置的應用程式保護設定](../protection-settings-for-windows-10-devices.md)。
