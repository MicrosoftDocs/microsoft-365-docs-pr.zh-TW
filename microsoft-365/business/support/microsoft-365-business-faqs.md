# <a name="microsoft-365-business-frequently-asked-questions"></a>Microsoft 365 商務常見問題

## <a name="general"></a>一般

### <a name="what-is-microsoft-365-business"></a>什麼是 Microsoft 365 商務版？ 

Microsoft 365 是一種整合式解決方案，其搭配一流的高效率工具、安全性和裝置管理功能，適用于中小型企業。 包括：

**一組商務生產力與共同作業工具** 
* Word、Excel、PowerPoint、Outlook、OneNote、Publisher 和 Access 
* Exchange、OneDrive、商務用 Skype、Microsoft 團隊及 SharePoint。 
* Office 中的商務應用程式（預定、MileIQ<sup>[1](#footnote1)</sup>）。 
 
**企業級裝置管理與安全性功能** 
- 協助提供 Office 365 Advanced 威脅防護方案1和 Windows Defender 利用防護等外部威脅（如網路釣魚和複雜的惡意程式碼）的保護。 
-  使用 Office 365 資料遺失防護原則和 Azure 資訊保護方案1，協助控制和管理敏感資訊的存取方式及傳送方式。 
-  使用 Exchange Online 封存協助保護、保留及備份資料。  
-   適用于 Office 的應用程式保護，以及具有 Intune 應用程式保護的其他行動應用程式。 
-  Windows 10 電腦、MacOS 及行動裝置的裝置管理，含 Intune 裝置管理。 
-  使用多重要素驗證（MFA）、自助密碼重設和條件式存取的身分識別保護。
-  跨各裝置的各&mdash;裝置保護的一致性安全性設定; 跨裝置保護公司資料;Windows Defender，它永遠在開啟及最新狀態。 
 
 
**簡化裝置部署和使用者設定** 
* 單一管理主控台以設定及管理使用者和裝置 
* 在 Windows 10 電腦上自動安裝 Office 應用程式。 
* 永遠保持最新的 Office + Windows 10。 
* 使用 Windows AutoPilot 簡化電腦的部署。

**其他功能** 
- Microsoft 365 商務客戶也可以存取 Windows 虛擬桌面和 Office 共用電腦啟用功能。 請閱讀[Microsoft 365 商務版博客](https://techcommunity.microsoft.com/t5/Microsoft-365-Business-Blog/bg-p/Microsoft365BusinessBlog)以深入瞭解。 另請參閱[Microsoft 365 商務服務說明](https://docs.microsoft.com/office365/servicedescriptions/microsoft-365-service-descriptions/microsoft-365-business-service-description)。

### <a name="who-should-consider-adopting-microsoft-365-business"></a>誰應考慮採用 Microsoft 365 商務版？ 

Microsoft 365 商務版是完整的雲端式安全性解決方案，可讓您：  
- 使用複雜的網路釣魚和勒索軟體防護，針對高級 cyberthreats 防禦您的企業。 
- 使用加密來控制敏感資訊的存取，使資料不會與未經授權查看的使用者意外共用。 
- 保護連接至您公司的裝置，並協助保護 iOS、Android、Windows 和 Mac 裝置上的資料，並保證您的 Windows 裝置更新為最新狀態。

Microsoft 365 商務版完全與 Office 365 整合，提供您熟悉的管理、計費和全天候支援的位置。 其包含的企業級技術是針對員工少於300名員工所建立。 

### <a name="how-can-i-get-microsoft-365-business-for-my-business"></a>如何取得適合我的公司使用的 Microsoft 365 商務版？ 

Microsoft 365 商務版可以透過<a href="https://www.microsoft.com/solution-providers/search" target="_blank">Microsoft 合作夥伴</a>購買，也可以直接從<a href="https://www.microsoft.com//microsoft-365/business" target="_blank">microsoft</a>購買。 在選擇是否要直接向 Microsoft 購買或透過 Microsoft 合作夥伴購買之前，應該先考慮將您員工的能力，並維持 IT 基礎結構的需求。 Microsoft 合作夥伴可協助您部署及管理您的 IT 基礎結構，包括 Microsoft 解決方案。

### <a name="how-much-does-microsoft-365-business-cost"></a>Microsoft 365 商務版價格多少？ 

如果直接向 Microsoft 購買，根據_年度合約_，每一名使用者每個月支付 20 美元即可使用 Microsoft 365 商務版。 透過 Microsoft 合作夥伴購買時，價格可能因合作夥伴提供的服務以及 Microsoft 365 商務版計價模式的不同而異。 

### <a name="is-there-a-cap-to-how-many-microsoft-365-business-seats-a-customer-can-have"></a>有限制客戶最多可以擁有多少 Microsoft 365 商務版基座數嗎？ 

Microsoft 365 商務版專為中小型公司設計，具有低的 IT 複雜性需求。 客戶可以為其組織購買高達300的 Microsoft 365 業務授權。 客戶可以混合和搭配雲端訂閱。 這樣一來，客戶就可以將 Microsoft 365 企業版授權新增至相同的帳戶。 
 
當客戶考慮由多種訂閱類型所組成的環境時，他們應與其受信任的 IT 顧問合作，以判斷 Microsoft 365 商務和 Microsoft 365 企業使用不同的功能來保護和管理應用程式和資料的最佳方式，以判斷各種訂閱的安全性。

### <a name="can-i-combine-microsoft-365-business-with-other-microsoft-subscription-offerings"></a>是否可以將 Microsoft 365 商務版和其他 Microsoft 訂閱優惠方案組合在一起？

是的，客戶可以將其 Microsoft 365 商務版訂閱與來自 Azure、Dynamics 365、Enterprise Mobility + Security 及 Office 365 的方案與附加元件組合在一起。

### <a name="is-everyone-in-my-business-required-to-have-a-microsoft-365-business-subscription"></a>我公司的每個人是否都需要 Microsoft 365 商務版訂閱？

否，儘管使用 Microsoft 365 商務版訂閱管理裝置的使用者才能使用安全性與管理的權益，並不是每個人都需要 Microsoft 365 商務版訂閱。 
 
標準化 IT 環境可協助減少維護和安全性成本隨時間，而且是企業應該盡力達成的狀態。 不過，我們知道某些小型和中型客戶在升級其硬體時，主要是在更新其軟體，一段時間內已滿。 企業可以在其組織的一部分部署 Microsoft 365 商務版，但是如果需要提供機密企業資料最佳保護和一致的共同作業體驗，建議部署至所有的使用者。

### <a name="how-can-i-know-if-the-hardware-and-software-i-run-today-is-compatible-with-microsoft-365-business"></a>如何知道我今天執行的硬體和軟體是否與 Microsoft 365 商務版相容？ 

如果您目前執行的硬體執行 Windows 7 專業版或更新版本，它可能符合 Microsoft 365 企業版的最低需求。 某些 Windows 10 功能（如 Cortana、Windows Hello 及多點觸控功能）需要只有在更新的電腦上才能使用的特定硬體。 如需詳細資訊，請參閱<a href="https://www.microsoft.com/windows/windows-10-specifications" target="_blank">Windows 10 專業版系統需求</a>。 
 
現有的傳統型 (Win32) 應用程式相容性應該也很好，大部分現有的應用程式不需要任何變更就能運作。 客戶及其信任的 IT 顧問應該閱讀建議的應用程式測試程序以了解 <a href="https://docs.microsoft.com/windows/deployment/planning/windows-10-compatibility#recommended-application-testing-process" target="_blank">Windows 10 相容性</a>，並且檢閱 <a href="https://products.office.com/office-system-requirements#subscription-plans-section" target="_blank">Office 系統需求</a>以確保平順轉換到 Microsoft 365 商務版。 

### <a name="what-is-windows-10-business"></a>什麼是 Windows 10 商務版？ 

Windows 10 商務版是一組雲端服務和裝置管理功能，可補充 Windows 10 專業人員，並啟用 Microsoft 365 商務的集中式管理和安全性控制。

Windows 10 商務版也附帶特定的 Microsoft Defender 功能，例如控制資料夾存取和網路型執行緒的網路保護。 它也包含 Windows AutoPilot，也就是一種可簡化新 Windows 10 電腦部署的服務。

如果您的裝置具有 Windows 7、8和8.1 專業版的授權，則 Microsoft 365 Business 會提供升級至 Windows 10 專業版，這是部署 Windows 10 商務版的必要條件。
> [!IMPORTANT]
> 當您升級時，您必須提供原始的產品金鑰，否則升級將無法運作。 

### <a name="how-does-microsoft-365-business-help-support-our-bring-your-own-device-byod-policy"></a>Microsoft 365 商務版如何協助支援我們的「攜帶您自己的裝置」(BYOD) 原則？ 

許多員工寧願使用自己的行動電話或平板電腦來存取個人和工作資訊，而不是依照每一種目的執行多個裝置。 使用個人裝置工作雖然很平常，卻也增加商務資訊遭到不當利用的風險。 許多競爭激烈的行動數據保護方案要求使用者切換到裝置上的特定模式，或使用另一種讓使用者覺得受干擾而避免使用的複雜機制。 
 
Microsoft 365 商務版為客戶提供一種簡單但強大的方式，可讓員工使用其個人裝置進行工作，同時讓公司能夠防止這些裝置存取、保留和/或共用商務資訊。 更加明確： 
* **Office 行動應用程式的應用程式保護**可協助保護 office 資料。在 IOS 和 Android 行動裝置上包含電子郵件、行事曆、連絡人及檔，強制執行諸如在未連線到服務的情況下自動刪除商務資料等原則，這要求資訊只會儲存在商務 OneDrive 中，需要 PIN 碼/指紋驗證才能存取 office 應用程式，並防止公司資料從 Office 應用程式複製到個人應用程式。 透過 Intune 對其他行動應用程式的行動應用程式管理，也適用于 Microsoft 365 商務訂閱者。 
  
* **Windows 10 電腦適用的行動裝置管理**可讓企業選擇設定並執行功能，例如 Windows Defender 防護惡意程式碼入侵、自動更新，並在規定的時間後關閉螢幕。 此外，在 Windows 10 裝置遺失或被盜的情況下，您可以透過系統管理中心徹底擦除商務應用程式和資料。 

* **IOS 的裝置管理，Android & MacOS**功能可協助企業安全地管理包含 IOS、Android、Windows 和 MacOS 裝置的各種裝置生態系統。 

### <a name="how-does-microsoft-365-business-help-protect-pcs-in-my-organization-from-malicious-attacks"></a>Microsoft 365 商務版如何協助保護組織中的電腦使其免受惡意攻擊？ 

使用 Microsoft 365 商務版管理的電腦受 Windows Defender 保護。 Windows 10 上 1 種防毒功能，比其他任何解決方案保護更多電腦免受病毒、惡意程式碼、間諜軟體及其他威脅攻擊。  透過 Microsoft 365 商務版，組織可以確保 Windows Defender 保護已執行，且在其所有 Windows 10 裝置上永遠保持最新狀態。  
 
Windows 10 商務版也包含 Windows Defender 利用防護（一組新的入侵防護功能）。 它的其中一個功能（即「控制」資料夾存取）會鎖定資料夾，並防止未授權的應用程式存取使用者的重要檔案，以停止勒索軟體。 

### <a name="whats-the-difference-between-office-365-business-premium-microsoft-365-business-and-microsoft-365-enterprise"></a>Office 365 商務版 Premium、Microsoft 365 商務版和 Microsoft 365 企業版之間的差異為何？ 

Microsoft 有各種生產力及安全性管理產品，可在升級其桌面和裝置基礎結構時考慮，每個專案都有越來越強大的功能和功能。 

**Office 365 商務版**可透過 Office 365 應用程式和服務提供同類最佳的生產力，但不包括 Microsoft 365 商務的應用程式保護和裝置管理功能。

**Microsoft 365 Business**將 Office 365 應用程式和服務與高級安全性功能結合，以協助保護您的企業免受高級 cyberthreats，保護您的資料並管理您的裝置。 它包括可管理裝置和資料原則的簡化管理主控台。 許多小型至中型企業最適合使用 Microsoft 365 商務服務。 
 
**Microsoft 365 Enterprise**是一組授權計畫，可提供更高層級的相容性和安全性管理，並可透過 Microsoft 365 公司為企業客戶及具有超過300使用者的客戶設計。 此外，Microsoft 365 企業版方案也提供額外的功能，包括商務智慧和分析工具。

### <a name="can-i-switch-my-office-365-plan-to-microsoft-365-business"></a>是否可以將我的 Office 365 方案轉換成 Microsoft 365 商務版？ 

是的，客戶可以將合格的 Office 365 方案轉換成 Microsoft 365 商務版。 根據客戶目前的計畫，每月的費用可能會減少或增加。

### <a name="in-what-regions-is-microsoft-365-business-available"></a>哪些地區有提供 Microsoft 365 商務版？ 

Microsoft 365 商務版可供所有可供使用 Office 365 的合作夥伴及客戶使用。 請參閱<a href="https://products.office.com/business/international-availability" target="_blank">語言、國家及地區的 Office 365 國際版可用性</a>清單。 

### <a name="is-there-a-microsoft-365-business-trial-i-may-use-to-evaluate-the-offer"></a>是否有 Microsoft 365 商務版試用版可讓我用來評估優惠？

可用於 Csp 的 Microsoft 365 商務版試用版。 「直接客戶」的試用版稍後會提供。 
 
### <a name="what-should-customers-and-partners-know-before-running-microsoft-365-business-within-their-organization"></a>客戶與合作夥伴在其組織中執行 Microsoft 365 商務版之前應該知道什麼？ 

想要體驗 Microsoft 365 商務版完整功能的客戶，必須在其現有桌上型電腦上執行 Windows 7、8.1 或 10 Pro<sup>[2](#footnote2)</sup> 。 如果他們還沒有這麼做，現有的 Windows 10 專業版電腦應執行 Creators Update。 

## <a name="deployment"></a>部署

### <a name="what-should-customers-consider-when-planning-a-microsoft-365-business-deployment"></a>客戶在規劃 Microsoft 365 商務版部署時，應考慮哪些事項？

成功部署 Microsoft 365 商務版的最直接途徑是參與 Microsoft 合作夥伴的作業。 他們接受過大量訓練，遇到過各種不同的客戶案例，因此絕對擁有最佳條件了解您的環境和需求。 擁有資深 IT 人員的客戶，可以使用 <a href="https://support.office.com/article/496e690b-b75d-4ff5-bf34-cc32905d0364" target="_blank">Microsoft 365 商務版快速入門</a>來協助部署 Microsoft 365 商務版。 

### <a name="does-microsoft-365-business-include-the-full-capabilities-of-microsoft-intune"></a>Microsoft 365 商務版是否包含完整的 Microsoft Intune 功能？

是的，已授權 Microsoft 365 商務訂戶使用完整的 Intune 功能，以供 iOS、Android、MacOS 及其他跨平臺裝置管理使用。 在 Microsoft 365 商務中的簡化管理主控台中無法使用的功能，例如，協力廠商的應用程式管理和 WiFi 設定檔的設定、VPN 憑證，可在完整的 Intune 主控台中管理。 
  

### <a name="does-azure-active-directory-azure-ad-premium-p1-come-with-microsoft-365-business"></a>Azure Active Directory （Azure AD）高 P1 是否隨附 Microsoft 365 商務版？ 

先前的 Microsoft 365 公司具有 Azure AD Premium Plan 1 的子集，包括條件式存取、自助密碼重設和 Multi-Factor 驗證。 現在，我們正在將完整的 Azure AD 優質 P1 授權，推廣至 Microsoft 365 商務訂閱者（初次向新的客戶，再到目前的訂戶）。 如需詳細資訊，請參閱[Microsoft 365 商務服務說明](https://docs.microsoft.com/office365/servicedescriptions/microsoft-365-service-descriptions/microsoft-365-business-service-description)和這[篇博客](https://techcommunity.microsoft.com/t5/small-and-medium-business-blog/azure-active-directory-premium-p1-is-coming-to-microsoft-365/ba-p/1275496)。
 
 
### <a name="does-microsoft-365-business-allow-customers-to-manage-macs"></a>Microsoft 365 商務版是否允許客戶管理 Mac？ 

Intune 可協助您安全地管理 iOS、Android、Windows 和 MacOS 裝置。 
 
### <a name="what-is-windows-autopilot"></a>什麼是 Windows AutoPilot？ 

Windows AutoPilot 是一種服務，可簡化新 Windows 10 電腦部署作業。 當使用者第一次&mdash;登入 Microsoft 365 商務時，就可以執行此程式，而不需要透過利用 Microsoft 365&mdash;業務的集中式管理控制來接觸裝置。 針對正在執行 Windows 10 專業版 Creators Update (或更新版本) 且已恢復出廠預設值的現有電腦，也可以使用 Windows AutoPilot。 有關 Windows AutoPilot 的詳細資訊，請參閱<a href="https://blogs.technet.microsoft.com/windowsitpro/2017/06/29/modernizing-windows-deployment-with-windows-autopilot/" target="_blank">2017 年6月的博客文章</a>。

## <a name="compatibility"></a>相容性

### <a name="can-i-add-office-365-add-ons-to-microsoft-365-business"></a>我可以在 Microsoft 365 商務版中加入 Office 365 附加元件嗎？ 

可以加入到 Office 365 商務進階版的所有附加元件，都可以加入到 Microsoft 365 商務版。 這表示您可以購買 Office 365 Cloud App Security、Advanced 合規性、威脅情報、MyAnalytics、PowerBI Pro 和音訊會議。 
 
### <a name="can-i-add-phone-system-and-calling-plans-to-microsoft-365-business"></a>我可以在 Microsoft 365 商務版中加入電話系統和通話方案嗎？

不行，電話系統和通話方案會保留給有更多進階需求的客戶。 需要這些功能的客戶，應該查看 Microsoft 365 企業版方案。 
 
### <a name="can-microsoft-365-business-customers-use-windows-defender-advanced-threat-protection"></a>Microsoft 365 商務版客戶是否可以使用 Windows Defender 進階威脅防護？ 

否，需要 Windows Defender 進階威脅防護的客戶，需要 Windows 10 企業版 E5 或 Microsoft 365 企業版 E5。

### <a name="where-can-i-learn-how-to-sell-microsoft-365-business"></a>何處可了解如何銷售 Microsoft 365 商務版？ 

目前正在銷售 Office 365 的合作夥伴，可以使用相同的諮詢銷售方法來銷售 Microsoft 365 商務版。 此外，我們還會為您的銷售團隊引進更多資源和訓練，以瞭解客戶的現有桌面環境、Active Directory 依賴性、行動性和安全性，以與客戶相關的方式，有效地交流 Microsoft 365 業務的完整價值。 在位於 [https://partners.office.com/microsoft365business](https://partners.office.com/microsoft365business) 的 Office 合作夥伴入口網站上可找到這些資源。 

### <a name="how-can-microsoft-365-business-help-partners-increase-the-profitability"></a>Microsoft 365 商務版如何協助合作夥伴提高獲利？ 

Microsoft 365 商務版可透過更高的作業效率和銷售其他服務所帶來的收益，協助合作夥伴降低成本。 Forrester Research 在 2017 年 6 月所做Microsoft 365 商務版總體經濟影響 (TEI) 研究 (https://www.microsoft.com/microsoft-365/partners/PartnerTEI) 顯示，Microsoft 365 商務版對於合作夥伴獲利有正面影響。 

合作夥伴在 TEI 研究中報告，他們對於使用 Microsoft 365 商務版的預期： 

- \[一次性\] 部署及諮詢服務收益增加 20% 點數
- 受管理服務的市場觸及率增加 10% 點數
- 諮商和 \[進行中\] 受管理服務利潤 (因降低成本而獲利) 增加 8% 點數

### <a name="what-resources-are-available-to-partners-to-sell-deploy-and-support-microsoft-365-business"></a>哪些資源可供合作夥伴用於銷售、部署及支援 Microsoft 365 商務版？

Microsoft 提供多樣的雲端解決方案提供者合作夥伴適用資源選項，可供行銷、銷售及支援 Microsoft 365 商務版。 這些資源可在 [https://partners.office.com/microsoft365business](https://partners.office.com/microsoft365business) 找到。

### <a name="what-up-sell-opportunities-does-microsoft-365-business-give-partners"></a>Microsoft 365 商務版提供合作夥伴哪些向上行銷機會？ 

Microsoft 365 商務版可讓合作夥伴透過建立實體且安全的平臺來銷售其他服務，以及升級現有產品和服務，以維持客戶信任的 advisor 位置。 Microsoft 365 商務版可讓您立即利用 Exchange Server、Exchange Online 或 Office 365 商務基本版，對客戶進行升級討論。 合作夥伴可能也會從增加的受管理服務和/或 peruser 支援費用獲得更多收入。 
 
使用 Microsoft 365 商務中包含的新 Windows AutoPilot 功能，由於部署物流和成本，已不願銷售新 Windows 裝置的合作夥伴，因此會發現這項機會更具吸引力。 在內部部署和行動裝置安全性上自信的客戶也更可能投資于更多服務，例如 Dynamics 365。 
 
### <a name="should-partners-sell-microsoft-365-business-over-other-plans-from-microsoft"></a>合作夥伴應透過 Microsoft 的其他方案銷售 Microsoft 365 商務版嗎？

Microsoft 雲端解決方案提供者應一律使用最適合客戶企業需求和預算的方式銷售方案。 例如，如果客戶必須遵守隱私權和安全性規定，CSP 可能會銷售 Microsoft 365 商務和任何附加元件，以協助客戶滿足其需求，或建議使用 Microsoft 365 Business Enterprise SKUs 所提供的高級安全性和管理。 
 
### <a name="some-of-my-customers-have-devices-that-arent-genuine-will-microsoft-365-business-make-these-devices-genuine"></a>我的某些客戶的裝置不是正版;Microsoft 365 商務版會使這些裝置成為正版？ 

Microsoft 365 公司不會以非正版版本的 Windows，真品。 Microsoft 365 商務版提供升級權益，可讓執行正版 Windows 7、8或 8.1 Pro 的客戶升級為最新的 Windows 10 專業版。 

### <a name="what-is-the-gdpr-and-how-does-microsoft-365-business-help-customers-with-their-compliance-obligations"></a>什麼是 GDPR，以及 Microsoft 365 商務版如何協助客戶達成其合規義務？

一般資料保護規定（GDPR）是一項全新的隱私權法律，可讓派駐人員的「個人資料」更進一步控制其「個人資料」，並要求組織維持該個人資料的完整性。 GDPR 要求組織會控制或處理與歐盟居民相關的個人資料，僅使用符合個人資料的 GDPR 需求的協力廠商資料處理程式。 Microsoft 已在 2017 年 3 月對這些保證提供履約保證。 在 Microsoft 如何協助他們符合其他 GDPR 義務方面有問題的客戶，應瞭解 Microsoft 365 商務中可用的高級規範與安全性功能（例如，Azure 資訊保護、資料遺失防護、高級威脅防護等等）和其他套件（例如，Microsoft 365 企業版 E5）。 如需詳細資訊，請瀏覽 [www.microsoft.com/gdpr](https://www.microsoft.com/gdpr)。
