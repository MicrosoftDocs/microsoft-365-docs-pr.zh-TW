# <a name="microsoft-365-business-frequently-asked-questions"></a>Microsoft 365 商務版常見問題集

## <a name="general"></a>一般

### <a name="what-is-microsoft-365-business"></a>什麼是 Microsoft 365 商務版？ 
Microsoft 365 是整合式的解決方案，聚集所有同等級最佳的生產力工具、 安全性和裝置管理功能適用於小型至中型企業。 

**商務生產力和共同作業工具的整體集合** 
* Word、Excel、PowerPoint、Outlook、OneNote、Publisher 和 Access 
* Exchange、OneDrive、商務用 Skype、Microsoft Teams、SharePoint 
* 從 Office (Bookings、 Outlook Customer Manager、 MileIQ<sup>[1](#footnote1)</sup>、 Microsoft 開立發票<sup>[1](#footnote1)</sup>) 的商務應用程式 
 
**企業級裝置管理與安全性功能** 
- 協助提供從外部威脅，例如網路釣魚保護 & 複雜的惡意程式碼與 Office 365 進階威脅防護和 Windows Defender 惡意探索防護 
-  協助管理如何敏感資訊的控制項 & 存取和傳輸 （可在夏季，2018年） 的 Office 365 資料外洩防護原則和 Azure 資訊保護計劃 1 
-  協助保護、 保留和備份您的資料與 Exchange Online 封存  
-   Office & 的應用程式保護使用 Intune 應用程式防護其他行動裝置 app 
-  適用於 Windows 10 電腦，MacOS & Mobiles 裝置與 Intune 裝置管理裝置管理 
-  保護公司資料跨裝置-Windows Defender，always on 和最新的一致安全性設定跨裝置- 
 
 
**簡化裝置部署和使用者設定** 
* 單一管理主控台用於設定及管理使用者和裝置 
* 自動安裝 Windows 10 電腦上的 Office 應用程式 
* 讓 Office + Windows 10 一律維持在最新狀態 
* 透過 Windows AutoPilot 精簡電腦部署

### <a name="who-should-consider-adopting-microsoft-365-business"></a>誰應考慮採用 Microsoft 365 商務版？ 
Microsoft 365 商務版是專為小中型企業 (SMB) 設計的新解決方案，它將 Office 365 的同等級最佳生產力和共同作業能力與裝置管理和安全性方案整合在一起，以保護商務資料。 Microsoft 365 商務版客戶已準備好將他們的 IT 作業移動到雲端，且有意願積極保護公司和員工擁有裝置上的資料。

### <a name="how-can-i-get-microsoft-365-business-for-my-business"></a>如何取得適合我的公司使用的 Microsoft 365 商務版？ 
Microsoft 365 商務版可透過 <a href="https://www.microsoft.com/solution-providers/search" target="_blank">Microsoft 合作夥伴</a>購買，或直接從 <a href="https://www.microsoft.com//microsoft-365/business" target="_blank">Microsoft</a> 購買。 在選擇是否要直接向 Microsoft 購買或透過 Microsoft 合作夥伴購買之前，應該先考慮將您員工的能力，並維持 IT 基礎結構的需求。 Microsoft 合作夥伴可以協助您部署及管理您的 IT 基礎結構，包括 Microsoft 解決方案。

### <a name="how-much-does-microsoft-365-business-cost"></a>Microsoft 365 商務版價格多少？ 
如果直接向 Microsoft 購買，根據_年度合約_，每一名使用者每個月支付 20 美元即可使用 Microsoft 365 商務版。 透過 Microsoft 合作夥伴購買時，價格可能因合作夥伴提供的服務以及 Microsoft 365 商務版計價模式的不同而異。 

### <a name="is-there-a-cap-to-how-many-microsoft-365-business-seats-a-customer-can-have"></a>有限制客戶最多可以擁有多少 Microsoft 365 商務版基座數嗎？ 
Microsoft 365 商務版是專為 IT 複雜需求程度低至中的中小型企業而設計。 客戶最多可為其組織購買 300 份 Microsoft 365 商務版使用權。 客戶可以混搭雲端訂閱，如此一來，根據其組織的 IT 需求，客戶可新增 Microsoft 365 企業版授權至相同帳戶。 
 
在考慮由多種訂閱類型組成的環境時，客戶應該配合其所信任的 IT 顧問來判斷管理及保護各種訂閱方案所適用的最佳方式，因為 Microsoft 365 商務版和 Microsoft 365 企業版使用不同的功能來保護及管理應用程式和資料。

### <a name="can-i-combine-microsoft-365-business-with-other-microsoft-subscription-offerings"></a>是否可以將 Microsoft 365 商務版和其他 Microsoft 訂閱優惠方案組合在一起？ 
是的，客戶可以將其 Microsoft 365 商務版訂閱與來自 Azure、Dynamics 365、Enterprise Mobility + Security 及 Office 365 的方案與附加元件組合在一起。

### <a name="is-everyone-in-my-business-required-to-have-a-microsoft-365-business-subscription"></a>我公司的每個人是否都需要 Microsoft 365 商務版訂閱？ 
否，儘管使用 Microsoft 365 商務版訂閱管理裝置的使用者才能使用安全性與管理的權益，並不是每個人都需要 Microsoft 365 商務版訂閱。 
 
標準化 IT 環境有助於在一段時間內降低維護及保安成本，因此各家企業都在努力達到這種狀態。 但我們發現某些中小型企業客戶主要是在一段較長時間後升級硬體時，才更新他們的軟體。 企業可以在其組織的一部分部署 Microsoft 365 商務版，但是如果需要提供機密企業資料最佳保護和一致的共同作業體驗，建議部署至所有的使用者。

### <a name="how-can-i-know-if-the-hardware-and-software-i-run-today-is-compatible-with-microsoft-365-business"></a>如何知道我今天執行的硬體和軟體是否與 Microsoft 365 商務版相容？ 
如果您目前執行的硬體執行 Windows 7 專業版或更新版本，它可能符合 Microsoft 365 企業版的最低需求。 Cortana、 等 Windows Hello，多點觸控某些 Windows 10 功能需要只有較新的電腦的特定硬體。 如需其他資訊，請參閱 <a href="https://www.microsoft.com/windows/windows-10-specifications" target="_blank">Windows 10 專業版系統需求</a>。 
 
現有的傳統型 (Win32) 應用程式相容性應該也很好，大部分現有的應用程式不需要任何變更就能運作。 客戶及其信任的 IT 顧問應該閱讀建議的應用程式測試程序以了解 <a href="https://docs.microsoft.com/windows/deployment/planning/windows-10-compatibility#recommended-application-testing-process" target="_blank">Windows 10 相容性</a>，並且檢閱 <a href="https://products.office.com/office-system-requirements#subscription-plans-section" target="_blank">Office 系統需求</a>以確保平順轉換到 Microsoft 365 商務版。 

### <a name="what-is-windows-10-business"></a>什麼是 Windows 10 商務版？ 
Windows 10 商務版是一組雲端服務和裝置管理功能，這些功能彌補了 Windows 10 專業版的不足，並且啟用 Microsoft 365 商務版的集中式管理和安全性控制。 Windows 10 商務版也隨附 Windows AutoPilot，這項服務簡化新 Windows 10 電腦的部署。 如果您的裝置授權適用於 Windows 7、8 和 8.1 專業版，Microsoft 365 商務版提供升級至 Windows 10 專業版的好處，而這正是部署 Windows 10 商務版的必要條件。 

### <a name="how-does-microsoft-365-business-help-support-our-bring-your-own-device-byod-policy"></a>Microsoft 365 商務版如何協助支援我們的「攜帶您自己的裝置」(BYOD) 原則？ 
許多員工寧願使用自己的行動電話或平板電腦來存取個人和工作資訊，而不是依照每一種目的執行多個裝置。 使用個人裝置工作雖然很平常，卻也增加商務資訊遭到不當利用的風險。 許多競爭激烈的行動數據保護方案要求使用者切換到裝置上的特定模式，或使用另一種讓使用者覺得受干擾而避免使用的複雜機制。 
 
Microsoft 365 商務版提供客戶簡單但強大的方式，讓員工以工作時的能力提供商務使用其個人的裝置，來防止這些裝置無法存取、 保留，及/或共用的商務資訊. 更加明確： 
* **適用於 Office 行動裝置版應用程式的應用程式防護**可協助應用程式保護 Office 資料，包括 iOS 和 Android 行動裝置上的電子郵件、行事曆、連絡人及文件，方法是強制執行如在規定時間內若未連接服務便刪除商務資料等原則、要求資訊僅儲存到商務用 OneDrive、要求提供 PIN/指紋驗證存取 Office 應用程式，以及預防公司資料從 Office 應用程式複製到個人應用程式。 針對其他行動裝置 app 透過 Intune 行動應用程式管理也適用於 Microsoft 365 商務版訂閱者。 
  
* **Windows 10 電腦適用的行動裝置管理**可讓企業選擇設定並執行功能，例如 Windows Defender 防護惡意程式碼入侵、自動更新，並在規定的時間後關閉螢幕。 此外，可以透過系統管理中心完全清除遺失或遭竊 Windows 10 裝置的商務用應用程式和資料。 

* **裝置管理的 iOS、 Android & MacOS**功能可協助企業安全地管理包括 iOS、 Android、 Windows 及 macOS 裝置各種裝置生態系統。 

### <a name="how-does-microsoft-365-business-help-protect-pcs-in-my-organization-from-malicious-attacks"></a>Microsoft 365 商務版如何協助保護組織中的電腦使其免受惡意攻擊？ 
使用 Microsoft 365 商務版管理的電腦受 Windows Defender 保護。 Windows 10 上 1 種防毒功能，比其他任何解決方案保護更多電腦免受病毒、惡意程式碼、間諜軟體及其他威脅攻擊。  使用 Microsoft 365 商務版，企業可確保 Windows Defender 保護是執行中且一律最其所有 Windows 10 裝置上。  
 
Windows 10 商務版也包含 Windows Defender 惡意探索防護，一組新的入侵防護功能。 下列其中一個其功能，受控資料夾存取權，停止勒索軟體中其追蹤鎖定資料夾，並防止未經授權的應用程式存取使用者的重要檔案。 

### <a name="whats-the-difference-between-office-365-business-premium-microsoft-365-business-and-microsoft-365-enterprise"></a>Office 365 商務進階版、Microsoft 365 商務版及 Microsoft 365 企業版之間有何差異？ 
Microsoft 擁有小中型企業客戶在升級桌面和裝置的基礎結構時可能考量的各種生產力和安全性管理方案，而每一個方案提供越來越強大的特性及功能。 

**Office 365 商務進階版**提供含 Office 365 應用程式和服務同等級最佳生產力，但不包含 Microsoft 365 商務版的應用程式防護和裝置管理功能。

**Microsoft 365 商務版**將 Office 365 應用程式和服務，以及行動裝置版的應用程式管理 Windows 10 專業版組合在一起以啟用遠端管理，並協助保護裝置使其免受病毒和惡意程式碼的入侵。 它包括可管理裝置和資料原則的簡化管理主控台。 許多小型至中型企業可以使用 Microsoft 365 商務版最佳提供服務。 
 
**Microsoft 365 企業版**是一組的授權計劃，透過 Microsoft 365 商務版提供增強的層級的合規性和安全性管理，且為企業客戶和客戶已超過 300 位使用者所設計。 此外，Microsoft 365 企業版方案提供額外的功能包括商務智慧及分析工具。

### <a name="can-i-switch-my-office-365-plan-to-microsoft-365-business"></a>是否可以將我的 Office 365 方案轉換成 Microsoft 365 商務版？ 
是的，客戶可以將合格的 Office 365 方案轉換成 Microsoft 365 商務版。 視顧客目前使用的方案而定，每月的費用可能變低或變高。

### <a name="in-what-regions-is-microsoft-365-business-available"></a>哪些地區有提供 Microsoft 365 商務版？ 
有 Office 365 可用的所有合作夥伴和客戶，均可使用 Microsoft 365 商務版。 請參閱 <a href="https://products.office.com/business/international-availability" target="_blank">Office 365 國際可用的語言、國家和地區清單</a>。 

### <a name="is-there-a-microsoft-365-business-trial-i-may-use-to-evaluate-the-offer"></a>是否有 Microsoft 365 商務版試用版可讓我用來評估優惠？ 
Microsoft 365 商務版試用版適用於 Csp。 試用版直接客戶將會是稍後。 
 
### <a name="what-should-customers-and-partners-know-before-running-microsoft-365-business-within-their-organization"></a>客戶與合作夥伴在其組織中執行 Microsoft 365 商務版之前應該知道什麼？ 
針對想要體驗 Microsoft 365 商務版完整功能的客戶，須在其現有桌上型電腦上執行 Windows 7、8.1 或 10 專業版<sup>[](#footnote2)</sup>。 如果他們還沒有這麼做，現有的 Windows 10 專業版電腦應執行 Creators Update。 

## <a name="deployment"></a>部署

### <a name="what-should-customers-consider-when-planning-a-microsoft-365-business-deployment"></a>客戶在規劃 Microsoft 365 商務版部署時，應考慮哪些事項？ 
成功部署 Microsoft 365 商務版的最直接途徑是參與 Microsoft 合作夥伴的作業。 他們接受過大量訓練，遇到過各種不同的客戶案例，因此絕對擁有最佳條件了解您的環境和需求。 擁有資深 IT 人員的客戶，可以使用 <a href="https://support.office.com/article/496e690b-b75d-4ff5-bf34-cc32905d0364" target="_blank">Microsoft 365 商務版快速入門</a>來協助部署 Microsoft 365 商務版。 

### <a name="does-microsoft-365-business-include-the-full-capabilities-of-microsoft-intune"></a>Microsoft 365 商務版是否包含完整的 Microsoft Intune 功能？ 
是，Microsoft 365 商務版訂閱者已授權給 for iOS、 Android、 MacOS 和其他跨平台裝置管理使用完整的 Intune 功能。 不像 3rd 方應用程式管理和設定 WiFi 設定檔、 VPN 憑證等，在 Microsoft 365 商務版，簡化的管理主控台中可用的功能可以管理在完整的 Intune 主控台。 您必須使用 Configuration Manager 來管理 EMS 授權所涵蓋的用戶端權限。 SCCM 相關的查詢，請參閱<a href="https://docs.microsoft.com/en-us/sccm/core/understand/product-and-licensing-faq#bkmk_csp" target="_blank">SCCM 常見問題集</a>
  

### <a name="does-azure-active-directory-premium-p1-come-with-microsoft-365-business"></a>Azure Active Directory 進階版 P1 並未隨附 Microsoft 365 商務版？ 
Microsoft 365 商務版建置於各種 Microsoft 技術上，它共用 Azure Active Directory 的某些功能，但並非完整版本。 建立 Microsoft 365 商務版中的管理原則與安全性依賴一些 Azure 的功能，但不會納入所有功能 （例如自助服務功能、 條件式存取功能，以及報告）。 客戶可以選擇以 Microsoft 365 商務版的附加元件形式購買 Azure Active Directory 進階版 P1 或 P2。 請[Microsoft 365 商務版服務說明](https://technet.microsoft.com/library/b042d1fd-c997-46d8-8c02-1df4c932f148)如需詳細資訊，參閱 
 
 
### <a name="does-microsoft-365-business-allow-customers-to-manage-macs"></a>Microsoft 365 商務版是否允許客戶管理 Mac？ 
Intune 可協助您安全地管理 iOS、 Android、 Windows 及 macOS 裝置。 
 
### <a name="what-is-windows-autopilot"></a>什麼是 Windows AutoPilot？ 
Windows AutoPilot 是一種服務，可簡化新 Windows 10 電腦部署作業。 使用者第一次登入 Microsoft 365 商務版時，便可完成此程序，IT 人員完全不需要碰觸裝置，只要利用 Microsoft 365 商務版來集中管理控制項即可。 針對正在執行 Windows 10 專業版 Creators Update (或更新版本) 且已恢復出廠預設值的現有電腦，也可以使用 Windows AutoPilot。 Windows AutoPilot 詳細可以位於<a href="https://blogs.technet.microsoft.com/windowsitpro/2017/06/29/modernizing-windows-deployment-with-windows-autopilot/" target="_blank">此年 6 月，2017年部落格貼文</a>。

## <a name="compatibility"></a>相容性

### <a name="can-i-add-office-365-add-ons-to-microsoft-365-business"></a>我可以在 Microsoft 365 商務版中加入 Office 365 附加元件嗎？ 
可以加入到 Office 365 商務進階版的所有附加元件，都可以加入到 Microsoft 365 商務版。 這表示，您可以購買進階威脅防護、進階安全性管理、Office 365 雲端 App 安全性、進階合規性、威脅情報、MyAnalytics、PowerBI 專業版與音訊會議。 
 
### <a name="can-i-add-phone-system-and-calling-plans-to-microsoft-365-business"></a>我可以在 Microsoft 365 商務版中加入電話系統和通話方案嗎？ 
不行，電話系統和通話方案會保留給有更多進階需求的客戶。 需要這些功能的客戶，應該查看 Microsoft 365 企業版方案。 
 
### <a name="can-microsoft-365-business-customers-use-windows-defender-advanced-threat-protection"></a>Microsoft 365 商務版客戶是否可以使用 Windows Defender 進階威脅防護？ 
否，需要 Windows Defender 進階威脅防護的客戶，需要 Windows 10 企業版 E5 或 Microsoft 365 企業版 E5。 
 


### <a name="can-customers-create-hosted-windows-10-vms-with-a-microsoft-365-business-subscription"></a>客戶是否可以使用 Microsoft 365 商務版訂閱建立裝載的 Windows 10 VM？ 
否，需要虛擬化的客戶應該購買 Windows 10 企業版或 Microsoft 365 企業版訂閱。

目前正在銷售 Office 365 的合作夥伴，可以使用相同的諮詢銷售方法來銷售 Microsoft 365 商務版。 此外，我們正推出適合您銷售小組使用的更多資源和訓練，讓他們認識客戶現有桌面環境、Active Directory 依賴、行動通訊和安全性的需求，以和客戶有效地溝通 Microsoft 365 商務版的整體價值。 在位於 [http://partners.office.com/microsoft365business](http://partners.office.com/microsoft365business) 的 Office 合作夥伴入口網站上可找到這些資源。 

### <a name="how-can-microsoft-365-business-help-partners-increase-the-profitability"></a>Microsoft 365 商務版如何協助合作夥伴提高獲利？ 
Microsoft 365 商務版可透過更高的作業效率和銷售其他服務所帶來的收益，協助合作夥伴降低成本。 Forrester Research 在 2017 年 6 月所做Microsoft 365 商務版總體經濟影響 (TEI) 研究 (https://partners.office.com/TEIBusiness) 顯示，Microsoft 365 商務版對於合作夥伴獲利有正面影響。 

合作夥伴在 TEI 研究中報告，他們對於使用 Microsoft 365 商務版的預期： 

- \[一次性\] 部署及諮詢服務收益增加 20% 點數
- 受管理服務的市場觸及率增加 10% 點數
- 諮商和 \[進行中\] 受管理服務利潤 (因降低成本而獲利) 增加 8% 點數

### <a name="what-resources-are-available-to-partners-to-sell-deploy-and-support-microsoft-365-business"></a>哪些資源可供合作夥伴用於銷售、部署及支援 Microsoft 365 商務版？
Microsoft 提供多樣的雲端解決方案提供者合作夥伴適用資源選項，可供行銷、銷售及支援 Microsoft 365 商務版。 這些資源可在 [https://partners.office.com/microsoft365business](https://partners.office.com/microsoft365business) 找到。

### <a name="what-up-sell-opportunities-does-microsoft-365-business-give-partners"></a>Microsoft 365 商務版提供合作夥伴哪些向上行銷機會？ 
Microsoft 365 商務版可讓合作夥伴藉由建立紮實且安全的平台供銷售額外的服務，以及升級現有的產品及服務，維持其在客戶心中受信任的顧問定位。 Microsoft 365 商務版現可利用 Exchange Server、Exchange Online 或 Office 365 商務基本版，提供與客戶討論升級的商機。 合作夥伴也可以從提高受管理服務和/或每個使用者支援費用，獲取其他收益。 
 
有了 Microsoft 365 商務版內含的新 Windows AutoPilot 功能，以往因已部署後勤和投入成本而勉強銷售新 Windows 裝置的客戶，現在會發現這個商機有更大的吸引力。 對於內部部署及行動裝置安全性相當有自信的客戶，更可能願意投資於額外的服務，例如 Dynamics 365。 
 
### <a name="should-partners-sell-microsoft-365-business-over-other-plans-from-microsoft"></a>合作夥伴應透過 Microsoft 的其他方案銷售 Microsoft 365 商務版嗎？ 
Microsoft 雲端解決方案提供者應一律使用最適合客戶企業需求和預算的方式銷售方案。 例如，如果客戶必須遵循隱私權與安全性法規，CSP 可以銷售 Microsoft 365 商務版加上可協助客戶符合其需求任何附加元件，或可建議 Microsoft 365 商務版 E Sku 所提供的進階安全性與管理。 
 
### <a name="some-of-my-customers-have-devices-that-are-not-genuine-will-microsoft-365-business-make-these-devices-genuine"></a>我的部分客戶使用的裝置不是正版，Microsoft 365 商務版是否能將這些裝置變成正版？ 
Microsoft 365 商務版無法將非正版 Windows 變成正版。 但 Microsoft 365 商務版的確可提供升級的好處，讓這些執行正版 Windows 7、8 或 8.1 專業版的客戶升級至最新正版的 Windows 10 專業版。 
 
### <a name="what-support-is-available-to-csp-partners-for-the-microsoft-365-business-preview"></a>針對 Microsoft 365 商務版預覽版的雲端解決方案提供者合作夥伴提供支援何種？ 
現今適用於 CSP 合作夥伴 (頂級支援和進階支援方案) 的相同支援管道已經針對 Microsoft 365 商務版接受相關訓練，並準備好為合作夥伴提供支援。 

### <a name="what-is-the-gdpr-and-how-does-microsoft-365-business-help-customers-with-their-compliance-obligations"></a>什麼是 GDPR，以及 Microsoft 365 商務版如何協助客戶達成其合規義務？
一般資料保護法規 (GDPR) 是完整的新隱私權法，可讓歐洲經濟共同體 (EU) 居民更能管控他們的「個人資料」，並要求組織維護個人資料的完整性。 GDPR 要求組織控制或處理繫結至歐盟居民的個人資料，如此只會使用符合 GDPR 對於個人資料處理要求之協力廠商資料處理器。 Microsoft 已在 2017 年 3 月對這些保證提供履約保證。 有任何疑問 Microsoft 可以如何協助他們符合其其他 GDPR 義務的客戶應該了解進階合規性和安全性功能可作為附加元件 （例如 Azure 資訊保護，資料外洩防護，進階選項威脅防護等等）和其他套件 (例如 Microsoft 365 企業版 E5)。 如需詳細資訊，請瀏覽 [www.microsoft.com/gdpr](https://www.microsoft.com/gdpr)。
