# <a name="microsoft-365-business-frequently-asked-questions"></a>Microsoft 365 商務版常見問題集

## <a name="general"></a>一般

### <a name="what-is-microsoft-365-business"></a>什麼是 Microsoft 365 商務版？ 
Microsoft 365 是整合式的解決方案，聚集所有同等級最佳的生產力工具、 安全性和裝置管理功能適用於小型至中型企業。 包括：

**一群商務生產力和共同作業工具** 
* Word、Excel、PowerPoint、Outlook、OneNote、Publisher 和 Access 
* Exchange、 OneDrive、 Skype for Business、 Microsoft Teams 和 SharePoint。 
* 從 Office (Bookings、 MileIQ<sup>[1](#footnote1)</sup>) 的商務應用程式。 
 
**企業級裝置管理與安全性功能** 
- 可協助保護您免於網路釣魚和複雜的惡意程式碼與 Office 365 進階威脅防護計劃 1 和 Windows Defender 惡意探索防護等外部威脅。 
-  協助控制及管理如何敏感資訊存取和傳輸與 Office 365 資料外洩防護原則和 Azure 資訊保護方案 1。 
-  協助保護、 保留，並備份您的資料與 Exchange Online Archiving。  
-   適用於 Office 和使用 Intune 應用程式防護其他行動裝置應用程式的應用程式防護。 
-  裝置管理 Windows 10 電腦、 MacOS 及行動裝置與 Intune 裝置管理。 
-  多重要素驗證、 自助密碼重設與條件式存取的身分識別保護。
-  跨裝置的一致安全性設定&mdash;跨裝置; 公司資料保護Windows Defender，它一定是上和最新狀態。 
 
 
**簡化裝置部署和使用者設定** 
* 單一管理主控台用於設定及管理使用者和裝置 
* 自動安裝的 Windows 10 電腦上的 Office 應用程式。 
* 保持最新的 Office + Windows 10。 
* 精簡的電腦透過 Windows AutoPilot 部署。

**其他權利** 
- Microsoft 365 商務版客戶也可以存取 Windows 虛擬桌面和共用電腦啟用 Office 功能。 閱讀[Microsoft 365 商務版部落格](https://techcommunity.microsoft.com/t5/Microsoft-365-Business-Blog/bg-p/Microsoft365BusinessBlog)了解詳細資訊。 另請參閱[Microsoft 365 商務版服務說明](https://docs.microsoft.com/office365/servicedescriptions/microsoft-365-service-descriptions/microsoft-365-business-service-description)。

### <a name="who-should-consider-adopting-microsoft-365-business"></a>誰應考慮採用 Microsoft 365 商務版？ 
Microsoft 365 商務版是完整的雲端安全性解決方案，可讓您：  
- 防禦針對具有複雜網路釣魚和勒索軟體防護的進階 cyberthreats 貴公司。 
- 使用加密來保護資料不小心與有人未獲授權才能看見共用控制敏感資訊的存取權。 
- 安全連線到您的業務，並協助保護您的資料在 iOS、 Android、 Windows 及 Mac 裝置安全，以及您的 Windows 裝置上最新的裝置。

Microsoft 365 商務版是完全整合以提供您熟悉的位置與管理、 帳單和 24 x 7 支援的 Office 365。 以少於 300 員工建置適用於企業的企業級技術所組成。 

### <a name="how-can-i-get-microsoft-365-business-for-my-business"></a>如何取得適合我的公司使用的 Microsoft 365 商務版？ 
Microsoft 365 商務版可透過<a href="https://www.microsoft.com/solution-providers/search" target="_blank">Microsoft 合作夥伴</a>或直接向<a href="https://www.microsoft.com//microsoft-365/business" target="_blank">Microsoft</a>購買。 在選擇是否要直接向 Microsoft 購買或透過 Microsoft 合作夥伴購買之前，應該先考慮將您員工的能力，並維持 IT 基礎結構的需求。 Microsoft 合作夥伴可協助您部署和管理您的 IT 基礎結構，包括 Microsoft 解決方案。

### <a name="how-much-does-microsoft-365-business-cost"></a>Microsoft 365 商務版價格多少？ 
如果直接向 Microsoft 購買，根據_年度合約_，每一名使用者每個月支付 20 美元即可使用 Microsoft 365 商務版。 透過 Microsoft 合作夥伴購買時，價格可能因合作夥伴提供的服務以及 Microsoft 365 商務版計價模式的不同而異。 

### <a name="is-there-a-cap-to-how-many-microsoft-365-business-seats-a-customer-can-have"></a>有限制客戶最多可以擁有多少 Microsoft 365 商務版基座數嗎？ 
Microsoft 365 商務版專為小中型企業使用低到中型 IT 複雜性需求。 客戶可以購買為其組織的最多 300 個 Microsoft 365 商務版授權。 客戶可以混合及比對雲端訂用帳戶。 因此，根據其 IT 需求，客戶可能會新增 Microsoft 365 企業版授權到相同的帳戶。 
 
當客戶假設環境中多個訂閱類型所組成時，他們應該使用來決定最佳方式來管理並保護各種訂用帳戶為 Microsoft 365 商務版和 Microsoft 365 企業版使用其信任 IT 顧問保護和管理應用程式和資料的不同功能。

### <a name="can-i-combine-microsoft-365-business-with-other-microsoft-subscription-offerings"></a>是否可以將 Microsoft 365 商務版和其他 Microsoft 訂閱優惠方案組合在一起？ 
是的，客戶可以將其 Microsoft 365 商務版訂閱與來自 Azure、Dynamics 365、Enterprise Mobility + Security 及 Office 365 的方案與附加元件組合在一起。

### <a name="is-everyone-in-my-business-required-to-have-a-microsoft-365-business-subscription"></a>我公司的每個人是否都需要 Microsoft 365 商務版訂閱？ 
否，儘管使用 Microsoft 365 商務版訂閱管理裝置的使用者才能使用安全性與管理的權益，並不是每個人都需要 Microsoft 365 商務版訂閱。

### <a name="how-can-i-know-if-the-hardware-and-software-i-run-today-is-compatible-with-microsoft-365-business"></a>如何知道我今天執行的硬體和軟體是否與 Microsoft 365 商務版相容？ 
如果您目前執行的硬體執行 Windows 7 專業版或更新版本，它可能符合 Microsoft 365 企業版的最低需求。 Cortana、 等 Windows Hello，多點觸控某些 Windows 10 功能需要只有較新的電腦的特定硬體。 請參閱如需詳細資訊的<a href="https://www.microsoft.com/windows/windows-10-specifications" target="_blank">Windows 10 專業版的系統需求</a>。 
 
現有的傳統型 (Win32) 應用程式相容性應該也很好，大部分現有的應用程式不需要任何變更就能運作。 客戶及其信任的 IT 顧問應該閱讀建議的應用程式測試程序以了解 <a href="https://docs.microsoft.com/windows/deployment/planning/windows-10-compatibility#recommended-application-testing-process" target="_blank">Windows 10 相容性</a>，並且檢閱 <a href="https://products.office.com/office-system-requirements#subscription-plans-section" target="_blank">Office 系統需求</a>以確保平順轉換到 Microsoft 365 商務版。 

### <a name="what-is-windows-10-business"></a>什麼是 Windows 10 商務版？ 
Windows 10 商務版是一組的雲端服務和裝置管理功能，可補充 Windows 10 專業版和啟用集中式的管理以及 Microsoft 365 商務版的安全性控制。

Windows 10 商務版也隨附於某些進階版 Microsoft Defender 功能，例如控制資料夾存取和網路防護的 web 式執行緒。 此外，它還包含 Windows AutoPilot，簡化部署新的 Windows 10 電腦適用的服務。

如果您有裝置已授權的 Windows 7，8，和 8.1 專業版、 Microsoft 365 商務版提供升級到 Windows 10 專業版，也就是部署 Windows 10 商務版的必要條件。
> [!IMPORTANT]
> 您必須提供原始的產品金鑰，當您升級時，否則將無法運作升級。 

### <a name="how-does-microsoft-365-business-help-support-our-bring-your-own-device-byod-policy"></a>Microsoft 365 商務版如何協助支援我們的「攜帶您自己的裝置」(BYOD) 原則？ 
許多員工寧願使用自己的行動電話或平板電腦來存取個人和工作資訊，而不是依照每一種目的執行多個裝置。 使用個人裝置工作雖然很平常，卻也增加商務資訊遭到不當利用的風險。 許多競爭激烈的行動數據保護方案要求使用者切換到裝置上的特定模式，或使用另一種讓使用者覺得受干擾而避免使用的複雜機制。 
 
Microsoft 365 商務版提供客戶簡單但強大的方式，讓員工以工作時的能力提供商務使用其個人的裝置，來防止這些裝置無法存取、 保留，及/或共用的商務資訊。 更加明確： 
* **Office 行動裝置應用程式的應用程式防護**可協助保護 Office 資料，藉由執行強制原則，例如自動刪除未連線至服務、 需要的資訊只會儲存在商務用 OneDrive，需要存取 Office 應用程式] 中，pin 碼/指紋驗證和防止公司資料從 Office 應用程式複製到個人應用程式的指定時間後的 [商務資料包括電子郵件、 行事曆、 連絡人及 iOS 和 Android 的行動裝置上的文件。 針對其他行動裝置 app 透過 Intune 行動應用程式管理也適用於 Microsoft 365 商務版訂閱者。 
  
* **Windows 10 電腦適用的行動裝置管理**可讓企業選擇設定並執行功能，例如 Windows Defender 防護惡意程式碼入侵、自動更新，並在規定的時間後關閉螢幕。 此外，您可以完全商務應用程式和透過系統管理中心的資料抹除遺失或遭竊的 Windows 10 裝置。 

* **裝置管理的 iOS、 Android & MacOS**功能可協助企業安全地管理各種裝置生態系統，包括 iOS、 Android、 Windows 及 MacOS 裝置。 

### <a name="how-does-microsoft-365-business-help-protect-pcs-in-my-organization-from-malicious-attacks"></a>Microsoft 365 商務版如何協助保護組織中的電腦使其免受惡意攻擊？ 
使用 Microsoft 365 商務版管理的電腦受 Windows Defender 保護。 Windows 10 上 1 種防毒功能，比其他任何解決方案保護更多電腦免受病毒、惡意程式碼、間諜軟體及其他威脅攻擊。  使用 Microsoft 365 商務版，組織可確保 Windows Defender 保護是執行中且一律最其所有 Windows 10 裝置上。  
 
Windows 10 商務版也包含 Windows Defender 惡意探索防護，一組新的入侵防護功能。 下列其中一個其功能，受控資料夾存取權，停止勒索軟體鎖定資料夾，並防止未經授權的應用程式存取使用者的重要檔案。 

### <a name="whats-the-difference-between-office-365-business-premium-microsoft-365-business-and-microsoft-365-enterprise"></a>Office 365 商務進階版、 Microsoft 365 商務版，以及 Microsoft 365 企業版之間的差異為何？ 
Microsoft 有各種生產力和安全性管理供應項目，小型至中型客戶可能會考慮升級其桌面與裝置基礎結構，每個將越來越多強大功能時。 

**Office 365 商務進階版**提供最佳的生產力與 Office 365 應用程式和服務，但不包含 Microsoft 365 商務版的應用程式防護和裝置管理功能。

**Microsoft 365 商務版**結合了 Office 365 應用程式和服務具有進階的安全性功能，協助保護您的商業針對進階 cyberthreats、 保護您的資料和管理您的裝置。 它包括可管理裝置和資料原則的簡化管理主控台。 許多小型至中型企業可以使用 Microsoft 365 商務版最佳提供服務。 
 
**Microsoft 365 企業版**是一組的授權計劃，透過 Microsoft 365 商務版提供增強的層級的合規性和安全性管理，且為企業客戶和客戶已超過 300 位使用者所設計。 此外，Microsoft 365 企業版方案提供額外的功能，包括商務智慧及分析工具。

## <a name="deployment"></a>部署

### <a name="what-should-customers-consider-when-planning-a-microsoft-365-business-deployment"></a>客戶在規劃 Microsoft 365 商務版部署時，應考慮哪些事項？ 
成功部署 Microsoft 365 商務版的最直接途徑是參與 Microsoft 合作夥伴的作業。 他們接受過大量訓練，遇到過各種不同的客戶案例，因此絕對擁有最佳條件了解您的環境和需求。 擁有資深 IT 人員的客戶，可以使用 <a href="https://support.office.com/article/496e690b-b75d-4ff5-bf34-cc32905d0364" target="_blank">Microsoft 365 商務版快速入門</a>來協助部署 Microsoft 365 商務版。 

### <a name="does-microsoft-365-business-include-the-full-capabilities-of-microsoft-intune"></a>Microsoft 365 商務版是否包含完整的 Microsoft Intune 功能？ 
是，Microsoft 365 商務版訂閱者已授權給 for iOS、 Android、 MacOS 和其他跨平台裝置管理使用完整的 Intune 功能。 不適用於 Microsoft 365 商務版中的簡化的管理主控台 like 協力廠商應用程式管理和設定 WiFi 設定檔，VPN 憑證，您可以管理完整的 Intune 主控台中的功能。 
  

### <a name="does-azure-active-directory-premium-p1-come-with-microsoft-365-business"></a>Azure Active Directory 進階版 P1 並未隨附 Microsoft 365 商務版？ 
Microsoft 365 商務版包含選取 Azure AD Premium P1 (AADP P1) 功能，例如自助密碼重設與 AD 回寫、 Azure MFA 和條件式存取。 它不包含 AADP P1 的全部內容。 如需詳細資訊，請參閱[Microsoft 365 商務版服務說明](https://technet.microsoft.com/library/b042d1fd-c997-46d8-8c02-1df4c932f148)。
 
 
### <a name="does-microsoft-365-business-allow-customers-to-manage-macs"></a>Microsoft 365 商務版是否允許客戶管理 Mac？ 
Intune 可協助您安全地管理 iOS、 Android、 Windows 及 MacOS 裝置。 
 
### <a name="what-is-windows-autopilot"></a>什麼是 Windows AutoPilot？ 
Windows AutoPilot 是一種服務，可簡化新 Windows 10 電腦部署作業。 當使用者登入 Microsoft 365 商務版第一次可完成此程序&mdash;而不屬於觸摸裝置的 IT&mdash;利用 Microsoft 365 商務版的集中式的管理控制項。 針對正在執行 Windows 10 專業版 Creators Update (或更新版本) 且已恢復出廠預設值的現有電腦，也可以使用 Windows AutoPilot。 <a href="https://blogs.technet.microsoft.com/windowsitpro/2017/06/29/modernizing-windows-deployment-with-windows-autopilot/" target="_blank">此年 6 月 2017年部落格文章</a>中，可以找到有關 Windows AutoPilot 的詳細資訊。

## <a name="compatibility"></a>相容性

### <a name="can-i-add-office-365-add-ons-to-microsoft-365-business"></a>我可以在 Microsoft 365 商務版中加入 Office 365 附加元件嗎？ 
可以加入到 Office 365 商務進階版的所有附加元件，都可以加入到 Microsoft 365 商務版。 這表示您可以購買 Office 365 雲端 App 安全性、 進階合規性、 威脅情報、 MyAnalytics、 PowerBI Pro 及音訊會議。 
 
### <a name="can-i-add-phone-system-and-calling-plans-to-microsoft-365-business"></a>我可以在 Microsoft 365 商務版中加入電話系統和通話方案嗎？ 
不行，電話系統和通話方案會保留給有更多進階需求的客戶。 需要這些功能的客戶，應該查看 Microsoft 365 企業版方案。 
 
### <a name="can-microsoft-365-business-customers-use-windows-defender-advanced-threat-protection"></a>Microsoft 365 商務版客戶是否可以使用 Windows Defender 進階威脅防護？ 
否，需要 Windows Defender 進階威脅防護的客戶，需要 Windows 10 企業版 E5 或 Microsoft 365 企業版 E5。

Microsoft 365 商務版可透過更高的作業效率和銷售其他服務所帶來的收益，協助合作夥伴降低成本。 Forrester Research 在 2017 年 6 月所做Microsoft 365 商務版總體經濟影響 (TEI) 研究 (https://www.microsoft.com/microsoft-365/partners/PartnerTEI) 顯示，Microsoft 365 商務版對於合作夥伴獲利有正面影響。 

合作夥伴在 TEI 研究中報告，他們對於使用 Microsoft 365 商務版的預期： 

- \[一次性\] 部署及諮詢服務收益增加 20% 點數
- 受管理服務的市場觸及率增加 10% 點數
- 諮商和 \[進行中\] 受管理服務利潤 (因降低成本而獲利) 增加 8% 點數

Microsoft 提供多樣的雲端解決方案提供者合作夥伴適用資源選項，可供行銷、銷售及支援 Microsoft 365 商務版。 這些資源可在 [https://partners.office.com/microsoft365business](https://partners.office.com/microsoft365business) 找到。

### <a name="what-up-sell-opportunities-does-microsoft-365-business-give-partners"></a>Microsoft 365 商務版提供合作夥伴哪些向上行銷機會？ 
Microsoft 365 商務版可讓合作夥伴，維護與客戶位置會保持信任的顧問建立實心且安全的平台，上面銷售其他服務及升級現有的產品和服務。 Microsoft 365 商務版可提供機會讓客戶現在可以使用 Exchange Server、 Exchange Online 或 Office 365 商務基本版與已升級的討論。 協力廠商，也可能會獲得更多收益增加受管理的服務和/或 peruser 支援費用從。 
 
使用新的 Windows AutoPilot 功能中包含 Microsoft 365 商務版，以及已被願意部署後勤因為銷售新的 Windows 裝置的合作夥伴，成本會發現這個機會更吸引人。 有自信其內部部署和行動裝置的安全性中的客戶也是較容易投入更多的服務，例如 Dynamics 365。 
 
### <a name="should-partners-sell-microsoft-365-business-over-other-plans-from-microsoft"></a>合作夥伴應透過 Microsoft 的其他方案銷售 Microsoft 365 商務版嗎？ 
Microsoft 雲端解決方案提供者應一律使用最適合客戶企業需求和預算的方式銷售方案。 例如，如果客戶必須遵守的隱私權和安全性法規，雲端解決方案提供者可能銷售 Microsoft 365 商務版加上任何附加元件，協助客戶符合其需求或可能的進階的安全性與管理 Microsoft 365 商務版所提供建議企業 Sku。 
 
### <a name="some-of-my-customers-have-devices-that-arent-genuine-will-microsoft-365-business-make-these-devices-genuine"></a>我的部分客戶擁有不是正版; 的裝置將 Microsoft 365 商務版將這些裝置正版？ 
Microsoft 365 商務版不盡否則正版的 Windows 版本，正版。 Microsoft 365 商務版並未提供升級權益可讓客戶執行真正的 Windows 7、 8 或 8.1 專業版升級至最新、 正版版本的 Windows 10 專業版。 

### <a name="what-is-the-gdpr-and-how-does-microsoft-365-business-help-customers-with-their-compliance-obligations"></a>什麼是 GDPR，以及 Microsoft 365 商務版如何協助客戶達成其合規義務？
一般資料保護法規 (GDPR) 是完整的新隱私權法，可讓歐洲經濟共同體 (EU) 居民更能管控他們的「個人資料」，並要求組織維護個人資料的完整性。 GDPR 會要求組織只使用協力廠商資料處理器符合個人資料處理的 GDPR 需求歐盟居民會受限於該控制項或程序個人資料。 Microsoft 已在 2017 年 3 月對這些保證提供履約保證。 有任何疑問 Microsoft 可以如何協助他們符合其其他 GDPR 義務的客戶應該了解 Microsoft 365 商務版 （例如，Azure 資訊保護資料中可用的安全性功能與進階合規性外洩防護、 進階威脅防護，依此類推） 和其他套件 (例如，Microsoft 365 企業版 E5)。 如需詳細資訊，請瀏覽 [www.microsoft.com/gdpr](https://www.microsoft.com/gdpr)。
