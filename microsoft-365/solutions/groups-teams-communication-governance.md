# <a name="communications-governance-for-collaboration-scenarios"></a>共同作業案例的通訊管理

Microsoft 365 提供許多控制項，可協助您管理組織中的通訊，包括訊息和會議設定。 請查看這些選項，並考慮其如何對應至您的業務需求、資料敏感度，以及您的使用者需要共同作業的人員範圍。

下表提供可在 Microsoft 365 中使用之通訊控制的快速參考。 以下各節提供進一步的資訊。

||控制可在頻道中開始和回復文章的使用者。|[在 Microsoft 小組中設定及管理通道仲裁](https://docs.microsoft.com/microsoftteams/manage-channel-moderation-in-teams)|
|Meetings|||

## <a name="messaging"></a>消息

您可以控制哪些聊天及通道郵件功能（例如：編輯或刪除郵件）可供 Microsoft 小組中的使用者使用郵件原則使用。 您可以為不同的使用者和群組建立不同的原則。

[管理 Teams 中的訊息原則](https://docs.microsoft.com/microsoftteams/messaging-policies-in-teams)

您可以設定頻道仲裁，以控制誰可以開始新的文章，以及如何在小組頻道回復文章。

[在 Microsoft 小組中設定及管理通道仲裁](https://docs.microsoft.com/microsoftteams/manage-channel-moderation-in-teams)

## <a name="meetings"></a>Meetings

您可以控制小組會議參與者可使用的功能，包括排程、內容共用、參與者及音訊和影片原則。

[管理 Teams 中的會議原則](https://docs.microsoft.com/microsoftteams/meeting-policies-in-teams)

您可以為小組會議控制匿名加入，以允許具有會議連結的任何人加入會議。

[允許匿名使用者加入會議](https://docs.microsoft.com/microsoftteams/meeting-settings-in-teams#allow-anonymous-users-to-join-meetings)


## <a name="communication-compliance"></a>通訊合規性

Microsoft 365 中的通訊法規遵從性可讓您檢查冒犯性語言的通訊、機密資訊，以及與內部及法規標準相關的資訊。 可監視聊天通訊、信箱和 Yammer 訊息，以產生警示。 使用管理工具，您可以快速識別並對具有原則相符的郵件採取動作。

[Microsoft 365 中的通訊法規遵從性](https://docs.microsoft.com/microsoft-365/compliance/communication-compliance)
