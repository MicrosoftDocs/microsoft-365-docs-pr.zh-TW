# <a name="step-1-create-a-test-base-account"></a>步驟1：建立測試基底帳戶

如果您沒有 Azure 訂閱，請先建立 [免費帳戶](https://azure.microsoft.com/en-us/free/) ，再開始。

## <a name="enter-details-for-test-base-account"></a>輸入測試基本帳戶的詳細資料
 
1. 在 Azure 入口網站中搜尋「 **測試基礎** 」。

![建立測試基本帳戶搜尋影像](Media/CreateTestAccount1.png)

2. 按一下 [ **新增** ] 以建立測試基底帳戶。

![按一下 [新增] 以建立帳戶](Media/CreateTestAccount2.png)

3.  通讀 [ ```Terms of Use``` 選取] 核取方塊，以確認您對的滿意度 ```Terms of Use``` 。

![回顧使用條款](Media/CreateTestAccount3.png)

4.  在下列需求下填入正確資訊： 
    -   訂閱：資源群組
    -   實例詳細資料： Name。

**目前，測試基底只支援標準的定價層。**

![選取詳細資料中的 [訂閱]、[資源群組] 及 [輸入]](Media/CreateTestAccount4.png)

5.  最後，按一下 ```Review + Create``` 以驗證及啟用新建立的帳戶。
