# <a name="windows-server-application-testing"></a>Windows伺服器應用程式測試 

使用 Microsoft 365 的測試基底，您現在可以針對 Windows Server 2016 和2019驗證應用程式，包括伺服器 Core！

若要開始針對 Microsoft 365 測試基礎上的 Windows Server 2016 和2019作業系統驗證上傳的應用程式，請遵循下列步驟：

1.   登入我們的自助上架入口網站。 從左導覽功能表中，選取 ```Upload new package``` ```Package catalogue``` 並填妥 [測試詳細資料] 底下的位置。

2.  選取 ```Security updates``` [作業系統] 更新類型：

![選取安全性更新](Media/selecting-security-updates.png)

3. 在 [要測試的 OS 版本] 底下，選取適用的作業系統版本。 您可以選取 [Windows server 作業系統版本] 或 [伺服器及用戶端] 作業系統的組合。

![選取 OS 版本](Media/selecting-OS-versions.png)

4. 提供其他必要資訊、複查所提供的詳細資料，並上傳應用程式套件。 上傳後，您可以在 [管理套件] 功能表索引標籤上，查看套件狀態。


5. 若要從您的應用程式驗證中查看測試結果和 insights，以用於 Windows Server 2016 和2019的預先發行安全性更新，請移至 [測試摘要] 頁面或 [安全性更新結果] 頁面。

![查看測試結果](Media/access-test-results.png)
