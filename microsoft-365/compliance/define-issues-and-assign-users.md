# <a name="define-issues-and-assign-users-in-advanced-ediscovery-classic"></a>定義問題，並在高級 eDiscovery (傳統) 中指派使用者

> [!NOTE]
> 進階電子文件探索需要具有進階合規性附加元件的 Office 365 E3，或適用於您組織的 E5 訂閱。如果您沒有該方案，且想要嘗試進階電子文件探索，您可以[註冊 Office 365 企業版 E5 試用版](https://go.microsoft.com/fwlink/p/?LinkID=698279)。 
  
在 [Advanced eDiscovery] 中，一或多個問題可以在案例內定義。 定義問題允許進一步的主題分類。 當連接至新案例時，會提供單一的預設問題。 您可以編輯預設的問題名稱和指派使用者給問題。 
  
## <a name="adding-or-editing-an-issue-and-assigning-users"></a>新增或編輯問題並指派使用者

1. 在 [ **相關性 \> 關聯性設定** ] 索引標籤中 \> 選取 **問題**。
    
    ![相關性設定問題](../media/dfd8f9ef-b167-4ed9-980e-00ae98a97169.png)
  
2. 若要新增問題，請按一下 * * + * * 圖示。 隨即會顯示 [ **新增問題** ] 對話方塊。 
    
    ![相關性設定新增問題](../media/c8e94982-139a-472a-b85d-282f2d742046.png)
  
    若要編輯問題，請按一下 [ **編輯** ] 圖示。 
    
3. 在 [ **發行名稱**] 中，輸入對事例具有描述性和重要的名稱。 
    
4. 在 [ **描述**] 中，輸入問題的相關資訊。
    
5. 選取 [ **啟用同時訓練** ] 核取方塊，以啟用選項。 此設定可讓多個檢閱者同時 (個別的範例) 中處理相同的問題。 
    
6. 在 [ **指派使用者給問題**] 的 [ **所有使用者** ] 清單中，選取要指派給問題的使用者，然後按一下向右箭號，將使用者新增至 [ **選取的使用者** ] 清單。 視需要重複。 在上方顯示的視窗中，"Admin" 會顯示為選取的使用者。 
    
    > [!NOTE]
    > 在相關性訓練週期之前或之後，都可以修改使用者對問題的指派。 
  
7. 在 [ **選取的使用者**] 中，從選取使用者名稱旁的下拉式清單中，選取下列其中一個採樣模式： 
    
  - **開啟**：可查看及標記檔案。 這是預設設定。
    
  - **空閒**：可以查看檔案;標記為選用。
    
  - **Off**：無法查看或標記檔案。
    
8. 新增問題後，請按一下 **[確定]**。
    
## <a name="deleting-issues"></a>刪除問題

可能會刪除問題 (意義上，從資料庫中移除) 之後立即移除，而不是該問題的實際工時。 
  
1. 在 [ **相關性 \> 關聯性設定** ] 索引 **卷** 標中，選取 [問題]。
    
2. 選取要從資料庫中刪除的問題，然後按一下 [ **刪除**]。
    
3. 隨即顯示確認訊息。 按一下 **[是]** 以確認。 
    
4. 按一下 [確定]。
