# <a name="commodity-futures-trading-commission-cftc-rule-131c-d-united-states"></a>貿易委員會 (CFTC) 規則 1.31(c-d) 美國境內的市售 Future

## <a name="about-cftc-rule-13c-d"></a>關於 CFTC 規則 1.3(c-d)

[市售 Future 貿易委員會](https://www.cftc.gov/)(CFTC) 獨立美國聯邦機構，規定了市售 future 和選項市場及多最近的交換市場。  
  
長期 CFTC 規則 1.31 定義 SEC Rule 17a-4(f) 所建立的記錄保留需求。 此外，它會指定必須維護的五年的電子記錄以及原始資料會保留 「 隨時都能存取 「 前兩年期間和可供整個期間委員會或正義美國部門檢查保留期間。  
  
在 2017 年， [CFTC 修訂其規則](https://www.cftc.gov/sites/default/files/idc/groups/public/@lrfederalregister/documents/file/2017-11014a.pdf)、 修改和現代化採用較少規定、 以原則為基礎提供更有彈性方式可以維護記錄的標準其 recordkeeping 規定。 這會使規則詳細技術中性，啟用管制的實體選擇最適合企業的技術同時保有 「 確保 recordkeeping 程序的可靠性。 「 保護措施 修訂的規則會移除組織維護兩年，原始的記錄，但仍保留的五年維護期間，harmonizes 作法公司 CFTC 和 SEC 所規範的需求

## <a name="microsoft-and-cftc-rule-131c-d"></a>Microsoft 和 CFTC 規則 1.31(c-d)

財務服務的客戶，表示下列其中一個領域中受到最嚴厲管制產業受限於複雜的佈建像保留金融交易和相關的通訊中的非可清除與非可修改的狀態。 之間最規定是規則 1.31 的美國市售 Future 貿易委員會 (CFTC) 的規定管制接管工作，保留書籍和記錄電子儲存媒體上的實體的嚴格要求。 儲存的記錄必須竄改與沒有變更或刪除它們之後指定的保留期間內的能力。 Microsoft Azure 不變的 Blob 儲存與原則鎖定和 Microsoft Office 365 with 保留鎖定有助於金融機構符合 CFTC 規則 1.31(c-d) 的儲存需求。

### <a name="microsoft-azure"></a>Microsoft Azure

若要評估與 CFTC 規則 1.31(c-d) Azure 合規性，Microsoft 保留獨立評估公司專門進行記錄管理及資訊治理規範，Cohasset 會產生關聯。 在產生報告中， [CFTC 1.31 (c)–(d) 合規性評估： Microsoft Azure 儲存體](https://servicetrust.microsoft.com/ViewPage/MSComplianceGuide?command=Download&downloadType=Document&downloadId=19b08fd4-d276-43e8-9461-715981d0ea20&docTab=4ce99610-c9c0-11e7-8c2c-f908a777fa4d_GRC_Assessment_Reports)，Cohasset 驗證[Azure 不變的 Blob 儲存體](https://docs.microsoft.com/azure/storage/blobs/storage-blob-immutable-storage)使用原則鎖定選項時，用來保留時間為基礎的 Blob，以非可清除與非可讀寫的 （蠕蟲） 格式時, 符合原則為基礎的需求 CFTC 規則。 每個 Blob （記錄） 受到防止修改、 覆寫，或刪除之前所需的保留期間已經過期和已釋放任何關聯的合法持有。 軟體提供者和機密的工作負載的協力廠商可以立即自信地仰賴 Azure 不變的 Blob 儲存體做為保留記錄的一次商店雲端解決方案。 金融機構現在可以建置自己運用這些功能時剩餘相容的應用程式。

### <a name="microsoft-office-365"></a>Microsoft Office 365

若要評估 Office 365 的合規性 CFTC 規則 1.31(c-d)，Microsoft 參與專門負責前置獨立的法律公司法規問題，Covington & Burling、 LLP。 在產生報告中，[在 Microsoft Office 365、 資料保留和規則 17a-4 符合性的封存](https://go.microsoft.com/fwlink/?linkid=830440)，Covington 進行驗證，[以保留鎖定的 Office 365](https://docs.microsoft.com/office365/securitycompliance/retention-policies#locking-a-retention-policy)也包含封存功能，可讓受管制的客戶將資料儲存在以協助他們符合 CFTC 需求保留記錄的方式。

在 Office 365 中封存可協助保留廣泛的資料，包括電子郵件、 語音信箱、 共用文件、 立即訊息和協力廠商資料。 特別是，在 Office 365 中封存可讓客戶若要設定全域或細微通訊保留原則，可將資料儲存在已定義的期間內和 beyond 非可重複燒錄、 非可清除的格式。
