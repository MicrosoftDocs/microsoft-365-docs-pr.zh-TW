# <a name="test-relevance-analysis-in-office-365-advanced-ediscovery"></a>在 Office 365 高级电子数据展示中测试相关性分析

> [!NOTE]
> 進階電子文件探索需要具有進階合規性附加元件的 Office 365 E3，或適用於您組織的 E5 訂閱。如果您沒有該方案，且想要嘗試進階電子文件探索，您可以[註冊 Office 365 企業版 E5 試用版](https://go.microsoft.com/fwlink/p/?LinkID=698279)。 
  
通过高级电子数据展示中的"测试"选项卡，您可以测试、比较和验证处理的总体质量。 这些测试在批处理计算后执行。 通过标记集合中的文件，专家可以最终确定每个标记的文件是否与案例实际相关。 
  
在单问题和多问题方案中，测试通常按问题执行。 每个测试后都可以查看结果，并且测试结果可以使用指定的测试文件返工。
  
## <a name="testing-the-rest"></a>测试其余的

"测试其余"测试用于验证剔除决策，例如，仅根据最终高级电子数据展示结果查看特定相关性截止分数以上的文件。 专家在选定的截止分数下查看文件样本，以评估该集中的相关文件数。
  
此测试提供"审阅"集和"测试休息"总体之间的统计信息和比较。 审核集的结果按培训期间的相关性计算。 结果包括基于设置和输入参数的计算，例如：
  
- 测试样本中文件数的样本统计信息，并确定相关文件。 
    
- "审阅"集和其余项的"填充"参数的表格比较，例如，文件数、相关文件的估计数量、估计的丰富度以及查找其他相关文件的平均成本。 费用参数设置可以由管理员设置。
    
1. 打开**相关性\>测试**选项卡。 
    
2. 在"**测试"** 选项卡中，**单击"新建测试"。** **将显示"创建测试"** 对话框，如以下示例所示。 
    
    ![相關性測試的其他結果](media/46e6898a-f929-4fd0-88d9-6f91d04b6ce2.png)
  
3. **在"测试名称"** 和"**描述"** 中键入名称和说明。
    
4. 在"**测试类型"** 列表中，**选择"测试其余"**
    
5. 在"**问题/类别"** 列表中，选择问题名称。 
    
6. 在"**加载"** 列表中，选择负载。 
    
7. **在"读取 %"** 中，接受默认值或为截止相关性分数选择一个值。 
    
8. **以"设置大小"** 或接受默认值。 请注意，还原图标将还原默认值。
    
9. **单击"开始标记"。** 生成测试样本。
    
10. 查看并标记"**相关性\>标记"** 选项卡中的每个文件，完成后，单击"**计算"。**
    
11. 在"测试"选项卡中，可以**单击"查看结果"** 以查看测试结果。 下图显示了一个示例。 
    
    ![測試其他結果](media/b95744a9-047d-4c29-992d-04fa7e58e58a.png)
  
在上图中，表**的"示例参数"** 部分包含有关专家标记的示例中的文件数以及该示例中的相关文件数的详细信息。 
  
表**的"填充参数"** 部分包含测试结果，包括分数低于所选截止分数的"审阅"文件组和分数高于所选截止分数的"其余"文件填充。 对于每个总体，将显示以下结果： 
  
- 包括具有读取百分比的文件 - 声明截止
    
- 文件总数 
    
- 相关文件的估计数量 
    
- 估计的丰富性 
    
- 查找另一个相关文件的平均审阅成本
    
## <a name="testing-the-slice"></a>测试切片

"测试切片"测试执行类似于"测试休息"测试的测试，但执行相关性读取百分之指定的文件集的段。
  
1. 打开**相关性\>测试**选项卡。 
    
2. 在"**测试"** 选项卡中，**单击"新建测试"。** **将显示"创建测试"** 对话框。 
    
3. **在"测试名称"****和"说明"** 中键入信息。
    
4. 在"**测试类型"** 列表中，**选择"测试切片"。**
    
5. 在"**问题"** 列表中，选择问题名称。 
    
6. 在"**加载"** 列表中，选择负载。 
    
7. **在"读取 % "中，** 接受默认的低范围值和高范围值，或选择截止相关性分数的值。 
    
8. **在"设置大小"** 中，选择一个值或接受默认值。
    
    还原图标将还原默认值。
    
9. **单击"开始标记"。** 生成测试样本。
    
10. 查看并标记"**相关性\>标记"** 选项卡中的每个文件，完成后，单击"**计算"。** 
    
11. 在"测试"选项卡中，可以**单击"查看结果"** 以查看测试结果。
