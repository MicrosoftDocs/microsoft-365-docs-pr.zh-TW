# <a name="breach-notification-under-the-gdpr"></a>GDPR 規定的外洩通知

Microsoft 十分重視一般資料保護規定 (GDPR) 的責任。有關 Microsoft 服務如何防止個人資料外洩，以及若發生外洩會如何回應和通知您的詳細資訊，請參閱下列主題：

- [Office 365](gdpr-breach-Office365.md) 
- [Azure](gdpr-breach-Azure.md)
- [Dynamics 365](gdpr-breach-Dynamics365.md)
- [Microsoft 支援服務與專業服務](gdpr-breach-Microsoft-Support-Professional-Services.md)

如需 Microsoft 如何偵測及回應個人資料外洩的詳細資訊，請參閱服務信任入口網站中的 [GDPR 規定的資料外泄通知](https://servicetrust.microsoft.com/ViewPage/GDPRBreach)。
