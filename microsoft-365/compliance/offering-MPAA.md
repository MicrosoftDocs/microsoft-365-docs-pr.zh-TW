## <a name="mpaa-overview"></a><span data-ttu-id="b7118-105">MPAA 概觀</span><span class="sxs-lookup"><span data-stu-id="b7118-105">MPAA overview</span></span>

## <a name="microsoft-and-mpaa"></a><span data-ttu-id="b7118-108">Microsoft 和 MPAA</span><span class="sxs-lookup"><span data-stu-id="b7118-108">Microsoft and MPAA</span></span>

<span data-ttu-id="b7118-109">在 2016 年 2 月，Microsoft Azure 成為第一個成功完成獨立 MPAA 稽核人員正式評定的超大型多租用戶雲端服務，並符合所有三個 MPAA 內容安全性最佳做法架構：通用、應用程式和雲端安全性指導方針。</span><span class="sxs-lookup"><span data-stu-id="b7118-109">In February 2016, Microsoft Azure became the first hyperscale, multitenant cloud service to successfully complete a formal assessment by independent MPAA auditors and comply with all three of the MPAA content security best practices frameworks: Common, Application, and Cloud Security Guidelines.</span></span>

<span data-ttu-id="b7118-110">MPAA 評定涵蓋通用指導方針的 48 個安全性主題，以及應用程式與雲端安全性指導方針的其他六個主題。</span><span class="sxs-lookup"><span data-stu-id="b7118-110">The MPAA assessment covers 48 security topics in the Common Guidelines, and an additional six in the Application and Cloud Security Guidelines.</span></span> <span data-ttu-id="b7118-111">這些內容是建立在產業公認的安全性標準 (例如 ISO/IEC 27001 和 NIST 800-53) 之上，並且符合最佳做法 (例如雲端安全性聯盟 (CSA) 雲端控制基礎)。</span><span class="sxs-lookup"><span data-stu-id="b7118-111">These are built on industry-accepted security standards such as ISO/IEC 27001 and NIST 800-53, and are aligned to best practices, such as the Cloud Security Alliance (CSA) Cloud Controls Matrix.</span></span>

<span data-ttu-id="b7118-112">Azure 合規性的正式評定代表與主要工作室業務往來的公司能夠使用 Azure 來協助減少通常與安全架設、管理、儲存和發佈內容的 IT 成本，同時遵守 MPAA 需求。</span><span class="sxs-lookup"><span data-stu-id="b7118-112">The formal assessment of Azure compliance means that companies who do business with major studios can use Azure to help reduce the IT costs that are normally associated with the secure creation, management, storage, and distribution of content — all while complying with MPAA requirements.</span></span> <span data-ttu-id="b7118-113">Azure 媒體服務、儲存體、虛擬網路和其他超過 30 個服務，提供雲端中的內容工作流程引擎，比起傳統內部部署生產程序更加安全，且在保護媒體資產下游更加有效。</span><span class="sxs-lookup"><span data-stu-id="b7118-113">Azure Media Services, Storage, Virtual Networks, and more than 30 other services provide a content workflow engine in the cloud that is more secure and scalable than traditional on-premises production processes and more effective at protecting media assets downstream.</span></span>
