# <a name="us-section-508"></a>美國區段 508

## <a name="about-us-section-508"></a>關於美國區段 508

美國國會 1998年和 2000 需要使其電子版與資訊技術 (EIT) 產品，例如軟體、 硬體、 電子化內容和支援文件，可供存取的聯邦機構中速度健法案行動不便人士使用。 508 條款 1973 （29 美國代碼 §794d），美國人力健法案的速度，因為斷定聯邦機構採購、 維護和使用 EIT 可確保行動不便人士的聯邦員工有相當於存取權，並使用的方式相對於其他聯邦員工 EIT 及資料。  
  
因為它是主要的軟體和美國聯邦與狀態政府雲端服務提供者，就有一個 Microsoft 致力於遵守所有相關的政府標準和相容性控制。 藉由遵守這些[機密性的協助工具標準](https://go.microsoft.com/fwlink/?linkid=2052068)，Microsoft 可以確保所有客戶 — 內部和外部政府 — 可以使用 Microsoft 服務和產品。

## <a name="microsoft-and-us-section-508"></a>Microsoft 和美國區段 508

美國 508 條款規定的 Microsoft 遵守指向進行技術和資料存取的所有客戶的承諾。 美國 508 是美國特有的協助工具需求，補充 EN-US 301 549 （歐洲） 和 WCAG （國際）。
