# <a name="working-with-microsoft-compliance-score-preview"></a>使用 Microsoft 規範評分（預覽）

本文說明如何使用合規性分數的重要元素。 您將會瞭解如何使用**改進的動作**來管理您的合規性工作流程。 您也將瞭解如何使用您的**解決方案**和**評估**頁面上的資訊，以及如何產生報告。

## <a name="manage-your-workflow-with-improvement-actions"></a>使用改進動作管理工作流程

改進相容性活動的**改進動作**。 每個改進動作都提供詳細的執行指導，以協助您與資料保護法規和標準相符。 動作可指派給組織中的使用者，以執行實施和測試工作。 您也可以在改進動作中儲存檔、記事及記錄狀態更新。

## <a name="view-your-improvement-actions"></a>查看您的改善動作

合規性分數儀表板會顯示您的**重要改進動作**，也就是具有最重要問題的最可用點數。

若要在 [**改進動作**] 頁面上查看您的所有改進動作，請選取儀表板上的 [**改進動作**] 索引標籤。 或者，選取 [在您的儀表板上的主要改進動作清單下，**查看所有改進動作**]。

如果您有很長的動作清單，篩選您的視圖可能會很有説明。 在 [動作] 清單的右上角選取 [**篩選**]。 當 [**篩選**] 飛入窗格出現時，根據規定和標準、解決方案和群組選取您的準則。 您也可以選取右上角的 [**群組**]，以自訂您的視圖。 從下拉式功能表中，選取 [按群組、解決方案、類別、動作類型或狀態來查看]。

此頁面的預設視圖不會顯示已**通過**測試狀態的改善動作。 若要查看已通過測試的動作，請檢查 [**篩選**] 飛入窗格中的 [已**傳遞**] 方塊。 僅限具有**超過**您分數之測試狀態的動作。

[改進動作] 頁面會顯示每個改進動作的下列資料點：

- **分數影響**：完成動作時，整體得分會增加的點數
- **規章**：動作相關的規章或標準
- **群組**：您指派動作的群組
- **解決方案**：您可以用來執行動作的解決方案
- **評估**：針對動作所在的評估（組織控制以符合特定符合性目標）
- **類別**：相關的資料保護類別（例如，保護資訊、管理裝置等等）
- **測試狀態**：
    - **None** -沒有記錄的狀態更新
    - **未評估**-尚未開始測試
    - **Not in 範圍**-不會從相容性分數計算中排除，而且不會增加您的分數
    - 已**部分測試**-測試尚未完成
    - **失敗的高風險**測試實施失敗，且適用的標準不相容的風險很高
    - 已成功測試**傳遞**的執行
- **指向**[已完成]：顯示可獲得的最大點數。

### <a name="improvement-actions-details"></a>改進動作詳細資料

每個改進動作都有詳細資料頁面。 此頁面提供完成建議動作的詳細執行指示，以解決**在 [一覽**標頭] 底下所列的相關標準與法規需求。

[詳細資料] 頁面是您可以啟動建議動作的所在位置。 您也可以將工作指派給其他使用者、更新狀態，以及附加附注和檔。

若要查看改進動作的詳細資料頁面：

1. 移至您的 [改進動作] 頁面。
2. 按一下或點擊 [預期的改進動作] 列中的任何地方，這會開啟其 [詳細資料] 頁面。

您可以在畫面的右上角選取向上或向下箭號，輕易地查看清單中的下一個或上一個改善動作。 如果您已在 [改進動作] 頁面上篩選清單，向上或向下移動會移至該篩選清單中的下一個專案。

## <a name="assign-improvement-actions"></a>指派改進動作

若要開始執行改進動作，您可以自行執行工作或指派給其他使用者。 指派的人員可能是：

- 企業原則擁有者
- IT 實施者
- 另一個員工負責執行任務

在識別適當的人員之後，請確定他們在合規性分數（合規性管理員、合規性資料管理員、安全性管理員或全域管理員）中擁有足夠的[角色](compliance-score-setup.md#set-user-permissions-and-assign-roles)，才能執行工作，請採取下列步驟： 

1. 從 [改進動作詳細資料] 頁面中，選取畫面左上方旁邊的 [**編輯狀態**]。 

2. 在 [編輯狀態] 飛入窗格中，按一下或點擊 [**指派給**] 方塊，它會填入使用者的**建議人員**清單。 您可以從清單中選取使用者，或輸入要指派動作之人員的電子郵件地址。

3. 選取 [**儲存並關閉**] 以完成工作分派。 指派的使用者將會收到已獲指派改進動作的電子郵件，然後可以從其儀表板開啟改進動作。

然後，已指派的使用者可以執行實施指示中所述的建議動作。

## <a name="perform-work-and-store-documentation"></a>執行工作和儲存庫檔

當您執行實施工作時，您可以將檔案和記事直接上傳至 [**附注與檔**] 區段中的 [改進] 動作。  此環境是一種安全且集中的存放庫，可協助您示範控制的滿意度，以符合法規標準和規定。 任何具有唯讀存取權的使用者都可以讀取此區段中的內容。 只有具有編輯許可權的使用者可以上傳和下載檔案，並輸入或編輯附注。

[**附注與檔**] 區段中的欄位包括：

**上傳檔**

- 選取 [**管理檔**] 以上傳任何相關檔案。
- 當 [管理檔] 飛出窗格開啟時，選取 [**新增**檔]，然後從您的系統選取檔案。 公認的檔案類型： 
  - 檔（.doc、.xls、.ppt、.txt、.pdf）
  - 影像（.jpg，.png）
  - 影片（mkv）
  - 壓縮檔（.zip、rar）
- 當您的檔案解析窗格中之後，請選取 [**關閉]，** 這會自動儲存檔附件。 接著，您將會看到 [**上傳**的檔] 下方所列的檔案。 
- 若要下載或刪除檔，請從檔案清單下選取 [**管理檔**]。 在快顯視窗窗格上，按一下或點擊檔列，以將其反白顯示，然後選取 [**下載**] 或 [**刪除]。**

**執行、測試及其他附注**

- 若要在這三個欄位中新增附注，請選取 [**編輯執行附注**] 下的任何欄位。
- 當快顯視窗開啟時，請在 [文字] 欄位中輸入記事，然後選取 [**儲存並關閉**]。
- 若要編輯附注，請選取 [**編輯執行附注**]，進行編輯，然後選取 [**儲存並關閉**]。

[附注] 欄位中沒有字元限制。 建議您保留附注摘要，讓您能輕鬆地從 [改進動作詳細資料] 頁面中進行查看及編輯。

## <a name="change-improvement-action-status"></a>變更改進動作狀態

您可以記錄每個改進動作的「實施」狀態和「日期」和「測試狀態」和「日期」。 以下是可用的欄位及狀態選項：

- **實施狀態**：從下列 [狀態] 選項中選取：
    - **尚未執行**-尚未執行的動作
    - 已**執行**的動作
    - **替代執行**-如果您使用其他協力廠商工具，或採取其他不包含在 Microsoft 建議中的動作，請選取此選項。
    - **計畫的**動作已計畫執行
    - **不在範圍內**-與您的組織不相關的動作選項;選取此狀態會將動作從計分中排除。取消選中它會將動作納入計分
- **實施日期**：實施狀態設定為 [已**實現**] 或 [**替代實現**] 時可用的欄位;在行事曆快顯視窗中切換以選取日期
- **測試狀態**：從下列選項中選取：
    - **未評估**-尚未開始測試
    - 已成功測試**傳遞**的執行
    - **失敗的低風險**-測試失敗、低風險
    - **失敗之中度風險**-測試失敗，中度風險
    - **失敗的高風險**-測試失敗，高風險
- **測試日期**：切換到 [行事曆] 彈出式功能表以選取日期

> [!NOTE]
> 任何具有編輯許可權的使用者都可以編輯「實施」和「測試狀態」欄位，而不只是**指派給**使用者。

## <a name="assign-improvement-action-to-assessor-for-completion"></a>將改進動作指派給 assessor 以完成

完成工作並上傳證據之後，下一步是設定實施狀態和日期，並將 [改進] 動作指派給 assessor 進行驗證。

評估者類型包括：

- 在組織內執行控制項驗證的內部評估者
- 外部評估者檢查、驗證及驗證合規性（例如，協力廠商獨立組織，以審核 Microsoft 雲端服務）

Assessor 會驗證工作並檢查檔，並選取適當的測試狀態。

**如果測試狀態為「已傳遞**」：動作已完成，而且**點數**會顯示已取得的可能點數的完整數目，並符合您的整體符合性分數。

**如果測試狀態為任何「失敗」的程度**：動作不符合需求，而且 assessor 可將其指派回適當的使用者進行其他工作。

## <a name="solutions-page"></a>解決方案頁面

[**解決方案] 頁面**是另一個著手執行改進動作以提升得分的起點。 

若要移至 [解決方案] 頁面，請選取儀表板上的 [**方案**] 索引標籤，或選取 [在您的儀表板的右上方] 中，選取 [**查看所有****會影響分數**的解決方案]。

[解決方案] 頁面會顯示以方案組織的盈餘分析和潛在點數。 從這個視圖中查看剩下的點數和改善動作，可協助您瞭解哪些解決方案需要更立即關注。

### <a name="filtering-your-solutions-view"></a>篩選您的解決方案視圖

若要篩選您的解決方案視圖：

1. 從評估清單的左上角選取 [**篩選**]。
2. 在 [飛入**篩選**] 窗格上，勾選 [所需的準則] （[標準] 和 [法規、解決方案、動作類型、合規性管理員群組、類別）] 旁邊。
3. 選取 [**套用**] 按鈕。 篩選窗格會關閉，您會看到篩選的視圖。

您也可以將您的視圖修改成按群組、產品或法規來查看評估，方法是從評估清單上方的 [**群組**] 下拉式功能表中選取群組的類型。

### <a name="taking-actions-from-the-solutions-page"></a>從 [解決方案] 頁面採取動作

[解決方案] 頁面會顯示您組織中已連接至 [改進動作] 的解決方案。 該表列出每個方案對您整體分數的貢獻，在該解決方案中取得的分數提升點，以及在該解決方案中群組的其餘改進動作數目，也會增加您的分數。

您可以使用下列兩種方式從這個畫面採取動作：

1. 在您預定方案的列上，選取 [**其他動作**] 欄底下的超連結號碼。 您會看到 [改進動作] 畫面的篩選視圖，顯示該解決方案的未測試改進動作。

2. 在您預定方案的列上，選取 [**開啟方案**] 欄下的 [**開啟**]。 您會在 Microsoft 365 和 Office 365 安全性與合規性中心中看到解決方案或位置，您可以採取建議的動作。

## <a name="assessments-page"></a>評估頁面

「評估」頁面會列出您為組織所選擇的評估。 您的相容性分數分母是由所有追蹤的評估所決定。 您新增的評估程度越高，您在 [改進動作] 頁面上看到的改善動作越多，而得分分母的越高。

若要取得評估頁面，請選取儀表板上的 [**評估**] 索引標籤。

在此頁面上，您可以快速查看每個評估的重要資訊：

- **狀態**：評估中所有改進動作的完成狀態會列為下列其中一項：
    - **不相容**：評估中的改進動作尚未執行且已成功測試;尚未開始工作
    - **進行中**：工作正在實施或測試改進動作中進行中;例如，評估中的「改進」動作已獲指派以進行工作，正處於實施及測試的處理常式中
- **評估進度**：評估的最後完成工作所占的百分比（以成功測試的控制項數目衡量）。
- **客戶管理的動作**：已完成的動作數目，可滿足客戶管理控制項的實施
- **Microsoft 受管理的動作**：已完成的動作數目，可滿足 Microsoft 管理控制項的實施
- **評估群組**：您建立的群組名稱，評估所在位置
- **相關服務**：相關聯的 Microsoft 365 服務
- **相關規章**：評估所要滿足的規章標準、原則或法律

### <a name="default-assessments"></a>預設評估

根據預設，您會在 [評估] 頁面上看到 Microsoft 365 資料保護基準評估。 合規性分數也提供數項預先設定的評估（[查看完整清單](compliance-score.md#templates)）。 您可以新增更多評估，以涵蓋合規性管理員中的其他法規和標準。

### <a name="managing-assessments"></a>管理評估

在公開預覽期間，您會移至 [合規性管理員] 工具，以發貨、自訂及管理評估。

在 [**評估**] 頁面的 [評估] 頁面上，選取 [在**合規性管理員中的「管理評估**] （位於評估清單頂端）。 此連結會帶您前往合規性管理員中的**評估**儀表板。

評估清單頂端的 [其他] 連結，會帶您前往 [合規性管理員] 中的 [**控制項資訊**] 儀錶**板，顯示**可促成您的合規性分數的 Microsoft 控制項。

### <a name="filtering-your-assessments-view"></a>篩選評估視圖

篩選您的評估：

1. 從評估清單的左上角選取 [**篩選**]。
2. 在 [飛入**篩選**] 窗格上，勾選所需準則（標準及規章，合規性管理員群組）旁的複選。
3. 選取 [**套用**] 按鈕。 篩選窗格會關閉，您會看到篩選的視圖。

您也可以將您的視圖修改成按群組、產品或法規來查看評估，方法是從評估清單上方的 [**群組**] 下拉式功能表中選取群組的類型。

### <a name="managing-improvement-actions-within-an-assessment"></a>管理評估中的改進動作

在 [**客戶管理的動作**] 欄的 [評估] 清單中，選取預定評估的列上的連結文字。 這會為您顯示改進動作頁面的篩選視圖，顯示該評估中的動作。

## <a name="reporting"></a>Reporting

您可以在合規性分數中匯出所有改進動作的報告。 從 [**改進動作**] 頁面中，選取您的螢幕左上角的 [**匯出**] 清單中的動作清單。 這會產生包含所有改進動作的 Excel 工作表，以及 [**改進動作**] 頁面上顯示的篩選類別。

您也可以遵循下列步驟，從合規性管理員匯出報告：

1. 在 [合規性管理員] 中，移至您的 [**控制項資訊**儀表板]。
2. 您將會看到 [**評估**] 索引標籤和 [**範本**] 索引標籤。  
3. 若要匯出評估：請選取 [**評估**] 索引標籤。使用 [**群組**] 和 [**評估**] 下拉式功能表，選取您要匯出的評估。 選取靠近螢幕右上方的 [**匯出**]。 將下載 Excel 檔案。 它包含以控制項分組的動作清單，包含您的實施及測試詳細資料。
4. 若要匯出範本：請選取 [**範本**] 索引標籤，然後從**範本**下拉式功能表中選擇您要匯出的範本。 選取靠近螢幕右上方的 [**匯出**]。 將下載 Excel 檔案。 它包含以控制項分組的動作清單，包含您的實施及測試詳細資料。
