# <a name="data-protection-impact-assessments-guidance-for-data-controllers-using-microsoft-azure"></a><span data-ttu-id="386b3-103">資料保護影響評估：資料控制者使用 Microsoft Azure 的指引</span><span class="sxs-lookup"><span data-stu-id="386b3-103">Data Protection Impact Assessments: Guidance for Data Controllers Using Microsoft Azure</span></span>

<span data-ttu-id="386b3-p101">在一般資料保護規定 (GDPR) 下，資料控制者若要處理「可能會對自然人的權利和自由造成高度風險」的作業，必須準備資料保護影響評估 (DPIA)。Microsoft Azure 本身不具有使用它的資料控制者應該建立 DPIA 的項目。然而，是否需要 DPIA，要視控制者「如何」\*\* 部署、設定、使用 Microsoft Azure 的詳細資料和相關內容而定。</span><span class="sxs-lookup"><span data-stu-id="386b3-p101">Under the General Data Protection Regulation (GDPR), data controllers are required to prepare a Data Protection Impact Assessment (DPIA) for processing operations that are “likely to result in a high risk to the rights and freedoms of natural persons.” There is nothing inherent in Microsoft Azure itself that would necessarily require the creation of a DPIA by a data controller using it. Rather, whether a DPIA is required will be dependent on the details and context of *how* the data controller deploys, configures, and uses Microsoft Azure.</span></span>

<span data-ttu-id="386b3-107">本文件旨在提供資料控制者有關 Microsoft Azure 的資訊，協助他們判斷是否需要 DPIA，以及若需要，應包含哪些詳細資料。</span><span class="sxs-lookup"><span data-stu-id="386b3-107">The purpose of this document is to provide data controllers with information about Microsoft Azure that will help them to determine whether a DPIA is needed and, if so, what details to include.</span></span>

> [!Note]
> <span data-ttu-id="386b3-108">Microsoft 在本文件中不提供任何法律建議。</span><span class="sxs-lookup"><span data-stu-id="386b3-108">Microsoft is not providing any legal advice in this document.</span></span> <span data-ttu-id="386b3-109">本文件僅供參考。</span><span class="sxs-lookup"><span data-stu-id="386b3-109">This document is being provided for informational purposes only.</span></span> <span data-ttu-id="386b3-110">我們鼓勵客戶與其隱私權主管和法律顧問合作，以確定與使用 Microsoft Azure或其他任何 Microsoft 線上服務相關之任何 DPIA 的必要性和內容。</span><span class="sxs-lookup"><span data-stu-id="386b3-110">Customers are encouraged to work with their privacy officers and legal counsel to determine the necessity and content of any DPIAs related to their use of Microsoft Azure or any other Microsoft online service.</span></span>
