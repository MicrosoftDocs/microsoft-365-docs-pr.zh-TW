# <a name="federal-financial-institutions-examination-council-ffiec"></a>聯邦金融機構檢查委員會 (FFIEC)

## <a name="ffiec-overview"></a>FFIEC 概觀

聯邦金融機構考試委員會 (FFIEC) 是可包含五個銀行業管理者所負責的金融機構在美國境內的美國聯邦政府考試正式 interagency 主體。 FFIEC 檢查程式教育版 Office 會發佈 IT 檢查手冊適用於欄位 examiners 從 FFIEC 成員行政機關。

[FFIEC 稽核 IT 檢查手冊](https://ithandbook.ffiec.gov/it-booklets/audit.aspx)含的指引來評估品質和 IT 稽核的效率這些 examiners 金融機構和 Tsp 程式。 具體而言，其包含提及 SOC 1、 SOC 2 和 SOC 3 證明報告的美國國家標準的 Certified 公開會計 (AICPA) 做為獨立稽核報告的範例。 不過，FFIEC 建議金融機構不是完全依賴這些報告中所包含的資訊，但也使用驗證及監視詳細[FFIEC 外包技術服務 IT 檢查手冊](https://ithandbook.ffiec.gov/it-booklets/outsourcing-technology-services.aspx)中所述的程序。

## <a name="microsoft-and-ffiec"></a>Microsoft 和 FFIEC

Microsoft Azure、 Microsoft Power BI 和 Microsoft Office 365 內建以符合嚴格需求提供雲端服務的財務服務機構。 我們支援的一部分，我們提供相關指導，以協助您遵守 FFIEC 稽核需求資訊技術和時要使用 Azure SOC attestations 追求您 FFIEC 合規性責任的能力。

Accelerate 金融服務部署：[下載 Azure 安全性與合規性 FFIEC 金融服務藍圖](https://servicetrust.microsoft.com/ViewPage/FFIECBlueprint)

若要協助符合其 FFIEC 合規性需求，以 Azure 金融機構用戶端，Microsoft 已開發:

- [雲端安全性診斷工具 * *](https://aka.ms/FFIEC-CSDT) ，協助您更有效率地執行 Azure 服務的風險評定。 工具 （Excel 試算表） 功能 19 資訊安全性網域 （例如網路和系統的安全性和資訊與風險管理） 的追蹤的財務服務法規和其他相關的標準，以及 FFIEC IT 需求檢查手冊。 此工具說明 Azure 遵守技術服務提供者 (Tsp) 適用於每個需求的方式。
- [Azure 安全性與合規性藍圖 FFIEC 規範服務工作負載](https://servicetrust.microsoft.com/ViewPage/FFIECBlueprint)，隨附的診斷工具。 它還提供使用 Azure 的雲端服務和 FFIEC 需求與風險評定指導方針的客戶符合性考量指導方針。

若要進一步協助您遵守 FFIEC 需求，Microsoft 雲端服務會提供[SOC 證明報告](offering-SOC.md)所產生的獨立 CPA 公司。 例如，SOC 1 類型 2 證明根據 AICPA SSAE 18 標準 （請參閱 AT C 區段 105） 取代 SAS 70，且適用於報表上以進行金融報告特定控制項。 SOC 報告包含在指定的監視期間達到相關的控制項的目標上的 Microsoft 控制項效率的稽核人員的意見。 金融機構追求 FFIEC 特有合規義務資產上 Azure、 Power BI 和 Office 365 部署時，可以使用此正式的稽核。

## <a name="frequently-asked-questions"></a>常見問題集

**可以使用 Microsoft SOC 標準的合規性我機構符合 FFIEC 合規義務？**

若要協助您符合這些義務，Microsoft 會提供相關我們的合規性 SOC 標準上面所述的特定資訊。 不過，最後，它會是您自行以判斷與特定的法律和法規機構適用是否符合我們的服務。 FFIEC 也建議的 「 稽核報告或評論的使用者不應該依賴察覺若要確認 TSP 的內部控制環境報告中所包含的資訊。 他們應該使用其他驗證及監視程序所述更完整 FFIEC IT 檢查手冊[外包技術手冊](https://ithandbook.ffiec.gov/it-booklets/outsourcing-technology-services.aspx)。 」
