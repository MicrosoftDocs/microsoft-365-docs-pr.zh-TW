# <a name="compliance-offering-european-standards-en-301-549-reports"></a>合規性方案：歐洲標準 EN 301 549 報告

## <a name="about-en-301-549"></a>關於 EN 301 549

適用於公開訊息採購的協助工具需求和通訊技術 (ICT) 產品與服務闡明於 EN 301 549 中，這是一組針對 ICT 產品和服務的標準，其中包括網站、軟體和數位裝置。 EN 301 549 由歐洲電信標準協會 (ETSI) 於 2014 年發佈，以回應歐洲聯盟委員會的要求，並於 2018 年更新版本 2.1.2。 它適用於政府和公家機關採購使用。 EN 301 549 在條款 9.2 中針對 web 協助工具納入 WCAG 2.1 標準。

由於 Microsoft 是歐洲主要的軟體和雲端服務提供者，因此致力於遵守所有相關[政府標準和合規性控制](https://go.microsoft.com/fwlink/p/?linkid=2051708)。 透過遵守這些大範圍的協助工具標準，Microsoft 確保政府內部和外部的所有客戶可以使用 Microsoft 服務和產品。

## <a name="microsoft-and-en-301-549"></a>Microsoft 和 EN 301 549

Microsoft 遵守 EN 301 549 標準表明其承諾對所有客戶提供協助工具。 EN 301 549 是歐洲專屬的協助工具標準，可補充條款 508 (美國) 和 WCAG 2.1 (國際)。
