# <a name="configure-irm-to-use-azure-rights-management"></a>設定 IRM 以使用 Azure Rights Management

使用 Office 365 郵件加密 (OME) 的新功能時，您不再需要個別設定 IRM。 Microsoft 不建議使用舊版 OME 和 IRM 搭配 Azure Rights Management 設定新的部署。 如需有關新 OME 功能的詳細資訊，請參閱[Office 365 郵件加密常見問題](./ome-faq.yml)。 如果您已準備好開始使用組織內的新 OME 功能，請參閱[設定以 Azure 資訊保護為基礎的新 Office 365 郵件加密功能](./set-up-new-message-encryption-capabilities.md)。
