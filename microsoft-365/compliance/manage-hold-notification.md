# <a name="manage-hold-notifications"></a>管理保留通知

已初始化之後您法律保留通知工作流程，您可以在進階電子文件中使用通訊工作流程來追蹤通訊的狀態。 [通訊] 索引標籤包含清單中的進階電子文件探索案例內的所有通知。 您可以檢視詳細資料，例如 custodians 已指派或已認可的通知數目。

## <a name="monitor-acknowledgments"></a>監視 acknowledgments

從 [**通訊**] 索引標籤中選取通訊之後，您可以檢視已認可保留通知的 custodians 的清單。 

1. 在合規性中心] 中，移至**eDiscovery > 進階電子文件**。

2. 選取 [大小寫，然後按一下 [**通訊**] 索引標籤。

3. 選取要顯示 [ **Custodian 通訊**彈出式視窗] 頁面上的通訊。

Custodians 相關聯的所選的通訊清單會顯示在 [通訊] 彈出式頁面。 深入了解有關多少認可收到並有多少未執行，也會顯示此頁面。 [] 頁面上也會顯示哪些 custodians 已傳送的通知，他們會收到保留通知。

## <a name="re-send-a-hold-notice"></a>重新傳送保留通知

有時候，custodians 遺失日常工作中的電子郵件追蹤的記錄。 或 custodian 可能長時間執行訴訟暫止的情況，請連絡您或其他人，並要求您重新傳送通知。 當您管理合法持有通知的通訊工作流程，您可能需要重新傳送的通知，以將它帶回 」 上的使用者信箱 」。

若要重新將保留通知傳送給 custodian:

1. 在進階電子文件中，選取 [大小寫，然後按一下 [**通訊**] 索引標籤。

2. 選取要顯示 [ **Custodian 通訊**彈出式視窗] 頁面上的通訊。

3. 按一下 [**更多 > 重新傳送保留通知**]。

4. 在**重新傳送保留通知**彈出式頁面上，選取 [custodians 您要重新傳送通知及輸入選用的原因。

5. 按一下 [傳送通知給選取 custodians**重新傳送**。

如果 custodian 尚未認可保留通知、 提醒及呈報工作流程會重新啟動。 如果 custodian 已認可的保留明，custodian 會收到一份原始保留通知。

> [!NOTE]
> 您只可以重新合法持有通知傳送至 custodians 指派給通訊。

4. 按一下 **[儲存]**。
