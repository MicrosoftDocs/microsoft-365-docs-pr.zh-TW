# <a name="process-data-for-an-investigation"></a>處理調查的資料

處理是檔案識別、內嵌檔和附件擴充、文字提取、OCR (光學字元) 識別的處理常式，以及該內容的索引。  

新增人員加入 [ **興趣** ] 索引標籤時，會處理 Microsoft 365 的所有已部分索引項目目，讓它們完全可供搜尋。  同樣地，當資料從 Microsoft 365 或非 Microsoft 365 資料來源新增至 [ **證據** ] 索引標籤時，也會處理內容。

[資料調查 () 預覽] 中的 [ **處理** ] 索引標籤，可提供所有這些案例的 *高級索引* 狀態的透明度。

如需詳細資訊，請參閱下列文章。

- [調查的高級資料索引](index-data-people-of-interest.md)

- [處理資料時發生補救錯誤](error-remediation.md)
