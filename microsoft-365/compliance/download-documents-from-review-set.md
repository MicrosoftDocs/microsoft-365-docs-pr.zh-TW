# <a name="download-selected-documents-from-a-review-set"></a>從審閱集下載選取的檔

下載提供一種簡單的方式，可從原生格式的審閱集下載內容。 「高級 eDiscovery」中的下載工具會利用瀏覽器的資料傳輸功能。 下載準備好時，會出現瀏覽器提示。 使用此方法下載的檔案會壓縮為容器檔案中，且會包含專案層級檔案。 這表示如果您選取下載附件，您會收到包含附件的電子郵件。 同樣地，如果您匯出的是嵌入在 Word 檔中的 Excel 試算表，下載中也會包含嵌入 Excel 試算表的 Word 檔。 當您下載專案時，會保留最後一個修改的資料屬性，而且可以視為檔案屬性。

若要在高級 eDiscovery 案例中從審閱集下載內容，請先選取您要下載的檔案，然後選取 [**動作專案**] [  >  **下載**]。
