# <a name="united-kingdom-cyber-essentials-plus"></a>英國網路基本版加上

## <a name="uk-cyber-essentials-plus-overview"></a>英國網路基本版加上概觀

網路基本版是英國政府供電配置專門設計來協助組織評估，並降低從常見的網路安全性威脅，讓 IT 系統的風險。 網路 Essentials 配置會識別組織備妥其 IT 系統內的安全性控制的網路安全性標準。 網路 Essentials 配置是所有的英國政府供應商，處理個人資料的需求。 網路 Essentials 徽章可協助組織示範的能力：

- 識別可協助組織更妥善地防範常見的網路威脅的潛在風險。
- 示範組織已採用保護客戶資料的適當安全性控制。
- 成為相容網路安全性的基本需求的英國政府期望與合格叫的英國 government 合約。

網路 Essentials 配置專為英國政府供應商，以識別潛在的弱點 IT 系統和軟體，以利用客戶資料。 方法已定義兩個不同的層級的憑證：

- 網路基本版是第一層，同時也包括自我評估對組織而言，若要檢查其 IT 基礎結構的最重要 IT 安全性控制。 回應分別是由外部憑證主體檢閱。
- 網路基本版加上提供的相同的控制項涵蓋範圍為網路基本版和也包含其他保證等同系統測試透過 [已授權的協力廠商憑證的主體已實作控制項。

## <a name="microsoft-and-uk-cyber-essentials-plus"></a>Microsoft 和英國網路基本版加上

Microsoft Azure 已得到網路基本版加上徽章，且符合[網路基本版結構描述](https://go.microsoft.com/fwlink/p/?linkid=2099398)中所述的需求。 Azure 生產系統經常測試和稽核，以提供全球業界領導規範公事包的證據。

[Azure 網路基本版加上的憑證](https://aka.ms/AzureCyberEPlusCert)，由它套用至 Azure 我們全域作業，是可供下載。
