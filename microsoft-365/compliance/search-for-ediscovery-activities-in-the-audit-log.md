# <a name="search-for-ediscovery-activities-in-the-office-365-audit-log"></a>在 Office 365 审核日志中搜索电子数据展示活动

在安全&合规性中心或通过运行相应的 Windows PowerShell cmdlet 执行的内容搜索和电子数据展示相关活动将记录在 Office 365 审核日志中。 当管理员或合规性管理员（或任何分配了电子数据展示权限的用户）在安全&合规性中心执行以下内容搜索和电子数据展示相关任务时，将记录事件：
  
- 创建和管理电子数据展示案例
    
- 创建、启动和编辑内容搜索
    
- 执行内容搜索操作，如预览、导出和删除搜索结果
    
- 为内容搜索配置权限筛选
    
- 管理电子数据展示管理员角色
    
> [!IMPORTANT]
> 本文中描述的活动只是使用安全&合规性中心执行电子数据展示任务的结果。 不包括使用 Exchange 在线中的就地电子数据展示工具或 SharePoint Online 中电子数据展示中心执行的电子数据展示任务。 
  
有关搜索 Office 365 审核日志、所需的权限以及导出搜索结果的详细信息，请参阅[在"安全&合规性中心"中搜索审核日志。](search-the-audit-log-in-security-and-compliance.md)
  
## <a name="how-to-search-for-and-view-ediscovery-activities"></a>如何搜索和查看电子数据展示活动

目前，您必须执行一些具体操作才能在 Office 365 审核日志中查看电子数据展示活动。 方法如下。
  
1. 移至 [https://protection.office.com](https://protection.office.com)。
    
2. 使用公司或學校帳戶登入 Office 365。
    
3. 在左侧窗格中，单击"**搜索"，****然后单击"审核日志搜索"。**
    
4. 在"**活动"** 下拉列表中，**在"电子数据展示"活动**下，单击一个或多个要搜索的活动。 或者，您可以**单击"电子数据展示"活动**来搜索所有与电子数据展示相关的活动。 
    
    > [!NOTE]
    > "活动"下拉列表还包括一组名为**eDiscovery cmdlet 的活动，** 这些活动将从 cmdlet 审核日志中返回记录。 
  
5.  选择日期和时间范围以显示该时间段内发生的电子数据展示事件。 
    
6. 在"**用户"** 框中，选择要显示搜索结果的一个或多个用户。 将此框留空以返回所有用户的条目。 
    
7. **单击"搜索"** 以使用搜索条件运行搜索。 
    
8. 显示搜索结果后，可以**单击"筛选结果"** 以筛选或排序生成的活动记录。 遗憾的是，您无法使用筛选来显式排除某些活动。 
    
9. 要查看有关活动的详细信息，请单击搜索结果列表中的活动记录。 
    
    将显示"**详细信息**飞出"页，其中包含事件记录中的详细属性。 要显示其他详细信息，请单击"**更多信息"。** 有关这些属性的说明，请参阅[电子数据展示活动的详细信息](#detailed-properties-for-ediscovery-activities)部分。

## <a name="detailed-properties-for-ediscovery-activities"></a>电子数据展示活动的详细属性

下表**描述了在搜索结果**中列出的电子数据展示**活动的详细信息"** 页上包含的属性。 导出审核日志搜索结果时，这些属性也包含在 CSV 文件中。 请注意，电子数据展示活动的审核日志记录不会包括下面列出的所有详细属性。 
  
> [!TIP]
> 导出搜索结果时，CSV 文件包含**名为"详细信息"** 的列，其中包含多值属性中下表中描述的详细属性。 您可以使用 Excel 中的"电源查询"功能将此列拆分为多个列，以便每个属性都有自己的列。 这将允许您对这些属性的一个或多个属性进行排序和筛选。 有关详细信息，请参阅[搜索审核日志](search-the-audit-log-in-security-and-compliance.md#step-4-export-the-search-results-to-a-file)中的"将搜索结果导出到文件"部分。 
  
|**屬性**|**描述**|
|:-----|:-----|
|案例  <br/> |已创建、更改或删除电子数据展示案例的标识 （GUID）。  <br/> |
|客户端应用程序  <br/> |eDiscovery cmdlet 活动具有此属性的**EMC**值。 这表示活动是通过使用安全&合规性中心 GUI 或在 PowerShell 中运行 cmdlet 执行的。  <br/> |
|客户端 IP  <br/> |记录活动时使用的设备的 IP 地址。 IP 地址以 IPv4 或 IPv6 地址格式显示。  <br/> |
|客户端申请ID  <br/> | 对于电子数据展示活动，此属性通常为空。  <br/> |
|CmdletVersion  <br/> |组织中运行的安全&合规性中心版本的内部版本。  <br/> |
|CreationTime  <br/> |电子数据展示活动完成时的日期和时间（UTC）。  <br/> |
|有效组织  <br/> |Office 365 组织的名称。  <br/> |
|交换位置  <br/> |包含在内容搜索中或置于电子数据展示案例中的保留的 Exchange 联机邮箱。  <br/> |
|排除  <br/> |从内容搜索或电子数据展示案例中保留的邮箱或网站位置。  <br/> |
|扩展属性  <br/> |来自内容搜索、内容搜索操作或电子数据展示案例中保留的其他属性，例如对象 GUID 和执行活动时使用的相应 cmdlet 和 cmdlet 参数。  <br/> |
|Id  <br/> |报表条目的 ID。 ID 唯一标识审核日志条目。  <br/> |
|非PII参数  <br/> |与操作属性中标识的 cmdlet 一起使用的参数（不带任何值）的列表。 此属性中列出的参数与参数属性中列出的参数相同。  <br/> |
|对象Id  <br/> |由"操作"属性中列出的活动创建、更改或删除的对象的 GUID 或名称（例如，内容搜索或电子数据展示案例）。 此对象还在审核日志搜索结果中的"项目"列中标识。  <br/> |
|ObjectType  <br/> |用户创建、删除或修改的 eDiscovery 对象的类型;例如内容搜索操作（预览、导出或清除）、电子数据展示案例或内容搜索。  <br/> |
|作業  <br/> |对应于所执行电子数据展示活动的操作的名称。  <br/> |
|组织ID  <br/> |Office 365 组织的 GUID。  <br/> |
|參數  <br/> |与相应 cmdlet 一起使用的参数的名称和值。  <br/> |
|公用文件夹位置  <br/> |Exchange Online 中包含在内容搜索中或置于电子数据展示案例中的保留状态的公用文件夹位置。  <br/> |
|查詢  <br/> |与活动关联的搜索查询，如内容搜索或基于查询的保留。  <br/> |
|记录类型  <br/> |记录指示的操作类型。 值**18**表示与[电子数据展示 cmdlet 活动](#ediscovery-cmdlet-activities)部分中列出的活动相关的事件。 值为**24**表示与"[如何搜索和查看电子数据展示活动"](#how-to-search-for-and-view-ediscovery-activities)部分中列出的活动相关的事件。  <br/> |
|结果状态  <br/> |指示操作（在操作属性中指定）是否成功。  <br/> |
|安全合规性中心事件类型  <br/> |指示活动是安全&合规性中心事件。 所有电子数据展示活动对于此属性的值将为**0。**  <br/> |
|共享点位置  <br/> |包含在内容搜索中或置于电子数据展示案例中的保留的 SharePoint 联机网站。  <br/> |
|StartTime  <br/> |启动电子数据展示活动时协调通用时间 （UTC） 中的日期和时间。  <br/> |
|UserId  <br/> |执行导致记录记录的活动（在"操作"属性中指定）的用户。 请注意，系统帐户（如 NT AUTHORITY_SYSTEM）执行电子数据展示活动的记录也包含在审核日志中。  <br/> |
|用户密钥  <br/> |UserId 属性中标识的用户的替代 ID。 对于电子数据展示活动，此属性的值通常与 UserId 属性相同。  <br/> |
|用户服务计划  <br/> |组织使用的 Office 365 订阅。 对于电子数据展示活动，此属性通常为空。  <br/> |
|用户类型  <br/> |执行操作的用户类型。 以下值指示用户类型。  <br/> 0 常规用户。 2 Office 365 组织的管理员。 3 微软数据中心管理员或数据中心系统帐户。 4 系统帐户。 5 应用程序。 6 服务主体。 |
|版本  <br/> |指示已记录的活动的版本号（由操作属性标识）。  <br/> |
|工作量  <br/> |发生活动的 Office 365 服务。 对于电子数据展示活动，值**为"安全合规中心"。**  <br/> |
