# <a name="themes-in-advanced-ediscovery"></a>高級 eDiscovery 中的主題

個人如何撰寫檔？ 他們通常會從一或多個想要在檔中傳遞的觀點開始，並使用與創意相符的文字加以撰寫。 構思的流行越多，與該觀點相關的文字越常見。 這會告訴人們也會如何使用檔。 從閱讀檔中瞭解的重要一點是檔嘗試傳遞的想法、哪些創意會出現在何處，以及構思之間的關係。

這可以擴充為使用者想要使用一組檔的方式。 他們想要查看哪些構思會出現在集合中，以及哪些檔正在談論這些觀點。 此外，如果他們找到感興趣的特定檔，他們想要看到討論類似創意的檔。

「高級 eDiscovery」中的主題功能會嘗試透過分析審閱集中所討論的 *主題* ，並為審閱集中的檔指派主題，以模仿有關檔的原因。 在 [Advanced eDiscovery] 中，主題會進一步執行一個步驟，並識別每個檔中的 *主要主題* 。 主要主題是顯示在檔中的最常用主題。

## <a name="how-does-themes-work"></a>主題如何運作？

Themes 功能會分析具有審閱集中之文字的檔，以分析顯示在審閱集中所有檔的常見主題。 「高級 eDiscovery」會將這些主題指派給他們顯示的檔。 它也會以主題所代表檔中所用的字來標出每個主題。 因為檔可包含各種類型的主體，所以高級 eDiscovery 常常會為檔指派多個主題。 檔中顯示最醒目的主題是指定為其主要主題。
