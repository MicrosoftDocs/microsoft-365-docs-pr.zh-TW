# <a name="export-case-data-in-advanced-ediscovery"></a>在 Advanced eDiscovery 中匯出案例資料

有三種方式可以從審查集匯出資料：

**下載：** 使用瀏覽器) 一組小型的原生檔案，下載 (。 這是匯出小型資料集的最快方法。 此方法會保留原生檔案名。 如需詳細資訊，請參閱 [從審閱集下載檔案](download-documents-from-review-set.md)。

**匯出：** 自訂匯出的資料。 這包括匯出檔案的中繼資料、原生檔案、文字檔，以及已儲存到 PDF 檔案的修訂文件。 當匯出的資料準備好之後，您可以將資料從 [ **匯出** ] 索引標籤下載到本機電腦。 如需詳細資訊，請參閱 [從審閱集匯出檔](export-documents-from-review-set.md)

**新增至其他審閱集：** 將資料從一個複查集複製到不同的審閱集。 如需詳細資訊，請參閱 [將資料從一個複查集新增至另一個複查集](add-data-to-review-set-from-another-review-set.md)。

> [!NOTE]
> 在 Microsoft 365 中，會將資料散列，而這些雜湊會在輸出檔案中提供，以供驗證之用。 這會透過審計記錄檔和報告功能（例如收集統計資料、負載設定報告及匯出報告）補充， (包括匯出載入檔案) 。
