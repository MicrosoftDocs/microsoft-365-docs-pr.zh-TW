# <a name="gramm-leach-bliley-act-glba"></a>Gramm-Leach-Bliley Act (GLBA)

## <a name="glba-overview"></a>GLBA 概觀

Gramm-leach-bliley 金融服務業現代化法案 (GLBA) 是美國法律新式金融服務產業，讓商業和投資銀行、 證券公司及保險公司合併彙算，並解決保護使用者隱私權相關的問題。 它需要聯邦貿易委員會 (FTC) 和其他金融服務管理者可以實作法規以解決這類的隱私權佈建為財務隱私權規則，以及保護規則。 以保護機密消費者資料 GLBA 需求適用於至取用者，例如手續費、 投資建議與保險提供財務產品和服務的金融機構。 FTC 會負責強制執行規範。
