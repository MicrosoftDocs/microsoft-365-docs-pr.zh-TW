# <a name="close-or-delete-an-investigation"></a>關閉或刪除調查

## <a name="close-an-investigation"></a>關閉調查

 在您完成調查並採取適當的補救動作後，您可能會想要保留您收集的證據，但請關閉此案例，以便日後能夠輕易參考。 一旦您關閉調查，它仍然會列在安全性 & 合規性中心的「資料調查」頁面上。 已關閉案例的詳細資料、保留、搜尋及成員都會保留。 日後，您可以重新開啟封閉式調查。

若要結束調查：

1. 在 [**設定**] 索引標籤上，選取 [**調查資訊**]。

2. 選取 [**關閉調查**]。 


## <a name="delete-an-investigation"></a>刪除調查

有時候，您可能想要銷毀證據，因為它包含不想要保留的機密或機密資料。 若要避免進一步的資料外泄，您應該考慮刪除調查。

若要刪除調查：

1. 在 [**設定**] 索引標籤上，選取 [**調查資訊**]。

2. 選取 [**刪除調查**]。
