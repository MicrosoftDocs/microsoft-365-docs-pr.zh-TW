# <a name="how-to-submit-feedback-on-app-governance"></a>如何提交有關應用程式控管的意見反應 

為能提交有關應用程式控管附加元件對於 Microsoft Cloud App Security 的意見反應：

- 在任何要提交意見反應的入口網站頁面右下角，按一下黑色的意見反應按鈕。

- 如果要提交功能意見反應：
  1. 在此文字方塊中提供星級評等和註解  
  1. 如果您選取 **包含此螢幕擷取畫面**，您目前所在的頁面螢幕擷取畫面將會進行分享。  
  1. 如果您選取 **您可以針對此意見與我聯絡**，您的電子郵件地址將會與 Microsoft 共用。
  1. 選取 **提交** 以傳送您的意見反應。

- 如果您要提交有關惡意應用程式活動的意見反應：

  1. 提供五顆星評等（這是必要欄位）。
  1. 在文字方塊中，輸入您標誌為惡意應用程式的應用程式名稱和應用程式識別碼，以及任何您可以提供的支持詳細資料。
  1. 如果您選取 **包含此螢幕擷取畫面**，您目前所在的頁面螢幕擷取畫面將會進行分享。  
  1. 如果您選取 **您可以針對此意見與我聯絡**，您的電子郵件地址將會與 Microsoft 共用。
  1. 選取 **提交** 以傳送您的意見反應。
