# <a name="compliance-offering-reserve-bank-of-india-rbi-and-insurance-regulatory-and-development-authority-of-india-irdai"></a>合規性供應項目： 保留銀行印度 (RBI) 和保險法規及開發授權單位的印度 (IRDAI)

## <a name="about-rbi-and-irdai"></a>關於 RBI] 和 [IRDAI

[印度的預約銀行](https://www.rbi.org.in/)(RBI)、 印度的中央金融機構，[保險法規和的印度開發授權單位](https://www.irdai.gov.in/Defaulthome.aspx?page=H1)(IRDAI)，以及[電子韓國與資訊技術](https://meity.gov.in/content/information-technology-act)(MeitY) 組成的監督銀行保險的組織中，主要的財務產業管理者其中三項和市場的基礎結構機構。 其指示詞包含外包和風險管理指引的控管自己的敏感資料的隱私權規則與合規性需求。

外包和風險管理指導方針包括：

- [管理風險和程式碼管理辦法外包的財務服務由銀行上的指導方針](https://rbidocs.rbi.org.in/rdocs/notification/PDFs/73713.pdf)(RBI) 地址管制的銀行會公開時外包金融服務，而且可協助您確保外包不會阻礙 RBI 主管角色的風險。 RBI 不需要先前核准的銀行外包金融服務; 搜尋不過，不應該外包核心銀行函數，例如內部稽核和符合性功能。
- [在電子化的資訊安全性的指導方針銀行技術風險管理，以及網路 Frauds](https://rbidocs.rbi.org.in/rdocs/content/PDFs/GBS300411F.pdf) (RBI)。 金融機構必須報告其中的規模和活動的性質重大或需要大量的資料與服務提供者印度外部共用的外包排列方式。 本指南適用於特別是如果儲存或處理印度之外操作資料。
- [外包的印度負責確認保戶規定的活動](https://www.irdai.gov.in/ADMINCMS/cms/frmGeneral_Layout.aspx?page=PageNo3149&flag=1)(IRDAI)。 每年保險組織所需回報外包至 IRDAI 的核心活動某些支援功能的財務一年中的關閉 45 天內。 （在 Microsoft[檢查清單](https://servicetrust.microsoft.com/Documents/TrustDocuments?command=Download&downloadType=Document&downloadId=26f4af15-2771-4cd4-a7c7-9328149f9453&docTab=6d000410-c9e9-11e7-9a91-892aae8839ad_Compliance_Guides)中的頁面 7 說明構成 「 支援的核心活動的函數 」。

使用雲端服務的財務公司也必須遵守的隱私權規則，包括[資訊技術 （合理的安全性作法及程序，並機密個人資料或資訊） 的規則，2011年](https://meity.gov.in/sites/upload_files/dit/files/GSR313E_10511\(1\).pdf)(MeitY)。 開發以加強印度的資料保護法律，這些規則控管的保護和機密個人資料的處理。

## <a name="microsoft-rbi-and-irdai"></a>Microsoft、 RBI 及 IRDAI

若要協助指南金融機構印度考慮到雲端外包商務功能，Microsoft 已發佈印度的金融機構合規性檢查清單。 檢閱並完成[檢查清單](https://servicetrust.microsoft.com/Documents/TrustDocuments?command=Download&downloadType=Document&downloadId=26f4af15-2771-4cd4-a7c7-9328149f9453&docTab=6d000410-c9e9-11e7-9a91-892aae8839ad_Compliance_Guides)，財務組織可以採用 Microsoft business 雲端服務有信心它們遵守適用的法規需求。

當印度金融機構外包至雲端的商務活動時，他們必須遵循印度預約銀行指導方針管理風險，以及解決的問題之資訊技術的使用中。 他們也必須遵守的資料安全性和隱私權需求資訊技術 (MeitY) 與韓國的電子產品所建立。 此外，保險組織必須遵循保險法規及開發授權單位的印度 (IRDAI) 所發佈的外包指導方針。

Microsoft 檢查清單可協助在印度財務來進行到期盡職調查 「 評估 」 的 Microsoft business 雲端服務的公司，並包括：

- 針對 context 法規橫向概觀。
- 檢查清單可設定約定需要解決的問題，並將對應對這些法規義務的 Microsoft Azure、 Microsoft Dynamics 365 和 Microsoft Office 365 服務。 檢查清單可以用做為工具，來測量的法規架構的規範和用於記載合規性，提供內部結構協助進行的 Microsoft business cloud services 自己風險評定的客戶。
