# <a name="export-case-data-in-advanced-ediscovery-classic"></a>在高級 eDiscovery （古典）中匯出案例資料

> [!NOTE]
> 進階電子文件探索需要具有進階合規性附加元件的 Office 365 E3，或適用於您組織的 E5 訂閱。如果您沒有該方案，且想要嘗試進階電子文件探索，您可以[註冊 Office 365 企業版 E5 試用版](https://go.microsoft.com/fwlink/p/?LinkID=698279)。 
  
在 [高級 eDiscovery] 中的匯出程式可讓您匯出「高級 eDiscovery」內容和外部評論結果。 
  
## <a name="guidelines-for-exporting-data"></a>匯出資料的指導方針

- 例如，您可以檢查從 Advanced eDiscovery 產生的匯出資料，例如，將相關的元素散佈至檔檢查小組或匯出至外部檔審閱工具。
    
- 您可以在任何時刻及之後的高級 eDiscovery 處理過程中，產生及操縱匯出輸出。 在標準案例中，相關性訓練和批次計算成功完成之後，便會達到重要的結果。 例如，您可以在繼續之前，產生1000最相關檔案的範例，並評估高級 eDiscovery 效能。
    
- 將大量的資料匯出至資料庫時，可能會對必要的資料庫大小和高級 eDiscovery 資料庫所需的設定產生重大影響。
