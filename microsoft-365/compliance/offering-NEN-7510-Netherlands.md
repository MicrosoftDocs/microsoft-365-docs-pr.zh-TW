## <a name="frequently-asked-questions"></a><span data-ttu-id="f90dd-120">常見問題集</span><span class="sxs-lookup"><span data-stu-id="f90dd-120">Frequently asked questions</span></span>

<span data-ttu-id="f90dd-134">Microsoft Online Services 提供許多控制措施，可協助荷蘭醫療保健組織符合 NEN 7510 合規性需求。</span><span class="sxs-lookup"><span data-stu-id="f90dd-134">Microsoft Online Services provides many controls that help organizations within Dutch Healthcare with their NEN 7510 compliance needs.</span></span> <span data-ttu-id="f90dd-135">不過，組織必須以其自身的實作選擇、其他技術控制措施及系統管理程序來補充廠商的保證。</span><span class="sxs-lookup"><span data-stu-id="f90dd-135">However, an organization needs to complement those vendor assurances with their own implementation choices, additional technology controls, and administrative processes.</span></span> <span data-ttu-id="f90dd-136">此報告顯示，在適用控制措施的完整清單中已經有超過 94% 的直接涵蓋範圍。</span><span class="sxs-lookup"><span data-stu-id="f90dd-136">The report shows already over 94% direct coverage of the full list of applicable controls.</span></span> <span data-ttu-id="f90dd-137">對於其餘的控制措施，Microsoft 會在報告中，針對如何證明這些控制措施的合規性，提供指導方針。</span><span class="sxs-lookup"><span data-stu-id="f90dd-137">For the remaining controls, Microsoft provides guidance in the report on how compliance with those controls can be demonstrated.</span></span>
