# <a name="work-with-processing-errors-in-advanced-ediscovery"></a>使用 Advanced eDiscovery 中的處理錯誤

*處理* 程式是檔案識別、內嵌檔和附件、文字提取和光學字元辨識的處理常式， (OCR) ) 影像檔及該內容的後續編制索引。  

當您在 [**來源**] 索引標籤上，將保管人和非保管人資料來源新增至案例時，會處理 Microsoft 365 中所有已部分索引的專案，讓它們完全可供搜尋。 同樣地，當內容同時從 Microsoft 365 和非 Microsoft 365 資料來源新增至審閱集時，也會處理此內容。

進階電子文件探索中的 **[處理]** 索引標籤，可深入解析不同處理案例的進階索引編製狀態。

如需詳細資訊，請參閱下列文章：

- [進階的監管人資料索引](indexing-custodian-data.md)

- [處理資料時發生補救錯誤](error-remediation-when-processing-data-in-advanced-ediscovery.md)

- [單一項目錯誤補救](single-item-error-remediation.md)
