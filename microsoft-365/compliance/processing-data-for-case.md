# <a name="work-with-processing-errors-in-advanced-ediscovery"></a>使用進階電子文件中的錯誤處理

*處理*是檔案識別碼，內嵌文件和附件、 文字擷取及 OCR （光學字元辨識） 的圖像檔案的擴充和後續編製索引內容的程序。  

當您新增 custodian **Custodians** ] 索引標籤上的案例時，使其完整搜尋處理 Office 365 的所有已局部編製索引項目。 同樣地，當內容會新增至 Office 365 和非 Office 365 的資料來源設定檢閱，也會處理此內容。

[進階電子文件的 [**處理**] 索引標籤提供深入的不同處理案例進階編製索引的狀態。
