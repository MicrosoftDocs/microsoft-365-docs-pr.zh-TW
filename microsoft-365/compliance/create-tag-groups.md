# <a name="create-tag-groups"></a>建立標記群組

在您或其他人員可以在審閱集中對檔進行標記之前，必須先建立標記。 您可以建立包含子標記的標籤群組來執行此動作。 在審閱集中審閱檔時，標記會顯示在 [標記] 面板中。

若要建立標記群組：

1.  在 [審閱集] 中，按一下 [ **管理審閱集**]。

    ![按一下 [管理審閱集]](../media/ED-managews.png)

2.  在 [ **標記** ] 磚中，按一下 [ **管理標記**]。

    ![按一下 [標記] 磚中的 [管理標記]](../media/ED-managetags.png)

在標記管理內，可以建立標記以符合您的案例需求。 您可以從建立標記群組開始。

1.  按一下 [ **新增區段**]。

    ![新增標記群組](../media/ED-addtagsection.png)

    當您儲存時，預覽畫面會隨之更新，這樣您就可以預覽標記面板，而不必關閉標記管理畫面，並回到工作集。

2. 輸入標題和選用的描述。 

3. 若要在該群組內建立標記，請按一下新標記群組上的省略號，以在該新區段中建立標記。
    
    ![在標記群組中建立標記](../media/ED-createtag.png)

   - **選項標記** 會強制使用者從一個標記群組中選取一個標籤。
   
   - **核取方塊標記** 可讓使用者選取標記的任何組合。 |

## <a name="nested-tags"></a>嵌套標記

若要嵌套標記，請按一下標記旁邊的省略號，然後選取要新增的新標記。

![嵌套標記](../media/ED-tagnesting.png)
