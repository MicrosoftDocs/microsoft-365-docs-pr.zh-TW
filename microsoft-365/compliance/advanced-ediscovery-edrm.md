- <span data-ttu-id="f2cac-109">**識別。**</span><span class="sxs-lookup"><span data-stu-id="f2cac-109">**Identification.**</span></span> <span data-ttu-id="f2cac-110">在您找出調查中潛在感興趣的人員之後，您可以將其新增為保管人 (也稱為「 *資料保管人*」，因為它們可能具有與調查) 相關的資訊，以供高級 eDiscovery 案例使用。</span><span class="sxs-lookup"><span data-stu-id="f2cac-110">After you identify potential persons of interest in an investigation, you can add them as custodians (also called *data custodians*, because they may possess information that's relevant to the investigation) to an Advanced eDiscovery case.</span></span> <span data-ttu-id="f2cac-111">將使用者新增為保管人後，就很容易保留、收集和審查保管人檔。</span><span class="sxs-lookup"><span data-stu-id="f2cac-111">After users are added as custodians, it's easy to preserve, collect, and review custodian documents.</span></span>

- <span data-ttu-id="f2cac-112">**保存。**</span><span class="sxs-lookup"><span data-stu-id="f2cac-112">**Preservation.**</span></span> <span data-ttu-id="f2cac-113">為了保留及保護與調查相關的資料，「高級 eDiscovery」可讓您在案例中對保管人相關聯的資料來源進行合法保留。</span><span class="sxs-lookup"><span data-stu-id="f2cac-113">To preserve and protect data that's relevant to an investigation, Advanced eDiscovery lets you place a legal hold on the data sources associated with the custodians in a case.</span></span> <span data-ttu-id="f2cac-114">您也可以將非 custodial 資料置於保留狀態。</span><span class="sxs-lookup"><span data-stu-id="f2cac-114">You can also place non-custodial data on hold.</span></span> <span data-ttu-id="f2cac-115">Advanced eDiscovery 也有內建的通訊工作流程，因此您可以將法律封存通知傳送給保管人，並追蹤其確認。</span><span class="sxs-lookup"><span data-stu-id="f2cac-115">Advanced eDiscovery also has a built-in communications workflow so you can send legal hold notifications to custodians and track their acknowledgments.</span></span>

- <span data-ttu-id="f2cac-116">**收集。**</span><span class="sxs-lookup"><span data-stu-id="f2cac-116">**Collection.**</span></span> <span data-ttu-id="f2cac-117">在您識別 (和保留) 與調查相關的資料來源之後，您可以在高級 eDiscovery 搜尋中使用內建的搜尋工具，並從 custodial 資料來源中收集即時資料， (和非 custodial 的資料來源（如果適用) 可能與案例相關）。</span><span class="sxs-lookup"><span data-stu-id="f2cac-117">After you identified (and preserved) the data sources relevant to the investigation, you can use the built-in search tool in Advanced eDiscovery search for and collect live data from the custodial data sources (and non-custodial data sources, if applicable) that may be relevant to the case.</span></span>

- <span data-ttu-id="f2cac-118">**處理。**</span><span class="sxs-lookup"><span data-stu-id="f2cac-118">**Processing.**</span></span> <span data-ttu-id="f2cac-119">在您收集所有與案例相關的資料之後，下一步是處理它以進一步複查和分析。</span><span class="sxs-lookup"><span data-stu-id="f2cac-119">After you've collected all data relevant to the case, the next step is process it for further review and analysis.</span></span> <span data-ttu-id="f2cac-120">在 [Advanced eDiscovery] 中，您在集合階段中所識別的就地資料會複製到 Azure 存放位置， (稱為「 *複查集* 」) ，可為您提供案例資料的靜態視圖。</span><span class="sxs-lookup"><span data-stu-id="f2cac-120">In Advanced eDiscovery, the in-place data that you identified in the collection phase is copied to an Azure Storage location (called a *review set*), which provides you with a static view of the case data.</span></span> 

- <span data-ttu-id="f2cac-121">**檢討。**</span><span class="sxs-lookup"><span data-stu-id="f2cac-121">**Review.**</span></span> <span data-ttu-id="f2cac-122">將資料新增至審閱集後，您就可以查看特定的檔，並執行額外的查詢，將資料縮小至與案例最為相關的資料。</span><span class="sxs-lookup"><span data-stu-id="f2cac-122">After data has been added to a review set, you can view specific documents and run additional queries to reduce the data to what is most relevant to the case.</span></span> <span data-ttu-id="f2cac-123">此外，還可以批註及標記特定檔。</span><span class="sxs-lookup"><span data-stu-id="f2cac-123">Also, can annotate and tag specific documents.</span></span>

- <span data-ttu-id="f2cac-124">**分析。**</span><span class="sxs-lookup"><span data-stu-id="f2cac-124">**Analysis.**</span></span> <span data-ttu-id="f2cac-125">「高級 eDiscovery」提供整合式分析工具，可協助您進一步剔除您決定不與調查相關的複查集的資料。</span><span class="sxs-lookup"><span data-stu-id="f2cac-125">Advanced eDiscovery provides integrated analytics tool that helps you further cull data from the review set that you determine isn't relevant to the investigation.</span></span> <span data-ttu-id="f2cac-126">除了減少相關資料的數量之外，高級 eDiscovery 也可協助您組織內容，讓審查程式更輕鬆且更有效率，以節省法律考評成本。</span><span class="sxs-lookup"><span data-stu-id="f2cac-126">In addition to reducing the volume of relevant data, Advance eDiscovery also helps you save legal review costs by letting you organize content to make the review process easier and more efficient.</span></span>
