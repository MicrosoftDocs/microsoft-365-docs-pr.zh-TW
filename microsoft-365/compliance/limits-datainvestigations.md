# <a name="data-investigations-preview-limits"></a>資料調查（預覽）限制

本文說明資料調查中的限制（預覽）。

## <a name="investigation-limits"></a>調查限制

下表列出資料調查（預覽）中的調查限制。 
    
  |**限制的描述**|**限制**|
  |:-----|:-----|
  |調查數目上限。  <br/> |50  <br/> |
  |可以新增至調查的檔總數（針對調查中的所有證據集）。  <br/> |1 百萬  <br/> |
  |每次負載的檔案大小總計。  <br/> |100 GB  <br/> |
  |單一檔案的大小上限。   <br/> |100 MB <sup>1</sup> <br/> |
  |從單一檔案解壓縮的字元數上限。  <br/> |10000000 <sup>1</sup> <br/> |
  |單一檔案中的內嵌專案深度。  <br/> |25 <sup>1</sup> <br/> |
|||
> [!NOTE]
><sup>1</sup>超過單一檔限制的任何專案會顯示為處理錯誤。
