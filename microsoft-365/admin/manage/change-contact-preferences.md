# <a name="change-your-contact-preferences-for-communications-from-microsoft"></a>變更您對來自 Microsoft 之通訊的連絡人喜好設定

使用 [**連絡人喜好**設定] 頁面更新組織的連絡人資訊，並選擇您想要接收的 Microsoft 365 產品相關通訊。
  
如需如何變更系統管理員行動電話號碼和電子郵件地址的資訊，請閱讀[更新 Microsoft 365 中的系統管理員電話號碼和電子郵件地址](update-phone-number-and-email-address.md)。
  
如果您想了解如何變更公司設定檔資訊 (如組織的名稱與地址、主要公司電話號碼和技術連絡人資訊)，請參閱[變更貴組織地址、技術連絡人電子郵件和其他資訊](change-address-contact-and-more.md)。
  
## <a name="to-update-your-information"></a>更新您的資訊
  
1. 在系統管理中心中，選取頁首右右邊的 [**設定**] 圖示。

2. 選取 [**連絡人喜好**設定]。

3. 選擇您想要收到的產品相關通訊資料類型。

4. 鍵入要接收這些通訊的商務電話、商務行動電話或企業電子郵件。
    您可以使用任何電子郵件地址（包括您的 Microsoft 電子郵件地址）來接收這些通訊。 如果您沒有新增連絡人偏好的電子郵件地址，則會使用您的備選電子郵件地址（**個人資訊**頁面）。

5. 選擇 [儲存]****。
  
針對 Office 365 企業版、Microsoft 365 Business Standard 和 Microsoft 365 Business Basic，當您第一次註冊為 Microsoft 365 的管理員時，您的使用者可能會自動註冊以接收產品相關通訊。
