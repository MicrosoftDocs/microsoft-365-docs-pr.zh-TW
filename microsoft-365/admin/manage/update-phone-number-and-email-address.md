# <a name="update-your-admin-phone-number-and-email-address-in-office-365"></a>更新 Office 365 中系統管理員電話號碼和電子郵件地址

本文說明系統管理員可以如何變更 Office 365 中的商務電話與電子郵件。
  
如果您想了解如何變更公司設定檔資訊 (如公司名稱與地址、公司電話號碼和技術連絡人資訊)，請參閱[變更貴組織地址、技術連絡人電子郵件和其他資訊](change-address-contact-and-more.md)。
  
## <a name="to-update-your-name-phone-number-and-email-address"></a>更新您的姓名、電話號碼與電子郵件地址

使用 [**個人資訊**] 頁面來變更您的名稱、 行動電話號碼和備用電子郵件地址。 備用電子郵件地址用於重要通知，例如重設您的系統管理員密碼 （不您的電腦系統管理員密碼）。 
  
1. 瀏覽至 Microsoft 365 系統管理中心。

2. 在標頭中，選取您的設定檔圖示\>**我的帳戶** \> **個人資訊**。

3. 在 [**連絡人詳細資料**] 區段中，選取 [編輯]。 更新您的行動裝置，手機和備用電子郵件地址。 請確認您使用的備用電子郵件地址與 Office 365 電子郵件地址不同。

    > [!IMPORTANT]
    > 備用電子郵件地址與行動電話號碼會需要重設您的系統管理員密碼 （不您的電腦系統管理員密碼）。

4. 當您完成時，選取 [**儲存**。
