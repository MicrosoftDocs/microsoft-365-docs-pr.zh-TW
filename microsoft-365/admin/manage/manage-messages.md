# <a name="manage-messages-in-message-center"></a>在訊息中心管理郵件

訊息中心文章是有關新功能和變更功能的重要宣告。 本文提供有關您可以使用郵件進行哪些事情的詳細資訊，以協助您的組織中的變更管理。
  
## <a name="read-a-message"></a>閱讀郵件

若要移至消息中心，請移至系統[管理中心](https://go.microsoft.com/fwlink/p/?linkid=2024339)，然後選取 [**狀況**  >  **訊息中心**]。 

選取郵件標題，以在讀取窗格中開啟它。 每封郵件都包含變更摘要、變更對您有何影響的說明、您如何準備組織以進行變更的相關資訊，以及（在大多數情況下）您可以在其中深入瞭解的其他資訊連結。 即使郵件到期，其他資訊仍可繼續使用。

## <a name="give-feedback-on-a-post"></a>提供文章的意見反應

在郵件中心，您可以選取郵件以查看詳細資料。

如果您想要提供郵件的意見反應，請選取 [**贊**] 或 [不**喜歡**] 圖示，並在出現的文字方塊中提供意見反應。 不要提供任何個人資訊。 您也可以選擇 **[確定] 以與我聯繫以取得此意見**反應，然後選取 [**傳送**]。

## <a name="share-a-message"></a>共用郵件

看到其他人需要採取行動的訊息嗎？ 您可以透過電子郵件與任何使用者共用郵件內容：
  
1. 選取郵件以加以開啟，然後選取 [**共用**]。
  
2. 若要共用郵件，請輸入最多兩個以冒號分隔的電子郵件地址。 您可以傳送給個人和群組電子郵件地址。 您也可以選擇透過電子郵件接收郵件的副本（郵件會移至您的主要電子郵件地址），或新增個人訊息以提供擁有更多內容的收件者。
  
3. 選取 [**共用**] 以傳送電子郵件。
    
## <a name="get-a-link"></a>取得連結

需要繼續追蹤另一位管理員，以確保他們知道變更並採取動作？ 您可以產生在電子郵件或立即訊息中共用的連結，例如，它會直接將使用者連線到該郵件。 您共用連結的人員必須是 Microsoft 365 中組織的系統管理員，否則他們無法存取連結的郵件。

::: moniker range="o365-worldwide"

1. 選取郵件以加以開啟。

2. 選取 [**複製到剪貼簿**（連結）] 圖示的連結。

3. 允許網頁存取您的剪貼簿。 直接連結會複製到您的剪貼簿。

4. 使用 Ctrl+V 或按一下滑鼠右鍵，然後選取 [**貼**上] 以插入連結。

::: moniker-end

::: moniker range="o365-germany"

1. 選取郵件以加以開啟。

2. 選擇**More** \> [從動作欄**取得連結**]。

3. 允許網頁存取您的剪貼簿。 直接連結會複製到您的剪貼簿。

4. 使用 Ctrl+V 或按一下滑鼠右鍵，然後選取 [**貼**上] 以插入連結。

::: moniker-end

::: moniker range="o365-21vianet"

1. 選取郵件以加以開啟。

2. 選擇**More** \> [從動作欄**取得連結**]。

## <a name="read-and-unread-states"></a>讀取及未讀的狀態

未讀取之訊息中心內的任何郵件都會以粗體顯示。 開啟郵件將其標記為已讀取。 您可以將郵件標示為「未讀取」。

::: moniker range="o365-worldwide"

1. 在郵件中心的主頁面上，選取訊息。

2. 選取郵件頂端的 [**標示為未讀取**（信封）] 圖示。
  
::: moniker-end

::: moniker range="o365-germany"

1. 選取郵件以加以開啟。

2. 選取 [**其他** \> **標記為未讀取**]。

3. 關閉郵件。

::: moniker-end

::: moniker range="o365-21vianet"

1. 選取郵件以加以開啟。

2. 選取 [**其他** \> **標記為未讀取**]。

3. 關閉郵件。

::: moniker-end

## <a name="archive-and-restore"></a>封存及還原

如果您看到與您不相關的訊息，或是您已處理過的訊息，您可以封存郵件以從郵件中心視圖中移除。 您在郵件中心看到的視圖是專門針對您的使用者帳戶，所以從您的視圖封存，不會影響其他系統管理員。

::: moniker range="o365-worldwide"

- 在郵件中心的主頁面上，選取訊息，然後選取**郵寄清單上方的**[封存]。

- 開啟郵件，**然後選取郵件**底部的 [封存]。

需要取回已封存的郵件嗎？ 沒關係。
  
1. 選取消息中心頂端的 [已封存的**郵件**] 索引標籤。 已封存郵件的清單隨即顯示。 

2. 選取郵件，選取 [**還原**]，郵件會還原成使用中的郵件。

::: moniker-end

::: moniker range="o365-germany"

有兩種方式可封存郵件。
  
- 選取出現在郵件識別碼右側的 X。 郵件會從您的清單中移除。 或者，

- 開啟郵件，**然後選取郵件**頂端的 [封存]。

需要取回已解除的訊息嗎？ 沒關係。
  
1. 從 [**視圖**] 下拉式功能表中選取 [已封存的**郵件**] 篩選器。 已封存郵件的清單隨即顯示。

2. 在郵件識別碼的右側選取 [**還原**]，郵件會還原成使用中的郵件。

::: moniker-end

::: moniker range="o365-21vianet"

有兩種方式可封存郵件。
  
- 選取出現在郵件識別碼右側的 X。 郵件會從您的清單中移除。 或者，

- 開啟郵件，**然後選取郵件**頂端的 [封存]。

需要取回已封存的郵件嗎？ 沒關係。
  
1. 從 [**視圖**] 下拉式功能表中選取 [已封存的**郵件**] 篩選器。 已封存郵件的清單隨即顯示。

2. 在郵件識別碼的右側選取 [**還原**]，郵件會還原成使用中的郵件。

::: moniker-end
