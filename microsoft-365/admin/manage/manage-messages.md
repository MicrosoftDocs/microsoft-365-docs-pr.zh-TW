# <a name="manage-messages-in-message-center"></a>管理訊息中心中的郵件

訊息中心文章是新的和已變更之功能的相關重要宣告。 本文提供您可以運用郵件以協助您組織中的變更管理事項的相關詳細資訊。
  
## <a name="read-a-message"></a>已讀取郵件

若要移至 [訊息中心，移至[管理中心](https://go.microsoft.com/fwlink/p/?linkid=2024339)]，然後選取 [**健康情況** > **訊息中心**。 

選取 [讀取窗格中開啟的郵件標題。 每一封郵件包含進行變更，陳述式的變更可能會影響您，如何準備您的組織變更，並 （在大多數情況下），您可以了解更多的其他資訊連結的相關資訊的摘要。 即使在郵件過期之後，才可供，會繼續的其他資訊。

::: moniker range="o365-worldwide"
 
> [!NOTE]
> 如果您使用的不是新的 Microsoft 365 系統管理中心，您可以選取位於首頁頂端的 **[試用新的系統管理中心] **開關將它開啟。

::: moniker-end

## <a name="share-a-message"></a>共用郵件

請參閱其他人需要處理的郵件？ 您可以與任何使用者共用郵件的內容，透過電子郵件：
  
1. 選取郵件以開啟它，然後再選取 [**共用**。
  
2. 若要共用的郵件，請輸入以冒號分隔的兩個電子郵件地址。 您可以傳送給人員和群組電子郵件地址。 （選用） 您可以選擇透過電子郵件 （郵件會移至您在 Office 365 中的主要電子郵件地址） 接收郵件的副本或新增個人訊息收件者提供更多的內容。
  
3. 選取 [**共用**]，以傳送電子郵件。

## <a name="dismiss-and-restore"></a>關閉並還原

如果您看到訊息不屬於您，或也許您已已經在其上動作，您可以解除要從 [訊息中心] 檢視中移除的訊息。 您看到訊息中心中檢視是專屬於您的使用者帳戶，因此訊息從您的檢視不會影響其他系統管理員。

::: moniker range="o365-worldwide"

- 在 [訊息中心] 的 [主要] 頁面中，選取郵件，，然後選取**解除]** 清單上方的郵件。

- 開啟的郵件，然後選取 [**解除**底部的郵件。

需要得到 dismissed 的郵件嗎？ 不會造成問題。
  
1. 選取頂端的 [訊息中心**Dismissed 郵件**] 索引標籤。 關閉飛出的郵件清單隨即顯示。 

2. 選取郵件中，選取 [**還原**]，然後將郵件還原至使用中的郵件。

::: moniker-end
