# <a name="change-your-organizations-address-technical-contact-and-more"></a>變更貴組織的地址、技術連絡人等等
  
您可以變更您的組織設定檔，例如您的組織名稱、 地址、 電話和技術連絡人。 **您必須是全域系統管理員才能更新這項資訊。**
  
若要變更您的帳單或訂閱相關聯的地址，請參閱[變更商務用 Office 365 的帳單地址](../../commerce/billing-and-payments/change-your-billing-addresses.md)。

## <a name="edit-organization-information"></a>編輯組織資訊

> [!IMPORTANT]
> 您無法變更您訂閱的國家或地區。 因為國家或地區其中總部貴組織決定哪些服務及其都可以使用您、 稅金和帳單 [貨幣] 資料中心的位置。 若要變更貴組織的國家或地區，請註冊新的帳戶，選擇所需的國家或地區，並購買新訂閱。

若要變更您的公司設定檔] 頁面上的其他資訊：
  
1. 在系統管理中心，移至 [**設定** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2118715" target="_blank">設定</a>] 頁面。

2. 在 [**組織設定檔**] 索引標籤上選取 [**組織資訊**]。

3. 更新您的組織的詳細資訊，然後選取 [**儲存變更**。

    每個欄位的說明都提供如下。

## <a name="what-do-these-fields-mean"></a>這些欄位代表什麼意思？

|**Field**  |**描述**  |
|---------|---------|
|名稱  <br/>   | 在這裡輸入的名稱就是使用者將會在下列頁面上看到的名稱：  <br/>  登入頁面：如果使用者以公司或學校電子郵件地址設定其他 Microsoft 帳戶，他們可能會在登入頁面上看到組織名稱。這可協助他們區別 公司或學校帳戶 與其他帳戶，以識別要用哪個帳戶登入。  <br/>  組織設定檔連結和頁面：組織設定檔的連結會顯示組織名稱。  <br/>  Yammer 導覽： 中 Yammer，左側導覽窗格中會使用組織名稱為家用 Yammer 網路的名稱。  <br/> OneDrive 同步處理用戶端： 組織名稱會顯示在 [windows 檔案總管及 Mac 的檔案路徑、 OneDrive 活動中心]、 [OneDrive 雲端圖示，和 [OneDrive 設定視窗的工具提示上尋找工具。 目前，更新組織名稱不會更新它已設定的用戶端。 <br/>        |
|地址、縣/市、州/省、郵遞區號  <br/>     | 在此處輸入的地址是您會看到上您的帳單，在 [銷售到： 購買人地址帳單上的會與您組織的地址在您的設定檔] 頁面上相同 （請參閱[了解帳單以取得商務用 Office 365](../../commerce/billing-and-payments/understand-your-invoice2.md)）。  <br/>        |
|國家或地區  <br/>    | 這是公司總部所在的國家或地區。選取的國家或地區會決定您適用的服務、您國家或地區的稅金及帳單貨幣，以及最接近您的資料中心的位置 (請參閱 [Microsoft Office 授權限制](https://office.microsoft.com/redir/FX103037529))。  <br/>附註： 一旦選取國家或地區無法變更。 如果您要變更選項，必須取消訂閱後再次註冊。 如需此程序，[請連絡支援](../contact-support-for-business-products.md)的協助。        |
|電話  <br/>     | 這是您公司的主要電話號碼，通常是公司總部的號碼。  <br/>        |
|技術連絡人  <br/> |這是管理您 Office 365 訂閱之主要技術人員的電子郵件地址。此人員會收到 Office 365 服務狀態的相關通訊。  <br/> |
|慣用語言  <br/> |此慣用語言會決定 Microsoft 傳送至您組織之所有通訊的語言。當您註冊時，此設定會決定 SharePoint Online 使用的語言，也就是使用者在您小組網站上看到的語言。如果您在註冊後變更語言喜好設定，日後所有的通訊就會以最新選取的語言傳送。    <br/> 附註： 使用 SharePoint Online 的語言無法變更。           |

## <a name="change-your-alternate-email-address"></a>變更您的備用電子郵件地址

身為系統管理員，您可以變更後您備用電子郵件地址重設密碼。

1. 移至<a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">Microsoft 365 系統管理中心</a>。

2. 在標頭中，選取您的設定檔] 圖示，然後選取 [**我的帳戶]**。

3. 在左邊，選取 [**個人資訊**。

4. 在 [**連絡人詳細資料**] 卡片中，選取 [**編輯**]。

5. 編輯 [**備用電子郵件**] 方塊中的地址，然後選取 [**儲存**]。

若要了解變更其他設定檔資訊，請參閱[變更連絡人喜好設定](change-contact-preferences.md)或[變更顯示語言](https://support.office.com/article/6f238bff-5252-441e-b32b-655d5d85d15b.aspx)。
  
### <a name="email-signatures"></a>電子郵件簽名
  
，您可以變更 Outlook Web App 中的電子郵件簽名。如需詳細資訊，請參閱[郵件設定](https://support.office.com/article/30c69a79-efc6-42d2-b740-4bf1c1f8a01c.aspx)。
