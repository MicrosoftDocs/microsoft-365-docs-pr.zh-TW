# <a name="message-center"></a>訊息中心

若要追蹤即將到來的變更，包括新增及變更的功能、規劃的維護或其他重要宣告，請移至 <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">訊息中心</a>。
  
若要開啟訊息中心：

::: moniker range="o365-worldwide"

- 在系統管理中心中，移至 [ **狀況** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">訊息中心</a>]。

::: moniker-end

::: moniker range="o365-germany"

- 在系統 <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">管理中心</a>中，移至 [ **狀況** > **訊息中心**]。

::: moniker-end

::: moniker range="o365-21vianet"

- 在系統 <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">管理中心</a>中，移至 [ **狀況** > **訊息中心**]。

::: moniker-end

您也可以在行動裝置上使用 [Microsoft 365 系統管理應用程式](https://go.microsoft.com/fwlink/p/?linkid=627216) 來查看訊息中心，這是使用推播通知保持最新的絕佳方式。

### <a name="major-updates"></a>主要更新

您可以從 [**標記**] 下拉式清單中選取 **主要更新**，以查看主要更新。

需要採取動作時，會提前至少30天的主要更新，而且可能包括：
  
- 變更每日生產力，例如收件匣、會議、委派、共用和存取

- 對主題、網頁元件及其他可能影響自訂功能的元件所做的變更

- 增加或減少可查看的容量，例如儲存區、規則、專案或期間的數目

- 產品署名的變更可能：

  - 導致使用者混淆，

  - 導致技術支援人員處理及參考材料的變更，或

  - 變更 URL

- 新服務或應用程式

- 需要系統管理動作的變更 (獨佔預防或修正問題) 

- 資料儲存位置的變更
  
### <a name="preferences"></a>偏好

如果管理是分佈于整個組織，您可能不想或需要查看所有 Microsoft 365 服務的帖子。 每個系統管理員都可以：

- 設定可控制郵件中心顯示哪些郵件的喜好設定。
- 篩選郵件
- 設定電子郵件首選項以接收所有郵件的每週摘要、僅限主要更新的電子郵件，以及資料隱私權訊息的電子郵件。  

::: moniker range="o365-worldwide"

1. 在郵件中心的頂端，選取 [ **喜好** 設定]。

2. 在 [ **自訂視圖** ] 索引標籤中，確定針對您要監視的每一個服務，都選取了核取方塊。 清除您想要從您的訊息中心視圖中篩選出之服務的核取方塊。

3. 預設會開啟摘要電子郵件，並將其傳送至您的主要電子郵件地址。 若要停止接收每週摘要，請清除 [**電子郵件]** 索引標籤中的 [**從郵件中心傳送電子郵件通知**] 核取方塊。 

   您也可以輸入最多兩個電子郵件地址（以分號分隔）。

   您也可以選擇要取得的電子郵件，以及您所選取之服務的每週摘要。

4. 選取 [ **儲存** ] 以保留您的變更。
  
::: moniker-end

::: moniker range="o365-germany"

1. 在郵件中心的頂端，選取 [ **編輯郵件中心喜好** 設定]。

2. 針對您要監視的每一項服務，確定此切換設定為 [ **開啟** ]。 使用切換將您要篩選掉之服務的設定變更為 [ **關閉** ]。

3. 預設會開啟摘要電子郵件，並將其傳送至您的主要電子郵件地址。 若要停止接收每週摘要，請將 [以 **每週摘要傳送我的郵件** ] 設定為 [ **關閉**]。 

   主要更新的電子郵件通知是一個不同的控制項。 如果您想要收到有關主要更新的電子郵件通知，請確認已 **開啟****主要更新的傳送電子郵件**。 將設定變更為 [ **關閉** ]，以停止取得主要更新的電子郵件。 

   若要收到有關資料隱私權訊息的電子郵件通知，請確認已 **開啟**[**傳送我的電子郵件的資料隱私權訊息**]。 若要停止接收這些通知，請將設定變更為 [ **關閉**]。  (資料隱私權郵件不會包含在每週摘要中。 ) 

   您可以選取或清除您的主要電子郵件地址，但不能變更。 若要指定每週的電子郵件摘要傳送目標的電子郵件地址，請確認 [ **傳送我的郵件的每週更新摘要** 為 **開啟**]。 若有兩位以上的人員應該取得摘要電子郵件，請輸入 Microsoft 365 群組或通訊群組清單的電子郵件地址。

4. 選取 [ **儲存** ] 以保留您的變更。<br/>

::: moniker-end

::: moniker range="o365-21vianet"

1. 在郵件中心的頂端，選取 [ **喜好** 設定]。

2. 在 [ **自訂視圖** ] 索引標籤中，確定針對您要監視的每一個服務，都選取了核取方塊。 清除您想要從您的訊息中心視圖中篩選出之服務的核取方塊。

3. 預設會開啟摘要電子郵件，並將其傳送至您的主要電子郵件地址。 若要停止接收每週摘要，請清除 [**電子郵件]** 索引標籤中的 [**從郵件中心傳送電子郵件通知**] 核取方塊。 

   您也可以輸入最多兩個電子郵件地址（以分號分隔）。 <br><br/>您也可以選擇要取得的電子郵件，以及您所選取之服務的每週摘要。

4. 選取 [ **儲存** ] 以保留您的變更。<br/>

## <a name="unsubscribe-from-message-center-emails"></a>從郵件中心電子郵件取消訂閱

1. 預設會開啟摘要電子郵件，並將其傳送至您的主要電子郵件地址。 若要停止接收周曆摘要，請選取 [ **喜好** 設定]，然後按一下 [ **電子郵件**]。

    - 取消選取 [ **傳送我的郵件的每週摘要** ] 核取方塊。
    - 主要更新的電子郵件通知是一個不同的控制項。 如果您不想收到有關主要更新的電子郵件通知，請確認未選取 [ **將我的主要更新傳送給我的電子郵件** ] 核取方塊。
    -   若要停止接收資料隱私權訊息的電子郵件通知，請確認未選取 [ **將電子郵件傳送給我的資料隱私權訊息** ] 核取方塊。   (資料隱私權郵件不會包含在每週摘要中。 ) 

2. 選取 [ **儲存** ] 以保留您的變更。<br/>
