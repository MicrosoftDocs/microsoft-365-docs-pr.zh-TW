# <a name="message-center"></a>訊息中心


若要追蹤即將到來的變更，包括新增及變更的功能、規劃的維護或其他重要宣告，請移至<a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">訊息中心</a>。 
  
若要開啟訊息中心：

::: moniker range="o365-worldwide"

- 在系統管理中心中，移至 [**狀況** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">訊息中心</a>]。

::: moniker-end

::: moniker range="o365-germany"

- 在系統<a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">管理中心</a>中，移至 [**狀況** > **訊息中心**]。

::: moniker-end

::: moniker range="o365-21vianet"
 
- 在系統<a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">管理中心</a>中，移至 [**狀況** > **訊息中心**]。

::: moniker-end

您也可以在行動裝置上使用[Microsoft 365 系統管理應用程式](https://go.microsoft.com/fwlink/p/?linkid=627216)來查看訊息中心，這是使用推播通知保持最新的絕佳方式。

### <a name="messages"></a>郵件

郵件中心會以表格的形式顯示所有使用中郵件的視圖。 依預設，它會顯示最近的郵件的清單頂端。 您可以選取一個索引標籤，以查看所有使用中的**郵件**、**高重要性**的郵件、**未讀取的郵件**，以及已清除的**郵件**。 您也可以使用 [**篩選**] 下拉式功能表，查看特定郵件類別中的所有郵件。

以下是您將在每一欄中看到的資訊的快速綜述。 選取任何欄標題，以變更排序次序。
  
|**欄**|**描述**|
|:-----|:-----|
|核取記號  <br/> |選取 [欄標題] 欄中的核取記號會選取目前顯示的所有郵件。 選取一或多封郵件旁的核取記號可讓您對這些郵件採取動作。  <br/> |
|應用程式圖示 <br/> |圖示表示要套用郵件的應用程式。<br/> |
|郵件標題  <br/> |郵件標題是即將進行之變更的簡短描述。 如果不會顯示完整的標題，請將游標放在它上方，整個標題就會出現在一個彈出框中。  <br/> |
|更多選項 <br/> |更多選項可讓您解除郵件、標示為已讀或未讀取的郵件，或與另一個系統管理員共用。若要還原已關閉的郵件，請選取 [已清除的**郵件**] 索引標籤，選取郵件旁的核取記號，然後選取 [**還原**]。 <br/> |
|主要更新 <br/> |此欄中的驚嘆號表示主要更新。 <br/> |
|行動方式  <br/> |如果我們進行的變更需要您在特定期限採取動作，我們才會有日期。 由於我們很少使用 [**法案**] 欄，因此如果您在這裡看到一些內容，您應該特別注意。  <br/> |
|類別  <br/> | 郵件是透過下列三種類別之一來識別： <br/><br/> **防止或修正問題**：通知您可能影響組織的已知問題，並可能要求您採取動作以避免服務中斷。 防止或修正問題與服務健康情況訊息不同，因為它們會提示您主動避免發生問題。 <br/> <br/> **規劃變更**：通知您 Microsoft 365 的變更可能需要您採取行動以避免服務中斷。 例如，我們將告訴您，對系統需求的變更或有關要移除之功能的變更。 我們嘗試提供至少30天的變更，以要求系統管理員保證服務保持正常運作。 <br/> <br/> **敬請掌握**：告訴您您的組織中所開啟的新增或更新功能。 這些功能通常會在[Microsoft 365 藍圖](https://go.microsoft.com/fwlink/?linkid=2070821)中先宣告。 <br/><br/>您也可能會將依照服務等級協定的計畫維護告知您。 規劃的維護可能會導致停機時間，您或您的使用者將無法存取 Microsoft 365、特定功能或服務（例如，電子郵件或 OneDrive 商務）。  <br/> |
|上次更新  <br/> |發佈或上次更新郵件的日期。  <br/> |
|郵件識別碼  <br/> |Microsoft 會依郵件識別碼追蹤我們的郵件中心文章。 如果您想要提供意見反應或您對特定郵件的支援，可以參考此識別碼。  <br/> |

在[郵件中心管理郵件](manage-messages.md)，以取得有關您可以使用郵件進行的工作的詳細資訊。
  
### <a name="major-updates"></a>主要更新

您可以在郵件中心的頂端選取 [**高重要性**] 索引標籤，以查看主要更新。

需要採取動作時，會提前至少30天的主要更新，而且可能包括：
  
- 變更每日生產力，例如收件匣、會議、委派、共用和存取

- 對主題、網頁元件及其他可能影響自訂功能的元件所做的變更

- 增加或減少可查看的容量，例如儲存區、規則、專案或期間的數目

- 產品署名的變更可能：

  - 導致使用者混淆，

  - 導致技術支援人員處理及參考材料的變更，或

  - 變更 URL

- 新服務或應用程式

- 需要系統管理動作（獨佔預防或修正問題）的變更

- 資料儲存位置的變更
  
### <a name="preferences"></a>偏好

如果管理是分佈于整個組織，您可能不想或需要查看所有 Microsoft 365 服務的帖子。 每個系統管理員都可以：

- 設定可控制郵件中心顯示哪些郵件的喜好設定。
- 篩選郵件
- 設定電子郵件首選項以接收所有郵件的每週摘要、僅限主要更新的電子郵件，以及資料隱私權訊息的電子郵件。  

::: moniker range="o365-worldwide"

1. 在郵件中心的頂端，選取 [**編輯喜好**設定]。

2. 確定針對您要監視的每一個服務，都選取了核取方塊。 清除您想要從您的訊息中心視圖中篩選出之服務的核取方塊。

3. 預設會開啟摘要電子郵件，並將其傳送至您的主要電子郵件地址。 若要停止接收每週摘要，請清除 [**傳送每週更新的電子郵件摘要**] 核取方塊。 <br/><br/>主要更新的電子郵件通知是一個不同的控制項。 如果您想要收到有關主要更新的電子郵件通知，請確認已選取 [**將我的電子郵件傳送至主要更新**]。 清除此核取方塊可停止取得主要更新的電子郵件。 <br><br/>選取 [**傳送我的電子郵件給我的資料隱私權訊息**] 旁邊的核取方塊，以接收資料隱私權訊息的個別電子郵件通知（資料隱私權郵件不會包含在每週摘要中）。 <br><br/>您可以選取或清除您的主要電子郵件地址，但不能變更。 若要指定傳送資料隱私權郵件的其他電子郵件地址，請選取 [**其他電子郵件地址**] 核取方塊，然後輸入您要傳送電子郵件的其他電子郵件地址。 若有兩個人應該取得資料隱私權電子郵件，請輸入 Microsoft 365 群組或通訊群組清單的電子郵件地址。

4. 選取 [**儲存**] 以保留您的變更。
  
::: moniker-end

#### <a name="display-messages-in-your-preferred-language"></a>以您慣用的語言顯示訊息
  
我們使用機器翻譯以您慣用的語言自動顯示郵件。 有關如何設定語言的詳細資訊，請閱讀[郵件中心文章的語言翻譯](language-translation-for-message-center-posts.md)。
