# <a name="message-center-in-office-365"></a>在 Office 365 訊息中心


若要追蹤的即將到來的變更，包括新的和已變更的功能、 計劃的維護或其他重要宣告，前往<a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">訊息中心</a>。 這是我們將張貼正式的宣告，可讓您採取積極變更管理。 每個文章會提供您的計劃的變更與它可能會如何影響您的使用者，與出的詳細資訊連結可協助您準備的高階概觀。
  
若要開啟 [訊息中心，請先確定您已選取**嘗試在新版系統管理中心**切換在系統管理中心首頁上，然後：

::: moniker range="o365-worldwide"

- 在系統管理中心，移至 [**健康情況** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">訊息中心</a>。

::: moniker-end

::: moniker range="o365-germany"

- 在<a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">系統管理中心</a>，移至 [**健康情況** > **訊息中心**。

::: moniker-end

::: moniker range="o365-21vianet"
 
- 在<a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">系統管理中心</a>，移至 [**健康情況** > **訊息中心**。

::: moniker-end

您也可以使用行動裝置上的[Microsoft 365 Admin app](https://go.microsoft.com/fwlink/p/?linkid=627216)來檢視訊息中心，也就是保持最新的推入通知的絕佳方式。

### <a name="major-updates"></a>主要的更新

主要更新可以檢閱藉由選取頂端的 [訊息中心的 [**高重要性**] 索引標籤。

需要採取行動時，主要的更新會傳達至少 30 天前。
  
主要更新的定義如下：
  
- 每日的產能，例如收件匣、 會議、 委派、 共用及存取的變更

- 變更的佈景主題、 網頁組件及其他元件，可能會影響自訂的功能

- 增加或減少可見的容量，例如存放區，規則、 項目或工期數目

- 產品的品牌的變更可能：

  - 一般使用者，造成混淆

  - 服務台程序和參照材料，變更會導致或

  - 變更 URL

- 將新的服務或應用程式

- 需要 （之外防止或修正問題） 系統管理員採取動作的變更

- 變更您的資料儲存位置

當您開啟讀取其主要更新後時，您會看到的標頭**重大更新**、 誰它**適用於**，以及目前的推行狀態和如何影響您的描述。
