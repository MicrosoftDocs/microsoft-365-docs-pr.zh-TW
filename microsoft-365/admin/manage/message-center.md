# <a name="message-center"></a>訊息中心

若要追蹤即將到來的變更，包括新增及變更的功能、規劃的維護或其他重要宣告，請移至 <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">訊息中心</a>。
  
若要開啟訊息中心：

::: moniker range="o365-worldwide"

- 在系統管理中心中，移至 [ **狀況** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">訊息中心</a>]。

::: moniker-end

::: moniker range="o365-germany"

- 在系統 <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">管理中心</a>中，移至 [ **狀況** > **訊息中心**]。

::: moniker-end

::: moniker range="o365-21vianet"

- 在系統 <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">管理中心</a>中，移至 [ **狀況** > **訊息中心**]。

::: moniker-end

您也可以使用行動裝置上的[Microsoft 365 系統管理應用程式](https://go.microsoft.com/fwlink/p/?linkid=627216)來查看訊息中心，這是使用推播通知保持最新的絕佳方式。

若要取消訂閱郵件中心電子郵件，請參閱本文中 [的從郵件中心電子郵件取消訂閱](#unsubscribe-from-message-center-emails) 。

### <a name="major-updates"></a>主要更新

您可以從 [**標記**] 下拉式清單中選取 **主要更新**，以查看主要更新。

需要採取動作時，會提前至少30天的主要更新，而且可能包括：

- 變更每日生產力，例如收件匣、會議、委派、共用和存取
- 對主題、網頁元件及其他可能影響自訂功能的元件所做的變更
- 增加或減少可查看的容量，例如儲存區、規則、專案或期間的數目
- 產品署名的變更可能：
  - 導致使用者混淆，
  - 導致技術支援人員處理及參考材料的變更，或
  - 變更 URL
- 新服務或應用程式
- 需要系統管理動作的變更 (獨佔預防或修正問題) 
- 資料儲存位置的變更
  
### <a name="preferences"></a>偏好

如果管理是分佈于整個組織，您可能不想或需要查看所有 Microsoft 365 服務的帖子。 每個系統管理員都可以：

- 設定可控制郵件中心顯示哪些郵件的喜好設定。
- 篩選郵件
- 設定電子郵件首選項以接收所有郵件的每週摘要、僅限主要更新的電子郵件，以及資料隱私權訊息的電子郵件。  

::: moniker range="o365-worldwide"

1. 在郵件中心的頂端，選取 [ **喜好** 設定]。

2. 在 [ **自訂視圖** ] 索引標籤中，確定針對您要監視的每一個服務，都選取了核取方塊。 清除您想要從您的訊息中心視圖中篩選出之服務的核取方塊。

3. 預設會開啟摘要電子郵件，並將其傳送至您的主要電子郵件地址。 若要停止接收每週摘要，請清除 [**電子郵件]** 索引標籤中的 [**從郵件中心傳送電子郵件通知**] 核取方塊。 

   您也可以輸入最多兩個電子郵件地址（以分號分隔）。

   您也可以選擇要取得的電子郵件，以及您所選取之服務的每週摘要。

4. 選取 [ **儲存** ] 以保留您的變更。
  
::: moniker-end

## <a name="give-feedback-on-a-post"></a>提供文章的意見反應

在訊息中心，您可以選取訊息以查看詳細資訊。

如果要對訊息提供意見反應，請在詳細資料窗格中，選取訊息詳細資料窗格底部的 **喜歡** 或 **不喜歡** 圖示，並在顯示的文字輸入框中提供選用的意見反應。 請勿提供任何個人資料。 您也可以選取 **「有關此意見反應，可以聯絡我」**，然後選取 **[傳送]**。

## <a name="share-a-message"></a>共用訊息

看到了他人需要採取動作的訊息？ 您可以透過電子郵件與任何使用者共用訊息內容：
  
1. 選取要開啟的郵件，然後選取 **[共用]**。
  
2. 若要共用訊息，請輸入最多兩個用冒號分隔的電子郵件地址。 您可以傳送至個人和群組的電子郵件地址。 或者，您可以選擇在電子郵件中接收訊息的副本 (訊息將轉到您的主要電子郵件地址) 或新增個人訊息以向收件者提供更多内容。
  
3. 選取 **[共用]** 以傳送電子郵件。

## <a name="get-a-link"></a>取得連結

需要跟進另一個管理員，以確保他們知道的變更和採取動作？ 例如，您可以產生連結以在電子郵件或立即訊息中共用，該連結將使用者直接連線至該訊息。 您共用連結的人員必須具有訊息中心的權限。 請參閱 [不具訊息中心權限的系統管理員角色](message-center.md#admin-roles-that-dont-have-access-to-the-message-center) 以獲得詳細資訊。

1. 選取訊息以開啟。

2. 選取 **複製連結**。

3. 使用 Ctrl+V 或以滑鼠右鍵按一下並選取 **貼上** 以將連結插入你所想要的文件。

## <a name="read-and-unread-states"></a>已讀取和未讀取狀態

訊息中心中任何未讀取的訊息都將以粗體顯示。 開啟訊息會將其標示為已讀取。 您可以將訊息標示為未讀取。

在訊息中心的主頁面上，選取 **更多選項** 訊息旁的省略符號，然後選取 **標記為未讀取**。

您也可以開啟訊息，並在詳細資料面板中將其標記為未讀取。
  
## <a name="archive-and-restore"></a>封存與還原

若看到與您無關的訊息，或者您可能已經對其採取了動作，您可以將該訊息封存以將其從收件匣中移除。 您在 [訊息中心] 看到的檢視是您的使用者帳戶特定的，因此從您的檢視封存它不會影響其他系統管理員。 有兩種方法可以封存郵件。

- 在訊息中心 的主頁面上，選取訊息，然後選取訊息清單上方的 **封存**。
- 開啟訊息，然後選取訊息窗格頂端的 **封存**。

需要取回封存的訊息？ 沒問題。
  
1. 選取訊息中心頂部的 **封存** 索引標籤。 此時會出現已封存訊息的清單。

1. 選取訊息，然後選取 **還原**，訊息將還原至收件匣。

## <a name="favorite-messages"></a>我的最愛訊息

若要將訊息標記為我的最愛，請將游標停留在訊息標題上方，您就會看到 **我的最愛** :::image type="icon" source="../../media/favorite-star.png" border="false"::: 星號，您可以在 **更多選項** 省略符號之後選取。 將訊息標示為我的最愛之後，您也可以排序和篩選郵件。

## <a name="scroll-messages-in-the-message-pane"></a>在訊息窗格中捲動訊息

當您在讀取窗格中開啟訊息時，您可以使用窗格頂端的 **向上** 和 **向下** :::image type="icon" source="../../media/updownarrows.png" border="false"::: 鍵來移至清單中的下一則或上一則訊息。

## <a name="track-your-message-center-tasks-in-planner"></a>在 Planner 中追蹤訊息中心的工作

Microsoft 365 訊息中心會提供許多與 Microsoft 365 服務變更相關的可採取動作資訊。 要追蹤哪些變更需要工作完成、時間，與執行人員，以及追蹤每項工作的完成都不是容易的事。 您也可能想要記下某事，並將它標記為稍後檢查。 只要將 Microsoft 365 系統管理中心的訊息同步處理至 Microsoft Planner 時，即可以執行這些以及更多作業。 如需詳細資訊，請參閱[在 Planner 中追蹤訊息中心的工作](/office365/planner/track-message-center-tasks-planner)。

有關 [訊息中心] 的概觀，請參閱 [Microsoft 365 中的訊息中心](message-center.md)。 或者，要了解如何設定語言喜好設定以啟用 [訊息中心] 文章的機器翻譯功能，請參閱[訊息中心文章的語言翻譯功能](language-translation-for-message-center-posts.md)。 如果您想程式設計一種替代方法來取得即時服務健康情況資訊和訊息中心通訊，請參考 [Microsoft 365 服務通訊 API 概觀](/previous-versions/office/developer/o365-enterprise-developers/jj984343(v=office.15))。

## <a name="unsubscribe-from-message-center-emails"></a>從郵件中心電子郵件取消訂閱

1. 預設會開啟摘要電子郵件，並將其傳送至您的主要電子郵件地址。 若要停止接收周曆摘要，請選取 [ **喜好** 設定]，然後按一下 [ **電子郵件**]。
    - 取消選取 [ **傳送我的郵件的每週摘要** ] 核取方塊。
    - 主要更新的電子郵件通知是一個不同的控制項。 如果您不想收到有關主要更新的電子郵件通知，請確認未選取 [ **將我的主要更新傳送給我的電子郵件** ] 核取方塊。
    - 若要停止接收資料隱私權訊息的電子郵件通知，請確認未選取 [ **將電子郵件傳送給我的資料隱私權訊息** ] 核取方塊。   (資料隱私權郵件不會包含在每週摘要中。 ) 

2. 選取 [ **儲存** ] 以保留您的變更。
