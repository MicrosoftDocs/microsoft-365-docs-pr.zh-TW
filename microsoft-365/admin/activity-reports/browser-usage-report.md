# <a name="microsoft-365-reports-in-the-admin-center---microsoft-browser-usage"></a>Microsoft 365系統管理中心報告-Microsoft 瀏覽器使用量

[Microsoft 365 **報告**] 儀表板會顯示您組織中產品的活動概況。 它可讓您深入瞭解個別產品層級報告，讓您更深入瞭解各項產品內的活動。 請參閱[報告概觀主題](activity-reports.md)。 在 [Microsoft 瀏覽器使用量] 報告中，您可以深入瞭解 Internet Explorer、舊版 Microsoft Edge 及新的 Microsoft Edge 用法。 流量報告是以使用 Microsoft browser 所存取的 Microsoft 365 線上服務為基礎。

 > [!NOTE]
 > 您必須是全域系統管理員、全域讀取者或報告讀取者 Microsoft 365 或 Exchange、SharePoint 或商務用 Skype 管理員以查看報告。

## <a name="how-to-get-to-the-microsoft-browser-usage-report"></a>如何取得 Microsoft 瀏覽器使用方式報告

1. 在系統管理中心中，移至 **[報告]** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">[使用量]</a> 頁面。 
2. 在 [儀表板] 主頁上，按一下 [Microsoft browser 使用狀況卡片上的 **View more** ] 按鈕。

## <a name="interpret-the-microsoft-browser-usage-report"></a>解讀 Microsoft browser usage report

![Microsoft 瀏覽器使用方式報告](../../media/95557c88-24ee-417d-a828-96ba00b17aaf.png)

|項目|描述|
 |:-----|:-----|
 |1. <br/> |您可以針對過去7天、30天、90天或180天的趨勢，查看 **Microsoft browser usage** report。  <br/> |
 |2. <br/> |每個報告中的資料通常會涵蓋過去7天。 <br/> |
 |3. <br/> |[**每日** 作用中使用者] 圖表會顯示 Microsoft Edge 舊版 Microsoft Edge 和 Internet Explorer 的每日使用者計數，以用於存取 Microsoft 365 服務。 <br/> |
 |4.<br/>|[作用中的 **使用者**] 圖表會顯示使用 Microsoft Edge、舊版 Microsoft Edge 和 Internet Explorer 的使用者總數，在選取的時段內用來存取 Microsoft 365 服務的使用者人數。<br/>|
 |5.<br/>|表格顯示每個使用者層級的資料明細。您可以新增或移除表格中的欄位。  <br/><br/>**Username** 是使用 Microsoft 瀏覽器連線至 Microsoft 365 服務之使用者的電子郵件地址。<br><br/>**使用 Microsoft Edge** 會顯示刻度標記，如果使用者使用 Microsoft Edge 連接至 Microsoft 365 服務。<br/><br/>**使用舊版 Microsoft Edge** 會顯示刻度標記，如果使用者使用舊版 Microsoft Edge 連接至 Microsoft 365 服務。<br/><br/>如果使用者使用 internet explorer 連接至 Microsoft 365 服務，則 **internet explorer** 會顯示刻度標記。 |
 |6.<br/>|選取 [ **選擇欄** ] 圖示，以新增或移除報告中的欄。|
 |7.<br/>|您也可以選取 [**匯出**] 連結，將報表資料匯出至 Excel .csv 檔案。 這會匯出所有使用者的資料，並可讓您進行簡單的匯總、排序及篩選，以進行進一步分析。 如果您的使用者少於100，您可以在報表本身的資料表中進行排序和篩選。 如果您有超過100的使用者，為了進行篩選和排序，您將需要匯出資料。|
