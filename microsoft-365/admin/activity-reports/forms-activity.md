# <a name="microsoft-365-reports-in-the-admin-center---forms-activity"></a>在系統管理中心-表單活動的 Microsoft 365 報告

Microsoft 365**報告**儀表板顯示您的活動概觀跨組織中的產品。 此功能可讓您深入了解個別產品層級報表，更加深入解析各產品內的活動。 請參閱[報告概觀主題](activity-reports.md)。
  
例如，您可以了解每位授權使用 Microsoft Forms 查看其互動表單使用者的活動。 它也可以協助您了解共同作業的程度所建立的表單和表單使用者回應的數量的層級。
  
> [!NOTE]
> 您必須是全域系統管理員、 全域讀者或報告讀取者，在 Microsoft 365 或 Exchange、 SharePoint 或 Skype 商務系統管理員，才能查看報告。 

## <a name="how-to-get-to-the-forms-activity-report"></a>如何取得表單活動報告

1. 在系統管理中心中，移至 **[報告]** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">[使用量]</a> 頁面。

    
2. 從 [**選取 [報告]** 下拉式清單，選取 [**表單** \> **活動**。

## <a name="interpret-the-email-activity-report"></a>解讀電子郵件活動報告

您可以透過檢視使用者的電子郵件活動查看 [**活動**] 和 [**使用者**] 圖表。 

![表單活動報告](../../media/adminformsactivity.png)

|||
|:-----|:-----|
|1.  <br/> |[**表單活動**] 報告可檢視的趨勢，過去 7 天、 30 天、 前 90 天或 180 天。 不過，如果您在報表中選取的特定一天，表格 (7) 將會顯示資料 28 天，從目前的日期 （不是該報告產生的日期）。  <br/> |
|2.  <br/> |每個報告中的資料通常涵蓋的最後一個 24 到 48 小時。  <br/> |
|3.  <br/> |[**使用者**] 檢視可協助您了解在作用中表單使用者人數的趨勢。 使用者會被視為作用中，如果他或她已執行過周圍表單活動 （建立、 編輯、 檢視、 因故而停止運作） 或在特定時間期間內回應表單。  <br/> |
|4.  <br/> |[**活動**] 檢視可協助您了解在作用中使用者人數的趨勢。 使用者在指定的期間內若曾經執行過檔案活動 (儲存、同步處理、修改或共用) 或瀏覽過頁面，就會被視為作用中使用者。<br/> 附註： 活動可發生多次單一表單中，但只會算做一個使用中的表單。 例如，您可以建立表單，並繼續編輯相同表單多次一段指定的時間，但它只會算做一個單一表單。 不過如果使用者送出多個回應的相同表單，每個回應仍為個別的回應，因此會計算多次。 <br/> |
|5.<br/>|在 [**使用者**] 圖表中，Y 軸是唯一的使用者數目。 X 軸是唯一的使用者會在作用中的日期。 圖例會：<br/><br/>**設計者**表示使用者已建立或編輯表單。<br/>**回應程式**表示使用者已送出表單的回應。<br/> **使用者總數**表示使用者已設計工具或回應程式的公司中的任何人。<br/><br/> 在 [**活動**] 圖表中，Y 軸是唯一的表單或回應的計數。 X 軸是發生在表單或回應活動的日期。 圖例會：<br/><br/>**建立的表單**是唯一的使用者已建立的表單計數。<br/> **在回應中的 Signed**的計數登入的組織中的使用者收到的回應。<br/> **匿名回應**是組織中的使用者接收到匿名回應的計數。<br/><br/>|
|6.<br/>|您可以篩選看圖表所選取項目在圖例中的數列。 例如，在 [使用者] 圖表中，選取設計者、 回應程式或使用者總數以查看只與其個別相關的資訊。 變更此選取項目不會變更其下格線表格中的資訊。|
|7.<br/>|下表顯示每個使用者層級的活動明細。圖例會：<br/><br/>**Username**是 Microsoft Forms 執行該活動之使用者的電子郵件地址。<br/>**[上次活動日期 (UTC)** 是由所選取的日期範圍內的使用者執行表單活動的最晚日期。 若要查看特定日期發生的活動，請直接選取圖表中的日期。<br/><br/>這會篩選表格，顯示檔案活動資料僅針對該特定日期執行該活動的使用者。<br/><br/>**建立的表單數目**為使用者建立的表單數目。<br/> **回應的表單數目**是使用者已送出的回覆表單的數目。|
|8.<br/>|選取 [**管理欄]** 圖示來新增或移除報告中的欄。|
|9.<br/>|您也可以選取 [**匯出**] 連結，來匯出到 Excel.csv 檔案的報表資料。 這會匯出所有使用者的資料，並可讓您進行簡單彙總，排序和篩選，以便進一步分析。 如果您有 100 個使用者，您可排序及篩選中報表本身的表格。 如果您有 100 個以上的使用者，在排序和篩選器中，您必須將資料匯出。|
