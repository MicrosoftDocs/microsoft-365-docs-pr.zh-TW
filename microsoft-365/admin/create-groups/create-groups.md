# <a name="create-an-office-365-group-in-the-microsoft-365-admin-center"></a>在 Microsoft 365 系統管理中心建立 Office 365 群組
  
雖然使用者可以從 Outlook 或其他應用程式]，建立 Office 365 群組身為系統管理員，您可能需要建立或刪除群組、 新增或移除成員，以及自訂其運作方式。 在 Microsoft 365 系統管理中心是執行這項操作的位置。 

> [!TIP]
> Office 365 連線 Yammer 群組必須在 Yammer 中建立，但可以在其他 Office 365 群組像在 Microsoft 365 系統管理中心中管理。 若要深入了解，請參閱 [Yammer 與 Office 365 群組](https://support.office.com/article/d8c239dc-a48b-47ab-b85e-6b4b8191a869.aspx)。 

::: moniker range="o365-worldwide"

> [!NOTE]
> 如果您使用的不是新的 Microsoft 365 系統管理中心，您可以選取位於首頁頂端的 **[試用新的系統管理中心] **開關將它開啟。

::: moniker-end

## <a name="create-an-office-365-group"></a>建立 Office 365 群組

::: moniker range="o365-worldwide"

1. 在系統管理中心，移至 [**群組** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2052855" target="_blank">群組</a>] 頁面。

::: moniker-end

::: moniker range="o365-germany"

1. 在[系統管理中心](https://go.microsoft.com/fwlink/p/?linkid=848041)，移至 [**群組** > **群組**] 頁面。 

::: moniker-end

::: moniker range="o365-21vianet"

1. 在[系統管理中心](https://go.microsoft.com/fwlink/p/?linkid=850627)，移至 [**群組** > **群組**] 頁面。 

::: moniker-end

2. 選取 [**新增群組**。
  
3. 在 [**選擇群組類型**] 頁面上，選取 [ **Office 365**，然後選取 [**下一步**。

4. 在 [**基礎**] 頁面上，輸入名稱] 群組中，並 （選擇性） 的描述。 選取 **[下一步]**。
    
5. 在 [**編輯設定**] 頁面上，輸入群組的唯一的電子郵件地址，選擇 [隱私權選項和是否想要新增 Microsoft Teams，然後選取 [**下一步**。
    
6. 在 [**擁有者**選擇一或多個人員會指定用來管理群組的名稱。 任何群組擁有者都能從群組收件匣刪除電子郵件。 其他成員無法從群組收件匣刪除電子郵件。 選取 **[下一步]**。
    
7. 檢閱您的設定進行任何變更之後，選取 [**建立群組**]。

8. 選取 [**關閉**]。
    
## <a name="add-members-to-the-group"></a>將成員新增至群組

一旦建立群組之後，您可以新增成員並設定其他設定。

::: moniker range="o365-worldwide"
  
使用者可以[自行加入或要求核准](https://support.office.com/article/Join-a-group-in-Outlook-2e59e19c-b872-44c8-ae84-0acc4b79c45d)，或您可以將他們新增現在。


1. 在系統管理中心中，重新整理頁面，因此會顯示新的群組中，選取**群組** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2052855" target="_blank">群組</a>]，然後選取 [您想要將成員新增至群組的名稱。
    
2. 在 [**成員**] 索引標籤中，選取 [**檢視所有及管理成員**。

3. 選取 [新增成員]****。
    
4. 選取您想要新增的使用者，然後選取 [**儲存**。
    
5. 選取三次 [**關閉**]。 
    
群組會出現在 Outlook 中指派給它的成員。

::: moniker-end

## <a name="who-can-delete-email-from-the-group-inbox"></a>哪些人可以從群組收件匣刪除電子郵件？

群組擁有者可以從群組收件匣中刪除任何電子郵件，無論他們是否為初始作者。
  
成員可以刪除電子郵件交談從群組收件匣如果他們啟動，而且僅使用網頁型 Outlook （以滑鼠右鍵按一下 [電子郵件，然後選擇 [**刪除**]）。 Outlook App (Outlook 2016) 不支援這項操作。
  
從群組信箱刪除電子郵件時，它是不會從任何群組成員的個人信箱中刪除。
