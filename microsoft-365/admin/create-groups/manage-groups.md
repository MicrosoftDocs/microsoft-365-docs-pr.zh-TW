# <a name="manage-an-office-365-group-in-the-microsoft-365-admin-center"></a>管理 Microsoft 365 系統管理中心中的 Office 365 群組

您必須[建立 Office 365 群組](create-groups.md)並新增群組成員之後，您可以設定您的群組。 您可以編輯群組名稱或描述、 擁有者或成員、 管理並指定外部寄件者可以電子郵件至群組，以及是否要將群組交談的複本傳送到成員。 

## <a name="edit-a-group-name"></a>編輯群組名稱

::: moniker range="o365-worldwide"

1. 在系統管理中心，移至 [**群組** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2052855" target="_blank">群組</a>] 頁面上，然後選取您想要變更群組] 核取方塊。

2. 選取 [**編輯名稱**。

3. 在 [**編輯詳細資料**] 窗格中，輸入 [] 群組中的新名稱，然後選取 [**儲存**。

::: moniker-end

::: moniker range="o365-germany"

1. 在系統管理中心，移至 [**群組** \> **群組**] 頁面上，然後選取您想要變更群組的名稱。

2. 選取 [**名稱**] 旁邊的 [**編輯**]。

3. 在 [**編輯詳細資料**] 窗格中，輸入 [] 群組中的新名稱，然後選取 [**儲存**。

::: moniker-end

::: moniker range="o365-21vianet"

1. 在系統管理中心，移至 [**群組** \> **群組**] 頁面上，然後選取您想要變更群組的名稱。

2. 選取 [**名稱**] 旁邊的 [**編輯**]。

3. 在 [**編輯詳細資料**] 窗格中，輸入 [] 群組中的新名稱，然後選取 [**儲存**。

::: moniker-end

::: moniker range="o365-worldwide"

2. 選取您想要管理以開啟 [設定] 窗格的群組名稱。

3. 在 [**成員**] 索引標籤的 [**擁有者**，選取 [**檢視所有及管理擁有者**，並執行下列其中一項：

    - 若要新增擁有者，請選取 [**新增擁有者**，並搜尋或選取擁有者，，然後選取**儲存**。 
    - 若要移除擁有者，搜尋或選取 [群組擁有者、 選取的人員名稱旁邊的**X** ，然後選取**儲存**。 

::: moniker-end

2. 旁**擁有者**，選取 [**編輯**]，然後執行下列其中一項：

    - 若要新增擁有者，請選取 [**新增擁有者**，並搜尋或選取擁有者，，然後選取**儲存**。 
    - 若要移除擁有者，搜尋或選取 [群組擁有者、 選取的人員名稱旁邊的**X** ，然後選取**儲存**。

## <a name="send-copies-of-conversations-to-group-members-inboxes"></a>傳送至群組成員的收件匣的交談一份
  
當您使用管理中心建立群組時，預設使用者不會收到群組的電子郵件和會議邀請傳送給收件匣的複本。 他們將需要移至群組，查看交談和會議。 您可以變更此設定在系統管理中心。

當您啟用此設定時，群組成員會收到一份群組電子郵件和會議邀請傳送至其 Outlook 收件匣。 群組成員可以閱讀和刪除此電子郵件複本，且不會影響任何其他人。 電子郵件複本仍會存在群組收件匣中。

群組成員可以選擇退出接收這些電子郵件選擇停止在 Outlook 中的群組。

::: moniker range="o365-worldwide"

1. 在系統管理中心，移至 [**群組** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2052855" target="_blank">群組</a>] 頁面上，，然後選取 [您想要變更群組的名稱。 

2. 如果您想要讓成員在他們自己的收件匣中收到份群組郵件和行事曆項目，請在 [**設定**] 索引標籤上選取 [**傳送群組交談和活動群組成員的複本**]。

3. 選取 **[儲存]**。

::: moniker-end

::: moniker range="o365-germany"

1. 在系統管理中心，移至 [**群組** \> **群組**] 頁面上，，然後選取 [您想要變更的群組。

2. 選取 [**名稱**] 旁邊的 [**編輯**]。

3. 如果您想要讓成員在他們自己的收件匣中收到份群組郵件和行事曆項目，請開啟至**上**的 [**傳送群組交談和活動將群組成員的收件匣的複本**。

4. 選取 **[儲存]**。

::: moniker-end

::: moniker range="o365-21vianet"

1. 在系統管理中心，移至 [**群組** \> **群組**] 頁面上，，然後選取 [您想要變更的群組。

2. 選取 [**名稱**] 旁邊的 [**編輯**]。

3. 如果您想要讓成員在他們自己的收件匣中收到份群組郵件和行事曆項目，請開啟至**上**的 [**傳送群組交談和活動將群組成員的收件匣的複本**。

4. 選取 **[儲存]**。

::: moniker-end

## <a name="let-people-outside-the-organization-email-the-group"></a>讓外部組織電子郵件至群組的人員

如果您想要有公司電子郵件地址，例如 info@contoso.com 很好時，此選項。
 
::: moniker range="o365-worldwide"

1. 重新整理您的系統管理中心頁面上，讓新的群組會出現。

2. 在系統管理中心的 [群組] 清單中，選取您要變更群組的名稱，然後選取 [**設定**] 索引標籤上的 [**允許外部寄件者電子郵件至此群組**。
    
4. 選取 **[儲存]**。

::: moniker-end

::: moniker range="o365-germany"

3. 設定將**讓外部組織電子郵件至群組的人員**開關切換為**上**。
    
4. 選取 **[儲存]**。

::: moniker-end
