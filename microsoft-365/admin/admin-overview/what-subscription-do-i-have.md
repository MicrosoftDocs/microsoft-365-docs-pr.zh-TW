# <a name="what-subscription-do-i-have"></a>我有何種訂閱？

如果您是系統管理員，您可以驗證您的組織已移至 [系統管理中心的訂閱。
  
 **不是系統管理員？** 請參閱[我擁有哪些 Office 365 商務版產品或授權？](https://support.office.com/article/f8ab5e25-bf3f-4a47-b264-174b1ee925fd.aspx)

::: moniker range="o365-worldwide"

> [!NOTE]
> 如果您使用的不是新的 Microsoft 365 系統管理中心，您可以選取位於首頁頂端的 **[試用新的系統管理中心] **開關將它開啟。

1. 在系統管理中心，移至 [帳單]**** \> [產品與服務]<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank"></a> 頁面。

2. 您的訂閱會顯示在**產品 & 服務**] 頁面上，搭配訂閱名稱，此訂閱，且其狀態的相關資訊。 若要查看緊縮的檢視，請選取**表格檢視**。

::: moniker-end
  
::: moniker range="o365-germany"

1. 在 [系統管理中心中，移至**帳單** \><a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">訂閱</a>] 頁面。  

2. 如果您有只有一個訂閱，它會顯示在 [**訂閱**] 頁面上，連同訂閱名稱、 訂閱，以及其狀態的相關資訊。 在下列的螢幕擷取畫面中，您可以看到我擁有 Office 365 商務版訂閱。

    ![The Subscriptions page that shows which subscription you have as well as its status.](../../media/4d51dfcc-e9f3-4414-964a-6ef182f49eba.png)
  
3. 如果您有多個訂閱，您會看到訂閱詳細資訊旁有一欄，列出已購買的訂閱和所有試用版訂閱。根據預設，系統會自動選取清單頂端列出的訂閱，並顯示該訂閱的詳細資訊。

    如果您有多個訂閱，選擇您想要查看詳細的資訊。 系統會使用該訂閱的相關資訊來更新訂閱卡。

    ![顯示多個訂閱依據其狀態的清單在系統管理中心的 [訂閱] 頁面。](../../media/548ab8e9-bf9c-46d1-8c7c-ef5b631f3faa.png)
  
    > [!NOTE]
    > 如果您有訂閱已到期或停用，系統就會依據訂閱的目前狀態列出。

::: moniker-end
