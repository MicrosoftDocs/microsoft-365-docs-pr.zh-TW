# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>取得 Microsoft 365 流量分析的最新版本

範本應用程式可能會以新的資料或新的視覺化項每年進行多次刷新。 您的現有實例仍可運作，但是如果您想要取得最新的版本，則必須建立新的實例，而且必須將任何自訂套用至新的實例。 請參閱[啟用 Microsoft 365 流量分析](enable-usage-analytics.md)。
