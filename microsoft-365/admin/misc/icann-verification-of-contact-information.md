# <a name="icann-verification-of-contact-information"></a>連絡人資訊的 ICANN 驗證

當您購買網域，或變更網域的連絡人資訊時，註冊機構需要若要驗證您遵守 numbers，ICANN 法規的連絡資訊。

進行驗證，電子郵件傳送至註冊者的地址。

 **若要確認您的連絡人資訊，請驗證電子郵件中選取的連結。** 如果沒有驗證網域擁有者的電子郵件地址的網域就會被擱置。 然後，移至系統管理中心中的[網域] 頁面上](https://admin.microsoft.com/adminportal/home?ref=Domains)，並檢查您網域的狀態。
