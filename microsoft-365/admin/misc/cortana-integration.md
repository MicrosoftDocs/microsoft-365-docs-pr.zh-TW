# <a name="cortana-in-office-365"></a>Office 365 中的 Cortana

Cortana，您的個人生產能力 assistant 可提供 AI 的體驗，以節省時間，並重點關注最重要的問題。 Cortana 可協助您的使用者在工作和生命週期中提高其整個公司的個人生產力。 當您使用有效的工作或學校帳戶登入時，使用者可以在 Office 365 生產力工具內取得雲端式 Cortana 協助服務。 Office 365 中的 Cortana 符合 Office 365 的企業級隱私權、安全性和合規性承諾。

做為個人的生產能力小幫手，Cortana 是專門設計用來透過像電子郵件、檔案、聊天等的方式，提供安全且安全地處理及365原因的功能，以節省時間、提高效率，並增強使用者的生產力。 

向前發展，我們的重點是在企業生產力上的小娜。

- 與其他 Office 365 服務一致，Office 365 中的 Cortana 符合[線上服務條款（OST）](https://www.microsoft.com/en-us/licensing/product-licensing/products)中所反映的企業級隱私權、安全性和合規性承諾。

- 新的雲端式 Cortana 協助服務（例如簡介電子郵件和播放我的電子郵件）將會在 Office 365 中使用 Cortana 來啟用，並且完全符合這些承諾。 這些功能目前適用于在 US 地理位置具有信箱的使用者，以及適用于含美國信箱的跨國公司員工的英文（美國）市場。

- 使用者可以透過用戶端應用程式（如 Outlook Mobile）上的「Office 365 服務」（如 Outlook Mobile）中所述的小娜，依照個別授權條款。

- 現有的消費者經驗（包括 Windows 10 中的 Cortana （版本1909及更舊版本），以及 iOS 和 Android 上的 Cortana app，都是由[Microsoft 服務合約](https://www.microsoft.com/en-us/licensing/product-licensing/products)和[Microsoft 隱私權聲明](https://privacy.microsoft.com/en-us/privacystatement)所控制。 這些術語也會在使用使用者認證登入時，于使用者提供給使用者的 Office 365 服務中管理 Cortana。

## <a name="what-data-is-processed-by-cortana-in-office-365"></a>Cortana 在 Office 365 中所處理的資料為何？

Office 365 中的 Cortana 處理來自使用者的查詢、需要用來完成使用者要求的 Office 資料，以及 Microsoft 系統執行服務所產生的其他遙測。 Office 365 中的小娜所收集的資料包含使用者的口述查詢（即從語音辨識轉譯）的文字標記法。 這項文字資料是客戶資料，而且會依照[線上服務條款](https://www.microsoft.com/en-us/licensing/product-licensing/products)加以管理。 它只用于開發及改善與[線上服務條款](https://www.microsoft.com/en-us/licensing/product-licensing/products)一致的機器教學模型。

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-in-office-365"></a>Office 365 中的 Cortana 客戶資料的管理模型為何？

與其他 Microsoft 365 和 Office 365 服務一致，Office 365 中的 Cortana 會受到保護，並受限於[線上服務條款](https://www.microsoft.com/en-us/licensing/product-licensing/products)。 這包括保護客戶資料以防意外遺失、篡改、未經授權的披露或存取，或非法銷毀的承諾集合。 客戶資料也服從嚴格訪問限制。 Microsoft 只會使用客戶資料來供應商定的服務，以及與這些服務相容的目的。 如需詳細資訊，請參閱下表。

## <a name="cortana-in-office-365-features"></a>Office 365 中的小娜功能

### <a name="play-my-emails"></a>播放我的電子郵件

播放我的電子郵件（透過 Outlook mobile 連線）是一種語音導向的免提體驗，可讓使用者透過其電話、耳機或連接的音訊裝置上的揚聲器收聽其收件匣。 使用者可以要求 Cortana 高聲閱讀他們最近的電子郵件，並要求 Cortana 採取諸如旗標、封存、刪除和略過訊息等動作。 在 commuting 或前往時，這項功能特別有助於追趕電子郵件。 當使用者在 [播放我的電子郵件] 中與 Cortana 交談時，語音音訊要求會直接進入 Office 365 中的 Cortana。 在 Office 365 雲端中處理使用者電子郵件語音讀出的文字。 在此程式期間，不會在使用者的行動裝置上處理任何 Office 365 資料，也不會儲存任何新的電子郵件資料。 在線上服務條款中，會依照[線上服務條款](https://www.microsoft.com/en-us/licensing/product-licensing/products)中的資料保護字詞，保留口述命令的記錄（例如「標為已讀取」、「下一步」和「旗標」）。

當電子郵件受到保護時，Cortana 會呼叫出來，並在閱讀郵件之前停頓片刻，讓使用者有足夠的時間來暫停播放或略過下一封郵件。 類似于私人電話，當使用者可能會 overheard 機密資訊的位置時，使用者應小心謹慎。 在這些情況下，當您在 Outlook mobile 中使用「播放我的電子郵件」時，建議您組織的員工在適當的環境中有磨損耳機。

### <a name="opt-out-of-play-my-emails"></a>選擇不播放我的電子郵件

使用下列步驟，個人可以選擇 [播放我的電子郵件]。

1. 開啟 Outlook mobile。

2. 移至 [**設定**]。
  
3. 選取 [**播放我的電子郵件**]。

4. 在您要停用的帳戶上，將切換移至 [關閉]。

### <a name="briefing-email"></a>簡報電子郵件

Cortana 會傳送個人化簡報電子郵件，其中包含您用來將其標記為 [已**完成**] 或 [排程焦點] 以完成工作的任務和承諾。 它也包含您一天的會議和相關檔摘要。 Cortana 會從使用者的 Office 365 訊息中提取資訊，並將其儲存在其 Office 365 信箱中，直到合併到簡報電子郵件為止。 在任何時候，您的 Office 365 信箱之外可以存取個人資料。

### <a name="how-to-opt-out-of-briefing-email"></a>如何選擇未用簡報電子郵件

系統管理員可以使用 Microsoft 365 系統管理中心的[切換](https://docs.microsoft.com/briefing/be-admin)設定其組織的簡報。 個人可以在郵件的頁尾選取 [取消訂閱]，以選擇不使用 Cortana 的簡報電子郵件。

我們將繼續引進 Office 365 功能中的更多 Cortana，以協助增加組織的生產力。

深入瞭解[Microsoft 規範服務](https://docs.microsoft.com/microsoft-365/compliance/offering-home)。

## <a name="how-is-cortana-in-office-365-different-from-other-cortana-features-i-may-have-previously-experienced"></a>Office 365 中的 Cortana 與我先前可能經歷的其他 Cortana 功能有何不同？

以下是兩種方式來考慮 Cortana 在企業中的運作方式：

**Office 365 中使用 cortana 的組織的新體驗**： office 365 服務中的 cortana 是專門設計用來符合組織的安全性和合規性需求。

1. 這是一項新服務，在此檔中會加以討論。

2. 針對受線上服務條款制約的服務，Microsoft 是資料處理器： Microsoft 只會收集客戶所要求的線上服務，並使用客戶資料，以提供客戶所要求的線上服務。 在歐盟的一般資料保護規定（GDPR）下，客戶是其資料的資料控制器。 請參閱[線上服務條款](https://www.microsoft.com/en-us/licensing/product-licensing/products)，並[為我們的商業雲端客戶引進更多隱私權透明性](https://blogs.microsoft.com/eupolicy/2019/11/18/introducing-privacy-transparency-commercial-cloud-customers/)。

3. 舉例來說，「我的電子郵件」是一種 Cortana 服務，您的使用者可以透過 Outlook 連線至 iOS，並利用 Office 365 中的 Cortana。

4. 系統管理員在使用 Microsoft 365 應用程式的企業版應用程式時，系統管理員一定會具有 Cortana 選用的連接體驗，類似選用的連線體驗。

**現有的消費者服務**： Cortana 選用聯機服務主要是針對使用者經驗所設計，目前是在 Windows 10 （版本1909和更舊版本）和在 IOS 和 Android 上的 Cortana 應用程式中傳遞。

1. 這些經驗可以啟用天氣、新聞及流量等功能。

2. 租使用者系統管理員可以控制 Windows 10 （版本1909和更舊版本）中的 Cortana 和 iOS 和 Android 上的 Cortana 應用程式是否可以允許 Cortana 連線到 Office 365 租使用者資料。

關閉 Cortana 對您組織的 Microsoft 主控資料的存取

1. 在 Microsoft 365 系統管理中心中，選取 [**設定** > **服務**]，然後選取 [ **Cortana**]。

2. 取消選取 [**允許這些 Cortana 選用的 Cortana 選用的連線經驗] 核取方塊，以使用您組織的 Microsoft 主控資料**來停用 Cortana 連線體驗。

3. 選取 [儲存變更]****。

針對[Microsoft 服務合約](https://go.microsoft.com/fwlink/p/?LinkId=2109174)和 [microsoft 隱私權聲明](https://privacy.microsoft.com/en-us/privacystatement)所控制的服務，microsoft 是資料控制器。 就像資料控制者一樣，Microsoft 會使用資料，依照[Microsoft 隱私權聲明](https://privacy.microsoft.com/en-us/privacystatement)來改進產品和服務。
