# <a name="getting-a-mailbox-not-found-error-in-outlook-on-the-web"></a><span data-ttu-id="a5ad4-103">是否收到在 Outlook 網頁版中找不到信箱的錯誤？</span><span class="sxs-lookup"><span data-stu-id="a5ad4-103">Getting a mailbox not found error in Outlook on the web?</span></span>

<span data-ttu-id="a5ad4-104">如果您正在使用網頁版 Outlook 並收到 **找不到信箱** 錯誤，表示您用於連結至網頁版 Outlook 的帳戶沒有 Exchange Online 的授權，因此，沒有與此帳戶相關聯的信箱。</span><span class="sxs-lookup"><span data-stu-id="a5ad4-104">If you're using Outlook on the web and you get a  **Mailbox couldn't be found for**  error, the account that you used to connect to Outlook on the web doesn't have an Exchange Online license and therefore, no mailbox is associated with the account.</span></span> 

## <a name="assign-a-license-to-your-account"></a><span data-ttu-id="a5ad4-105">指派授權至您的帳戶</span><span class="sxs-lookup"><span data-stu-id="a5ad4-105">Assign a license to your account</span></span>

<span data-ttu-id="a5ad4-106">您的系統管理員可以指派授權至您的帳戶，依照下列步驟進行：</span><span class="sxs-lookup"><span data-stu-id="a5ad4-106">Your admin can assign a license to your account by following these steps:</span></span>

1. <span data-ttu-id="a5ad4-107">開啟 [Microsoft 365 系統管理中心](https://portal.office.com/adminportal/home#/homepage) 並移至 **使用者** 章節底下的 **作用中的使用者**，選取看到此錯誤的使用者。</span><span class="sxs-lookup"><span data-stu-id="a5ad4-107">Open the  [Microsoft 365 admin center](https://portal.office.com/adminportal/home#/homepage)  and go to  **Active users**  under the  **Users**  section, and select the user who is seeing the error.</span></span>
1. <span data-ttu-id="a5ad4-108">在開啟的使用者頁面中，移至 **授權與應用程式** 章節，選取正確的 **位置** 值，並指派包含 Exchange Online 的授權 (展開授權以查看詳細資料)。</span><span class="sxs-lookup"><span data-stu-id="a5ad4-108">In the user page that opens, go to the  **Licenses and Apps**  section, select the appropriate  **Location**  value, and assign a license that contains Exchange Online (expand the license to see its details).</span></span> 
1. <span data-ttu-id="a5ad4-109">完成後，按一下 [儲存變更 **]**。</span><span class="sxs-lookup"><span data-stu-id="a5ad4-109">When you're finished, click  **Save changes**.</span></span>
