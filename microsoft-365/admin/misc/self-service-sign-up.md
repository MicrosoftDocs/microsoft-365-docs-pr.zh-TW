# <a name="using-self-service-sign-up-in-your-organization"></a>組織中使用自助登

我們隨時傾聽到您的意見反應，並容易了組織中的使用者註冊 microsoft 線上服務。 我們新呼叫這個註冊程序 」 自助註冊 」，因為您的使用者可以註冊用來支付訂閱，或使用可用的服務，不會要求您在其代理上採取動作的服務。
  
## <a name="how-self-service-sign-up-works"></a>運作方式自助註冊

下列範例說明如何自行註冊的學校的運作方式。 相同的程序適用於任何已啟用其租用戶中的自助程式的組織。
  
1. 學生和教師成員必須表示它們是與您的機構相關聯的學校電子郵件地址。 例如，電子郵件地址 jakob@uw.edu 可能會指出從華盛頓州大專院校版的學生。
    
2. 學生和教師移至[我們的網站](https://go.microsoft.com/fwlink/p/?LinkId=536628)，並使用他們的電子郵件地址來註冊您的組織所提供服務，這類 Office 365 專業增強版。 他們也可以註冊為我們提供其他可用服務。
    
3. 我們驗證其電子郵件地址，然後他們可以開始使用 Office 365、 Power BI 或其他服務立即。
    
4. 為企業系統管理員，您會看到誰已註冊訂閱在系統管理中心中檢視 [**訂閱**] 頁面。 如此一來您可以看到您的租用戶中的服務或無法辨識的新授權時。
