# <a name="signing-up-for-office-365-with-a-personal-email-address"></a>使用個人電子郵件地址註冊 Office 365

## <a name="when-you-skip-adding-a-custom-domain"></a>略過新增自訂網域

  
當您使用您的個人電子郵件地址註冊 Office 365，並略過新增[自訂網域](../get-help-with-domains/what-is-a-domain.md)後，就會限制您對特定功能的存取。 
  
如果您選擇現在新增自訂網域，您可以使用下列所有的高級功能：
  
|**功能**|**Office 365，含個人電子郵件**|**Office 365 與自訂網域**|
|:-----|:-----|:-----|
|**OneDrive <sup>1，2</sup>**| [個人 OneDrive](https://onedrive.live.com/about/en-us/plans/)| [商務用 OneDrive](https://onedrive.live.com/about/en-us/business/) |
|**Office 應用程式： Word、Excel、PowerPoint、OneNote、Outlook、Access （僅限 PC）**| 是 | 是 
|**商務應用程式<sup>3</sup> ： Microsoft 連線、開具發票、清單、預約、MileIQ 和 Outlook 客戶經理**| 否 | 是 
|**存取 Microsoft 365 admin center**| 限制存取（帳單、支援和網域設定） | 是
|**新增使用者**| 否 | 是
|**Office 365 服務： Exchange Online、商務用 Skype、Microsoft 團隊 SharePoint 線上**| 否 | 是
|**安全性和規範工具**| 否 | 是
