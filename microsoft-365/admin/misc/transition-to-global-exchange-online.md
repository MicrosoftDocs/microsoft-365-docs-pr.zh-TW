# <a name="update-your-mx-records-to-transition-to-the-global-exchange-online-service"></a>更新您的 MX 記錄以轉換為全域 Exchange Online 服務

1. 登入 [Microsoft 365 系統管理中心](https://admin.microsoft.com)，然後移至 **設定**  >  **網域**

2. 每個網域的狀態將會顯示在右側。 如果您組織的網域指向 [Microsoft Cloud 德國 Exchange Online]，您將需要更新您的 MX 記錄。

3. 按一下網域，然後按一下 [偵測 **到 DNS 錯誤]，按一下這裡以進行查看**。

4. 此頁面可讓您向您說明如何修正 MX 記錄。 如果您的網域註冊機構使用[網域連線](../setup/add-domain.md#registrars-with-domain-connect)，您可以按一下 [修正我的記錄] 上方的。 否則，您可以依照嚮導中的連結，進行註冊機的逐步 **指示** 。
