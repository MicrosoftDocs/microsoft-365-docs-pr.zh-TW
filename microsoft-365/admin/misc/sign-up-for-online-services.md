# <a name="your-domain-may-be-in-use-if-someone-else-signed-up-with-it"></a>您的網域可能正在使用中（如果其他人已簽署）

使用公司或學校電子郵件的使用者，不需透過 IT 部門就可以[註冊一些 Microsoft 線上服務](self-service-sign-up.md)。 例如，Office 365、Power BI 和 Rights Management Services 等服務。 藉由提供公司或學校電子郵件地址完成自助式註冊之後，使用者即可存取該服務所包含的功能，但他們不能管理服務 (如新增使用者、新增授權、管理網域等)。 
  
合格的管理員可以[接管帳戶的控制權](become-the-admin.md)來取得管理環境的能力。 其做法是確認他們擁有註冊時所用之公司或學校電子郵件地址所含的網域，同時在某些情況下，購買該服務的授權。
  
## <a name="how-does-the-self-service-signup-work"></a>自助式註冊如何運作？

 當某人使用與公司或學校相關的電子郵件地址 (例如 sara@contoso.com) 註冊時，該電子郵件地址的網域部分 (在這個範例中為 contoso.com) 會用來作為他們的 Office 365 網域名稱。 
  
其他所有以同一網域之電子郵件地址註冊的使用者 (假設是 rob@contoso.com) 都會新增到同一個帳戶。
  
在上述範例中，Sara 和 Rob 都可以使用他們註冊的服務，但是不能管理該項服務。 比如，他們無法購買更多授權、增加新的使用者，或對其他使用者強制執行原則。
