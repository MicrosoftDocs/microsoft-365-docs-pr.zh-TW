# <a name="microsoft-productivity-score-preview"></a>Microsoft 生產力評分（預覽）

Microsoft 365 提供生產力解決方案，可讓您的組織達到其業務目標。 生產力分數提供使用這些解決方案的洞察力，您可以轉換為工作的完成方式。它包含： 

- **Visibility**透過協助他們瞭解人員如何以其目前分數的方式處理員工經驗和技術經驗的類別，以及技術經驗（即將推出）。 
- **深入**瞭解，以找出提高體驗的機會。 
- 更新技能和系統的**動作**，讓每個人都能進行最佳的工作。 

分數和真知灼見會出現在兩個類別中–  

- **員工經驗：** 示範 Microsoft 365 如何協助建立高效且共同運作的員工，方法是量化人員如何在內容間進行共同作業、在任何位置進行共同作業、瞭解通訊樣式（即將推出），以及如何開發會議文化（即將推出） 

- **技術經驗**（即將推出）：協助您優化裝置體驗，例如主動修正常見的服務台問題，以及改善電腦的啟動時間和網路，以確保您的應用程式順利運作。  

## <a name="how-it-works"></a>運作方式

### <a name="areas-of-scoring"></a>計分區域 

在每個分數類別中，生產力分數可讓您跨區域的工作轉換獲得深入瞭解。 員工經驗中涵蓋的領域包括：

- 與內容共同作業  
- 隨處運作 
- 瞭解通訊樣式（即將推出） 
- 開發會議文化（即將推出） 

### <a name="scoring-user-actions"></a>計分使用者動作 

在每個地區內，我們會測量以資訊為基礎的主要動作，代表讓組織轉變為高生產力組織的運作方式。 針對每個區域，我們計算過去28天內執行這些動作的使用者百分比。 

「員工經驗」分數是策劃為橫跨所有地區的平均分數。 請注意，我們將不斷新增更多領域的員工經驗和技術經驗。 

### <a name="products-included-in-the-productivity-score"></a>生產排名中包含的產品 

生產力分數目前包括來自主要 Microsoft 365 工作負載 OneDrive、SharePoint、Word、Excel、PowerPoint、OneNote、Outlook、Yammer、小組、Skype 等的信號。 

您的分數會每日更新，並且反映過去28天（包括目前日期）完成的使用者動作。

## <a name="access-and-required-permissions"></a>存取權及必要許可權 

針對員工經驗，您必須訂閱與多位使用者搭配使用的 Microsoft 365 for business 方案。 

若要具有存取 Microsoft 生產力分數的許可權，您必須具備下列角色： 

- 全域系統管理員 
- Exchange 系統管理員 
- SharePoint 系統管理員 
- 商務用 Skype 系統管理員 
- Teams 系統管理員 
- 全域讀取者 
- 報告讀取者 

您可以選擇左側導覽中的 [**報表** > **生產力評分**]，從 Microsoft 365 系統管理員的首頁存取體驗。

## <a name="interpreting-productivity-score"></a>解讀生產力分數 

### <a name="learn-how-your-organization-works"></a>深入瞭解貴組織的運作方式 

他的生產力分數首頁會以百分比為基礎，提供目前的分數和歷程記錄，以及依據基準補充的各分評分類別中的區域的主要洞察力。 

![生產力排名首頁](../../media/pslanding.png)

1.**生產力分數**會以百分比和<numerator> / <denominator>格式列出，因此您也可以看到絕對點數（分子）及可能的最大點數。  
1. 此樞紐分析表可讓您選取要重點關注的分數類別。 在預覽中，您只能查看**員工經驗**，但選擇將很快展開以包含**技術經驗**。 
1. **對等基準**可讓您將目前的分數與像您一樣的組織進行比較。 基準量值的計算方式是一組類似組織內的平均量值。 此集合由具有類似啟用的使用者、地區、您擁有的授權類型、行業和 tenure 內部的 Microsoft 365 或 Office 365 中的組織所組成。 
1. [**得分類別**] 區段提供每個類別的基準成績明細。
1. **分數記錄**會顯示每個類別中的分數在過去6個月中的移動方式。
1. 對**內容**進行共同作業的主要洞察力會顯示此區域中的基準基準的相關量值，這些因素可用於評分。 選取 [**查看詳細**資料] 以查看 [區域詳細資料] 頁面。
1. 「**工作**中的主要洞察力」顯示此區域中的基準基準的相關量值，以貢獻分數。 選取 [**查看詳細**資料] 以查看 [區域詳細資料] 頁面。  

### <a name="insights-to-identify-opportunities-to-improve-experiences"></a>深入瞭解，以找出改善體驗的機會 

針對每個區域，選取主要洞察力中的 [**查看詳細**資料]，以查看顯示其他真知灼見的 [區域詳細資料] 頁面，相關的調研和動作，以提升您的分數。 .

所有區域詳細資料頁面均遵循下列結構： 

- 左上：保留首頁的主要洞察力。
- 左下方：支援具有上下文動作的真知灼見。  
- 許可權：學術研究示範區域與生產力之間的關係。

此外，所有的真知灼見都包含下列元素： 

- 標頭：針對每項洞察力追蹤的重要提示或結果。
- 摘要：進一步說明深入瞭解瞭解組織的生產力為何重要。 
- 資料視覺效果：以視覺方式衡量組織在深入瞭解中的測度的位置。
- 動作：內容指令，可協助每個洞察的更多相關結果，進而提升您的分數。

### <a name="area-detail-page--collaborating-on-content"></a>區域詳細資料頁面-與內容共同合作 

![在內容頁面上 Collaborationg。](../../media/pscollab.png)

在內容頁面上進行共同作業，包含「生產力分數」首頁的主要洞察力。 其包含下列元件：

1. 標頭：量化在內容上進行共同作業的使用者百分比（重要結果）。
1. 本文：更多有關如何增加共同作業帶來更高生產力的詳細資訊。
1. 視覺效果：這會透過基準的方式，對此區域的分數貢獻的每個元素進行量化。

    - **讀者**：在啟用 OneDrive 和 SharePoint 啟用使用者的基礎內，量化，在雲端（OneDrive 和 SharePoint）上存取或下載檔案的使用者。
    - **作者**：量化使用者修改、上傳、同步處理、存回或移動雲端（OneDrive 和 SharePoint）上的檔案，以 OneDrive 和 SharePoint 啟用的使用者在雲中的基礎上進行。
    - **合作者**：以 OneDrive 和 SharePoint 啟用的使用者為基礎，在雲端（OneDrive 和 SharePoint）上量化使用者共同作業。 若有兩位使用者在建立或修改 Word、Excel、PowerPoint、OneNote 或 PDF 雲端檔之後，他們會在28天的時段內進行讀取或編輯，則為合作者。

1. **為什麼很重要**的是在雲端檔案上與生產力相關的研究連結共同作業的摘要。 **閱讀 Forrester 中的研究中心**，以深入瞭解如何 conducive 共同作業以提升生產力的相關資訊。
1. 支援透過**多個**檔案共同作業的洞察力量化組織中的使用者如何在不同的檔案間進行協同作業。 
1. 有關**共用雲端**檔案的支援資訊，請量化如果使用者是在雲端共用內容時，在內部和外部共用之間的區別。
1. 如果使用者透過電子郵件共用檔案所使用的附件與雲端檔案的連結，則對**電子郵件雲端**檔案的支援將會量化。

考慮進行共同作業的檔案類型為 Word/Excel/PowerPoint/OneNote/PDF。

1. **為什麼很重要的一點**是從多個平臺的「資訊檢索」連結使用的摘要，到較高的生產力。 請**閱讀 Forrester 中的「調研**」，以提供更多關於如何在如何 conducive 工作以提升生產力的內容的資訊。
1. 支援**電子郵件**的深入瞭解在 outlook 中，透過使用 outlook 的使用者，在您組織中使用 outlook 的使用者人數。
1. 協助工具的支援洞察力**可量化組織**中的使用者如何在團隊中的使用者所使用的不同平臺上使用團隊。
1. 支援深入瞭解**存取 Office 檔**的情況量化組織中的使用者如何在 Word、excel、PowerPoint 或 OneNote 上使用之使用者所使用的 Word、excel、PowerPoint 和 OneNote。

### <a name="actions-for-update-skills-and-systems"></a>更新技能和系統的動作 

為了推廣 actionability，每個支援的知識都是內置動作，可協助您轉變您的員工經驗。其中一些可能是設定變更或認知活動。 在 [建議動作] 面板選項中，按一下您可能會考慮的動作。 目前已啟用對**內容進行合作**的動作。

![在 Office 檔上共同作業頁面。](../../media/psactions1.png)

1. **View 動作**會移至一個窗格，其中會列出與內容共同作業相關的所有動作。
1. **鼓勵使用者共同**作業，以與共同撰寫和 @mentions 等主題共同作業的教學影片清單。 
1. **自訂外部共用**設定會進入一個頁面，其中包含外部共用簡介，描述使用者共用時所發生的情況，視其共用及與其共用的方式而定。  
1. 以**共用連結取代附件**的頁面，說明如何分享 OneDrive 連結，而不是透過電子郵件的附件，以進行更好的協同作業。

選取任何動作按鈕會開啟右側的 [**建議的動作**] 窗格，其具有每個動作摘要及檔頁面的連結。
