# <a name="microsoft-productivity-score-preview"></a>Microsoft 生產力分數 （預覽）

Microsoft 365 提供可讓您的組織符合其業務目標的生產力解決方案。 生產力分數提供見解給使用這些解決方案，您可以轉換的運作方式取得完成。它包含： 

- 協助他們了解人員的運作方式與分解其目前分數的表單中所有類別的員工體驗與 （即將推出） 的技術經驗的**可見性**。 
- **深入了解**以識別要啟用增強的體驗的機會。 
- 若要讓每個人都可以盡善盡美更新技能與系統的**動作**。 

分數和見解有跨兩個類別 –  

- **員工體驗：** 顯示如何建立生產力而人力量化人員進行共同作業的內容，協助 Microsoft 365 使用無所不在，了解通訊樣式 （即將推出），以及開發會議文化特性 （即將推出） 

- **技術體驗**（即將推出）： 協助您最佳化您的裝置體驗例如主動修正 helpdesk 常見問題，以及改善電腦啟動時間與您的網路，以確保您的應用程式正常運作。  

## <a name="how-it-works"></a>運作方式

### <a name="areas-of-scoring"></a>計分區域 

在每個分數類別，生產力分數可提供見解上工作轉換不同區域。 員工體驗中所涵蓋的區域如下：

- 內容共同作業  
- 任何位置工作 
- 了解通訊樣式 （即將推出） 
- 開發會議文化特性 （即將推出） 

### <a name="scoring-user-actions"></a>計分使用者動作 

內每個區域中，我們會測量 research 型關鍵動作代表，讓組織能夠轉換成高生產力組織的運作方式。 針對每個區域我們計算過去 28 天內執行這些動作的使用者百分比。 

員工體驗分數是策劃平均分數為跨所有區域。 請注意，我們會保留加入新的多個層面，員工體驗和技術經驗經過一段時間。 

### <a name="products-included-in-the-productivity-score"></a>包含在生產力分數的產品 

生產力分數目前包含訊號的索引鍵的 Microsoft 365 工作負載用 OneDrive 中，從 SharePoint、 Word、 Excel、 PowerPoint、 OneNote、 Outlook、 Yammer、 microsoft Teams、 Skype。 

您的分數會每日更新並反映使用者動作 （包括目前的日期） 的 28 天內完成。

## <a name="access-and-required-permissions"></a>存取及必要的權限 

為員工使用體驗，您必須對 Office 365 商務版方案或 Microsoft 365 商務版方案的訂閱與多個使用者。 

若要存取 Microsoft 生產力分數的權限，您必須具備下列角色： 

- 全域系統管理員 
- Exchange 系統管理員 
- SharePoint 系統管理員 
- 商務用 Skype 系統管理員 
- Teams 系統管理員 
- 全域讀取者 
- 報告讀取者 

您可以存取體驗 Microsoft 365 系統管理員從首頁選擇 [**報告** > **生產力分數**的左方導覽中。
