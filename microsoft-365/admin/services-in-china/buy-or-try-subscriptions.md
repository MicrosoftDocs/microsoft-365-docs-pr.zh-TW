# <a name="buy-or-try-subscriptions-for-office-365-operated-by-21vianet"></a>購買或試用由世紀運作之 Office 365 的訂閱

> [!NOTE]
>  本文適用于中國由世紀運作的 Office 365。

## <a name="buy-or-try-office-365"></a>購買或試用 Office 365

您可以立即購買訂閱，也可以嘗試使用最多30天的 Office 365。 如果您願意，您可以購買您的試用訂閱，並保留您在試用版中設定的網域、資料和設定。

1. 移至 [ [比較商務用 Office 365 方案](https://go.microsoft.com/fwlink/p/?linkid=393691&amp;clcid=0x409) ]，然後選取您要購買的計畫名稱。 然後，選取 [ **立即購買**]。

2. 只填滿 **一些詳細資料** 頁面以建立帳戶。

3. 依照指示設定30天試用或完成購買。

## <a name="buy-your-trial-subscription"></a>購買您的試用訂閱

1. 以工作或學校帳戶[登入 Office 365](https://go.microsoft.com/fwlink/p/?linkid=513813)。

2. 如果您還沒有在「開始」頁面上，請在左上角選取 [ **Office 365** ]。

    ![使用此按鈕瀏覽至 Office 365 開始頁面](../../media/2fc597ab-ae33-4e5a-aec1-e60e48beac62.png)

3. 在頁面右上角的導覽列下方，選取 [ **購買**]。

    ![用於購買 Office 365 試用版的按鈕](../../media/73fba4ad-6879-460b-8ef1-f2efb2ee4104.png)

4. 在 [ **購買訂閱** ] 頁面上，您會看到您可以購買的不同方案。 您已嘗試的計畫是由 **試用** 旗標標所識別。

    > [!IMPORTANT]
    > 如果您購買的是不同的試用方案方案，您必須將您的授權從您的試用方案重新指派給新的計畫 (，您的90天寬限期會在您的試用方案到期) 後，才能保留您的資料、帳戶及設定。 否則，您會遺失資料、帳戶及設定。

5. 選取您要購買的訂閱，然後選取 [ **立即購買**]。

6. 請依照步驟取出。

## <a name="add-subscriptions-or-add-ons-to-an-existing-office-365-account"></a>將訂閱或附加元件新增至現有的 Office 365 帳戶

1. 在系統 [管理中心](https://go.microsoft.com/fwlink/p/?linkid=850627)中，移至 [ **帳單** \> **購買服務** ] 頁面。

2. 選取您想要購買的服務 () ，選取 [ **購買**]，然後選取 [ **立即查看**]。

3. 依照指示完成購買。

## <a name="payment-options"></a>付款選項

您可以使用下列方式支付訂閱費用：

- 發票

- 使用支付寶或中國銀聯的線上付款

付款的憑證會以 Fapiaos 的形式提供。 您可以將發票要求提交給我們的 [發票系統](https://go.microsoft.com/fwlink/p/?LinkId=395314) ，大約在您支付後三個 (3) 天。 如需詳細資訊，請參閱 [Apply for a 發票 For Office 365 （由世紀運作](apply-for-a-fapiao.md)）。

> [!NOTE]
>  不接受國際信用卡。
