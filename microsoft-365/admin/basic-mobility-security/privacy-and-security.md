# <a name="privacy-and-security-in-basic-mobility-and-security"></a>基本行動性和安全性中的隱私權和安全性

基本行動性和安全性是由 Microsoft Intune 所提供技術支援的雲端式服務，可協助您管理組織中所使用的行動裝置和保護其安全。 啟動基本行動性和安全性之後，您可以建立行動裝置管理原則。 然後，您可以將這些原則部署至已由組織中授權的 Microsoft 365 使用者註冊的行動裝置。

Microsoft Intune 會將資訊傳送至 Microsoft 365，以瞭解每個受管理裝置的相容性狀態，然後您可以產生報告，顯示貴組織中的受管理裝置是否符合所設定的原則。 若要深入瞭解 Microsoft 對隱私權及安全性的承諾，請參閱 [Microsoft 信任中心](https://www.microsoft.com/trust-center)。
