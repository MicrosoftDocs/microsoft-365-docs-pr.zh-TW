# <a name="privacy-and-security-in-basic-mobility-and-security"></a>基本行動性和安全性中的隱私權和安全性

基本行動性和安全性是雲端式服務，其 Microsoft Intune 會協助您管理組織中的行動裝置及保護其安全。 啟動基本行動性和安全性之後，您可以建立行動裝置管理原則。 然後，您可以將這些原則部署至已由組織中授權 Microsoft 365 使用者所註冊的行動裝置。

Microsoft Intune 會將資訊傳送給每個受管理裝置之相容性狀態的 Microsoft 365，然後您可以產生報告，顯示組織中的受管理裝置是否符合已設定的原則。 若要深入瞭解 Microsoft 對隱私權及安全性的承諾，請參閱 [Microsoft 信任中心](https://www.microsoft.com/trust-center)。
