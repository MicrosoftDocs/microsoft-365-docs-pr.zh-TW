# <a name="step-5---wipe-and-block-a-former-employees-mobile-device"></a>步驟 5-清除並封鎖離職員工的行動裝置

如果您的離職員工有組織電話，您可以使用 Exchange 系統管理中心來清除並封鎖該裝置，以便從裝置移除所有組織資料，而且該資料就無法再連接至 Office 365。 如果您的組織使用基本行動性和安全性來管理行動裝置，您可以使用基本行動性和安全性來清除並封鎖這些裝置。

## <a name="wipe-mobile-device-using-the-exchange-admin-center"></a>使用 Exchange 系統管理中心來清除移動裝置

1. 移至 <a href="https://go.microsoft.com/fwlink/p/?linkid=2059104" target="_blank">Exchange 系統管理中心</a>。
2. 在 Exchange 系統管理中心中，**流覽至 [** 收件者] [ \> **信箱**]。
3. 選取使用者，然後在 [行動 **裝置**] 底下，選取 [ **查看詳細資料**]。
4. 在 [行動 **裝置詳細資料**] 頁面的 [行動 **裝置**] 下，選取行動裝置，選取 [**清除資料** ![ 清除裝置] ](../../media/1c113a36-53cb-4974-884f-3ecd9535506e.png) ，然後選取 [**封鎖**]。
5. 選取 ****[儲存]。
   > [!TIP]
   > 請務必移除或停用內部部署 Blackberry Enterprise 服務中的使用者。 您也應停用使用者的任何 Blackberry 裝置。 如需有關如何停用使用者的特定步驟，請參閱 Blackberry Business Cloud Services Administration Guide。
