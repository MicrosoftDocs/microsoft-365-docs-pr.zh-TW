# <a name="assign-admin-roles"></a>指派系統管理員角色

如果您是購買您的 Microsoft 商務版訂閱的人員，您必須是全域系統管理員。這表示您必須對您的訂閱和您的產品不受限制的控制可以存取大部分的資料。

如需詳細資訊，請參閱[關於系統管理員角色](about-admin-roles.md)。

當您新增新的使用者，如果您沒有指派這些系統管理員角色他們位於*使用者角色*，並不需要系統管理員權限給任何 Microsoft 系統管理中心。 但是，如果您需要完成工作的協助，您可以指派系統管理員角色給使用者。 例如，如果您需要某人協助重設密碼，您不應該指派給全域系統管理員角色，您應該將它們指派密碼系統管理員角色。 有太多的全域系統管理員，無限制存取您的資料和線上商務，會造成安全性風險。

## <a name="assign-admin-roles"></a>指派系統管理員角色 

::: moniker range="o365-worldwide"

您可以指派給角色的使用者 2 不同的方式：

- 您可以移至使用者的詳細資料和**管理角色**來指派角色給使用者。
- 或者，您可以移至 [**角色**及選取的角色，並再將多位使用者新增至它。

### <a name="assign-admin-roles-to-users-using-roles"></a>將系統管理員角色指派給使用者使用角色

1. 在系統管理中心中，前往 [**角色**>若要檢視所有適用於貴組織的系統管理員角色的**角色**。
2. 選取您想要指派至使用者的系統管理員角色。
3. 選取 [**已指派系統管理員** > **新增**。
4. 輸入使用者的**顯示名稱**或**使用者名稱**，並再從建議清單中選取使用者。
5. 新增多個使用者，直到完成為止。
6. 選取 [**儲存**]，然後將使用者新增至指派系統管理員的清單。

### <a name="assign-a-user-to-an-admin-role-from-active-users"></a>從作用中的使用者將使用者指派給系統管理員角色

1. 在系統管理中心，移至 [**使用者** > [作用中使用者](https://go.microsoft.com/fwlink/p/?linkid=834822)頁面。

2. 在**作用中使用者**] 頁面上，選取您想要變更其系統管理員角色的使用者。 在彈出式視窗窗格] 旁**的角色**，選取 [**管理角色**。

3. 選取您想要指派給使用者的系統管理員角色。 如果您沒有看到您要尋找的角色，選取 [**全部顯示**在清單底部。

::: moniker-end

::: moniker range="o365-germany"

1. 在系統管理中心中，移至 **[使用者]** > <a href="https://go.microsoft.com/fwlink/p/?linkid=847686" target="_blank">[作用中使用者]</a> 頁面。

2. 在**作用中使用者**] 頁面上，選取您想要變更其系統管理員角色的使用者。 在彈出式視窗窗格中旁 [**角色**] 中，, 選取 [**編輯**]。 

    如果您沒有看到 [**編輯**] 選項，您不具有編輯權限，而且不能指派系統管理員角色給其他人。 在您將角色指派為您的業務要求的全域系統管理員。 在小型企業，企業擁有者 （人員購買您的訂閱） 是全域系統管理員。在大型企業版，IT 部門中的主要人員都是全域系統管理員。

3. 選取**自訂的系統管理員**，以查看我們為您制訂的角色清單。 如需每個角色的說明，請參閱[關於系統管理員角色。](about-admin-roles.md)

::: moniker-end

## <a name="didnt-work-for-you"></a>無法運作嗎？

您可能沒有正確的權限，因此您不需要指派系統管理員角色給其他使用者的存取。 請委託其他系統管理員為您指派角色。
