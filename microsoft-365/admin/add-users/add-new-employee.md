# <a name="add-a-new-employee-to-office-365"></a>將新進員工加入 Office 365

本文可協助您將新的員工上架至商務用 Office 365。 我們假設您是系統管理員，且您已[完成 Office 365 設定](../setup/setup.md)，而您現在已有人加入您公司的新電腦。
  
如果您的新員工需要 Office 365，而且您使用的是[office 365 方案](https://products.office.com/business/compare-office-365-for-business-plans)，可讓您在電腦上安裝 Word 和 Excel 等 office 應用程式。 
  
 **不是系統管理員？** [深入瞭解 office 365](https://support.office.com/article/office-basics-video-training-396b8d9e-e118-42d0-8a0d-87d1f2f055fb) ，以協助商務和家用使用者搭配 office 365 設定。 
  
 **您計畫中沒有任何 Office 應用程式？** 請遵循下列步驟，但略過安裝應用程式的區段。 請改為使用[Office 的 Online 版本](https://support.office.com/article/91a4ec74-67fe-4a84-a268-f6bdf3da1804.aspx)。 
  
以下是快速綜述： 
  
|**步驟**|**為何這麼做？**|
|:-----|:-----|
|[步驟1：為員工建立 Office 365 帳戶](#step-1-create-an-office-365-account-for-the-employee) <br/> |每次新員工加入您的公司時，為他們建立帳戶，讓他們可以開始使用 Office 365。  <br/> |
|[步驟2：授與員工的使用者識別碼和密碼](#step-2-give-the-employee-their-user-id-and-password) <br/> |當您建立帳戶時，您會取得可傳遞給您員工的識別碼和密碼，讓他們可以登入。  <br/> |
|[步驟3：說明登入的位置](#step-3-explain-where-to-sign-in) <br/> |登入位置是[https://www.office.com](https://www.office.com) <br/> |
|[步驟4：協助您的員工快速入門](#step-4-help-your-employee-get-started) <br/> |讓您的員工知道如何使用組織中 OneDrive 或任何小組網站。  <br/> |
   
## <a name="step-1-create-an-office-365-account-for-the-employee"></a>步驟1：為員工建立 Office 365 帳戶


如需相關指示，請參閱[個別或大量新增使用者至 Office 365-系統管理](add-users.md)說明。 當您設定新的員工時，您可以選擇將登入詳細資料傳送到員工的個人帳戶。 如此一來，他們會收到來自 Microsoft Online 服務小組的電子郵件，告訴他們如何登入 Office 365。
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>步驟2：授與員工的使用者識別碼和密碼


除非您將其傳送至他們的個人電子郵件地址，否則請列印員工的 Office 365 登入名稱和密碼，然後將其交給他們。 或透過電話告知其資訊。
  
因為他們還沒有 Office 365 電子郵件的存取權，所以請勿將資訊傳送給該電子郵件地址。
  
## <a name="step-3-explain-where-to-sign-in"></a>步驟3：說明登入的位置 


就像 Facebook、Amazon 或 Gmail，您的員工會登入使用 Office 365。 請提供下列登錄資訊：
  
- 登入[https://www.office.com](https://www.office.com)。
    
- 選取 [登**入**]，然後輸入使用者識別碼和密碼。
    
## <a name="step-4-help-your-employee-get-started"></a>步驟4：協助您的員工快速入門


與他們共用[Office 365 的員工快速設定](https://support.office.com/article/employee-quick-setup-b9700090-ce64-4046-ab92-ce8488a7bc0f)，以登入、安裝軟體、設定電子郵件等等。
