## <a name="reset-an-office-365-business-password-for-someone-else"></a>重設其他人的 Office 365 商務版密碼

僅適用於使用 Office 365 商務版方案的人，則這些步驟。 若要執行這些動作，您需要使用您的 Microsoft 365 系統管理員帳戶登入。 [什麼是系統管理員帳戶？](../admin-overview/admin-overview.md)

 
::: moniker range="o365-worldwide"

1. 在系統管理中心中，移至 **[使用者]** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">[作用中使用者]</a> 頁面。

::: moniker-end

::: moniker range="o365-germany"

1. 在系統管理中心中，移至 **[使用者]** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847686" target="_blank">[作用中使用者]</a> 頁面。

::: moniker-end

::: moniker range="o365-21vianet"

1. 在系統管理中心中，移至 **[使用者]** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850628" target="_blank">[作用中使用者]</a> 頁面。

::: moniker-end

2. 在 [**作用中使用者**] 頁面上選取的使用者，然後選取 [**重設密碼**。
    
3. 自動產生新密碼的使用者或建立一個用於，，然後選取 [**重設****重設密碼**頁面上，請遵循指示。  
    
4. 輸入使用者能讓他們收到新的密碼，並使用，以確保它們 got 待電子郵件地址。
 
  
## <a name="reset-my-office-365-tenant-admin-password"></a>重設我的 Office 365 租用戶系統管理員密碼

如果您忘記您的密碼，但您無法登入 Office 365 因為，例如，您的密碼會儲存在您的瀏覽器，請使用下列步驟： 
    
1. 在 Office 365 中，選取 [**設定** \> **Office 365** \> **個人資訊**。 
          
2. 請仔細檢查您的**連絡人詳細資料**和**備用電子郵件**正確。 如果沒有，現在變更它們。 
        
3. 登出 Office 365： 在右上角 （以上述影像、 **Diane**如下所示） 中選取您的姓名\>**登出**。 
        
4. 現在登入： 輸入您的使用者名稱\>**下一步** \> ，然後選取 [**忘記密碼**。 
    
5. 遵循精靈的指示，重設密碼的步驟。 若要確認您已適當人員來重設密碼，它就會使用替代的連絡資訊。 
    
如果您忘記您的密碼，而且無法登入： 
    
- 詢問另一個全域系統管理員在您的業務，為您重設您的密碼。
    
- 或者，您也可以[連絡 Microsoft 支援服務](https://support.office.com/article/contact-support-for-business-products-admin-help-32a17ca7-6fa0-4870-8a8d-e25ba4ccfd4b?ui=en-US&amp;rs=en-US&amp;ad=US#ID0EAADAAA=Phone_support_)。 
    
## <a name="reset-all-office-365-business-passwords-for-everyone-in-your-organization-at-the-same-time"></a>同時重設貴組織中所有人員的所有 Office 365 商務版密碼
<a name="bkmk_forgot"> </a>

這些步驟適用於有大量使用者的企業。如果您有數百或數千個使用者，請參閱下一節了解如何大量重設密碼。
  
1. 在系統管理中心中，移至 **[使用者]** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">[作用中使用者]</a> 頁面。
    
2. 選取 [**顯示名稱**旁邊選取 [所有人貴公司中的選項。 然後取消選取自己。 您無法在同一時間重設其他人的密碼重設自己的密碼。
    
3. 選取 [**重設密碼**。 

4. 依照 [**重設密碼**] 頁面上的指示，然後選取 [**重設**。  如果您選擇用於自動產生密碼，將顯示新的暫時密碼。   
    
5. 您可以在何處取得暫時密碼輸入電子郵件地址。 您必須通知他們暫時密碼是您的使用者。
