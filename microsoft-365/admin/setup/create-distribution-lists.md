# <a name="create-distribution-groups-in-the-microsoft-365-admin-center"></a>在 Microsoft 365 系統管理中心建立通訊群組
  
當您想要將電子郵件傳送給使用者群組，而不必輸入每個個別收件者的名稱時，通訊群組是由特定的討論主旨（例如「行銷」）所組織，或是共用常見工作，需要經常進行通訊的使用者。 它們也為您提供一種方式，讓您可以自動將電子郵件轉寄至多個電子郵件地址。
  
通訊群組有時稱為通訊群組清單。
  
> [!VIDEO https://www.microsoft.com/videoplayer/embed/e8747233-4f1c-4bf4-81cf-2a4ea9ccd80d?autoplay=false]
  
## <a name="create-a-distribution-group-list"></a>建立通訊群組（清單）

::: moniker range="o365-worldwide"

移至位於 <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a> 的系統管理中心。

::: moniker-end

::: moniker range="o365-germany"

移至位於 <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">https://portal.office.de/adminportal</a> 的系統管理中心。

::: moniker-end

::: moniker range="o365-21vianet"

移至位於 <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">https://portal.partner.microsoftonline.cn/adminportal</a> 的系統管理中心。

::: moniker-end

1. Select the app launcher icon and choose **Admin**.
    
    Can't find the app you're looking for? 從應用程式啟動器中，選取 [**所有應用程式**]，以查看您可以使用之應用程式的字母順序清單。 From there, you can search for a specific app. 
    
2. 選取左導覽窗格中的 [**群組** \> **群組**]，然後選取 [**新增群組**]。 
      
3. 在 [**選擇群組類型**] 欄位中，選擇 [**分配**]，然後選擇 **[下一步]**。
  
4. 輸入新群組的名稱、描述和電子郵件別名，並選擇是否要讓組織外部人員傳送電子郵件給群組。 
    
5. 按 [**新增**] 建立群組，然後查看群組，然後選擇 [**關閉**]。 
    
6. 若要將使用者新增至通訊群組，請參閱[新增使用者或連絡人至通訊群組](../email/add-user-or-contact-to-distribution-list.md)。
    
若要瞭解如何在 Outlook 中使用 outlook 2016 和 Outlook 網頁版中的群組，請參閱 Outlook 主題中的[使用連絡人群組（先前是通訊群組清單）](https://support.office.com/article/1c97fcb2-0ed4-41e6-b401-58f9d7d40e39.aspx) 。 
  
查看[通訊群組問題](https://docs.microsoft.com/office365/troubleshoot/groups/distribution-list-issues)以取得通訊群組問題的協助。
