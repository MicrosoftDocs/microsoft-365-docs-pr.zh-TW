# <a name="create-distribution-groups-in-the-microsoft-365-admin-center"></a>在 Microsoft 365 系統管理中心建立通訊群組
  
當您想要傳送電子郵件給一組人員，而不需要輸入每位個別收件者的名稱時，通訊群組 (通訊群組清單) 會按照特定討論主題 (例如「行銷」) 或共用需要經常溝通之常用工作的使用者整理。 它們也會提供一個方式，好讓系統自動將電子郵件轉會到多個電子郵件地址。

通訊群組有時也稱為通訊群組清單。
  
> [!VIDEO https://www.microsoft.com/videoplayer/embed/e8747233-4f1c-4bf4-81cf-2a4ea9ccd80d?autoplay=false]
  
## <a name="create-a-distribution-group-list"></a>建立通訊群組 (清單) 

::: moniker range="o365-worldwide"

移至位於 <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a> 的系統管理中心。

::: moniker-end

::: moniker range="o365-germany"

移至位於 <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">https://portal.office.de/adminportal</a> 的系統管理中心。

::: moniker-end

::: moniker range="o365-21vianet"

移至位於 <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">https://portal.partner.microsoftonline.cn/adminportal</a> 的系統管理中心。

::: moniker-end

1. Select the app launcher icon and choose **Admin**.
    
    Can't find the app you're looking for? 從應用程式啟動 **器選取所有** 應用程式，以查看按字母順序排列的可用應用程式清單。 From there, you can search for a specific app. 
    
2. 在 **左側** \> **導航** 窗格中選取群組群組，然後選取 **新增群組。** 
      
3. 在選擇 **群組類型欄位上****，選擇發佈**，然後選擇下一 **步**。
  
4. 輸入新群組的名稱、描述和電子郵件別名，並選擇是否要讓組織外部人員傳送電子郵件給群組。 
    
5. 按 **新增** 以建立群組，然後審查您的群組 **並選擇關閉。** 
    
6. 若要將使用者新增到通訊群組清單，請參閱新增使用者 [或連絡人至通訊群組](../email/add-user-or-contact-to-distribution-list.md)。
    
在 Outlook 主題中，瞭解如何在 Outlook 2016 和 Outlook 網頁版中的使用連絡人群組 ([通訊](https://support.microsoft.com/office/1c97fcb2-0ed4-41e6-b401-58f9d7d40e39)) 群組。 
  
查看通訊 [群組問題，](https://docs.microsoft.com/office365/troubleshoot/groups/distribution-list-issues) 以尋找通訊群組清單問題的協助。
