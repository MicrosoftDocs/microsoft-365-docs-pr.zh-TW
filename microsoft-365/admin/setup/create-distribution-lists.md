# <a name="create-distribution-groups-in-the-microsoft-365-admin-center"></a>在 Microsoft 365 系統管理中心中建立通訊群組
  
當您想要傳送電子郵件給群組的人員，而不必輸入每個個別收件者的名稱時使用，通訊群組被組織由特定的討論主題 （例如"Marketing") 或共用要求它們進行通訊的常見工作的使用者常見問題。 他們也會提供方法，讓您自動轉寄電子郵件傳送到多個電子郵件地址。
  
通訊群組有時也稱為通訊群組清單。
  
> [!VIDEO https://www.microsoft.com/videoplayer/embed/e8747233-4f1c-4bf4-81cf-2a4ea9ccd80d?autoplay=false]
  
## <a name="create-a-distribution-group-list"></a>建立通訊群組 （清單）

::: moniker range="o365-worldwide"

移至位於 <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a> 的系統管理中心。

::: moniker-end

::: moniker range="o365-germany"

移至位於 <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">https://portal.office.de/adminportal</a> 的系統管理中心。

::: moniker-end

::: moniker range="o365-21vianet"

移至位於 <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">https://portal.partner.microsoftonline.cn/adminportal</a> 的系統管理中心。

::: moniker-end

1. Select the app launcher icon and choose **Admin**.
    
    找不到您要的 App 嗎？請從應用程式啟動器選取 [所有 App]**** 以查看依字母順序排序的可用 Office 365 App 清單。您可從該處搜尋特定的 App。 
    
2. 選取**群組**\>左側的導覽窗格中，，然後選取 [**新增群組**中的**群組**。 
      
3. 在 [**選擇群組類型**] 欄位中，選擇 [**通訊群組**，，然後選擇 [**下一步**。
  
4. 輸入新的群組，名稱、 描述及電子郵件別名，然後選擇您是否要人員，組織外部的傳送電子郵件給群組。 
    
5. 按 [**新增**] 來建立群組，然後檢閱您的群組，然後選擇 [**關閉**]。 
    
6. 若要將使用者新增至您的通訊群組，請參閱[新增使用者或連絡人至 Office 365 通訊群組](../email/add-user-or-contact-to-distribution-list.md)。
    
請參閱如何使用 Outlook 2016 和在網頁型 Outlook 中的群組[使用連絡人群組 （先前稱為通訊群組清單） 在 Outlook 中](https://support.office.com/article/1c97fcb2-0ed4-41e6-b401-58f9d7d40e39.aspx)主題。 
  
請參閱[通訊群組問題](https://docs.microsoft.com/office365/troubleshoot/groups/distribution-list-issues)的通訊群組問題的說明。
