# <a name="download-perpetual-software-and-product-license-keys"></a>下載永久軟體和產品授權碼

本文說明如何針對透過雲端方案提供者 (CSP) 計畫購買的永久軟體，下載軟體和產品授權金鑰。

## <a name="before-you-begin"></a>開始之前

您必須是全域系統管理員，才可執行本文中的步驟。 如需詳細資訊，請參閱[關於系統管理員角色](../add-users/about-admin-roles.md)。

## <a name="download-software-and-product-license-keys"></a>下載軟體和產品授權碼

1. 在 Microsoft 365 系統管理中心中，移至 [帳單 **]**  >  [<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">您的產品</a>] 頁面。
2. 在 [ **產品** ] 索引標籤的 [ **軟體** ] 區段中，選取您要下載的軟體。
3. 在 [訂閱詳細資料] 頁面的 [ **下載 & 機碼** ] 區段中，選擇 [ **產品版本**]、[ **語言**] 及 [ **CPU] & 檔案類型**，然後選取 [ **下載**]。
4. 若要下載機碼，請選取 [ **複製機碼至剪貼** 簿]。
5. 在右窗格中，選取 [ **複製**]，然後關閉窗格。
6. 在安全的位置中貼上檔案中的機碼，然後在軟體安裝期間依照指示輸入它。 啟動已下載軟體的必要機碼。
