# <a name="add-your-company-branding-to-the-sign-in-page"></a>將貴公司商標新增到登登頁面

 您現在可以使用 Microsoft 365 訂閱 (Azure Active Directory) AD) 訂閱來自訂使用者看到的載入頁面。 
  
## <a name="add-company-branding-to-your-sign-in-page-and-access-panel-pages"></a>將公司商標新增至登入頁面及存取面板頁面

如果您有商務用 Microsoft 365、Microsoft Dynamics CRM Online、Enterprise Mobility Suite 或其他 Microsoft 服務的付費訂閱，表示您擁有 Azure Active Directory 的免費訂閱。 您可以使用 Azure Active Directory 來建立及管理使用者和群組帳戶，並新增公司商標至您的頁面。 若要啟用此訂閱並存取 Microsoft Azure 管理入口網站，您必須完成一次性的註冊程序。 之後，您就可以從使用 Azure Active Directory 的 Microsoft 服務來存取。 有關如何註冊您的 Microsoft 365 訂閱的指示，請參閱註冊免費的 [Azure Active Directory](https://go.microsoft.com/fwlink/p/?LinkID=527966)訂閱，並參閱在 Azure 中管理 [您的 Microsoft 365](https://go.microsoft.com/fwlink/p/?LinkId=620076) 訂閱目錄以參閱一般管理指示。 
  
下圖顯示您能在 Azure 中修改登入頁面的哪部分。
  
![您可以自訂之簽署頁面的區域。](../../media/screenshotbranding.png)
  
1. 大型插圖及/或其背景色彩
    
2. 橫幅標誌
    
3. 您也可以在此區域新增文字
    
除了登入頁面外，您還可以自訂 Azure 中的存取面板頁面。
  
如果您已準備好要新增商標、探索 Azure 內容組中的自訂選項：[將公司商標新增至登入及存取面板頁面](https://go.microsoft.com/fwlink/p/?LinkId=620077)。
