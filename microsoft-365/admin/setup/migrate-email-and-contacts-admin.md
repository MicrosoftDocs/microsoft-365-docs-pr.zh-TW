# <a name="migrate-email-and-contacts-to-office-365"></a>將電子郵件和連絡人移轉到 Office 365

匯入或複製到 Office 365 的電子郵件從 Gmail 或另一個電子郵件提供者。
  
 **想要此問題的協助嗎？**  [連絡商務用 Office 365 支援人員](../contact-support-for-business-products.md) 。 
  
您要使用的安裝在您的桌面，這項工作的 Outlook 版本。 Outlook 會包含在大部分的 Office 365[計劃](https://go.microsoft.com/fwlink/p/?LinkId=723731)。
  
### <a name="import-from-gmail"></a>從 Gmail 匯入

請遵循下列步驟來匯入電子郵件、 連絡人及行事曆從 Gmail Outlook 與 Office 365:
  
- [將 gmail 匯入至 Outlook](https://support.office.com/article/20fdb8f2-fed8-4b14-baf0-bf04b9c44bf7.aspx)
    
- [匯入連絡人至 Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8.aspx)
    
- [Google 日曆匯入](https://support.office.com/article/098ed60c-936b-41fb-83d6-7e3786437330)
    
> [!VIDEO https://www.microsoft.com/videoplayer/embed/c1e45708-9a92-475b-910b-f5aa40614d92?autoplay=false]
  
### <a name="import-from-outlook-desktop"></a>匯入從 Outlook （桌面版）

請遵循下列步驟，將電子郵件、 連絡人和行事曆從 Outlook 匯出至.pst 檔案，並再將該檔案匯入到 Office 365 的 Outlook:
  
1. [匯出電子郵件、 連絡人及行事曆](https://support.office.com/article/14252b52-3075-4e9b-be4e-ff9ef1068f91)
    
2. [匯入郵件、 連絡人及行事曆](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac)
    
如果您只想連絡人，請遵循下列步驟：
  
1. [從 Outlook 匯出連絡人](https://support.office.com/article/10f09abd-643c-4495-bb80-543714eca73f.aspx)
    
2. [匯入連絡人至 Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8.aspx)
    
若要開始處理程序，開啟 Outlook，然後選擇 [**檔案** \> **開啟&amp;匯出** \> **匯入/匯出**。

## <a name="see-other-email-accounts-in-outlook"></a>請參閱 < 在 Outlook 中的其他電子郵件帳戶

只想要看到您的電子郵件從另一個提供者 （例如 Gmail、 Yahoo 或 Live.com） 在 Outlook 中？ 匯入或移轉任何需要。 您可以設定 Outlook 或 Outlook Web App 以便您可以從您的 Office 365 信箱相同的地方存取您其他帳戶和傳送、 接收和讀取電子郵件傳送至這些帳戶。
  
### <a name="outlook-desktop"></a>Outlook （桌面版）

將帳戶，例如私人 Gmail 帳戶，新增至 Outlook。
  
- 開啟 Outlook，然後移至 [**檔案** \> **新增帳戶**。
    
需要更詳細的步驟嗎？ 請參閱[新增帳戶](https://support.office.com/article/6e27792a-9267-4aa4-8bb6-c84ef146101b)。

## <a name="multiple-mailboxes-admins-can-bulk-import-email-contacts-and-calendars-to-office-365"></a>多個信箱： 系統管理員可以大量匯入電子郵件、 連絡人及行事曆以 Office 365

根據您的來源電子郵件系統，您可以選擇從數個大量移轉方法。 閱讀決定哪一種方法適用於您的[方式移轉到 Office 365 的多個電子郵件帳戶](https://support.office.com/article/0a4913fe-60fb-498f-9155-a86516418842)。
