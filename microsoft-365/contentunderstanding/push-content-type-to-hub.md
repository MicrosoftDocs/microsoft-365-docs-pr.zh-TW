# <a name="push-content-types-to-a-hub"></a>將內容類型推入 hub

若要讓重要的內容類型可 SharePoint 文件庫和清單，您可以將它們推入您所選擇的中樞。 這會自動將其新增至與 hub 相關聯之網站上建立的任何新清單和程式庫，以及加入至 hub 的任何新網站。

若要讓此功能能夠運作，必須已發佈推入的內容類型。

將內容類型推入中樞

1. 在 SharePoint 系統管理中心，展開 [ **內容服務**]，然後按一下 [ **內容類型庫**]。

2. 按一下您要推送至集線器的內容類型。

3. 按一下命令列中的 [ **編輯** ]。
 
4. 按一下 **[選擇中心網站**]。
 
5. 選取所需的 hub 網站，然後按一下 **[確定]**。
 
6. 按一下 **[儲存]**。

當您第一次將內容類型推向現有的中樞 & 現有的關聯網站時，最多可能需要一小時的時間，才能從 hub 或關聯的網站進行訪問，以供網站中更新的設定。 任何新的 hub 關聯都不需要此等待，而且會在幾分鐘內反映設定。 

設定此設定後，具有這些設定的內容類型，將會在幾分鐘內，與 hub 產生任何新近關聯的網站。 一旦可用，任何已建立的新清單或文件庫將會在建立幾分鐘內自動新增內容類型。 只有當推入的內容類型直接或間接從檔內容類型派生時，才會將其新增至文件庫，而且只有在它不是直接或間接從檔內容類型派生時，才會將內容類型新增至清單。
