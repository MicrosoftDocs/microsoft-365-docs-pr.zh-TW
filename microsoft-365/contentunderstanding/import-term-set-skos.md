# <a name="import-a-term-set-using-a-skos-based-format"></a>使用以 SKOS 為基礎的格式匯入字詞集

您可以使用以 SKOS 為基礎的格式匯入字詞集合。 如需格式的詳細資訊，請參閱 [SharePoint 分類法 SKOS format reference](skos-format-reference.md)。

建議您將匯入檔案保持在20000個以下的字詞。 較大的檔案可增加驗證和匯入所需的時間。

1. 在 SharePoint 系統管理中心，展開 [ **內容服務**]，然後按一下 [ **字詞存放區**]。

2. 選取您要匯入字詞集的字詞群組。

3. 在命令列中，按一下 [匯 **入字詞集**]。
 
4.  如果您想要下載用作範本的範例檔案，請按一下 **sample-metadata** 以取得使用以 SKOS 為基礎之格式的範例檔案。
 
5.  建立包含字詞集的匯入檔案 & 您想要匯入的字詞。

6.  在 [ **檔案格式**] 底下，選取 [ **SKOS ( *) **。

7.  按一下 **[流覽]** ，然後流覽並新增您的匯入檔案。

8.  按一下 [匯入]****。 在匯入完成之前，請勿關閉面板。

順利匯入檔案時，會顯示成功訊息，而且會重新整理字詞存放區，您可以流覽至新建立的字詞集。
