# <a name="term-store-reports"></a>字詞存放區報告

您可以在 SharePoint 系統管理中心存取術語存放區的報告。 展開 [ **報告**]，然後按一下 [ **內容服務**]。

您可以使用下列報告：

### <a name="term-store-operations"></a>字詞存放區作業

在15天內，查看您字詞存放區中發生的最主要作業，並查看該期間內每日磁片量的趨勢。 當您想要深入瞭解字詞存放區中的變更性質，以及識別大量作業的趨勢時，這是很有用的。 

### <a name="term-store-composition"></a>字詞存放區組合

在一般 managed 字詞和企業關鍵字之間，查看字詞類型的散佈。 視需要將多個關鍵字移至受管理的字詞。 當您想要識別使用者是否積極使用關鍵字時，這是很有用的。 您可以查看它們，以識別受管理字詞的潛在候選人。

### <a name="open-and-closed-term-sets"></a>開放和關閉字片語

查看開啟和關閉的字詞集合的散佈。 如果您想要瞭解允許使用者新增字詞的字詞集數目，這會很有用。 如有需要，您可以將其轉換為 [已關閉] （例如，磁片區數量不斷增加）。 

### <a name="terms-without-synonyms"></a>不含同義字的字詞

在每一個尚未定義同義字的工作語言中，查看字詞數目。 這有助於識別您要定義同義字的語言。 同義字可協助使用者消除歧義，並尋找正確的字詞來標記其內容。
