# <a name="apply-a-retention-label-to-a-document-understanding-model-preview"></a>將保留標籤套用至檔理解模型 (預覽) 

> [!Note] 
> 本文內容適用于 Project Cortex 私人預覽。 [進一步瞭解專案 Cortex](https://aka.ms/projectcortex)。

</br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4CSoL]

</br>

您可以輕鬆地將 [保留標籤](https://docs.microsoft.com/microsoft-365/compliance/retention?view=o365-worldwide#retention-labels) 套用至 Project Cortex 中的檔理解模型。

保留標籤可讓您將保留設定套用至檔瞭解模型所識別的檔。  例如，您可能會想讓模型不僅會識別任何上傳至文件庫的合約更新檔，也可以對它們套用保留標記，以便永遠不會刪除這些檔。

您可以透過模型設定將保留標籤套用至檔理解模型。 

> [!Note]
> 若要讓您的模型使用保留標籤，必須 [在 Microsoft 365 規範中心建立併發布](https://docs.microsoft.com/microsoft-365/compliance/create-apply-retention-labels?view=o365-worldwide#how-to-create-and-publish-retention-labels)這些標籤。
