# <a name="find-answers-and-help-for-admins"></a>尋找系統管理員的答案及協助

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FGn5?autoplay=false]

如果您有疑問或需要 Microsoft 365 商務版 Premium 的協助，您可以使用數種選項取得協助。

## <a name="try-it"></a>試試看吧！

1. 使用您的工作認證登入 [office.com](https://office.com) ，然後選取 [ **管理**]。
1. 在系統管理中心的首頁上，選擇 [系統 **管理員訓練** ] 以查看您可以使用的訓練選項。
1. 選擇 [ **企業擁有** 者] 和 [系統管理員]，以進行小型對中型訓練。
1. 為大型或更複雜的企業選擇 **企業 IT 專業人員** 。
1. 請回到系統管理中心，選擇 [ **自訂安裝指導** 方針]，以查看指導您完成特定設定工作的指南數目。
1. 回到系統管理中心首頁，然後選擇 [ **訓練** ]，以取得特定應用程式的協助，並瞭解如何充分利用您每天所用的功能。
1. 回到系統管理中心，然後選擇 [ **需要協助** ] 以開啟支援助理。 輸入您的問題或您想要協助的主題，然後選取 [ **傳送**]。 選擇系統管理中心的連結，或選擇回答您問題的文章。
1. 如果您找不到問題的答案，您可以隨時聯繫支援人員，而且技術支援人員會很快對您作出回應。

您也可以使用下列資源來取得問題的協助：

- [aka.ms/learn365admin](../admin/index.yml)，搜尋所有系統管理員相關的工作及協助，例如重設密碼和指派 Microsoft 訂閱。
- [answers.microsoft.com](https://answers.microsoft.com)，可搜尋答案、開始討論，以及詢問其他使用者的問題。
- [techcommunity.microsoft.com](https://techcommunity.microsoft.com)，加入專家群組，並尋找適用于您公司的 microsoft 365 最佳作法。
