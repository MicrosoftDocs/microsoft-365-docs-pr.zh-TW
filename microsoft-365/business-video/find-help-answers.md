# <a name="find-docs-and-training"></a>尋找檔和訓練

如果您有疑問或需要協助 Microsoft 365 商務進階版，您有數個選項可以取得協助。

## <a name="browse-all-small-business-training"></a>流覽所有小型商務訓練

如需您需要在小型企業中使用 Microsoft 365 所有訓練支援的連結，請參閱[Microsoft 365 small business 訓練](index.yml)。 

## <a name="search-for-microsoft-365-admin-center-docs-and-resources"></a>搜尋 Microsoft 365 系統管理中心檔和資源

若要搜尋系統管理員相關的工作及協助，例如重設密碼和指派 Microsoft 訂閱，請參閱[Microsoft 365 系統管理中心](../admin/index.yml)說明。

## <a name="join-the-community"></a>加入社區

透過加入[Microsoft small business Community](https://smallbusiness.microsoft.com/)來開始交談、追蹤討論，以及與其他 Microsoft 365 小型企業使用者分享您的知識。

如果您有其他產品和服務的相關問題，請張貼問題，並追蹤相關討論，以[尋求 Microsoft Community](https://answers.microsoft.com)。
