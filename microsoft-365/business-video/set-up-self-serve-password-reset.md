# <a name="set-up-self-service-password-reset"></a>設定自助式密碼重設

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3AY8S?autoplay=false]

若要讓使用者重設自己的密碼，而不必每次與您聯繫，您可以在 Microsoft 365 商務進版中設定自助密碼重設。

## <a name="try-it"></a>試試看吧！

1. 在 Microsoft 365 系統管理中心的左側流覽窗格中，選取設定，然後選取安全性&**隱私權**。
1. 在 **讓人員重設自己的密碼** 下，選取 **Azure AD 系統管理中心**。
1. 在左側流覽窗格中 **，選取使用者**，然後在使用者 - **所有使用者** 頁面上，選取密碼 **重設**。
1. 選取 **全部** 以啟用自助密碼重設， **然後選取儲存**。

下一次使用者帳戶時，會要求他們輸入使用者識別碼和密碼。 他們選取 **了下** 一步，然後選擇是否要使用手機、電子郵件或兩者進行驗證。 他們輸入收到的驗證碼 **，選取驗證**， **然後選取完成**。 完成後，他們可以重設自己的密碼。
