# <a name="set-up-self-service-password-reset"></a>設定自助式密碼重設

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3AY8S?autoplay=false]

若要讓您的使用者重設自己的密碼，而不需要每次都與我們聯繫，您可以在 Microsoft 365 商務進階版中設定自助密碼重設。

## <a name="try-it"></a>試試看吧！

1. 在 [Microsoft 365 系統管理中心] 的左功能窗格中，選取 [**設定**]，然後選取 [**安全性 & 隱私權**]。
1. 在 [ **讓您的使用者重設自己的密碼**] 底下，選取 [**Azure AD 系統管理中心**]。
1. 在左功能窗格中，選取 [ **使用者**]，然後在 [ **使用者-所有使用者** ] 頁面上，選取 [ **密碼重設**]。
1. 選取 [ **全部** ] 啟用自助密碼重設，然後選取 [ **儲存**]。

使用者下一次登入其帳戶時，會要求他們輸入使用者識別碼和密碼。 請選取 **[下一步]**，然後選擇是否要使用電話、電子郵件或兩者進行驗證。 他們會輸入他們收到的程式碼、選取 [ **驗證**]，然後選取 **[完成]**。 完成後，他們就可以重設自己的密碼。
