# <a name="switch-from-google-workspace---overview"></a>從 Google Workspace 切換 - 概觀

完成下列步驟，將資料、電子郵件和使用者從 Google Workspace 移至商務用 Microsoft 365。


| 步驟  |描述  |
|---------|---------|
|步驟 1 |  [註冊](../sign-up.md) Microsoft 365 商務進版       |
|步驟 2 |   [設定 Microsoft 365 進行 Google Workspace 移移](set-up-microsoft-365-forgoogle.md)。 </br> 在此步驟中，您將驗證您擁有在 Google Workspace 中使用的網域、新增使用者，以及設定裝置的基本安全性。 |
|步驟 3 | [設定 Windows 裝置安全性原則](../secure-win10-pcs.md)。</br> Windows 安全性是在 Microsoft 365 系統管理中心的設定頁面中個別設定。 |
|步驟 4|[新增 Google Workspace 網域至 Microsoft 365](add-google-domain.md) </br> 確認您擁有用於電子郵件的網域後，您和所有其他使用者都可以開始使用舊電子郵件來註冊 Microsoft 365 商務進版。 |
|步驟 5 | [安裝 Office 應用程式和 Microsoft Teams。](../install-office.md)</br> 所有擁有 Microsoft 365 授權的人，都應該在工作裝置上安裝 Office App。|
|步驟 6 | [遷移每個人的電子郵件和日曆專案](migrate-email.md)。</br> 在此步驟中，您將執行 Exchange Online 移轉，以從 Google Workspace 移動每個人的電子郵件、日曆和連絡人。  |
|步驟 7 | [將網域連接至 Microsoft 365。](connect-domain-tom365.md) </br> 在您連結網域電子郵件之後，就會開始連至 Microsoft 365，而您所有的 Microsoft 365 服務都將能夠使用。|
|步驟 8|使用 [Mover 將每個人的資料從](mover-migrate-files.md) 雲端硬碟移到 OneDrive，以及從共用磁片磁碟機移到小組網站。</br> 在此步驟中，個人與共享磁片磁碟機中所有的資料均會複製並移至 Microsoft 365。|
|步驟 9| [停止 Google Workspace，](cancel-google.md) 但保留您的網域。 </br> 如果 Google 管理您的網域，即使您停止 Google Workspace 訂閱，也可以保留該網域。 如果您想要的話，也可以選擇將其移至另一個 DNS 主機。|
