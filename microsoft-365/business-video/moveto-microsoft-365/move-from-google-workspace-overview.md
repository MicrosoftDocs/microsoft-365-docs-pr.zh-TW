# <a name="switch-from-google-workspace---overview"></a>從 Google Workspace 切換-簡介

完成下列步驟，將您的資料、電子郵件和使用者從 Google Workspace 移至 Microsoft 365 for business。


| 步驟  |描述  |
|---------|---------|
|步驟 1 |  [註冊](../sign-up.md) Microsoft 365 商務版 Premium       |
|步驟 2 |   [為 Google Workspace 遷移設定 Microsoft 365](set-up-microsoft-365-forgoogle.md)。 </br> 在這個步驟中，您將會驗證您擁有 Google Workspace 中使用的網域、新增使用者，並設定裝置的基本安全性。 |
|步驟 3 | [設定 Windows 裝置的安全性原則](../secure-win10-pcs.md)。</br> Windows 安全性是在 Microsoft 365 系統管理中心的 [設定] 頁面上分開設定。 |
|步驟 4|[將 Google Workspace 網域新增至 Microsoft 365](add-google-domain.md) </br> 在您確認您擁有電子郵件所使用的網域後，其他所有使用者都可以開始使用舊的電子郵件登入 Microsoft 365 商務版 premium。 |
|步驟 5 | [安裝 Office 應用程式和 Microsoft 團隊](../install-office.md)。</br> 所有具有 Microsoft 365 授權的人員，都應該在其工作裝置上安裝 Office 應用程式。|
|步驟 6 | [遷移所有人的電子郵件和行事曆專案](migrate-email.md)。</br> 在此步驟中，您將執行 Exchange Online 遷移，以從 Google Workspace 移動所有人的電子郵件、行事曆和連絡人。  |
|步驟 7 | [將網域連接至 Microsoft 365](connect-domain-tom365.md)。 </br> 在您連接網域電子郵件將開始進入 Microsoft 365，而且所有的 Microsoft 365 服務都會運作。|
|步驟8|使用 [移動器，將每個人的資料](mover-migrate-files.md) 從磁片磁碟機移至 OneDrive，以及從共用磁片磁碟機移至小組網站。</br> 在此步驟中，會複製個人和共用磁片磁碟機中的所有資料並移至 Microsoft 365。|
|步驟 9| [終止 Google Workspace](cancel-google.md) ，但保留您的網域。 </br> 如果 Google 管理您的網域，即使您中斷 Google Workspace 訂閱後，也可以將它保留在這裡。 您也可以視需要選擇將它移至另一個 DNS 主機。|
