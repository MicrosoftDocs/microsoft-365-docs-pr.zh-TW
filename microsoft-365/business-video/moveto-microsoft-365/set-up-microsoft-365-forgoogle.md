# <a name="set-up-microsoft-365-for-google-workspace-migration"></a>為 Google Workspace 遷移設定 Microsoft 365

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4JD2Y?autoplay=false]

若要從 Google Workspace 遷移電子郵件、行事曆及連絡人，您必須在您的商務網域準備好搭配訂閱使用之前，設定 Microsoft 365 商務版的臨時網域。

## <a name="try-it"></a>試試看吧！ 

1. 登入 [office.com](https://office.com)，然後選擇 [系統 **管理**]。
1. 在 Microsoft 365 系統管理中心中，選擇 [ **移至引導安裝程式**]。 
1. （選用）安裝 Office 應用程式，然後選取 [ **繼續**]。 
1. 從 Google Workspace 遷移後，請選擇 [預設網域]，然後選取 [ **使用此網域**]。 
1. 輸入您的使用者名稱和所需的使用者名稱，並確認您的電子郵件地址，以便您接收使用者的密碼，然後選取 [ **新增使用者並指派授權**]。 
1. 輸入您的使用者將會收到其新認證的電子郵件地址，請選取 [ **傳送電子郵件並繼續**]。
1. 然後再次 **繼續** ，以連線您的暫存網域。 
1. 您可以選擇在組織中告知使用者小組，然後選取 [ **繼續**]。
1. 展開並查看您的 **Android 和 iOS 設定**，然後選取 [ **建立行動應用程式管理原則**]。
1. 視需要提供意見反應，然後選擇 [ **移至系統管理中心**]。

處理常式的下一個步驟是設定組織的保護。
