# <a name="sign-up-for-microsoft-365-business-premium-subscription"></a>註冊 Microsoft 365 商務進階版訂閱

## <a name="watch-sign-up-for-microsoft-365-business-premium"></a>觀賞：註冊 Microsoft 365 商務進階版

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3znhX?autoplay=false]

## <a name="try-it"></a>試試看吧！

請遵循下列步驟，註冊先前稱為 Microsoft 365 商務版) 的 Microsoft 365 商務進階版 (：

1. 若要註冊 Microsoft 365 商務進階版，請線上搜尋，然後選取。
2. 選擇 [  **商務** ]，然後  **立即購買**。
3. 輸入您的個人電子郵件地址，選取  **[下一步]** ，然後  **設定帳戶**。
4. 輸入您的姓名、電話號碼及公司名稱。 如果您&#39;d 像是 Microsoft 合作夥伴的說明，請選取 [我想要  **microsoft 共用我的資訊**] 旁邊的方塊。 選取  **[下一步]**。
5. 若要驗證您的電話號碼，請選取 [  **傳送驗證碼**]。 輸入您在電話上收到的程式碼，然後選取 [  **驗證**]。
6. 如果您已經有網域，請選擇 [  **立即取得 Microsoft 網域**]，然後輸入您的公司名稱。 這會變成您的暫時電子郵件地址。 它也會變成永久 SharePoint 和 OneDrive URLs 的一部分。 選取 [  **檢查可用性** ]，然後選取  **[下一步]**。
7. 輸入您的使用者名稱、密碼、確認您的密碼，然後選取 [  **註冊**  ] 以建立您的帳戶。
8. 輸入使用者數目、選擇計費計畫，然後選取  **[下一步]**。
9.  輸入您的公司&#39;s 位址，然後按  **[下一步]**。
10. 輸入您的信用卡資訊，然後  **按訂單**。 片刻之後，將會建立您的帳戶。
11. 此時您可以選取 [**繼續] 設定帳戶**，或者您可以登入 Microsoft 365 系統管理中心，然後完成安裝程式。
