# <a name="securely-share-files-externally"></a>安全地於外部共用檔案

## <a name="watch-securely-share-files-outside-your-organization"></a>觀賞：安全地共用組織外部的檔案

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE22Yf0?autoplay=false]

在商務 Microsoft 365 中，很容易與組織中的人員共用檔案，但有時候您也需要與組織外部的人員共用檔案。 若要安全地與客戶共用檔案，您可以傳送安全連結。

## <a name="try-it"></a>試試看吧！

1. 在 SharePoint 網站上的文件庫頂端，按一下 [**新增**]，然後選取 [**資料夾**]。
1. 為資料夾提供名稱，然後按一下 [ **建立**]。
1. Upload 您要共用至新文件庫的檔案，或選擇 [**新增**] 直接在文件庫中建立新的檔。
1. 在 [文件庫] 中選取新的資料夾，然後按一下 [ **共用** ] 按鈕。
1. 在 [ **傳送連結** ] 視窗中，按一下頂端的清單，然後選擇 [ **特定人員**]，選取 [ **允許編輯**]，然後 **選擇 [套用**]。
1. 在 [ **傳送連結** ] 視窗中，輸入您客戶的電子郵件地址，輸入訊息，然後選擇 [ **傳送**]。

    您的客戶將會收到一封具有驗證碼的電子郵件，他們必須自行傳送以存取您的資料夾。 若找不到電子郵件，他們應該檢查其 [垃圾郵件] 資料夾。
