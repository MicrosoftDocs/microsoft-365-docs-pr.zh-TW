# <a name="connect-your-pc-to-microsoft-365-business-premium"></a>將您的電腦連線至 Microsoft 365 商務版

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3yXh3?autoplay=false]

## <a name="try-it"></a>試試看吧！
在您設定 Microsoft 365 商務版 Premium (之前稱為 Microsoft 365 商務) 時，您和您的員工必須完成一些步驟，才能充分利用其安全性和管理功能。 這裡&#39;s 方式。

首先，請確認您已安裝 Windows 10 專業版：

1. 選擇 [  **開始** ]、[控制台] 中的 [搜尋]，然後選取該  **面板** 。
2. 選擇 [**系統和安全性**   >   **系統**]，然後確認您的 Windows 版本。

接下來，連接至 Microsoft 365 商務版：

1. 選取 [  **開始** ]，然後選取 [  **設定** ] 圖示。
2. 選取 [**帳戶**  >   **存取工作或學校**   >   **] [連線]**。
3. 選取 [  **將此裝置加入 Azure Active Directory**]。
4. 輸入您的 Microsoft 365 帳戶的電子郵件地址，然後選取  **[下一步]**。
5. 輸入您的密碼，然後選取 [登  **入** ]，然後確認您加入的是正確的組織。
6. 選取 [  **加入** ]，然後選取 [  **完成**]。

根據您公司設定 Microsoft 365 商務的方式，您可能會看到加密您的電腦的通知。 若要加密您的電腦：

1. 選取 [  **需要加密**  ] 訊息。
2. 選取 [  **&#39;我沒有安裝任何其他磁片加密軟體]，將所有磁片** 加密，然後選取  **[是]**。 您將會看到一個包含進度列的  **開始 BitLocker**  視窗。
3. 在 [  **選擇在啟動時解除鎖定磁片磁碟機**  ] 視窗中，選取 [ **輸入 PIN (建議)**]。
4. 輸入您的 PIN 碼，然後再次輸入以加以驗證。 選取 [  **設定 PIN**]。
5. 選擇用來儲存加密復原機碼的方法，然後選取  **[下一步]**。
6. 選擇您要對多少磁片磁碟機進行加密，然後選取  **[下一步]**。
7. 選擇 [加密模式]，選取  **[下一步]** ，然後選取 [  **繼續**]。 您的磁片磁碟機現在會進行加密。 您可以在加密處理常式期間繼續運作。

您的電腦現在受到 Microsoft 365 商務版的保護和管理。
