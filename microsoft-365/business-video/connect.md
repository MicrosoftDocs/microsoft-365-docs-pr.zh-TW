# <a name="connect-your-pc-to-microsoft-365-business-premium"></a>將您的電腦連接到 Microsoft 365 商務進版

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3yXh3?autoplay=false]

## <a name="try-it"></a>試試看吧！
設定 Microsoft 365 商務進 (之前稱為 Microsoft 365 商務版) 後，您和員工必須完成幾個步驟，以充分利用其安全性與管理功能。 以下是&#39;說明。

首先，請確認您已安裝 Windows 10 專業版：

1. 選擇  **開始** ，搜尋  **控制台** ，然後選取它。
2. 選擇 **系統及安全性**   >   **系統**，並確認您的 Windows 版本。

接下來，請連接至 Microsoft 365 商務版：

1. 選取  **開始** ，然後  **選取設定圖示** 。
2. 選取 **Access**  >   **公司或學校 Connect**   >   **的帳戶**。
3. 選取 **將這個裝置加入 Azure Active Directory。**
4. 輸入 Microsoft 365 帳戶的電子郵件地址，然後選取下一  **步**。
5. 輸入您的密碼，選取  **Sign in，** 然後確認您加入的是正確的組織。
6. 選取  **加入** ，  **然後選取完成**。

視貴公司設定 Microsoft 365 商務版方式的不同，您可能會看到加密電腦的通知。 若要加密您的電腦：

1. 選取需要  **加密的**  訊息。
2. 選取  **我&#39;** 任何磁片加密軟體，加密我的所有磁片，然後選取  **是**。 您將看見一個  **啟動 BitLocker**  視窗，以及一個進度列。
3. 在選擇 **啟動時如何解除** 鎖定磁片磁碟機的視窗中，選取輸入 PIN 碼 (**建議) 。**
4. 輸入您的 PIN，然後再次輸入以進行驗證。 選取 **設定 PIN。**
5. 選擇儲存加密修復金鑰的方法，然後選取下一  **步**。
6. 選擇要加密多少硬碟，然後選取下一  **步**。
7. 選擇加密模式  **，選取下** 一步，  **然後選取繼續**。 您的硬碟現在將會加密。 您可以在加密程式期間繼續工作。

您的電腦現在受到 Microsoft 365 商務版保護及管理。
