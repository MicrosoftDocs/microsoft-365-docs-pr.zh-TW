# <a name="move-files-to-sharepoint"></a>將檔案移至 SharePoint

> [!VIDEO https://www.microsoft.com/videoplayer/embed/65ec460e-296c-458f-b004-906756dfb3d0?autoplay=false]

註冊 Microsoft 365 商務進版之後，&#39;將公司工作檔案複製到 SharePoint。 SharePoint 是儲存每個人都需要存取之共用公司檔案的好位置。 這通常會取代檔案共用或網路磁碟機機的使用。

## <a name="try-it"></a>試試看吧！

1. 開啟 Microsoft Teams，然後開啟公司中每個人都能存取的團隊。
2. 選取  **檔案** ，然後選取在  **SharePoint 中開啟**。
3. 選取  **檔** 以前往網站上儲存所有專案的位置。
4. 選取 **同步，****然後選取是**。 這會將 SharePoint 網站的檔案同步處理至您的電腦。
5. 在檔案檔案&#39;，您現在會從 SharePoint 網站看到&#39;同步處理的檔資料夾。 如果您同步其他 SharePoint 網站，這些網站會顯示在此網站的上方或下方。 以滑鼠右鍵按一下同步處理的檔資料夾，然後以新視窗開啟該資料夾，即可將資料夾與您想要移動的公司檔案並排查看。
6. 選取所有您想要移動的檔案，然後拖曳到您同步的 SharePoint 資料夾。 檔案旁的藍色圓形箭號表示&#39;SharePoint 網站。 同步&#39;之後，系統會顯示綠色勾選記號。
7. 返回您的 SharePoint 網站以查看已同步處理至雲端的檔案。

除了直接從電腦處理檔案，您還可以在任何電腦的網頁瀏覽器中存取這些檔案。

現在，您可以存取 SharePoint 網站或 Teams 中的檔案。
