# <a name="install-outlook-and-office-apps-on-android-devices"></a>在 Android 裝置上安裝 Outlook 和 Office 應用程式

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWfquk?autoplay=false]

如果您的 Android 裝置是由 Microsoft 365 商務進階版所管理，則在您第一次安裝 Office 應用程式時，系統可能會提示您設定額外的安全性。 

## <a name="try-it"></a>試試看吧！

1. 在您的裝置上，選擇 Google Play 商店。
2. 搜尋您要安裝的 Office 應用程式。 在此範例中&#39;會使用 Outlook。
3. 在 [Outlook 歡迎] 畫面上，選擇 [**入門**]。
4. 在 [**新增帳戶**] 頁面上，輸入您的 Microsoft 365 電子郵件地址，然後選擇 [**繼續**]。
5. 在 [  **輸入密碼** ] 底下，輸入您的密碼，然後選擇 [登  **入**]。
6. 在下一個頁面上，如果您想要新增其他帳戶，請選擇 [  **繼續**  ]。 否則，請選擇 [  **略過**]。
7. 當系統提示您新增 Intune 公司入口網站應用程式時，請選擇 [**移至儲存區**]。
8. 安裝並開啟 Intune 公司入口網站應用程式，然後在看到您的組織保護此應用程式中的資料時，選擇  **[確定]**  。
9. 如果系統要求 PIN，請輸入您&#39;用來解除鎖定電話的4位數 PIN 碼，然後選擇 [  **傳送** ]，確認您的 PIN 碼，然後選擇 [  **傳送**  ]。
