# <a name="install-office-apps"></a>安裝 Office App 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/acce002c-0756-4b64-ac5d-2198ee96a9b1?autoplay=false]

## <a name="try-it"></a>試試看吧！

您可以在 [  **開始** ] 功能表上找到 Microsoft Office 應用程式。 如果您未看到&#39;，您可以自行安裝。

1. 移至 office.com。 您可能需要以您的工作帳戶登入。
2. 選取 [**安裝 Office**   >   **Microsoft 365 應用程式**   >   **執行**]，然後選取 **[是]**。
3. 將會安裝 Office 應用程式。 此程式可能需要數分鐘的時間。 完成後，請選取 [  **關閉**]。
4. 若要安裝 Microsoft 團隊，請移至 [office.com] 頁面，然後選擇 [  **小組**]。
5. 取得 Windows 應用程式，然後選取 [  **執行**]。 當安裝完成時，小組會顯示提示。

您已安裝的 Office app 現在會出現在 [  **開始** ] 功能表中。
