# <a name="install-the-microsoft-365-admin-mobile-app-on-your-phone"></a>在您的手機上安裝 Microsoft 365 Admin 行動裝置應用程式

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWe8Kw?autoplay=false]

您可以使用 Microsoft 365 Admin 行動裝置應用程式來管理使用者、監控服務健康情況，以及從您的手機執行其他系統管理員工作。

## <a name="try-it"></a>試試看吧！

1. 若要取得 Microsoft 365 Admin 行動裝置應用程式，請在您的裝置的應用程式存放區中搜尋  **Microsoft 365 Admin**  ，並加以安裝。
2. 使用您的全域系統管理員電子郵件地址登入，並查看儀表板，以查看服務健康情況、監控使用者授權，以及查看郵件和服務要求。
3. 以下是您可以使用 app 執行的一些常見工作：

### <a name="reset-a-users-password"></a>重設使用者的密碼

1. 在 [  **快速連結** ] 底下，選擇 [  **使用者**]。
2. 在 [  **使用者**  ] 頁面上，選擇使用者的名稱，然後選擇 [  **重設密碼**]。
3. 如果您想要使用者在初次登入時變更其新密碼，請選取 [  **在第一次登入時變更**  ] 核取方塊，然後選擇 [  **重設**]。
4. 要求確認時，請選擇  **[是** ]，然後按一下  **[確定]**。
5. 在 [  **共用密碼**  ] 視窗中，您可以選擇性地在電子郵件或文字訊息中傳送新密碼給使用者。

### <a name="add-a-user-to-a-group"></a>將使用者新增至群組

1. 在 [  **快速連結** ] 底下，選擇 [  **群組**]。
2. 在 [  **群組**  ] 頁面上，選擇您要新增使用者的群組，然後選擇 [  **編輯成員**]，然後再  **新增成員**。
3. 選擇您想要新增的使用者。
4. 要求確認時，請選擇  **[是** ]，然後按一下  **[確定]**。

### <a name="turn-on-notifications"></a>開啟通知

1. 在主功能表上，選擇 [  **設定**]。
2. 在 [  **應用程式設定**] 底下，選擇 [  **通知**]。
3. 選取 [  **服務健康情況**  與  **訊息中心**] 旁邊的核取方塊。
