# <a name="install-the-microsoft-365-admin-mobile-app-on-your-phone"></a>在手機上安裝 Microsoft 365 Admin 行動應用程式

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWe8Kw?autoplay=false]

您可以使用 Microsoft 365 Admin 行動應用程式來管理使用者、監視服務健康情況，以及從手機執行其他系統管理工作。

## <a name="try-it"></a>試試看吧！

1. 若要取得 Microsoft 365 Admin 行動裝置 App，請搜尋裝置 App Store 中的  **Microsoft 365 Admin**  並安裝它。
2. 使用全域系統管理員電子郵件地址進行登錄，並查看儀表板以查看服務健康情況、監視使用者授權，以及查看訊息和服務要求。
3. 以下是您可以使用應用程式執行一些一般工作：

### <a name="reset-a-users-password"></a>重設使用者的密碼

1. 在  **快速連結下** ，  **選擇** 使用者。
2. 在使用者  **頁面上**  ，選擇使用者的名稱，然後選擇重  **設密碼**。
3. 如果您想要要求使用者第一次登錄時變更新密碼，請選取第一次使用的變更核取方塊，然後選擇重設。 
4. 當系統詢問要確認時，選擇  **是** ，  **然後選擇確定**。
5. 在  **共用密碼視窗中**  ，您可以選擇性地以電子郵件或文字訊息傳送新密碼給使用者。

### <a name="add-a-user-to-a-group"></a>新增使用者至群組

1. 在  **快速連結下** ，選擇  **群組**。
2. 在群組 **頁面上**，選擇您想要新增使用者的群組，然後選擇編輯成員，然後 **新增成員**。 
3. 選擇您想要新增的使用者。
4. 當系統詢問要確認時，選擇  **是** ，  **然後選擇確定**。

### <a name="turn-on-notifications"></a>開啟通知

1. 在主功能表上  **，選擇設定**。
2. 在  **應用程式設定下**，  **選擇** 通知。
3. 選取服務健康情況和訊息 **中心****旁的核取方塊**。
