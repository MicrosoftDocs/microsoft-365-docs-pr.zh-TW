# <a name="set-up-outlook-for-microsoft-365-for-business-email"></a>為商務用 Microsoft 365 電子郵件設定 Outlook 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/9fe86884-8a83-42cc-bca9-61a12e6dad31?autoplay=false]

## <a name="try-it"></a>試試看吧！

安裝 Office 應用程式之後，您&#39;想要將 Outlook 設定為開始使用電子郵件、行事曆及連絡人。 這裡&#39;s 方式。

1. 開啟 [開始] 功能表。 搜尋 Outlook，然後選擇它。
2. 輸入您的 Microsoft 365 電子郵件地址，然後選取  **[連接]**。
3. 輸入您想要使用的任何其他電子郵件地址，例如您的先前或個人的電子郵件地址。 選取  **[下一步]**。
4. 出現提示時，請輸入密碼，然後選取 [登  **入**]。
5. 新增所有的帳戶之後，請選擇您是否要設定 Outlook 行動裝置，或等到以後為止。
6. 選取 [  **完成**]。 Outlook 下載您的電子郵件及其他資料可能需要數分鐘的時間。

現在您可以查看您新增之帳戶的電子郵件。 您也可以查看 [行事曆]、[連絡人] 和 [任務]。
