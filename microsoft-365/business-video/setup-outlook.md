# <a name="set-up-outlook-for-microsoft-365-for-business-email"></a>設定商務用 Microsoft 365 電子郵件的 Outlook 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/9fe86884-8a83-42cc-bca9-61a12e6dad31?autoplay=false]

## <a name="try-it"></a>試試看吧！

安裝 Office App 之後，&#39;Outlook 設定為開始使用電子郵件、日曆和連絡人。 以下是&#39;說明。

1. 開啟開始功能表。 搜尋 Outlook，然後選擇它。
2. 輸入您的 Microsoft 365 電子郵件地址，然後選取連結 **。**
3. 輸入任何其他您想要使用的電子郵件地址，例如您先前的或個人電子郵件地址。 選取下  **一個**。
4. 如果系統提示，請輸入密碼，然後選取Sign **in。**
5. 新增所有帳戶後，請選擇是否要設定 Outlook Mobile 或等到稍後再設定。
6. 選取  **完成**。 Outlook 可能需要數分鐘的時間才能下載您的電子郵件及其他資料。

現在您可以針對新增的帳戶來查看電子郵件。 您也可以查看您的日曆、連絡人和工作。
