# <a name="move-users-to-different-subscriptions"></a>將使用者移至不同的訂閱

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1SBE2?autoplay=false]

當您的使用者變更角色時，他們可能會需要目前 Microsoft 365 商務版 Premium 訂閱中無法使用的功能。 發生這種情況時，您可以新增包含這些功能的新訂閱，並將授權指派給需要這些功能的人員。

## <a name="try-it"></a>試試看吧！

### <a name="add-a-new-subscription"></a>新增訂閱

1. 在 Microsoft 365 系統管理中心的左功能窗格中，選取 [ **帳單**]，然後 **購買 [服務**]。
1. 選擇您想要新增的訂閱，然後選取 [ **購買**]。
1. 選擇您所需的支付方式和授權數目，然後選取 [ **立即查看**]。
1. 請複查您的訂單，然後選擇 **[下一步]**。
1. 確認您的總數，然後選取 [ **下單**]。

### <a name="change-the-license-assigned-to-a-user"></a>變更指派給使用者的授權

1. 在 Microsoft 365 系統管理中心的左功能窗格中，選取 [ **使用者**]，然後選取 [作用中的 **使用者**]。
1. 選取使用者的名稱，然後選取 [授權] **和 [app** s]。
1. 在 [ **授權**] 底下，您會看到新的訂閱。 選取新訂閱旁的核取方塊，並清除舊訂閱旁邊的核取方塊。
1. 選取 **[儲存變更]**。 新訂閱的授權現在已指派給此使用者。

### <a name="remove-an-unused-license"></a>移除未使用的授權

1. 在 Microsoft 365 admin center 中，選取 [ **帳單**]，然後選取 [ **產品] & 服務**]。
1. 選取您要變更的訂閱，然後選取 [ **新增/移除授權**]。
1. 使用向下鍵來減少此訂閱的授權數目，然後選取 [ **提交變更**]。 您現在擁有的使用者授權數目正確。
