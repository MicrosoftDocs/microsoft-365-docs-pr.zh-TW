# <a name="move-files-to-onedrive"></a>將檔案移至 OneDrive

> [!VIDEO https://www.microsoft.com/videoplayer/embed/d74b083c-1f44-43ea-8a14-2e1fc600b341?autoplay=false]

註冊 Microsoft 365 商務進版之後，&#39;現有的工作檔案複製到商務用 OneDrive。 OneDrive 是儲存您想要控制存取權之個人工作檔案或機密業務檔案的好位置。

## <a name="try-it"></a>試試看吧！

1. 在 Windows 中  **，選取開始** 按鈕。
2. 搜尋並選取 OneDrive。
3. 輸入您的公司電子郵件地址。
4. 選取  **簽署，** 然後選取下  **一步**。
5. 完成簡短的教學課程，然後選取 [  **開啟我的 OneDrive 資料夾**。 如果您也使用個人版 OneDrive，&#39;在這裡看到它。
6. 在檔案管中，前往您想要複製的檔案。
7. 選取檔案，然後將檔案拖放到 OneDrive 資料夾。 檔案旁的藍色圓形箭號表示&#39;同步處理至雲端。 同步&#39;之後，系統會顯示綠色勾選記號。

除了在網內處理 OneDrive 檔案之外，您還可以從網頁瀏覽器存取這些檔案：

1. 在網頁瀏覽器中，以公司office.com來使用帳戶。
2. 選擇 **[OneDrive。** 系統會列出您的所有檔案，而且您可以在瀏覽器中從任何電腦處理這些檔案。
