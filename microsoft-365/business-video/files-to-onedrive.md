# <a name="move-files-to-onedrive"></a>將檔案移至 OneDrive

> [!VIDEO https://www.microsoft.com/videoplayer/embed/d74b083c-1f44-43ea-8a14-2e1fc600b341?autoplay=false]

在您註冊 Microsoft 365 商務版 Premium 後，您&#39;想要將現有的工作檔案複製到商務 OneDrive。 OneDrive 是存放您想要控制存取之個人工作檔案或機密商務檔案的好地方。

## <a name="try-it"></a>試試看吧！

1. 在 [Windows] 中，選取 [  **開始** ] 按鈕。
2. 搜尋 OneDrive，然後選取。
3. 輸入您的工作電子郵件地址。
4. 選取 [登  **入** ]，然後選取  **[下一步]**。
5. 流覽簡短的教學課程，然後選取 [  **開啟我的 OneDrive] 資料夾**。 如果您也使用 OneDrive 的個人版本，您&#39;會在這裡看到。
6. 在檔案瀏覽器中，移至您要複製的檔案。
7. 選取檔案，然後將檔案拖放至您的 OneDrive 資料夾。 位於檔案旁邊的藍色圓形箭號表示它們&#39;會重新同步到雲端。 當他們&#39;執行同步處理時，會顯示綠色核取記號。

除了在本機使用 OneDrive 檔案之外，您還可以從網頁瀏覽器存取這些檔案：

1. 在您的網頁瀏覽器中，使用您的工作電子郵件登入 office.com。
2. 選擇 [  **OneDrive**]。 您會列出所有檔案，您可以在任何電腦上的瀏覽器中處理這些檔案。
