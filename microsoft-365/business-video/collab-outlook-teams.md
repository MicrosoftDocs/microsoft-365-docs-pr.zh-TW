# <a name="use-outlook-and-teams-together"></a>搭配使用 Outlook 與 Teams

當您與您的員工、合作夥伴、用戶端及小型企業中的其他人通訊時，請使用 Microsoft 365 中的 Outlook 和團隊。 在 Outlook 中，您會發現電子郵件、行事曆、連絡人及工作，而在小組中您會找到交談、檔案、會議，以及快速存取 Microsoft 和協力廠商應用程式。 以下摘要說明您在公司中的每個使用方式。

## <a name="microsoft-outlook"></a>Microsoft Outlook

- 檢查公司內部和外部的郵件，包括公司的新潛在客戶。
- 檢查您的行事曆、新增約會及排程會議。
- 以連絡人的身分，新增您用來做為連絡人之人員的名稱、號碼和電子郵件。
- 建立工作、記事及其他方式，以維持在短期和長期專案的最上層。

:::image type="content" source="../media/outlook-model.png" alt-text="Outlook 首頁的樣式化圖像。":::

## <a name="microsoft-teams"></a>Microsoft Teams

- 與您的小組進行豐富、即時的交談，並邀請用戶端或客人加入交談。
- 讓私人聊天或通話，並彼此共用您的畫面。
- 以員工和客人排程和加入會議。
- 從相關聯的 SharePoint 網站或商務 OneDrive 新增或共用檔案。
- 將更多應用程式（如 Planner、Power BI 或 GitHub）新增至團隊。

:::image type="content" source="../media/teams-model.png" alt-text="Microsoft 小組首頁的樣式化圖像。":::
