# <a name="use-outlook-and-teams-together"></a>搭配使用 Outlook 與 Teams

當您與小型企業中的員工、合作夥伴、客戶及其他人員通訊時，請使用 Microsoft 365 中的 Outlook 和 Teams。 在 Outlook 中，您可以找到電子郵件、月曆、連絡人及工作，而在 Teams 中，您可以找到交談、檔案、會議，以及快速存取 Microsoft 和協力廠商應用程式。 以下摘要說明在企業中可能如何使用每一項功能。

## <a name="microsoft-outlook"></a>Microsoft Outlook

- 查看公司內外的郵件，包括公司的新潛在客戶。
- 檢查您的月曆、新增約會及排程會議。
- 新增您合作人員的名稱、號碼和電子郵件。
- 建立工作、記事及其他方式，以隨時留意短期與長期專案。

:::image type="content" source="../media/outlook-model.png" alt-text="Outlook 首頁風格化影像。":::

## <a name="microsoft-teams"></a>Microsoft Teams

- 與小組進行豐富的即時交談，並邀請客戶或來賓加入交談。
- 進行私人聊天或通話，並和彼此分享您的螢幕畫面。
- 排程並加入與員工和來賓的會議。
- 從相關聯的 SharePoint 網站或商務用 OneDrive 新增或共用檔案。
- 新增更多應用程式，例如 Planner、Power BI 或 GitHub 至 Teams。

:::image type="content" source="../media/teams-model.png" alt-text="Microsoft Teams 首頁風格化影像。":::
