# <a name="overview-of-sharing-business-files"></a>共用商務檔案概觀

您需要與客戶、客戶、合作夥伴、供應商及公司中的其他人共用檔案。 如何共用檔案，讓每個人都可以尋找、開啟及編輯檔案？

## <a name="share-a-file-with-someone-outside-of-your-company"></a>與公司外部的人共用檔案

- 與公司外部的某人共用檔案 (例如客戶或客戶) 透過電子郵件共用檔案，但您只需要使用某個方法。
- 如果您需要回應，或想要他們填寫一些資訊並傳送回，您可以使用 SharePoint 或 Microsoft Teams 以編輯連結來共用。

深入瞭解如何使用 [安全連結與](securely-share-files-externally.md) 組織外部人員共用檔案，以及如何與任何人 [共用檔案連結](share-files-externally.md)。

## <a name="share-and-collaborate-with-a-client-or-business-partner"></a>與客戶或商務合作夥伴共用和共同合作

在專案中共用嗎？ 或作為與商務合作夥伴或供應商持續關係的一部分？ 您只要多加一點時間，就可以在 Microsoft Teams 中設定團隊，並持續提供一個可以共用和共同作業的地方。 與公司外部人員合作非常簡單，您甚至可以為其他供應商或合作夥伴設定個別的通道，讓一切保持一切簡單。

1. 建立團隊。
1. 上傳您的檔案。
1. 邀請您的合作夥伴或供應商以「來賓」使用者加入團隊，讓他們存取。

深入瞭解如何使用安全 [連結與組織外部人員共用](https://support.microsoft.com/office/7266f44e-3e06-4736-b9d3-0580c24bba34)檔案。

## <a name="share-inside-your-business"></a>在公司內共用

如果您使用在公司內分享的檔案，在 Microsoft Teams 中設定團隊也是一個很好的主意。 在那裡共用您的檔案，企業中的每個人都可以輕鬆尋找和編輯檔案，而且您瞭解一切在同一個頁面上。

1. 建立團隊並邀請您的小組加入。
1. 上傳您的檔案。

深入瞭解公司檔案[儲存和](https://support.microsoft.com/office/e4d98e10-3532-4eed-85d6-92728454e32b)[員工檔案儲存空間](https://support.microsoft.com/office/12dbe3e4-dbef-48f8-a90e-87f1bc607073)。
