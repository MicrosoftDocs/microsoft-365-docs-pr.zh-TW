# <a name="chat-with-employees-and-customers"></a>與員工和客戶聊天

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWedV6?autoplay=false]

您可以從內部與同事互動，或與來賓使用者交談，例如客戶或合作夥伴。 如果已為您的公司 [開啟來賓存取](/microsoftteams/set-up-guests) ，您只需輸入電子郵件地址，即可將客人新增至聊天。

## <a name="try-it"></a>試試看吧！

啟動1:1 或群組聊天，以與較小的群組進行共同作業，或私下進行互動，以及 pin 聊天以供快速存取。

1. 在  **聊天** 中，您可以讓1:1 或群組聊天給別人。
2. 選取 [**新增聊天**] :::image type="icon" source="../media/new-chat-icon.png" border="false"::: 。  
3. 輸入要聯繫的人員名稱或人員名稱。
4. 選取向左下箭號以將名稱新增至您的聊天室群組。

從聊天製作影片或音訊通話

- 選取 [  **影片** 通話] :::image type="icon" source="../media/video-call-icon.png" border="false"::: 或 [ **語音通話**] :::image type="icon" source="../media/audio-call-icon.png" border="false"::: 直接從聊天呼叫某人。
