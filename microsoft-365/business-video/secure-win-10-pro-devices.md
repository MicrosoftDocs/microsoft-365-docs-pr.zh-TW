# <a name="manage-windows-10-pro-device-policies"></a>管理 Windows 10 專業版裝置原則

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FYSL?autoplay=false]

您可以使用 Microsoft 365 商務版，確定 windows 10 裝置上已啟用 Windows Defender 防毒程式，而且 Microsoft 更新會自動下載至使用者的裝置。

## <a name="try-it"></a>試試看吧！

1. 登入 Microsoft 365 系統管理中心。
1. 在 [ **原則**] 底下，選擇 [新增原則]。
1. 在 [**新增原則**] 窗格的 [**原則名稱**] 下，輸入名稱，然後選取 [**原則類型**] 底下的 [ **Windows 10 裝置** 設定]。
1. 選擇 [ **安全的 Windows 10 裝置** ]，以查看子設定。
1. 請務必 **使用 Windows Defender 防毒軟體，協助保護電腦免受病毒和其他威脅** ，並 **將 windows 10 裝置自動保持在最** 新狀態。
1. 在 **誰會取得這些設定？** 預設會選取所有使用者，但您可以選擇 [ **變更** ]，以選取您建立的任何安全性群組。
1. 若要完成建立原則，請選擇 [ **新增**]。
1. 在 [ **新增原則** ] 頁面上，選擇 [ **關閉**]。
1. 在系統管理中心的首頁上，透過選擇 [原則] 並在 [**原則**] 頁面上查看您的 **原則，以** 確認新增原則。
1. 若要確認原則已生效，請在使用者的 Windows 10 裝置上，移至 [Windows 更新]，選擇 [ **高級選項**]，然後確認設定會變暗。

    然後，按一下 **[選擇如何傳送更新**]，並確認設定會變暗，而且會出現下列訊息： **部分設定是由您的組織隱藏或管理**。
