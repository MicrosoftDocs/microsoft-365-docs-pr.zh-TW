# <a name="buy-business-voice"></a>購買商務語音

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4Gryk?autoplay=false]

使用 Microsoft 365 商務語音，您可以撥打和接聽電話、使用音訊會議，以及設定通話功能表系統，將客戶路由傳送至右部門。 若要開始，您必須購買商務語音附加元件。

## <a name="try-it"></a>試試看吧！

1. 在 Microsoft 365 系統管理中心中，移至 [ **帳單**]，然後選取 [ **購買服務**]。
1. 向外移動，然後選擇 [ **附加** 元件]。 
1. 選取 [ **Microsoft 365 商務語音**]。

    如果您看不到它，您必須先購買包含 Microsoft 小組的商務訂閱。
1. 輸入您想要授權的使用者數目，然後選取 [ **立即查看**]。
1. 如有需要，請選取 [ **自動重新充電** ] 以購買通訊學分，並避免服務中斷。

    通訊信用信用用於國際通話、免付費電話號碼，以及其他通話紀要。
1. 選取 **[下一步]**。
1. 驗證您的支付方式。
1. 選取 [ **下單** ] 並 **繼續**。

您已成功新增商務語音附加元件。
