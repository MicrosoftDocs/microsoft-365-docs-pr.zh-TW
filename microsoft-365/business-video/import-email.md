# <a name="import-email-to-microsoft-365-for-business"></a>將電子郵件匯入 Microsoft 365 for business 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/40f7df36-9e24-44e5-8791-e9ed0dd8fd21?autoplay=false]

註冊 Microsoft 365 之後，您可能仍然會有電子郵件前往舊帳戶;例如，如果您使用 Gmail 或 Outlook 帳戶進行工作。 您可以繼續使用新舊帳戶，也可以將舊帳戶中的資料匯入新帳戶。

## <a name="try-it"></a>試試看吧！

### <a name="step-1-export-your-old-email"></a>步驟1：匯出舊的電子郵件。

1. 在 Outlook 中，  **選取 [** 檔案]、[開啟] 和 [ **匯出**]，然後匯 **入/匯出**。
2. 選取 [  **匯出至** 檔案]，然後選取  **[下一步]**。
3. 選取 [  **Outlook 資料檔案** ]，然後選取  **[下一步]**。
4. 選取舊的電子郵件帳戶。
5. 請確認您包含子資料夾。 然後選取  **[下一步]**。
6. 選取  **[流覽]**，然後移至容易記住的位置，例如您的桌面。 選取  **[確定**]，然後 **完成**。
7. 保留密碼欄位為空白，然後選擇 **[確定]**。

    您的舊電子郵件帳戶已匯出。

### <a name="step-2-import-your-old-email-into-your-new-email-account"></a>步驟2：將您的舊電子郵件匯入新的電子郵件帳戶

1. 在 Outlook 中，  **選取 [** 檔案]、[ **開啟**]、[  **匯出**]，然後匯 **入/匯出**。
2. 選取 [  **從其他程式或** 檔案匯入]，然後選取  **[下一步]**。
3. 選取 [  **Outlook 資料檔案** ]，然後選取  **[下一步]**。
4. 選取  **[流覽]**，然後移至您剛才匯出的檔案。 選取它，然後選取 [  **開啟**]，然後選取 **[下一步]**。
5. 請確認您包含子資料夾，而且您要將專案匯入新電子郵件帳戶中的相同資料夾中。 選擇  **[完成]**。
