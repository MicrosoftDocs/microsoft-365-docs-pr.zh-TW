# <a name="import-email-to-microsoft-365-for-business"></a>將電子郵件匯出至商務用 Microsoft 365 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/40f7df36-9e24-44e5-8791-e9ed0dd8fd21?autoplay=false]

註冊 Microsoft 365 之後，您可能仍有電子郵件進入舊帳戶;例如，如果您工作用 Gmail 或 Outlook 帳戶。 您可以繼續使用新舊帳戶，也可以將資料從舊帳戶將資料匯出到新帳戶。

## <a name="try-it"></a>試試看吧！

### <a name="step-1-export-your-old-email"></a>步驟 1：匯出舊的電子郵件。

1. 在 Outlook 中，  **選取檔案**、 **開啟和匯出**，然後 **匯出/匯出**。
2. 選取  **匯出至檔案，** 然後選取下一  **步**。
3. 選取  **Outlook 資料檔案，** 然後選取下一  **步**。
4. 選取您的舊電子郵件帳戶。
5. 確認您包含子資料夾。 然後選取下  **一步**。
6. 選取  **流覽**，然後前往容易記住的位置，例如您的桌面。 選取 **確定****，然後完成**。
7. 將密碼欄位保留空白，**然後選擇確定。**

    您的舊電子郵件帳戶已匯出。

### <a name="step-2-import-your-old-email-into-your-new-email-account"></a>步驟 2：將舊的電子郵件匯出至新的電子郵件帳戶

1. 在 Outlook 中，  **選取檔案**、 **開啟**、  **匯出**，然後 **匯出/匯出**。
2. 選取  **從另一個程式或檔案的匯出** ，然後選取下  **一步**。
3. 選取  **Outlook 資料檔案，** 然後選取下一  **步**。
4. 選取  **流覽**，然後前往您剛剛匯出的檔案。 選取它，然後選取開啟  **，** 然後選取下 **一個**。
5. 請確認您包含子資料夾，且您將專案輸入至新電子郵件帳戶中的同一個資料夾。 選擇  **完成**。
