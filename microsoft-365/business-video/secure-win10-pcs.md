# <a name="secure-windows-10-pcs"></a>保護 Windows 10 電腦

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4kcAF?autoplay=false]
 
## <a name="try-it"></a>試試看吧！  

在 Microsoft 365 商務版中設定裝置保護之後，請遵循下列步驟以保護您的 Windows 10 電腦。

1. 在 Microsoft 365 系統管理中心中  **，選擇設定**。
2. 在 **保護您 Windows 電腦的安全下**，選取的 **View。**
3. 請查閱 Windows  **10**  電腦安全性頁面上的資訊，以瞭解相關政策及對使用者的影響。
4. 選取  **開始使用**。
5. 在  **Windows 10 設定**  窗格中，查看選項並視需要調整。 將游標停留在工具提示上以深入瞭解該政策。
6. 在  **原則適用于哪些人？** ，選擇您的選擇將適用于貴組織的每個人或特定的安全性群組。
7. 選取  **儲存變更**。

Microsoft 365 商務版現在協助保護您的 Windows 10 電腦。
