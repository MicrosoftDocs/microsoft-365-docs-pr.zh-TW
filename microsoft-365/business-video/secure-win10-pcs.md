# <a name="secure-windows-10-pcs"></a>保護 Windows 10 電腦

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4kcAF?autoplay=false]
 
## <a name="try-it"></a>試試看吧！  

在 Microsoft 365 商務版中設定裝置保護之後，請遵循下列步驟來保護您的 Windows 10 電腦。

1. 在 Microsoft 365 系統管理中心，選擇 [**安裝**]。
2. 在 [**保護您的 Windows 電腦**] 底下，選取 [ **View**]。
3. 查看 [**保護您的 Windows 10 電腦**] 頁面上的資訊，讓您瞭解原則及其對使用者的影響。
4. 選取 [  **開始**]。
5. 在 [ **Windows 10 設定**] 窗格上，複查選項，並視需要調整。 將滑鼠停留在工具提示上方以深入瞭解原則。
6. 在 **[神秘是否應套用原則？** ，選擇是否要將您的選擇套用至組織中的所有人，還是套用至特定的安全性群組。
7. 選取 [  **儲存變更**]。

Microsoft 365 商務版現在會協助您保護 Windows 10 電腦。
