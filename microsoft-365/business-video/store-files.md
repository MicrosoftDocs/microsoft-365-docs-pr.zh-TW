# <a name="where-to-store-files"></a>儲存檔案的位置

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FTHX?autoplay=false]

在 Microsoft 365 中，您可以將工作儲存在商務用 OneDrive 或 SharePoint 網站上。 貴組織的每個人都擁有自己的商務用 OneDrive 文件庫，可以儲存個人工作檔案。 當您建立檔案並將其儲存到 OneDrive 資料夾時，檔案會同步處理至雲端，因此您可以隨時隨地存取這些檔案，包括網頁瀏覽器、電腦或行動裝置。 針對共用的檔案儲存空間和小組共同合作，請使用 SharePoint 網站。 SharePoint 網站也會同步到雲端，方便您隨時隨地存取。

## <a name="try-it"></a>試試看吧！

### <a name="store-or-share-files-with-onedrive"></a>使用 OneDrive 儲存或共用檔案

1. 在您的電腦上，使用 [檔案總管] 開啟 OneDrive。 或者，從 [Office 365](https://www.office.com/)的應用程式啟動器開啟 **OneDrive。**
2. 按一下共用，然後邀請其他人來查看檔或傳送連結，即可查看並上傳個人檔案，或共用文件或資料夾。 [深入了解](https://support.microsoft.com/office/9fcc2f7d-de0c-4cec-93b0-a82024800c07#os_type=onedrive_-_business)。

### <a name="collaborate-with-a-sharepoint-site"></a>使用 SharePoint 網站共同合作

1. 從 [Office 365，](https://www.office.com/)從應用程式啟動器開啟  **SharePoint，**  然後選取小組網站。
2. 將檔案新增到預設的檔位置，或建立更多檔位置，稱為文件庫。

您可以自訂 SharePoint 網站，為特定專案、小組或客戶建立子網站。 您也可以新增公司名稱、標誌及外觀與感覺，將網站轉換成企業內部網路。 [深入了解](https://support.microsoft.com/office/06bbadc3-6b04-4a60-9d14-894f6a170818)。
