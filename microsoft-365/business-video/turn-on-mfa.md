# <a name="turn-on-multifactor-authentication"></a>開啟多重要素驗證

## <a name="watch-turn-on-multifactor-authentication"></a>觀賞：開啟多重要素驗證

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2MuO3?autoplay=false]

多因素驗證表示您和您的員工必須提供一種以上的方式登入 Microsoft 365 是一種最簡單的方法來保護您的企業。

## <a name="try-it"></a>試試看吧！

1. 移至位於 [https://admin.microsoft.com](https://admin.microsoft.com) 的系統管理中心。
1. 選取 [**全部顯示**]，然後選擇 [ **Azure Active Directory 系統管理中心**]。
1. 選取 [ **Azure Active Directory**]、[**屬性**]、[**管理安全性預設值**]。
1. 在 [ **啟用安全性預設值**] 底下選取 **[是]** ，然後按一下 [ **儲存**]。

> [!NOTE]
> 針對新訂閱，會自動開啟安全的預設值。

在下一次選取的雇員登入時，系統會要求他們設定其電話上的 Microsoft Authenticator 應用程式，以進行第二種形式的驗證。
