<!--from Overview-->

Microsoft 受管理的桌上型電腦為客戶提供包羅萬象的 IT 解決方案之資料的使用者及釋出 IT 資源的更重要的工作。我們已開發這項服務根據我們常聽到的意見反應：
- 我的使用者想要的最新技術。
- 我想要更容易預測的成本。
- 我要聚焦我核心商務。 
- 我正在淹死的複雜性。 
- 我想要更策略。 
- 很難尋找足夠 IT 人才。  

Microsoft 受管理的桌上型電腦提供最佳使用者經驗搭配最新技術，由 Microsoft 所有備份。
