# <a name="adjust-conditional-access"></a>調整條件式存取

如果您在組織中使用[條件式存取](https://docs.microsoft.com/azure/active-directory/conditional-access/overview)原則，您必須將它們設定成排除特定帳戶，以便 Microsoft 受管理的電腦可以正常運作。

若要這樣做，請執行下列步驟：

1. 請參閱 「 回復步驟 」 一節[How To： 規劃在 Azure Active Directory 條件式存取部署](https://docs.microsoft.com/azure/active-directory/conditional-access/plan-conditional-access#rollback-steps)。
2. 請依照那里排除所有原則的*現代化工作場所服務帳戶*群組。


如果您有使用條件式存取的任何問題，請連絡系統管理員[支援](../working-with-managed-desktop/admin-support.md)。

3. [指派授權](assign-licenses.md)

8. [部署應用程式](deploy-apps.md)
