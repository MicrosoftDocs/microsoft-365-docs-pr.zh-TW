# <a name="set-up-microsoft-managed-desktop-devices"></a>設定 Microsoft 受管理的電腦裝置

您可以在 Microsoft 受管理的電腦中使用新的和現有的裝置。

## <a name="to-obtain-new-devices"></a>取得新裝置

我們建議使用我們認可的其中一個裝置合作夥伴。 您可以與 Microsoft 帳戶連絡人合作，以取得更多有關設定裝置合作關係的說明。 不過，本質上，此程式就像這樣：

1. 在 [[車間作業 Windows 10 專業版商務裝置](https://www.microsoft.com/windowsforbusiness/view-all-devices)] 網站上篩選 Microsoft 受管理的電腦，以查看目前建議的裝置清單。
2. 定購您想要搭配相容的映射使用的裝置的一個或多個範例。 順序可能需要 [特定的順序步驟](../service-description/device-images.md)。
3. [驗證](validate-device.md) 範例裝置。
5. 成功驗證之後，請定購裝置，使用核准的裝置夥伴。
6. 一旦到達，請執行下列其中一項操作：
    - [自行註冊新裝置](register-devices-self.md)
    - 與合作夥伴合作以註冊裝置
7. [讓您的使用者準備好](get-started-devices.md)使用 Microsoft 受管理的電腦裝置

## <a name="to-reuse-existing-devices"></a>重複使用現有的裝置

1. 檢查您現有的裝置是否符合我們的 [裝置需求](../service-description/device-requirements.md)。 您也可以使用可下載的 [準備情況評估檢查](../get-ready/readiness-assessment-downloadable.md) 程式，以確認指定的裝置符合必要的需求。 
2. 如果您重複使用現有的裝置，您可能必須重新映射。 如需影像選項，請參閱 [Device images](../service-description/device-images.md)。
3. 選取您想要重複使用之裝置的其中一個或多個範例，然後 [進行驗證](validate-device.md)。
4. 驗證成功之後，請執行下列其中一項操作：
    - [自行註冊現有裝置](register-reused-devices-self.md)
    - 與合作夥伴合作以註冊裝置
5. [讓您的使用者準備好](get-started-devices.md)使用 Microsoft 受管理的電腦裝置
