# <a name="set-up-microsoft-managed-desktop-devices"></a>設定 Microsoft 受管理的電腦裝置

您可以在 Microsoft 受管理的桌面中使用新的和現有的裝置。

## <a name="to-obtain-new-devices"></a>取得新裝置

我們建議使用我們認可的其中一個裝置合作夥伴。 您可以與 Microsoft 帳戶連絡人合作，以取得更多有關設定裝置合作關係的說明。 不過，本質上，此程式就像這樣：

1. 複查 [目前核准的裝置](../service-description/device-list.md)清單。
2. 使用認可的裝置夥伴，定購裝置。
3. 一旦到達，請執行下列其中一項操作：
    - [自行註冊新裝置](register-devices-self.md)
    - 與合作夥伴合作以註冊裝置
4. [讓您的使用者準備好](get-started-devices.md) 使用 Microsoft 受管理的桌面裝置

## <a name="to-reuse-existing-devices"></a>重複使用現有的裝置

1. 查看 [目前核准之裝置](../service-description/device-list.md) 的清單，以識別哪些裝置已核准，可搭配 Microsoft Managed Desktop 使用。 如果您重複使用現有的裝置，您必須重新映射。
2. 請執行下列其中一項動作：
    - [自行註冊現有裝置](register-reused-devices-self.md)
    - 與合作夥伴合作以註冊裝置
3. [讓您的使用者準備好](get-started-devices.md) 使用 Microsoft 受管理的桌面裝置
