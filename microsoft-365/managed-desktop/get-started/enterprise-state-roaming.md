# <a name="enable-enterprise-state-roaming"></a>啟用企業狀態漫遊

[Enterprise 狀態漫遊](/azure/active-directory/devices/enterprise-state-roaming-overview)可讓使用者安全地將使用者和應用程式設定資料同步處理至雲端。 這表示不管使用者登入哪 Windows 裝置，都有相同的經驗。 例如，如果您將其中一個 Microsoft 受管理的電腦裝置更換為新裝置，它的外觀和行為會與最後一個裝置完全相同。 Enterprise狀態漫遊是可為使用者設定的 Microsoft 受管理的電腦服務選用的功能，且不會包含或管理 Microsoft 受管理的電腦的一部分。

若要啟用 Enterprise 狀態漫遊，請依照 Azure Active Directory 中的[啟用 Enterprise 狀態漫遊](/azure/active-directory/devices/enterprise-state-roaming-enable)中的步驟進行。

>[!NOTE]
>如果您啟用 Enterprise 狀態漫遊，您的慣用語言清單將會覆寫裝置安裝期間選取的語言。 雖然使用者可以輕鬆地修正這種情況，但最初可能會造成不一致的當地語系化體驗。 在設定裝置之前，判斷 Enterprise 狀態漫遊是否適合您的使用者。
