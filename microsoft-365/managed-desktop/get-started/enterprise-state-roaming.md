# <a name="enable-enterprise-state-roaming"></a>啟用企業狀態漫遊

為了獲得 Microsoft Managed Desktop 的最佳體驗，請啟用 [企業狀態漫遊](/azure/active-directory/devices/enterprise-state-roaming-overview)，讓使用者安全地將使用者和應用程式設定資料同步處理至雲端。 這表示不管使用者登入哪一個 Windows 裝置，都具有相同的體驗。 例如，如果您將其中一個 Microsoft 受管理的桌面裝置更換為新的桌面裝置，其外觀和行為會與最後一個裝置完全相同。

若要啟用企業狀態漫遊，請依照在 [Azure Active Directory 中啟用企業狀態漫遊](/azure/active-directory/devices/enterprise-state-roaming-enable)中的步驟進行，然後回到本檔。

如果您對企業狀態漫遊有任何困難，請與系統管理員 [支援](../working-with-managed-desktop/admin-support.md)人員聯繫。
