# <a name="support-for-microsoft-managed-desktop"></a>Microsoft 受管理電腦的支援

Microsoft 會提供主動和反應性的事件管理。 Microsoft 在受管理的桌面系統管理員入口網站中追蹤事件。 它們會根據 [嚴重性定義](../working-with-managed-desktop/admin-support.md#sev)進行分類。

客戶可以聯繫 Microsoft 受管理的桌面作業，以進行下列作業：
- Microsoft 受管理桌面租使用者或設定的資訊要求
- 變更 Microsoft 受管理桌面裝置設定的要求
- 報告事件或中斷

## <a name="whats-included"></a>包含內容？

Microsoft 受管理的桌面支援包括：

- 專屬於 Microsoft 受管理的桌面裝置的工程師小組
- 使用 Microsoft 受管理桌面裝置之使用者的支援選項
- 授與 Microsoft 受管理桌面裝置的有限系統管理存取，以管理 Microsoft 受管理的桌面裝置 

支援的產品：

- Windows 10 搭配 Microsoft Defender 高級威脅防護 
- 下列 Microsoft 365 應用程式適用于企業應用程式： Outlook、Word、PowerPoint、Excel、商務用 Skype 用戶端、Microsoft 團隊 
- 商務用 Microsoft Store 
- 商務用戶端的 OneDrive 

支援詳細資料：

- 目前：美國、加拿大 (不包括魁北克) 、英國、比利時、盧森堡、荷蘭、澳大利亞和紐西蘭 (24x7x365)  
- 英文是唯一支援的電話及與客戶交談的語言 
- 我們與您公司的技術支援人員合作，而不是取代。企業營運 (LOB) 應用程式、網路資源等，仍然由您的技術人員處理 
- 「Test」群組中的 microsoft 受管理桌面裝置和裝置不是 Microsoft Managed Desktop 的一部分，已超出範圍
