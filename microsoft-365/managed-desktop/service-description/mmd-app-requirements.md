# <a name="microsoft-managed-desktop-app-requirements"></a>Microsoft 受管理的桌面應用程式需求

<!--This topic is the target for aka.ms/app-req. This is aka link is used from EA agreement for MMD. do not delete.-->

<!--Application addendum -->
 
Microsoft 受管理的桌面要求我們使用特定的方法來管理裝置，以保證裝置的效能、可靠性及可維修性。 [！注意] 如果您確定 Microsoft 受管理的桌面所採用的功能將無法為您運作，您可以要求[服務方案例外](customizing.md)。


|管理區域  |Microsoft 受管理的桌面方法  |
|---------|---------|
|裝置設定或原則管理     |  Microsoft Intune       |
|應用程式管理     | Microsoft Intune 和公司入口網站        |
|驅動程式部署     |  裝置、Windows Update 或 Intune 隨附的驅動程式       |
|裝置安全性     | 請參閱[裝置安全性](security.md#device-security)      |
|身分識別與存取管理     | 請參閱身分[識別和存取管理](security.md#identity-and-access-management)        |
|網路安全性     | 請參閱[網路安全性](security.md#network-security)        |
|資訊安全性     |  請參閱[資訊安全性](security.md#information-security)       |
|資料恢復     | 商務用 OneDrive        |
|核心生產力     | Microsoft 365 Apps 企業版    |
|瀏覽器     | Microsoft Edge        |




Microsoft 受管理的桌面可能會監視受管理裝置上執行的其他軟體。 如果對系統安全性、效能或可靠性有負面影響，您可能需要要求服務方案例外。
