# <a name="reassign-profiles"></a>重新指派設定檔

您可以使用管理入口網站，變更指派給裝置的 [裝置設定檔](../service-description/profiles.md) 。

> [!IMPORTANT]
> 變更裝置的設定檔將會擦除裝置上的所有資料，並導致重新開機。 在繼續之前，請確定已選取正確的裝置，然後再備份您可能想要保留的任何資料。

您選取的裝置設定檔會套用到您在第一個步驟中選取的所有裝置。 若要將個別裝置移至不同的設定檔，您必須針對每個裝置設定檔重複此程式。 

1. 在 Microsoft 端點管理員中， **** 選取   左窗格中的 [裝置]。 在功能表的 [ **Microsoft 受管理的電腦**] 區段中，選取 [ **裝置**]。  
2. 選取您要修改之裝置的核取方塊。 
3. 選取 [ **變更裝置設定檔**];飛入隨即開啟。
4. 使用下拉式功能表選取新的裝置設定檔。
5. 選取 [ **變更設定檔**]。
