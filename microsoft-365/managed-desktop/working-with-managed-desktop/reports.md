# <a name="work-with-reports"></a>使用報表

Microsoft 受管理的桌面提供其系統管理員在您的租使用者中所能使用的報告，以瞭解各方面的設備人口。 您可以在 [**報告**] 底下的 [[管理] 入口網站](https://aka.ms/mmdportal)中存取報告。


## <a name="device-compliance-report"></a>裝置合規性報告

主視圖會顯示符合 Microsoft Intune 中設定之規範原則的 Microsoft 受管理桌面裝置百分比。 您也可以查看所有 Microsoft 受管理的桌面裝置的特定資訊，或只查看不符合合規性原則的裝置，並匯出具有相同資訊的檔案。

深入瞭解[裝置符合性報告](device-compliance-report.md)。

## <a name="device-inventory-report"></a>裝置庫存報告

[**裝置**視圖] 提供 [**匯出全部**] 索引標籤，以[下載詳細的庫存報告](device-inventory-report.md)。
