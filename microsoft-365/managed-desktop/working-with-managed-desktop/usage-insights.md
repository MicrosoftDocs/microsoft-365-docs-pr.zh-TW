# <a name="usage-insights"></a>使用情況深入解析
此視圖提供 Microsoft 受管理桌面裝置的使用度量。 在這些情況下，如果應用程式正在執行中且處於焦點，便會被視為「使用中」。

若要查看使用狀況資料，請選取 [ **使用狀況** ] 索引標籤。

![使用狀況窗格。 裝置使用狀況圖表 (百分比使用狀況與時間) 左上方。 右上方的 [應用程式使用狀況] 表格。 以欄為底的裝置清單表格，具有欄裝置名稱、型號、序列值、顯示名稱、使用者名稱、目前狀態 (使用中、低或非使用中) 、總裝置使用量（以小時為單位）及使用中天數。](../../media/insights_usage.png)

## <a name="device-usage"></a>裝置使用量

在 [ **裝置使用量** ] 區域中，我們會顯示隨時間變化的報告裝置使用量層級，以及已註冊的裝置數目報告資料。 針對此頁面上顯示的所有資料，您可以使用右上角的下拉式功能表變更最近14天或28天中報告的資料的間隔。

我們會使用下列條款來分類使用方式層級：

- **Active：** 使用方式至少為55小時
- **低：** 使用方式介於8到55小時之間
- **非** 使用中：使用不到8小時




## <a name="application-usage"></a>應用程式使用

[ **應用程式使用量** ] 區域會以使用順序及受管理裝置的各自使用時間，來詳述應用程式。 它也會顯示使用指定之應用程式的受管理裝置總數。 選取 [ **搜尋** ] 以找出尚未列出的特定應用程式。


## <a name="device-details"></a>裝置詳細資料
詳細資料區域提供特定裝置的資訊，包括總裝置時數和時段內的使用中天數。 使用篩選器，將視圖限制為「使用中」、「非使用中」或「未報告」使用模式的裝置。
