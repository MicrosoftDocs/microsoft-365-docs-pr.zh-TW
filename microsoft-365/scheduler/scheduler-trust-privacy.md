# <a name="trust-and-privacy-in-scheduler-for-microsoft-365"></a>Microsoft 365 中的計畫程式信任和隱私權

排程器是一項獨特的產品，其人工智慧是在使用者意圖（通常是由於多義性或內容性參考）而不是在使用者的意圖上自信時，與人工協助擴充。 

## <a name="policies"></a>原則

- 所有客戶內容都儲存在客戶的承租人中。
- 排程器是一般資料保護法規 (GDPR) 相容性。
- 所有客戶資料都是在 Microsoft 365 信任和隱私權界限內處理。
- 排程者的人體助理是由 Microsoft 支援人員/資料處理者的個人資訊和高度機密資訊 **& 隱私權保證驗證的安全性** 。 
- 排程器服務不會使用或處理電子郵件附件。
- 「調度者」服務不會使用或處理加密的電子郵件。
- 排程器不會監視會議召集人或出席者的行事曆或收件匣。
