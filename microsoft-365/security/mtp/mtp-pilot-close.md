您目前處於結案和摘要階段。

您剛執行進位記憶體指令模擬，在網域控制站上遠端執行程式碼。 您看過 Microsoft Defender for Endpoint 和 Microsoft Defender for Identity 如何偵測並建立惡意活動的警示。 您也會在 Microsoft 365 資訊安全中心入口網站中，看到不同來源的警示與其他關係資訊如何傳遞到單一事件。 遇到這類整合時，SOC 分析師可以進行調查並採取必要動作。 您也建立了進一步搜尋查詢，以識別使用者開啟或儲存附件並依據該查詢建立偵測的內輸入電子郵件。

所有測試都結束後，程式已經結束。

最後一個輸出結果應為：

- 已完成的計分卡
- 試驗結果的詳細報告
- 如何向前移動的決策

將最終輸出結果報告呈現給內部專案關係 (，這是您于) 和 Microsoft[](https://docs.microsoft.com/microsoft-365/security/mtp/prepare-mtpeval)連絡人在準備階段時所識別的。 如此一來，可確保您有任何意見可用來改善產品與檔。

希望您喜歡這項模擬。 開始在組織中大規模實施您學習到的資訊，以充分使用整合式安全性解決方案。
