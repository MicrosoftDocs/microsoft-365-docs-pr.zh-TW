# <a name="investigate-users-in-microsoft-365-security-center"></a>在 Microsoft 365 安全性中心調查使用者

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

[!INCLUDE [Prerelease](../includes/prerelease.md)]

適用於：

- Microsoft 365 Defender

在調查過程中，您可能會發現使用者已遭到破壞。

Microsoft 365 的「安全性中心」使用者頁面會結合 Microsoft Defender for Endpoint、Microsoft Defender 身分識別及 Microsoft Cloud App Security (的資訊，取決於您) 的授權。 此頁面是調查使用者和潛在事件的理想開始位置。
![使用者頁面](../../media/m3d-userpage.png)

此頁面顯示使用者安全性風險的特定資訊。 這包括有助於評估風險的分數、最近的事件及警示，也就是對使用者整體風險的貢獻等等。

您可以從 Microsoft 365 安全性中心的多個區域存取此頁面。 您可以從 [ **使用者** ] 索引標籤中的特定事件存取此頁面。有些警示可能包含使用者為特定的受影響資產。 您也可以搜尋使用者。  

深入瞭解如何 [在此 Cloud App Security 教學課程中](/cloud-app-security/tutorial-ueba#:~:text=To%20identify%20who%20your%20riskiest,user%20page%20to%20investigate%20them)調查使用者和潛在風險。
