# <a name="whats-coming-to-microsoft-secure-score"></a>Microsoft 安全分數即將得到什麼

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

您可以在 Microsoft https://security.microsoft.com/securescore [365](overview-security-center.md)資訊安全中心找到 Microsoft 安全分數。

## <a name="proposed-changes"></a>建議變更

我們近期內將進行一些變更，讓 [Microsoft Secure Score](microsoft-secure-score.md) 更代表您的安全性工作，並改善可用性。 您的分數和可能的分數上限可能會變更。

### <a name="no-planned-changes"></a>無規劃的變更

目前沒有任何規劃的變更。
