# <a name="configure-and-manage-microsoft-defender-for-endpoint-capabilities"></a>設定及管理 Microsoft Defender for Endpoint 功能

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

適用於：

- [適用於端點的 Microsoft Defender](https://go.microsoft.com/fwlink/p/?linkid=2069559)

設定及管理所有 Microsoft Defender for Endpoint 功能，以取得組織的最佳安全性保護。 


## <a name="in-this-section"></a>本節內容 
主題 | 描述 
:---|:---
[設定攻擊面降減功能](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-attack-surface-reduction) |  透過確定設定設定正確，並套用利用緩解技術，這些功能可讓攻擊和利用。 
[設定下一代保護](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/configure-windows-defender-antivirus-features) | 設定下一代保護以捕捉所有的新興威脅類型。
[設定 Microsoft 威脅專家功能](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-microsoft-threat-experts) | 設定和管理您想要如何從 Microsoft 威脅專家取得 cybersecurity 威脅情報。
[設定 Microsoft 365 Defender 整合](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/threat-protection-integration)| 設定與 Microsoft Defender for Endpoint 整合的其他解決方案。
[管理和 API 支援](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/management-apis)| 將提醒納入您的 SIEM，或使用 APIs 建立自訂警示。 建立並組建 Power BI 報告。 
[設定 Microsoft Defender 安全性中心設定](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/preferences-setup) |  設定與入口相關的設定，例如一般設定、高級功能、啟用預覽體驗和其他。
