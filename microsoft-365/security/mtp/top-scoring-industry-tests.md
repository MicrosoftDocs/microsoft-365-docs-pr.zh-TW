## <a name="microsoft-365-defender"></a>Microsoft 365 Defender

[Microsoft 365 Defender](https://docs.microsoft.com/microsoft-365/security/mtp/microsoft-threat-protection) 是一套統一的預先和外泄後企業防護套件。 It natively coordinates detection， prevention， investigation， and response across endpoints， identities， email， and applications to provide integrated protection against sophisticated attacks.

Microsoft 365 Defender 將 [Microsoft Defender for Endpoint、Microsoft](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp)Defender for Office [365、Microsoft](https://www.microsoft.com/microsoft-365/exchange/advance-threat-protection) [Defender for Identity、Azure](https://azure.microsoft.com/features/azure-advanced-threat-protection/) [Active Directory 身分](https://docs.microsoft.com/azure/active-directory/identity-protection/overview-identity-protection)識別保護及 Microsoft Cloud App [安全性](https://www.microsoft.com/microsoft-365/enterprise-mobility-security/cloud-app-security) 功能合併為單一解決方案。

### <a name="mitre-demonstrated-real-world-detection-response-and-protection-from-advanced-attacks"></a>MITRE：示範實際偵測、回應，以及防止進一步攻擊

因此，MITRE 測試方法的核心就是模擬實際攻擊，以瞭解解決方案是否可以妥善偵測及回應。 雖然測試著重在端點偵測與回應，但MITRE 的模擬 APT29 攻擊橫跨多個攻擊網域，建立除了端點保護之外，讓防護者更上手的機會。 使用 Microsoft 365 Defender 超出端點的 Microsoft 擴充後人可見度。

- ATT&Microsoft 365 Defender 的 CK 評估 - 2020 年 5 月：實際 [偵測的領先者](https://www.microsoft.com/security/blog/2020/05/01/microsoft-threat-protection-leads-real-world-detection-mitre-attck-evaluation/)

    Microsoft 365 Defender 在攻擊鏈階段提供近 100% 的涵蓋範圍。 它提供對攻擊者活動的開箱即用可見。 可見度大幅減少安全性營運中心和仰賴特定組配置變更的廠商解決方案之手動工作。 Microsoft 365 Defender 的可見度也最小，減少攻擊者的未發現操作能力。

## <a name="next-generation-protection"></a>下一代保護

## <a name="endpoint-detection--response"></a>端點偵測&回應

Microsoft Defender 端點 [偵測與回應](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/overview-endpoint-detection-response) 功能提供接近即時且可採取動作的進一步攻擊偵測。 安全性分析人員可以有效地排定警示的優先順序、深入了解入侵的全貌，並採取回應動作來補救威脅。

針對 Endpoint 的 EDR 和端點保護功能，Microsoft Defender 已收到來自產業測試和出版物的正面結果。 [SC 實驗室在](https://www.scmagazine.com/home/reviews/sc-product-reviews-endpoint-security/) 2020 年 6 月評定端點安全性工具，並給予 Microsoft Defender 支援端點 [5/5 顆星](https://www.scmagazine.com/review/microsoft-defender-advanced-threat-protection/)。 他們針對端點功能使用一套完整的安全性功能來保護組織，防範現代威脅大權，這表示 Microsoft Defender 支援端點功能。 SC 實驗室也將端點安全性解決方案識別為系統管理與統一。 他們也確認端點保護的端點偵測與回應功能，因為攻擊鏈現在受到解決方案完整涵蓋。

### <a name="mitre-industry-leading-optics-and-detection-capabilities"></a>MITRE：業界頂尖的配件和偵測功能

MITRE 已測試產品偵測目標攻擊群組 APT3 常用技術的能力 (，也稱為採用來拉式圖或 UPS) 。 為了隔離偵測功能，所有防護及防護功能均已關閉。 Microsoft 很高興成為根據 ATT&CK 架構註冊為MITRE 評估的其中一位 EDR 廠商。 目前這個架構已廣泛成為攻擊者手法和策略的最全面目錄。

- ATT&CK 型評估的 Microsoft Defender for Endpoint - 2018 年 12 月：領先[企業與偵測功能](https://www.microsoft.com/security/blog/2018/12/03/insights-from-the-mitre-attack-based-evaluation-of-windows-defender-atp/)  |  [分析](https://techcommunity.microsoft.com/t5/Windows-Defender-ATP/MITRE-evaluation-highlights-industry-leading-EDR-capabilities-in/ba-p/369831)

    Microsoft Defender for Endpoint 全面涵蓋整個攻擊鏈中的駭客技術。 重點內容包括遙測的強項、威脅情報的強大功能，以及透過機器學習、結果及行為監視所提供進的自動偵測。

## <a name="to-what-extent-are-tests-representative-of-protection-in-the-real-world"></a>測試在什麼程度代表實際的保護？

獨立安全性產業測試旨在以無偏見的方式評估最佳的防毒軟體與安全性產品。 不過，在本文中強調的評估中，Microsoft 看到一組範圍更大且更廣泛的威脅。 在平均月份中，Microsoft 的安全性產品識別出超過 100 萬個新威脅。 即使獨立測試者可以取得並測試其中 1% 的威脅，在 20 或 30 項產品中，這也是一百萬次測試。 換句話說，惡意惡意攻擊的廣大程度使得您難以評估防範實際威脅的保護品質。

Microsoft Defender for Endpoint 內[](https://cloudblogs.microsoft.com/microsoftsecure/2017/12/11/detonating-a-bad-rabbit-windows-defender-antivirus-and-layered-machine-learning-defenses)的功能提供額外的保護圖層，不會納入產業防毒軟體測試，並解決一些最新且最複雜的威脅。 與 Defender for Endpoint 的其餘人員隔離 AV，可以完整瞭解 Microsoft 安全性堆疊在真實世界中的運作方式。 例如，攻擊面縮減和端點偵測&回應功能可協助防止惡意程式碼從一開始進入裝置。 我們已證實 Microsoft [Defender for Endpoint 元件](https://query.prod.cms.rt.microsoft.com/cms/api/am/binary/RE2ouJA) 發現 Microsoft Defender 防毒軟體在這些業界測試中未接的樣本。 它更代表 Microsoft 安全性套件如何有效保護實際的客戶。
