# <a name="monitor-and-view-reports-in-the-microsoft-365-security-center"></a>Microsoft 365 安全中心內的監視器和檢視報告

在 Microsoft 365 安全性中心提供保護和安全性狀態的快速摘要在整個 Microsoft 365 環境。

資訊安全中心包含**報告**] 區段中的功能的卡片涵蓋各種安全性分析師和系統管理員追蹤其日常作業的一部分的區域的主機。 在向下鑽研，卡提供詳細的報告和在某些情況下，管理選項。

## <a name="customize-views"></a>自訂檢視

根據預設，卡分成分為下列類別：
  
* [身分識別](monitor-and-report-identities.md)的使用者帳戶和認證
* [資料](monitor-data.md)-電子郵件和文件內容
* [裝置](monitor-devices.md)的電腦、 行動電話和其他裝置
* [應用程式](monitor-apps.md)-程式與附加的線上服務

**Group by 主題**，切換至重新排列卡和其群組到下列：

* **風險**-反白顯示實體，例如帳戶和裝置，可能會有風險的卡片。 這些卡也反白顯示的可能來源的風險，例如新的威脅行銷活動和特殊權限的雲端應用程式  
* **偵測趨勢**反白顯示新的威脅偵測、 異常行為及原則違規的卡片
* **設定和健康情況**-卡涵蓋設定資料庫和部署的安全性控制，包括管理服務的裝置上架狀態
* **其他**的未分類的其他主題下的所有其他卡片
