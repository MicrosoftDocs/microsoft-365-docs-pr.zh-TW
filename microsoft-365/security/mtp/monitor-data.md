# <a name="data-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Microsoft 365 資訊安全中心的資料監控與報告

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**[資料]** 類別可協助追蹤可能導致未經授權就外洩資料的使用者活動。 它們是現有 DLP 策略報告以及協力廠商 DLP 策略比對報告所提供。

您可以看到：

- 從雲端應用程式共用最多檔案的使用者
- 發生多少 DLP 策略比對
- 報告了多少 DLP 原則覆寫或誤判
- 透過 Microsoft Cloud App 安全性的協力廠商雲端服務中發生多少 DLP 策略比對
