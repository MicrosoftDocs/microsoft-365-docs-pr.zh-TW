# <a name="integrate-servicenow-tickets-into-the-microsoft-365-security-center-and-compliance-center"></a>將 ServiceNow 票證整合到 Microsoft 365 資訊安全中心與規範中心

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

>[!CAUTION]
>**ServiceNow 連接器的預覽期間已結束**<br>
>已不再提供此功能。 感謝您的意見和持續支援，我們決定接下來的步驟。

ServiceNow 是一個熱門的雲端運算平臺，可協助公司管理企業營運的數位工作流程。 其 Now 平臺有 IT 工作流程、員工工作流程和客戶工作流程。 [深入瞭解 ServiceNow](https://www.servicenow.com/)

Microsoft 已與 ServiceNow 合作，讓 IT 系統管理員能更輕鬆管理其在這兩個平臺上的票證和工作。 [Microsoft 365](overview-security-center.md) 安全性中心和 [Microsoft 365](https://docs.microsoft.com/microsoft-365/compliance/microsoft-365-compliance-center) 合規性中心已增強在 ServiceNow 中原生建立和追蹤票證的能力。
