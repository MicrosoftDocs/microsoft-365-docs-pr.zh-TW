<span data-ttu-id="2c121-138">開始劇集之前，請先 [在 GitHub 上](https://github.com/microsoft/Microsoft-threat-protection-Hunting-Queries/tree/master/Webcasts/TrackingTheAdversary) 存取對應的 Kusto CSL 檔案，並將其內容複寫到高級搜尋查詢編輯器。</span><span class="sxs-lookup"><span data-stu-id="2c121-138">Before starting an episode, access the corresponding [Kusto CSL file on GitHub](https://github.com/microsoft/Microsoft-threat-protection-Hunting-Queries/tree/master/Webcasts/TrackingTheAdversary) and copy its contents to the advanced hunting query editor.</span></span> <span data-ttu-id="2c121-139">當您觀賞劇集時，您可以使用複製的內容來追蹤喇叭，並執行查詢。</span><span class="sxs-lookup"><span data-stu-id="2c121-139">As you watch an episode, you can use the copied contents to follow the speaker and run queries.</span></span> 

<span data-ttu-id="2c121-140">在 CSL 檔案中，下列摘要會顯示一組完整的指導方針，並標示為批註 `//` 。</span><span class="sxs-lookup"><span data-stu-id="2c121-140">The following excerpt from a CSL file shows a comprehensive set of guidance marked as comments with `//`.</span></span>

```kusto

// Contains
// - Account information associated with the logon
// - The device which the account logged onto
// - The process which performed the logon
// - Network information (for network logons)
// - Timestamp
```

<span data-ttu-id="2c121-141">同一個 CSL 檔案包含下列所示批註之前和之後的查詢。</span><span class="sxs-lookup"><span data-stu-id="2c121-141">The same CSL file includes queries before and after the comments as shown below.</span></span> <span data-ttu-id="2c121-142">若要 [在編輯器中使用多個查詢](advanced-hunting-query-language.md#work-with-multiple-queries-in-the-editor)執行特定查詢，請將游標移至該查詢，然後選取 [ **執行查詢** ]。</span><span class="sxs-lookup"><span data-stu-id="2c121-142">To run a specific query with [multiple queries in the editor](advanced-hunting-query-language.md#work-with-multiple-queries-in-the-editor), move the cursor to that query and select **Run query**.</span></span>   

```kusto
DeviceLogonEvents

// DeviceLogonEvents
// A table containing a row for each logon a device enrolled in Microsoft Defender for Endpoint
```
