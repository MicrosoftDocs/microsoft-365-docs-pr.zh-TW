# <a name="set-the-time-zone-for-microsoft-365-defender"></a>設定 Microsoft 365 Defender 的時區

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**適用於：**
- Microsoft 365 Defender



Microsoft 365 Defender 可使用您的本機時區或 UTC 來顯示日期和時間資訊。 選取的時區會套用至 Microsoft 365 安全性中心以下功能所顯示的所有日期和時間資訊：
- 事件
- 自動化調查和修復 (包含控制中心)
- 進階搜捕

若要為這些功能設定時區，請移至 **設定**  >  **Microsoft 365 Defender**  >  **時區**。
