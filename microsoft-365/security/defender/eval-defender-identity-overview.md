# <a name="evaluate-microsoft-defender-for-identity-overview"></a>評估 Microsoft Defender 的身分識別概述


**適用於：**
- Microsoft 365 Defender

本文概述啟用和試驗 Microsoft Defender 身分識別的程式。 開始此程式之前，請確定您已複習[評估 Microsoft 365 Defender](eval-overview.md)的整體程式，且已[建立 Microsoft 365 Defender 評估環境](eval-create-eval-environment.md)。 
<br>

使用下列步驟來啟用和試驗 Microsoft Defender 身分識別。

![將 Microsoft Defender 身分識別新增至 Defender 評估環境的步驟](../../media/defender/m365-defender-identity-eval-steps.png)

下表說明圖例中的步驟。

| |步驟  |描述  |
|---------|---------|---------|
|1|[回顧架構需求和重要概念](eval-defender-identity-architecture.md)    | 瞭解適用于身分識別架構的 Defender，並確定您的環境符合架構必要條件。       |
|2 |[啟用評估環境](eval-defender-identity-enable-eval.md)     |   遵循步驟來設定評估環境。      |
|3 |[設定試驗](eval-defender-identity-pilot.md)     |   深入瞭解身分識別環境的基準設定，並試用身分識別教學課程的 Defender。     |
