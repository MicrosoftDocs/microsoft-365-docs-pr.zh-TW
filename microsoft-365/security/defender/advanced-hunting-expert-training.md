<span data-ttu-id="40df3-150">在 CSL 檔案中，下列摘要會顯示一組完整的指導方針，並標示為批註 `//` 。</span><span class="sxs-lookup"><span data-stu-id="40df3-150">The following excerpt from a CSL file shows a comprehensive set of guidance marked as comments with `//`.</span></span>

// - Network information (for network logons)
