# <a name="enable-and-pilot-microsoft-defender-for-office-365"></a>為 Office 365 啟用和試驗 Microsoft Defender

**適用於：**
- Microsoft 365 Defender

本文概述為 Office 365 啟用和試驗 Microsoft Defender 的程式。 開始此程式之前，請確定您已複習[評估 Microsoft 365 Defender](eval-overview.md)的整體程式，且已[建立 Microsoft 365 Defender 評估環境](eval-create-eval-environment.md)。 
<br>

使用下列步驟來啟用和試驗 Microsoft Defender 以供 Office 365。

![將 Microsoft defender for Office 新增至 Defender 評估環境的步驟](../../media/defender/m365-defender-office-eval-steps.png)

下表說明圖例中的步驟。

| |步驟  |描述  |
|---------|---------|---------|
|1|[回顧架構需求和重要概念](eval-defender-office-365-architecture.md)    | 瞭解 Office 架構的 Defender，並確定您的 Exchange Online 環境符合架構必要條件。       |
|2 |[啟用評估環境](eval-defender-office-365-enable-eval.md)     |   遵循步驟來設定評估環境。      |
|3 |[設定試驗 ](eval-defender-office-365-pilot.md)    |    建立試驗群組、設定保護，並熟悉主要功能及儀表板。     |
