# <a name="evaluate-microsoft-defender-for-endpoint-overview"></a>評估 Microsoft Defender for Endpoint 綜述

**適用於：**

- Microsoft 365 Defender


本文概述啟用和試驗 Microsoft Defender for Endpoint 的程式。 開始此程式之前，請確定您已複習[評估 Microsoft 365 Defender](eval-overview.md)的整體程式，且已[建立 Microsoft 365 Defender 評估環境](eval-create-eval-environment.md)。 
<br>

使用下列步驟來啟用和試驗 Microsoft Defender for Endpoint。

![將 Microsoft Defender for Endpoint 新增至 Defender 評估環境的步驟](../../media/defender/m365-defender-endpoint-eval-steps.png)


下表說明圖例中的步驟。

 |步驟   |描述
|---------|---------|
| [步驟1。回顧架構需求和重要概念](eval-defender-endpoint-architecture.md)    | 瞭解適用于端點架構和功能的 Defender。       |
|[步驟2。啟用評估環境](eval-defender-office-365-enable-eval.md)     |   遵循步驟來設定評估環境。      |
|[步驟3。設定試驗 ](eval-defender-office-365-pilot.md)    |    請確認您的試驗群組、執行類比，並熟悉主要功能與儀表板。     |
