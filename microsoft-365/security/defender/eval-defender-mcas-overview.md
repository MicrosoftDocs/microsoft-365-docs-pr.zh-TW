# <a name="evaluate-microsoft-cloud-app-security"></a>評估 Microsoft Cloud App Security

**適用於：**
- Microsoft 365 Defender


本文概述啟用和試驗 Microsoft Cloud App Security 及 Microsoft 365 Defender 的處理常式。 開始此程式之前，請確定您已複習[評估 Microsoft 365 Defender](eval-overview.md)的整體程式，且已[建立 Microsoft 365 Defender 評估環境](eval-create-eval-environment.md)。 
<br>

使用下列步驟來啟用和試驗 Microsoft cloud app security。

![將 Microsoft defender for Office 新增至 Defender 評估環境的步驟](../../media/defender/m365-defender-office-eval-steps.png)



|步驟  |描述  |
|---------|---------|
|[回顧架構需求和重要概念](eval-defender-mcas-architecture.md)    | 瞭解雲端 App 安全性架構，以及它如何與 Microsoft 365 Defender、Microsoft Defender 端點和 Azure Active Directory 整合。        |
|[啟用評估環境](eval-defender-mcas-enable-eval.md)     | 連線至入口網站、設定與使用 Defender 身分識別和/或組織網路裝置的整合，以及開始查看和管理雲端應用程式。         |
|[設定試驗 ](eval-defender-mcas-pilot.md)    | 將您的部署範圍設定為特定使用者群組、設定條件式存取應用程式控制，並嘗試保護您的環境的教學課程。       |
