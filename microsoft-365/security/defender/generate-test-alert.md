# <a name="generate-a-test-alert-in-your-microsoft-365-defender-evaluation-lab"></a>在 Microsoft 365 Defender 評估實驗室中產生測試警示  

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**適用於：**
- Microsoft 365 Defender

現在，您已完成 Microsoft 365 Defender 評估實驗室安裝和設定，您可以執行威脅攻擊類比，以體驗整合安全性解決方案的偵測、警示關聯及自我修復修復功能。  

1. 登入 https://security.microsoft.com
2. 在首頁上，尋找標為 [ **類比**] 的麻將牌。  如需在 Microsoft 365 Defender 中建立測試事件的步驟，請參閱逐步說明手冊。
