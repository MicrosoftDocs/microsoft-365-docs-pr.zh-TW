# <a name="how-to-help-ensure-that-a-message-isnt-marked-as-spam"></a>如何確保郵件不會被標示為垃圾郵件

如果您是 Exchange Online 或 Exchange Online Protection (EOP) 系統管理員，您可以協助確保透過服務傳送的電子郵件訊息不會被標示為垃圾郵件。 若要在[Office 365 中建立安全寄件者清單](create-safe-sender-lists-in-office-365.md)，請參閱更新後的秘訣和程式的版本。
