# <a name="manage-recipients-in-standalone-eop"></a>管理獨立 EOP 中的收件者

沒有 Exchange Online 信箱的獨立 Exchange Online Protection （EOP）組織支援下列類型的收件者：

- **郵件使用者**：郵件使用者是獨立 EOP 組織中的使用者帳戶基本類型。 郵件使用者在您的 EOP 組織中具有登入認證，但他們有外部電子郵件地址（其信箱位於您的 EOP 組織之外）。

  如需在 EOP 中管理郵件使用者的相關資訊，請參閱[管理 EOP 中的郵件使用者](manage-mail-users-in-eop.md)。

- **群組**：您可以建立下列類型的群組：

  - 通訊群組
  - 擁有郵件功能的安全性群組

  如需在 EOP 中管理群組的相關資訊，請參閱[管理 EOP 中的群組](manage-groups-in-eop.md)。
