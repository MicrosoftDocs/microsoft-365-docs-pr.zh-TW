# <a name="manage-schedules-for-multiple-reports-in-the-security-amp-compliance-center"></a>在安全性&amp;與合規性中心管理多個報告的計畫

在安全性&amp;與合規性中心中，有數個[報告和洞察力](reports-and-insights-in-security-and-compliance.md)可協助您的組織的安全性小組緩解並解決組織面臨的威脅。 如果您是組織的安全性小組成員，則可以管理一或多個報告的排程。 
  
## <a name="manage-schedules-for-reports"></a>管理報表的排程

> [!IMPORTANT]
> 確定您具備[安全性&amp;與合規性中心所指派](permissions-in-the-security-and-compliance-center.md)的必要許可權。 一般說來，全域管理員、安全性管理員及安全性讀者可以存取安全&amp;規範中心內的報告。 
  
![在 [安全性&amp;與合規性中心] \>中，選擇 [報表管理排程]](../../media/efa5e2f9-bf73-4f85-acea-f1ca7e2bca5e.png)

1. 移至 [https://protection.office.com](https://protection.office.com) 並登入。 這會帶您前往安全性 & 規範中心。

2. 在安全性&amp;與合規性中心，移至 [**報告** \> ] [**管理**排程]。
    
3. 選取清單中的項目。
    
4. 複查報表排程的資訊。
    
5. 視需要編輯或刪除排程，然後按一下 [**關閉**]。
