<span data-ttu-id="70349-105">在所有 Microsoft 365 組織中，Exchange Online Protection (EOP) 會掃描所有內送郵件是否有垃圾郵件、惡意程式碼及其他威脅。</span><span class="sxs-lookup"><span data-stu-id="70349-105">In all Microsoft 365 organizations, Exchange Online Protection (EOP) scans all incoming messages for spam, malware, and other threats.</span></span> <span data-ttu-id="70349-106">這些掃描的結果會新增至郵件中的下列標頭欄位：</span><span class="sxs-lookup"><span data-stu-id="70349-106">The results of these scans are added to the following header fields in messages:</span></span>
