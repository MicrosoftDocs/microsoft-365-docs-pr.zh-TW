# <a name="evaluate-microsoft-365-defender-for-endpoint-overview"></a>評估端點概覽的 Microsoft 365 Defender

**適用於：**

- Microsoft 365 Defender

![評估和試驗過程 Microsoft 365 Defender](../../media/defender/m365-defender-eval-process.png)

全面的安全性產品評估可能是一個複雜的程式，在實際執行端對端攻擊模擬之前，需要繁瑣的環境和裝置設定。 新增至複雜性是追蹤評估期間反映類比活動、警示和結果的難度。

Microsoft Defender for Endpoint 評估實驗室旨在消除裝置和環境設定的複雜性。 這可讓您：

- 重點是評估平臺的功能
- 執行模擬
- 請參閱動作中的預防、偵測和修正功能
<br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4woug]

使用簡化的設定體驗，您可以將重點放在執行您自己的測試案例和預先類比，以查看如何執行的 Defender for Endpoint。

您將擁有平臺強大功能的完整存取權，例如自動化調查、高級搜尋和威脅分析，可讓您測試已提供的 Defender for Endpoint 的綜合保護堆疊。

您可以將預先設定的 Windows 10 或 Windows 伺服器2019裝置，新增至適當的作業系統版本和正確的安全性元件，以及已安裝 Office 2019 Standard。

您也可以安裝威脅模擬器。 Defender for Endpoint 已與業界領先的威脅類比平臺合作，協助您測試用於端點功能的 Defender，而不需要離開入口網站。

 安裝您慣用的模擬器、在評估實驗室內執行案例，然後立即查看平臺的執行方式。 這項功能全都可免費取得，不需要額外收費。 您也可以輕鬆存取廣泛的模擬陣列，您可以從類比目錄存取及執行。
