# <a name="supported-operating-systems-and-platforms---threat-and-vulnerability-management"></a>支援的作業系統和平臺-威脅和弱點管理

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**適用於：**

- [適用於端點的 Microsoft Defender](https://go.microsoft.com/fwlink/?linkid=2154037)
- [威脅與弱點管理](next-gen-threat-and-vuln-mgt.md)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)

>想要體驗 Microsoft Defender for Endpoint？ [註冊免費試用版。](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-portaloverview-abovefoldlink)

開始之前，請確定您符合下列威脅和弱點管理的作業系統或平臺必要條件，讓裝置中的活動正確地進行。

>[!NOTE]
>受支援的威脅和弱點管理系統和平臺可能與 [Microsoft Defender For Endpoint list 的最低需求](minimum-requirements.md) 不同。

作業系統 | 安全性評估支援
:---|:---
Windows 7 | 作業系統 (OS) 弱點
Windows 8.1 | 作業系統 (OS) 弱點<br/>軟體產品弱點<br/>作業系統 (OS) 設定評估<br/>安全性控制設定評估<br/>軟體產品設定評估 |
Windows 10，版本1607-1703 | 作業系統 (OS) 弱點
Windows 10，版本1709或更新版本 |作業系統 (OS) 弱點<br/>軟體產品弱點<br/>作業系統 (OS) 設定評估<br/>安全性控制設定評估<br/>軟體產品設定評估
Windows Server 2008 R2 | 作業系統 (OS) 弱點<br/>軟體產品弱點<br/>作業系統 (OS) 設定評估<br/>安全性控制設定評估<br/>軟體產品設定評估
Windows Server 2012 R2 | 作業系統 (OS) 弱點<br/>軟體產品弱點<br/>作業系統 (OS) 設定評估<br/>安全性控制設定評估<br/>軟體產品設定評估
Windows Server 2016 | 作業系統 (OS) 弱點<br/>軟體產品弱點<br/>作業系統 (OS) 設定評估<br/>安全性控制設定評估<br/>軟體產品設定評估
Windows Server 2019 | 作業系統 (OS) 弱點<br/>軟體產品弱點<br/>作業系統 (OS) 設定評估<br/>安全性控制設定評估<br/>軟體產品設定評估
macOS 10.14 "Mojave" 以上 | 作業系統 (OS) 弱點<br/>軟體產品弱點 
Linux | 不支援的 (計畫)
