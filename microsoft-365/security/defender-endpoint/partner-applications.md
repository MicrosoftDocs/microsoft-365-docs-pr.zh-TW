# <a name="partner-applications-in-microsoft-defender-for-endpoint"></a>Microsoft Defender for Endpoint 中的夥伴應用程式 

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**適用於：**
- [適用於端點的 Microsoft Defender](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)


> 想要體驗適用於端點的 Microsoft Defender 嗎？ [注册免費試用版。](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-exposedapis-abovefoldlink)

Defender for Endpoint 支援協力廠商應用程式，協助增強平臺的偵測、調查和威脅智慧功能。


支援協力廠商解決方案，可協助您使用 Microsoft Defender for Endpoint 來進一步簡化、整合及協調其他廠商的防禦措施;讓安全小組能夠有效地回應新式威脅。

Microsoft Defender for Endpoint 會與現有的安全性解決方案無縫整合。 整合可提供下列解決方案的整合：

- SIEM
- 票據和 IT 服務管理解決方案
- 受管理的安全性服務提供者 (MSSP) 
- IoC 指示 ingestions 及符合
- 根據外部警示的自動裝置調查和修正
- 與安全性 orchestration 和 automation 回應的整合 (SOAR) 系統

## <a name="siem-integration"></a>SIEM 整合
端點的 Defender 可透過各種方法來支援 SIEM 整合。 這可能包括特殊的 SIEM 系統介面及現成的連接器、一般警示 API 啟用自訂執行，以及啟用警示狀態管理的動作 API。  如需詳細資訊，請參閱 [ENABLE SIEM integration](enable-siem-integration.md)。


## <a name="ticketing-and-it-service-management"></a>票證和 IT 服務管理 
票證發放解決方案整合有助於執行手動和自動回應程式。 當使用警示 API 來關閉票證時，終結點可協助您自動建立票證，並解決警示。 

## <a name="security-orchestration-and-automation-response-soar-integration"></a>SOAR) 整合的安全性 orchestration 和 automation 回應 ( 
Orchestration 方案可協助您建立行動手冊，並整合用於 Endpoint APIs 的資料模型和動作，以對協調回應（如裝置資料的查詢、觸發器裝置隔離、封鎖/允許、解析警示等等）。 

## <a name="external-alert-correlation-and-automated-investigation-and-remediation"></a>外部提醒關聯性及自動化調查和修正  
Defender for Endpoint 提供了獨特的自動化調查和修正功能，以在規模中促進事件回應。
  
整合自動化調查和回應功能及其他解決方案（如 IDS 和防火牆）可協助您解決警報，並將網路和裝置信號關聯的複雜性降到最低，以有效簡化裝置上的調查和威脅修正動作。  


外部提醒可以推入至 Defender for Endpoint。 這些警示會並排顯示，以及來自于來自 Defender for Endpoint 的其他裝置型提醒。 此視圖提供完整的警示內容，並可揭示攻擊的完整故事。  

## <a name="indicators-matching"></a>標記相符
您可以使用來自提供者和聚合器的威脅智慧，以維護和使用 (IOCs) 的損等指示器。


IoCs 的 Defender 可讓您透過關聯豐富遙測來建立警示，與這些解決方案整合，並作用於。  您也可以使用預防和自動化的回應功能封鎖執行，並在有相符的情況時採取修正動作。


目前的 Defender for Endpoint 支援 IOC 比對檔案和網路指示器。 封鎖可支援檔指示器。  

## <a name="support-for-non-windows-platforms"></a>支援非 Windows 平臺
用於 Windows 和非 Windows 平臺（包括行動裝置）的 Defender for Endpoint 提供集中的安全性作業體驗。 您可以在入口網站中查看不同支援作業系統 (OS) 中的警示，並更好地保護組織的網路。
