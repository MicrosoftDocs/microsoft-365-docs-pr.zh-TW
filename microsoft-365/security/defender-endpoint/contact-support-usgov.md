# <a name="contact-microsoft-defender-for-endpoint-support-for-us-government-customers"></a>聯繫 Microsoft Defender for US 政府客戶的端點支援

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**適用於：**
- [適用於端點的 Microsoft Defender](https://go.microsoft.com/fwlink/?linkid=2154037)

端點的 Defender 最近已升級支援程式，以提供更新式和高級的支援經驗。

## <a name="using-the-right-portal"></a>使用右側入口網站
為了開啟支援案例，您必須登入 Microsoft Defender for Endpoint 入口網站：

環境 | 入口網站 URL
:---|:---
在商業上的 GCC-M | [https://securitycenter.microsoft.com](https://securitycenter.microsoft.com)
GCC-M | [https://gcc.securitycenter.microsoft.us](https://gcc.securitycenter.microsoft.us)
GCC-H | [https://securitycenter.microsoft.us](https://securitycenter.microsoft.us)
DoD | [https://securitycenter.microsoft.us](https://securitycenter.microsoft.us)

如果您無法登入入口網站，您也可以使用 [電話](https://docs.microsoft.com/microsoft-365/admin/contact-support-for-business-products?view=o365-worldwide&tabs=phone&preserve-view=true)開啟支援案例。

## <a name="opening-a-support-case"></a>開啟支援案例
如需先決條件和指示，請參閱 [聯繫 Microsoft Defender For Endpoint support](contact-support.md)。
