# <a name="alerts-queue-in-microsoft-defender-security-center"></a>Microsoft Defender 資訊安全中心中的警示佇列

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**適用於：**
- [適用於端點的 Microsoft Defender](https://go.microsoft.com/fwlink/p/?linkid=2154037)

> 想要體驗 Defender for Endpoint？ [注册免費試用版。](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-exposedapis-abovefoldlink)

瞭解您可以如何查看和管理佇列，以便您可以有效調查實體（如裝置、檔案或使用者帳戶）上所看到的威脅。

## <a name="in-this-section"></a>本節內容
主題 | 描述 
:---|:---
[查看及組織警示佇列](alerts-queue.md) | 顯示網路中已標示的警示清單。
[管理警示](manage-alerts.md) | 瞭解如何管理提醒，例如變更其狀態、將其指派給安全性作業成員，以及查看警示的歷程記錄。
[調查警示](investigate-alerts.md)| 調查影響網路的警示，瞭解其含義及解決方法。
[調查檔案](investigate-files.md)| 調查與特定警示、行為或事件相關聯之檔案的詳細資料。 
[調查裝置](investigate-machines.md)| 調查與特定警示、行為或事件相關聯之裝置的詳細資料。 
[調查 IP 位址](investigate-ip.md) | 檢查您網路中裝置間可能的通訊和外部網際網路通訊協定 (IP) 位址。
[調查網域](investigate-domain.md) | 調查網域，查看您網路中的裝置和伺服器是否已與已知惡意的網域進行通訊。 
[調查使用者帳戶](investigate-user.md) | 識別具有最活躍之警示的使用者帳戶，並調查可能遭到破壞之認證的案例。
