# <a name="access-the-microsoft-defender-for-endpoint-community-center"></a>存取適用於端點的 Microsoft Defender 社群中心

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**適用於：**
- [適用於端點的 Microsoft Defender](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)

>想要體驗 Defender for Endpoint？ [注册免費試用版。](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-assignaccess-abovefoldlink)

「！附注」端點社區中心是一個位置，可讓社區成員瞭解、共同作業和分享產品的經驗。 

您可以探索幾個空格，以瞭解特定資訊：
- 公告 
- 新增功能
- 威脅情報


您可以透過數種方式存取社區中心：
- 在 Microsoft Defender 安全性中心導覽窗格中，選取 [ **社區中心**]。  隨即會開啟新的瀏覽器索引標籤，並帶您前往 Endpoint 技術社區頁面。 
- 透過 [Microsoft Defender For Endpoint Tech 社區](https://techcommunity.microsoft.com/t5/Windows-Defender-Advanced-Threat/ct-p/WindowsDefenderAdvanced) 頁面存取社區


您可以立即查看和讀取已在社區中發表的交談。 

若要取得群組內的完整體驗，例如能夠對帖子發表評論，您必須加入社區。 如需如何開始 Microsoft 技術群組的詳細資訊，請參閱 [Microsoft 技術社區：快速](https://techcommunity.microsoft.com/t5/Getting-Started/Microsoft-Tech-Community-Getting-Started-Guide/m-p/77888#M15)入門。
