## <a name="turn-on-preview-features"></a>開啟預覽功能

您將可以存取即將推出的功能，您可以在其中提供意見反應，以協助改善整體體驗，使其成為一般可用功能。

開啟預覽體驗設定，成為第一批嘗試即將推出的功能的人。

1. 在功能窗格中，選取 [**設定**] [  >  **高級功能**  >  **預覽功能**]。

2. **在 [開啟**] 和 [**關閉**] 間切換設定，然後選取 [**儲存喜好** 設定]。

## <a name="preview-features"></a>預覽功能

預覽版包含以下功能：

- [Web 內容篩選](web-content-filtering.md) <br> Web 內容篩選是 Microsoft Defender for Endpoint 中的 web 保護功能的一部分。 它可讓您的組織根據其內容類別別來追蹤和控制網站的存取權。 許多這類網站雖然並非惡意，但因相容性法規、頻寬使用量或其他考慮而可能會造成問題。

- [裝置健康情況和符合性報告](machine-reports.md) <br/> 裝置健康情況和合規性報告可提供組織中裝置的高層級資訊。

- [資訊保護](information-protection-in-windows-overview.md)<BR>
資訊保護是 Microsoft 365 企業版套件不可或缺的一部分，提供智慧防護以保持敏感性資料安全，同時在工作場所中啟用生產力。 Microsoft Defender for Endpoint 在 Microsoft 威脅防護中無縫整合，可為 Windows 裝置提供完整且完整的資料遺失防護 (DLP) 解決方案。

    >[!NOTE]
    >可從 Windows 10 版本1809部分取得。

- [板載 Windows Server 2019](https://docs.microsoft.com/microsoft-365/security/defender-endpoint/configure-server-endpoints#windows-server-version-1803-and-windows-server-2019) <BR> Microsoft Defender for Endpoint 現在新增 Windows Server 2019 的支援。 您可以使用 Windows 10 用戶端裝置的相同方法，將 Windows Server 2019 板載。
