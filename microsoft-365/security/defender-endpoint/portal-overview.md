# <a name="microsoft-defender-security-center-portal-overview"></a>Microsoft Defender 安全中心入口網站概述

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**適用於：**
- [適用於端點的 Microsoft Defender](https://go.microsoft.com/fwlink/p/?linkid=2146631)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)


>想要體驗 Defender for Endpoint？ [註冊免費試用版。](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-portaloverview-abovefoldlink) 

企業安全性小組可以使用 Microsoft Defender 安全中心來監控及協助回應潛在的潛在持續威脅活動或資料違規警報。

您可以使用 [Microsoft Defender 安全中心](https://securitycenter.windows.com/) 來執行下列作業：

- 從您的端點查看、排序及會審警示
- 搜尋有關觀察到的指標的詳細資訊，例如檔案及 IP 位址
- 變更 Microsoft Defender 的端點設定，包括時區及複查授權資訊

## <a name="microsoft-defender-security-center"></a>Microsoft Defender 安全中心

當您開啟入口網站時，您會看到：

-  (1) 導覽窗格 (選取功能窗格頂端的水平線以顯示或隱藏它) 
-  (2) 搜尋、社區中心、當地語系化、協助和支援、意見反應

 ![Microsoft Defender for Endpoint 入口網站](images/mdatp-portal-overview.png)

> [!NOTE]
> 只有在您的裝置使用 Microsoft Defender 防病毒作為預設即時保護的反惡意軟體產品時，才會顯示惡意程式碼相關的偵測。

您可以使用所有區段中可用的功能表選項來流覽入口網站。 請參閱下表，以取得每個區段的描述。

範圍 | 描述
:---|:---
**(1) 導覽窗格** | 使用功能窗格移動 **儀表板**、 **事件**、 **裝置清單**、 **警示佇列**、 **自動調查**、 **高級搜尋**、 **報告**、 **合作夥伴 & APIs**、 **威脅 & 弱點管理**、 **評估與教程**、 **服務健康** 情況、設定 **管理** 及 **設定**。 選取導覽窗格頂端的水平線，以顯示或隱藏它。
**儀錶 板** | 存取使用中的自動調查、作用中警示、自動調查統計資料、有風險的裝置、具有感應器問題的裝置、服務健康情況、偵測來源及每日裝置報告儀表板的狀態。
**事件** | 查看已匯總為事件的警示。
**裝置清單** | 顯示架至 Defender for Endpoint 的裝置清單、某些有關這些裝置的相關資訊，以及其曝光和風險層級。
**警示佇列** | 從組織中的裝置，查看所產生的警示。
**自動化調查** | 顯示已在網路中執行的自動化調查、觸發警示、每項調查的狀態和其他詳細資料，例如調查開始和調查的期間。
**進階搜捕** | 「高級搜尋」可讓您使用強大的搜尋和查詢工具，主動搜尋並調查整個組織。
**報告** | 查看詳細描述威脅防護、裝置健康情況和合規性、web 保護及弱點的圖形。
**合作夥伴 & APIs** | View the 支援的合作夥伴連線，增強平臺的偵測、調查和威脅智慧功能。 您也可以查看已連接的應用程式、API explorer、API 使用方式一覽及資料匯出設定。
**威脅 & 弱點管理** | 查看適用于裝置的 Microsoft 安全分數、披露分數、公開的裝置、易受攻擊的軟體，並針對主要安全性建議採取行動。
**評估與示教** | 管理測試裝置、攻擊模擬及報告。 透過試用環境中的引導式試驗，瞭解並體驗到端點功能。
**服務健康狀況** | 提供端點服務之 Defender 的目前狀態資訊。 您可以驗證服務健康情況是否良好，或是否有目前的問題。
**設定管理** | 顯示 boarded 裝置、組織的安全性基準、預測分析、web 保護覆蓋範圍，並可讓您在裝置上執行攻擊面管理。
**設定** | 顯示您在上架期間選取的設定，並可讓您更新行業偏好設定和保留原則週期。 您也可以設定其他設定設定，例如許可權、APIs、規則、裝置管理、IT 服務管理及網路評估。
**(2) 搜尋、社區中心、當地語系化、協助和支援、意見反應** | **搜尋** -搜尋依裝置、檔案、使用者、URL、IP、弱點、軟體及建議。 </br></br> **社區中心** -存取社區中心以瞭解產品的、共同作業和共用經驗。 </br></br>  **當地語系化** -設定時區。 </br></br>  說明 **與支援**-存取適用于 Endpoint 的 Defender 指南、Microsoft 和 Microsoft Premier support、授權資訊、模擬 & 教程、Defender for endpoint 評估實驗室，請參閱威脅專家。</br></br> **意見** 反應-提供您喜歡的意見或我們可以做什麼。

> [!NOTE]
> 針對高解析度 DPI 縮放問題的裝置，請參閱適用于 [高 DPI 裝置的 Windows 縮放問題](https://support.microsoft.com/help/3025083/windows-scaling-issues-for-high-dpi-devices) ，以取得可能的解決方案。
