# <a name="overview-of-attack-surface-reduction"></a>攻擊面縮小概觀

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**適用於：**
- [適用於端點的 Microsoft Defender](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)

> 想要體驗適用於端點的 Microsoft Defender 嗎？ [注册免費試用版。](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-exposedapis-abovefoldlink)


將您的組織易於 cyberthreats 和攻擊的位置降到最低，有助於減少攻擊面。 使用下列資源來設定組織中裝置和應用程式的保護。


> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4woug]


文章 | 描述
-|-
[攻擊面縮減](./attack-surface-reduction.md) | 利用可協助阻止惡意程式碼的智慧型規則，減少應用程式中的漏弱點 (攻擊面)。  (需要 Microsoft Defender 防病毒) 。
[硬體隔離](/windows/security/threat-protection/microsoft-defender-application-guard/md-app-guard-overview) | 在啟動和執行時，保護及維護系統的完整性。 透過地域和遠端認證來驗證系統完整性。 而且，使用 Microsoft Edge 的容器隔離來協助抵禦惡意網站。
[應用程式控制](/windows/security/threat-protection/windows-defender-application-control/windows-defender-application-control) | 使用應用程式控制，讓應用程式必須獲得信任才能執行。
[入侵防護](./exploit-protection.md) | 協助保護您組織使用的作業系統和應用程式。 Exploit protection 也可搭配協力廠商的防病毒解決方案使用。
[網路保護](./network-protection.md) | 將保護功能擴充至組織裝置上的網路流量和連線能力。  (需要 Microsoft Defender 防病毒) 
[Web 保護](./web-protection-overview.md) | 保護您的裝置免受網頁威脅，並協助您控制不想要的內容。
[受控資料夾存取權](./controlled-folders.md) | 協助防止惡意或可疑的應用程式 (包括檔案加密的勒索軟體) 惡意軟體變更您的重要系統資料夾中的檔案， (需要 Microsoft Defender 防毒軟體) 
[網路防火牆](/windows/security/threat-protection/windows-firewall/windows-firewall-with-advanced-security) | 防止未經授權的流量透過雙向網路流量篩選，流向或從組織的裝置。
[受攻擊面縮小常見問題集](./attack-surface-reduction-faq.md) | 有關攻擊面減少規則、授權等等的常見問題。
