# <a name="manage-recipients-in-standalone-eop"></a>在獨立版 EOP 中管理收件者

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]

**適用於**
-  [Exchange Online Protection 獨立](exchange-online-protection-overview.md)

獨立 Exchange Online Protection (EOP) 不含 Exchange Online 信箱的組織，都支援下列類型的收件者：

- **郵件使用者**：郵件使用者是獨立 EOP 組織中的使用者帳戶基本類型。 郵件使用者在您的 EOP 組織中具有登入認證，但是他們的信箱 (的外部電子郵件地址位於您的 EOP 組織) 之外。

  如需在 EOP 中管理郵件使用者的相關資訊，請參閱[管理 EOP 中的郵件使用者](manage-mail-users-in-eop.md)。

- **群組**：您可以建立下列類型的群組：

  - 通訊群組
  - 擁有郵件功能的安全性群組

  如需在 EOP 中管理群組的相關資訊，請參閱[管理 EOP 中的群組](manage-groups-in-eop.md)。
