# <a name="get-microsoft-365-for-campaigns"></a>取得 Microsoft 365 的活動

任何 Microsoft 365 商務版的人都可以使用此指引來設定電子郵件和共同作業的額外安全性。 不過, 在美國的行銷活動和政治各方符合 Microsoft 365 的特殊價格的資格。 目前可以使用這種特殊定價:
- 美國聯邦政治活動 (例如, 針對美國房屋、美國 Senate 或總統的 office)
- 美國國內層級的政治雙方。

如果您的活動或政治方合格, Microsoft 365 的市場活動就是 Microsoft 的最便宜的方案。  

如果您不符合特殊定價的資格, Microsoft 365 商務仍然是最具成本效益的方法, 可取得共同作業環境的完整安全性。 請參閱[註冊 Microsoft 365 以取得活動](m365-campaigns-sign-up.md)。 然後使用此文件庫來設定額外的安全性, 並協助您的小組安全地進行共同作業。 

## <a name="what-does-microsoft-365-for-campaigns-include"></a>Microsoft 365 對市場活動有何影響？
Microsoft 365 商務版的這項設定包含簡單的控制項, 可協助您和您的員工安全地共同作業: 
- **保護使用者身分識別**: 請確定使用者是在登入傳送電子郵件或存取檔案 (多重要素驗證) 時所聲稱的。
- **保護機密資訊**: 保護機密資訊, 以監視在組織外部共用的資訊 (資料遺失防護)。
- **保護行動裝置**: 保護行動裝置上的資料 (行動裝置應用程式保護原則)。
- 防止**惡意內容**: 掃描電子郵件附件 (高級威脅防護), 以防止對惡意內容的存取。
- **保護密碼**: 將密碼設為永不過期, 較安全且有助於防止工作 stoppages (密碼原則)。 


## <a name="what-does-it-cost-who-needs-it-and-what-is-the-commitment"></a>It 成本、需要 it 人員, 以及承諾是什麼？
如果您的市場活動符合每位使用者每月 $5 的特定價格的 Microsoft 365。 若要保護您的活動, 建議使用候選人、活動管理員、所有屬於活動或參與者的資深人員的授權, 而且通常是所有全職員工。 某些志願者員工可能也需要授權。 一般而言, 將授權指派給市場活動中需要受保護的電子郵件和裝置的任何人。
註冊 Microsoft 365 以進行行銷活動時, 沒有最小的承諾。 您可以每月支付您所需的授權, 並且隨時停止使用服務。

## <a name="how-do-i-qualify-for-special-pricing"></a>如何符合特殊定價的資格？

1. 請移https://m365forcampaigns.microsoft.com/至, 並提供一些您組織的詳細資訊。 您提供的詳細資料將有助於我們確認您在美國的國家層級政治活動或協力廠商。 當您完成此表單時, 沒有任何承諾。 
2. 當您完成表單之後, 會花幾天的時間來查看您的資訊。 
3. 在我們驗證您表示全國層級的政治活動或一方之後, 您將會收到 Microsoft 的電子郵件邀請。 您的邀請包含特定于您組織的註冊連結。 

當您收到邀請時, 請[註冊 Microsoft 365 以取得活動](m365-campaigns-sign-up.md)。
