# <a name="protect-yourself-against-phishing-and-other-attacks"></a>抵禦網路釣魚和其他攻擊

Microsoft 365 包含許多保護措施，以防範攻擊，但您可以採取一些措施來限制並降低線上攻擊的風險。

- **垃圾郵件或垃圾郵件**有許多原因可接收垃圾郵件，而並非所有的垃圾郵件都相同。 不過，您可以減少對您的功能，進而透過篩選出垃圾郵件，以降低遭受攻擊的風險。
- **網路釣魚**網路釣魚詐騙是指看似合法的電子郵件，但會嘗試取得您的個人資訊或竊取您的資金。
- **哄騙**詐騙者也可以使用稱為「哄騙」的技術，讓它看起來就好像您收到自己的電子郵件。 
- **惡意**代碼是可安裝在電腦上的惡意軟體，通常是在您按一下連結或從電子郵件開啟檔之後安裝。 惡意程式碼有多種類型（例如，勒索軟體，當您的電腦經過時），但您不想要使用任何類型的惡意程式碼。 

## <a name="best-practices"></a>最佳做法

**減少垃圾郵件**

請遵循下列[10 個秘訣的方式來協助減少垃圾郵件](https://support.microsoft.com/en-us/office/10-tips-on-how-to-help-reduce-spam-55f756e8-688b-41c3-a086-8f68ccc592f6)。

**報告！**

報告您收到的任何網路釣魚或其他騙局電子郵件。 選取郵件，然後選擇功能區上的 [**報告訊息**]。
以下是[報告垃圾郵件和網路釣魚電子郵件](https://support.office.com/article/Use-the-Report-Message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2)的詳細資訊。

**避免網路釣魚**
- 永不回復要求您傳送個人或帳戶資訊的電子郵件。
- 如果您收到的電子郵件看似可疑或要求您輸入這類資訊，請不要按一下取得公司網站的連結。
- 決不開啟任何附加至可疑電子郵件的檔案。
- 如果電子郵件看似來自公司，請透過電話或網頁瀏覽器聯繫公司的客戶服務，以查看電子郵件是否合法。
- 在 web 上搜尋電子郵件主題行後接 word 惡作劇，以查看其他人是否已舉報此騙局。

在[處理濫用、網路釣魚或哄騙時](https://support.office.com/article/Deal-with-abuse-phishing-or-spoofing-in-Outlook-com-0d882ea5-eedc-4bed-aebc-079ffa1105a3)，請閱讀五種常見的詐騙類型。

**確定您的電子郵件看似對其他人合法**

新增數位簽章以證明您的資訊可協助您的客戶信任您的通訊。 [使用數位簽章，查看安全郵件](https://support.office.com/article/secure-messages-by-using-a-digital-signature-549ca2f1-a68f-4366-85fa-b3f4b5856fc6)。
