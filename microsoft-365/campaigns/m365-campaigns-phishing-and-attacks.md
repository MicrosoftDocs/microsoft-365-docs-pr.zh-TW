# <a name="protect-yourself-against-phishing-and-other-attacks"></a>保護自己免受網路釣魚和其他攻擊

Microsoft 365 包含許多保護措施來防禦攻擊, 但是您可以自行做, 以限制和降低線上攻擊的風險。

- **垃圾郵件或垃圾郵件**有許多原因可能會收到垃圾郵件, 而不是所有的垃圾郵件都是相同的。 不過, 您可以減少對您的瞭解, 進而降低攻擊的風險, 方法是篩選出垃圾郵件。
- **網路釣魚**網路釣魚詐騙是一種看似合法但試圖取得您個人資訊或竊取您的金錢的電子郵件。
- **哄騙**詐騙者也可以使用稱為詐騙的技術, 讓它看起來就像是您收到自己的電子郵件一樣。 
- **惡意**代碼是可以安裝在電腦上的惡意軟體, 通常是在您按一下連結或從電子郵件開啟檔之後安裝。 有各種類型的惡意程式碼 (例如, 勒索軟體, 當您的電腦經過時), 但您不想要有任何的惡意程式碼。 


## <a name="best-practices"></a>最佳做法

**減少垃圾郵件**

請遵循下列[10 個有關如何協助減少垃圾郵件的秘訣](https://support.office.com/en-us/article/10-tips-on-how-to-help-reduce-spam-55F756E8-688B-41C3-A086-8F68CCC592F6)。

**報表!**

報告您收到的任何網路釣魚或其他詐騙電子郵件。 選取郵件, 然後在功能區上選擇 [**報告訊息**], 以報告郵件寄件者, 然後封鎖該郵件。
以下是有關[報告垃圾郵件和網路釣魚電子郵件](https://support.office.com/en-us/article/Use-the-Report-Message-add-in-b5caa9f1-cdf3-4443-af8c-ff724ea719d2)的詳細資訊。

**避免網路釣魚**
- 永遠不要回復要求您傳送個人或帳戶資訊的電子郵件。
- 如果您收到的電子郵件看似可疑或要求您輸入此類型的資訊, 請不要按一下接受您前往公司網站的連結。
- 永遠不要開啟任何附加至可疑的電子郵件的檔案。
- 如果電子郵件看似來自公司, 請透過電話或網頁瀏覽器與公司的客戶服務, 以查看電子郵件是否合法。
- 搜尋網頁中的電子郵件主旨行後接 word 惡作劇, 查看是否有其他人回報過此詐騙。

請閱讀關於[處理濫用、網路釣魚或詐騙](https://support.office.com/en-us/article/Deal-with-abuse-phishing-or-spoofing-in-Outlook-com-0d882ea5-eedc-4bed-aebc-079ffa1105a3)的五種常用詐騙類型。

**確定您的電子郵件看似對其他人合法**

協助您的客戶信任您的通訊-新增數位簽章, 以證明其來自您。 [使用數位簽章來查看安全郵件](https://support.office.com/en-us/article/secure-messages-by-using-a-digital-signature-549ca2f1-a68f-4366-85fa-b3f4b5856fc6)。
