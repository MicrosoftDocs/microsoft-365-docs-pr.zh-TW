# <a name="sign-in-to-microsoft-365-for-campaigns"></a>登入 Microsoft 365 的活動

## <a name="for-campaign-admins"></a>針對活動系統管理員
如果您註冊 Microsoft 365 以取得活動, 您就是 Microsoft 365 系統管理員。以下說明如何登入: 
1. 在步驟2的[註冊步驟](m365-campaigns-sign-up.md#steps-to-sign-up)2 中, 尋找我們傳送給我們的電子郵件地址的使用者名稱和密碼。
2. 在瀏覽器中, 移至 Microsoft 365 系統管理中心<a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a>。 
3. 輸入您的使用者名稱和密碼。 選取 [登**入**]。
4. 在頁面的右上方, 尋找 [在控制項**上預覽**]。 選取 [**在開啟時預覽**], 讓您可以使用[為您的市場活動設定](m365-campaigns-security-overview.md)的所有控制項的凹凸保護。

## <a name="for-campaign-staff"></a>針對活動員工
設定活動職員如[新增使用者](../business/add-users-m365b.md?toc=/microsoft-365/campaigns/toc.json)所述, 您也可以在 [**新增使用者**] 頁面上重設和重新傳送密碼。
所有的活動員工都可以登<a href="https://office.com" target="_blank">https://Office.com</a>入。
