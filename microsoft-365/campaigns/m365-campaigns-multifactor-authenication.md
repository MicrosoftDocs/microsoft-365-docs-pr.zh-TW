# <a name="set-up-multifactor-authentication"></a>設定多重要素驗證

多重要素驗證可為您的企業提供更多安全性。  當您的系統管理員要求您使用 MFA 時, 您可以設定 Microsoft 驗證者應用程式, 讓您使用手機安全地登入重要應用程式。 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2MmQR] 

若要深入瞭解, 請參閱在[Microsoft 365 商務版中設定多重要素驗證](https://support.office.com/article/a32541df-079c-420d-9395-9d59354f7225)

## <a name="use-the-outlook-app-in-your-devices"></a>在您的裝置中使用 Outlook 應用程式

一旦您的系統管理員要求您使用 MFA, 而且已將驗證應用程式設定為第二種驗證形式, 建議您安裝並僅使用 Outlook 應用程式來存取 Microsoft 365 的行銷活動電子郵件。 請參閱[設定行動裝置](../business/set-up-mobile-devices.md), 以瞭解如何在手機上安裝 Office 應用程式, 包括 Outlook。
