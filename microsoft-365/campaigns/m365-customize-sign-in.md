# <a name="customize-your-sign-in-page-with-a-privacy-and-consent-notice"></a>自訂您登入] 頁面上的隱私權和同意通知

政治活動可以讓更容易到檔案法律費用線上罪犯針對執法機構所新增的隱私權和同意的通知，以您登入] 頁面上。

您可以使用您的品牌，自訂您登入] 頁面上，您也可以新增文字，協助使用者登入，或指向出法律要求或取得 Microsoft 365 的行銷活動資源的存取限制。

## <a name="customize-the-text-on-your-sign-in-page"></a>自訂在您登入] 頁面上的文字

若要更新上登入的自訂項目，您必須是全域系統管理員。如需特定指示，請參閱[將公司商標新增](https://docs.microsoft.com/azure/active-directory/fundamentals/customize-branding)文章。

您可以更新的元素為：

- **登入頁面文字**

    這是簡單的就緒，可以新增隱私權並同意陳述式。
- 登入頁面背景影像
- 橫幅標誌
- Username 提示

如需隱私權和同意封通知，請參閱[搜尋並抓取電腦及取得中司法調查的電子舉證](https://www.justice.gov/sites/default/files/criminal-ccips/legacy/2015/01/14/ssmanual2009.pdf)PDF 檔案，特別是**附錄 a： 範例網路橫幅語言**的範例。
