# <a name="customize-your-sign-in-page-with-a-privacy-and-consent-notice"></a>使用隱私權和同意通知自訂登入頁面

在您的登入頁面上加入隱私權和同意通知，您的政治活動可讓執法部門輕鬆抵禦線上罪犯的法律費用。

您可以使用您的署名自訂登入頁面。 您也可以新增文字，以協助使用者登入，或指出法律需求或限制，以取得對行銷活動資源的 Microsoft 365 的存取權。

## <a name="design-customization-the-text-on-your-sign-in-page"></a>設計自訂登入頁面上的文字

若要更新登入頁面上的可自訂元素，您必須是全域系統管理員。如需詳細指示，請參閱[add company 署名](https://docs.microsoft.com/azure/active-directory/fundamentals/customize-branding)文章。

您可以更新的元素包括：

- **登入頁面文字**

     新增隱私權與同意聲明的簡易位置。
- 登入頁面背景圖像
- 橫幅標誌
- 使用者名稱提示

如需隱私權與同意通知的範例，請參閱附錄 A[搜尋和抓住電腦，並取得民事調查中的電子證據](https://www.justice.gov/sites/default/files/criminal-ccips/legacy/2015/01/14/ssmanual2009.pdf)。
