# <a name="create-a-communications-site-for-your-campaign"></a>建立您的行銷活動的通訊網站

通訊優先順序、 共用策略文件，並反白顯示即將來臨的事件的好方法是使用 SharePoint 中的通訊網站。 通訊網站會跨整個活動; 廣泛地共用項目它是您內部的行銷活動的網站。

## <a name="best-practices"></a>最佳做法

通訊網站中包含下列元素：

1. 新增您的行銷活動標誌與色彩做為標題影像和佈景主題
2. 導致策略、 與郵件、 重要的文件、 一個目錄，在**Hero 網頁組件**中的常見問題集。
3. 納入**文字網頁組件**至小組的候選陳述式。
4. 將活動事件新增至**事件網頁組件**，讓每個人都可以看到接下來什麼。
5. 新增人員可以使用或**影像圖庫網頁組件**共用的相片。

## <a name="set-it-up"></a>進行設定

1. 登入https://Office.com。
2. 在頁面左上角，選取 [應用程式啟動器圖示，然後選取 [ **SharePoint** ] 磚。 如果您沒有看到 [ **SharePoint** ] 磚，按一下 [**網站**] 磚] 或 [**全部**如果看不到 SharePoint。
3. 在 SharePoint 首頁頂端，按一下 [ **+ 建立網站**，並選擇**通訊網站**選項。

了解所有[關於站台的通訊](https://support.office.com/article/What-is-a-SharePoint-communication-site-94A33429-E580-45C3-A090-5512A8070732)，以及如何[建立 SharePoint Online 中的通訊網站](https://support.office.com/article/Create-a-communication-site-in-SharePoint-Online-7FB44B20-A72F-4D2C-9173-FC8F59BA50EB)。


## <a name="admin-settings"></a>系統管理員設定

如果您沒有看到 [ **+ 建立**網站] 連結，可能無法在 Microsoft 365 使用自助網站架設。 若要建立小組網站，請連絡您的組織中管理 Microsoft 365 的人員。 如果您的 Microsoft 365 系統管理員，請參閱[SharePoint Online 中的管理網站建立](https://docs.microsoft.com/sharepoint/manage-site-creation)若要啟用自助網站架設，為您的組織或[在新的 SharePoint 系統管理中心中的受管理的網站](https://docs.microsoft.com/sharepoint/manage-sites-in-new-admin-center)」 從 SharePoint Online 系統管理中心建立網站。
