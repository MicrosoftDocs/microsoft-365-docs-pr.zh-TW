# <a name="share-files-and-videos"></a>共用檔案和影片

當您需要控制誰可以查看您的檔案，以及誰可以編輯您的檔案時，您必須將檔案儲存在安全的位置，以確保正確套用許可權。 您可以使用 Microsoft Teams 來儲存檔案，然後使用 Microsoft Teams 或傳送 SharePoint 連結，在公司、實踐或活動內部或外部共用檔案。 傳送連結（而非電子郵件附件）表示您知道誰在查看及修改檔案，而且不需要您的許可權就能查看或修改這些檔案。

![Microsoft Teams 視窗的圖表，顯示 [檔案] 索引標籤，並取得功能表上的連結](../media/m365-democracy-teams-sharefiles.png)

您的檔案在 Microsoft Teams 和 SharePoint 中，您也可以一起處理檔案，並彼此檢查的變更。 使用 Microsoft Teams 在公司、實踐或活動中共用檔案。 如果您需要與組織外部的人員共用，可將其新增為小組的客人，或傳送安全的 SharePoint 連結。

您也可以使用 Microsoft Stream 在內部儲存和共用影片。 這些影片對公眾是無法查看的，所以它們非常適合內部活動郵件。

## <a name="best-practices"></a>最佳做法

使用下列方法可安全地共用檔案和視頻：

1. 將檔案儲存在 Microsoft Teams 或 SharePoint 中，並確定只有需要存取這些檔案的人員才會有這些檔案。
2. 當您想要共用時，請勿將檔案附加到電子郵件。 請改為選擇 [從 Microsoft Teams **取得連結**] 或 SharePoint，然後在電子郵件中傳送連結。
3. 若要在外部共用檔案，請將使用者新增為您的小組的來賓，或使用 SharePoint 以取得只共用該檔案的安全連結。
4. 使用 Microsoft Stream 來主控您想要讓活動看到的影片。
5. 使用 Microsoft Teams 或 SharePoint 儲存您需要的小組共同作業或分享的影片檔。
