# <a name="set-up-microsoft-365-for-campaigns"></a>設定 Microsoft 365 的活動

本文說明為市場活動設定 Microsoft 365 的程式。 任何 Microsoft 365 商務版的人都可以使用此指引來進行安全性。 

## <a name="admins-set-up-microsoft-365-for-campaigns"></a>系統管理員: 設定 Microsoft 365 的活動
下圖說明系統管理員如何設定 Microsoft 365 的市場活動。

![為市場活動設定 Microsoft 365 的步驟](media/M365-democracy-SetUpProcess.png)

對於符合特殊價格的行銷活動, 請[從 microsoft 要求邀請](https://m365forcampaigns.microsoft.com/), 然後[註冊 microsoft 365 以取得促銷活動](m365-campaigns-sign-up.md), 以開始使用。 若要完成安裝程式, 請[執行安裝精靈](../business/set-up.md?toc=/microsoft-365/campaigns/toc.json)來設定核心設定。

對於所有其他組織, 在[註冊 Microsoft 365 商務](../business/sign-up.md)版之後, 請執行[安裝精靈](../business/set-up.md?toc=/microsoft-365/campaigns/toc.json)來設定核心設定, 以完成安裝程式。

針對所有組織, 將安全性保護調高:[保護系統管理員帳戶](m365-campaigns-protect-admin-accounts.md)、[保護郵件和資料的存取](m365-campaigns-conditional-access.md), 以及[加強威脅防護](m365-campaigns-increase-protection.md)。


 ## <a name="everyone-set-up-your-devices"></a>每個人: 設定您的裝置 
 
使用者需要花幾分鐘的時間來設定裝置以與此環境搭配運作。 對於您的主要使用者 (即駭客最高價值的目標), 您可以設定及預先設定新的裝置。 如此一來, 他們就可以在使用 Microsoft 365 認證登入時立即開始。 

![使用者裝置設定程式](media/m365-democracy-user-device-setup.png)
  
若要設定使用者裝置: 
1. 每位使用者都[設定其行動裝置](../business/set-up-mobile-devices.md?toc=%2Fmicrosoft-365%2Fcampaigns%2Ftoc.json)。 
2. 對於未受管理的裝置, 使用者會設定其[pc 和 mac](m365-campaigns-protect-pcs-macs.md)。 對於主要人員, 建議您使用[受管理的裝置](../business/set-up-windows-devices.md?toc=/microsoft-365/campaigns/toc.json), 以獲得更好的保護。 對於所有裝置, 您會想要設定[多重要素驗證](m365-campaigns-multifactor-authenication.md)。 
3. 所有使用者都應該瞭解如何通過瞭解[網路釣魚和其他攻擊](m365-campaigns-phishing-and-attacks.md), 來保護自己和您的活動。 此[資訊圖表](m365-campaigns-protect-campaign-infographic.md)也可協助您的使用者瞭解如何協助保護您的活動免遭線上威脅。
