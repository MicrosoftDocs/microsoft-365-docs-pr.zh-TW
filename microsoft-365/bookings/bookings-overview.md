# <a name="microsoft-bookings"></a>Microsoft Bookings

Microsoft 預約使排程和管理約會成為一個輕而易舉。 預約包含 web 型預約行事曆，並與 Outlook 整合，以優化員工的行事曆，並讓您的客戶有彈性地預訂最適合他們的時間。 自動化的通知電子郵件會減少非展示和提高客戶滿意度，而組織節省時間，縮短重複排程工作。 透過內建的彈性和自訂的能力，可設計預約，以符合組織的許多不同部分的情況和需求。 [行事曆] 是 Exchange Online 中的信箱。

預約可讓您透過[Microsoft Teams](https://support.microsoft.com/office/overview-of-the-bookings-app-in-teams-7b8569e1-0c8a-444e-b712-d9968b05110b)和商務用 Skype，讓組織的會議成為線上會議的虛擬功能。 每個預約為線上會議的約會都會建立一個傳送給出席者的唯一會議連結，讓他們可以透過網頁瀏覽器、電話撥入或 Skype 或 Teams 應用程式加入。 預約也可以做為 Teams 中的應用程式，可讓您建立預約行事曆、指派員工，以及排程新的約會，而不會離開 Teams。

Bookings 有三個主要元件：

- 預約頁面，客戶和用戶端可以在此頁面上排程應提供服務或執行約會的教職員工成員約會。 此 Web 式排程頁面可透過直接連結、Facebook 頁面，甚至是透過網站內嵌的連結來共用。

- 一個 web 應用程式，其中包含一組以 web 為基礎的商務頁面，其中的預約行事曆擁有者及組織內的管理員可以定義約會類型和詳細資料、管理人員排程和可用性、設定上班時間，以及自訂約會的排程方式。 這些頁面可讓您使用多功能性，並可自訂預約行事曆，以符合個人或組織的不同需求。

- 這是一款因應企業需求的行動裝置應用程式，Bookings 行事曆擁有者和系統管理員可以在這裡查看所有約會、存取客戶清單和連絡人資訊，以及在外出時手動進行預約。
