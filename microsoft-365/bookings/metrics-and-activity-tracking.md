# <a name="microsoft-bookings-metrics-and-activity-tracking"></a>Microsoft 預約計量和活動追蹤

主要預約頁面上提供各種計量，可讓您輕鬆追蹤收入及客戶活動。

| 計量 | 描述 |
|:---|:---|
| 預定活動 | 30天活動期間會顯示過去的30天（包括 today）。 **預定** 的預約、 **預估的收入** 及已預約的 **客戶** 都會進行計算。 |
| 已進行預定 | 在今天或過去30天內所發生的預約數目。 您可以使用此資訊來查看您是否符合估計的銷售預期，並預測未來30天。 |
| 預估收入 | 這會顯示您透過預約取得的預估收入。 這是根據您為每個服務所定義的價格而計算。 這是預估，因為您在服務時間可能會收費收費、費用設定為每小時費用、您已有秘訣或客戶尚未支付。 例如，如果您的服務每小時充電 $10，而預約是2小時長，則預約只會估計該服務的 $10。<br/><br/>**重要：** 這只是預估，而且不會保證您的實際收入。 |
| 已預約客戶 | 目前和過去30天已預約約會的客戶人數。<br/><br/>**附注：** 此度量顯示已預約約會的客戶數目，而非約會總數。 例如，如果一個客戶在過去30天中預約了三個約會，而兩個客戶又做了一個預約，您就會看到3位客戶已預約。 |
