# <a name="comparison-chart-bookings-web-app-vs-bookings-teams-app"></a>比較表：預定 web app 與預約 Teams 應用程式

Teams 中的預約應用程式可讓調度程式處理其主要工作並變更部分設定。 不過，「預定」網頁包含的高級功能和設定尚未在 Teams 應用程式中使用。

此時會同時新增 Teams 應用程式功能，我們將繼續更新此清單。 如需詳細資訊，請參閱 [FAQ](bookings-faq.yml)中 **Microsoft Teams？區段中可用之原始預約 Web app 的所有功能**。

| 功能 | 預定 web 應用程式 | 預定 Teams app |
|:---|:---|:---|
| 建立新的預約行事曆 | 是 | 是 |
| 新增和移除預約行事曆中的職員 | 是 | 是 |
| 建立新的約會類型 | 是 | 是 |
| 排程線上預訂 | 是 | 是 |
| 編輯商務詳細資料 | 是 | 是 |
| 為組織外部人員加入具有來賓許可權的人員 | 是 | 否 |
| 排程人員或離線預約 | 是 | 否 |
| 排程群組或多客戶預訂 | 是 | 否 |
| 為您的公司設定標誌 | 是 | 否 |
| 設定上班時間 | 是 | 否 |
| 發佈自助排程頁面 | 是 | 否 |
| 管理客戶連絡人 | 是 | 否 |
| 設定員工的下班時間 | 是 | 否 |
