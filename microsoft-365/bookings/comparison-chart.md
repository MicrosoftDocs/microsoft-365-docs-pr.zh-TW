# <a name="comparison-chart-bookings-web-app-vs-bookings-teams-app"></a>比較表：預定 web 應用程式與預約小組應用程式

小組中的預約應用程式可讓計畫程式處理其主要工作並變更部分設定。 不過，「預定」網頁包含尚未在小組應用程式中使用的高級功能和設定。

「小組」應用程式的功能都是隨時新增，我們會繼續更新此清單。 如需詳細資訊，請參閱[FAQ](bookings-faq.md)中的 [ **Microsoft 團隊所有可用的原始預約 Web app 功能**] 一節。

| 功能 | 預定 web 應用程式 | 預定小組應用程式 |
|:---|:---|:---|
| 建立新的預約行事曆 | 是 | 是 |
| 新增和移除預約行事曆中的職員 | 是 | 是 |
| 建立新的約會類型 | 是 | 是 |
| 排程線上預訂 | 是 | 是 |
| 編輯商務詳細資料 | 是 | 是 |
| 為組織外部人員加入具有來賓許可權的人員 | 是 | 否 |
| 排程人員或離線預約 | 是 | 否 |
| 排程群組或多客戶預訂 | 是 | 否 |
| 為您的公司設定標誌 | 是 | 否 |
| 設定上班時間 | 是 | 否 |
| 發佈自助排程頁面 | 是 | 否 |
| 管理客戶連絡人 | 是 | 否 |
| 設定員工的下班時間 | 是 | 否 |
