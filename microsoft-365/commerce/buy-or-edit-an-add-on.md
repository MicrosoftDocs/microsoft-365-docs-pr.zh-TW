# <a name="buy-or-edit-an-add-on-for-office-365-for-business"></a>購買或編輯商務用 Office 365 的附加元件

商務用 Office 365 的其中幾個方案有您可以為訂閱購買的附加元件。為訂閱購買附加元件即可提供訂閱額外的功能。


::: moniker range="o365-worldwide"

> [!NOTE]
> 如果您使用的不是新的 Microsoft 365 系統管理中心，您可以選取位於首頁頂端的 **[試用新的系統管理中心] **開關將它開啟。

::: moniker-end

## <a name="update-an-add-on"></a>更新附加元件

您用來更新附加元件的步驟取決於此附加元件是傳統的附加元件或獨立附加元件。 

*傳統的附加元件*會連結至特定的訂閱。 如果您取消訂閱，也會取消相關聯的附加元件。
  
*獨立的附加元件*未連結至特定的訂閱。 獨立的附加元件以單獨訂閱上 [**訂閱**] 頁面 （在新版系統管理中心的 [**產品 & 服務**] 頁面上），或出現，而且有自己的到期日。 您管理任何其他訂閱的相同方式來管理的獨立附加元件。
  
### <a name="update-a-traditional-add-on"></a>更新的傳統的附加元件

::: moniker range="o365-worldwide"

1. 在系統管理中心，移至 [帳單]**** \> [產品與服務]<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank"></a> 頁面。

2. 在 [**產品 & 服務**] 頁面上，選取您要更新附加元件的訂閱。

3. 選取 [**附加元件**。 會顯示您已購買的附加元件。

4. 您想要變更的附加元件，選取 [**變更數量**]。

5. 輸入您想要訂閱，然後再選取 [**送出變更**的量變更。

## <a name="remove-an-add-on"></a>移除附加元件

您可以取消的獨立附加元件相同方式，您[取消訂閱](subscriptions/cancel-your-subscription.md)。 或者，您可以設定**週期性帳單**設為獨立的附加元件形式關閉藉由使用上述步驟。 不過，您無法移除傳統的附加元件形式購買之後。 如果您要移除的傳統的附加元件，請[連絡客戶支援的說明](../admin/contact-support-for-business-products.md)。
