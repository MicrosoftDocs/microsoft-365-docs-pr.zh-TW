# <a name="buy-a-subscription-to-microsoft-365-for-business-from-your-free-trial"></a>從免費試用版購買為商務用 Microsoft 365 的訂閱

您已嘗試 Microsoft 365，現在您已準備好購買。 從您的試用版購買 Microsoft 365 時，您有兩個選項：
  
- [購買與試用版相同的方案](#buy-the-same-plan-as-your-trial)
- [購買與試用版不同的方案](#buy-a-different-plan-than-your-trial)

> [!NOTE]
> 當您的免費試用期結束時，您的試用版訂閱將會自動轉換為付費訂閱。 付費訂閱會預設為您目前註冊的計畫。 您也可以遵循 [[購買與您的試用版不同的方案](#buy-a-different-plan-than-your-trial)] 區段中的步驟，購買不同的方案。

## <a name="buy-the-same-plan-as-your-trial"></a>購買與試用版相同的方案

步驟相當簡單，而且若您之後要購買與試用版相同的訂閱類型，便不必擔心會遺失資料、帳戶及設定。

::: moniker range="o365-worldwide"

1. 在系統管理中心，移至 [帳單]**** \> [產品與服務]<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank"></a> 頁面。

2. 在 [**產品 & 服務**] 頁面上，尋找您要購買的訂閱。 在 [**授權**] 區段中，選取 [**購買訂閱**]。

3. 選擇您訂閱的每月或每年承諾，然後選取 [**簽出**]。

4. 在下一個頁面上，確認訂閱，然後選取 [**簽出**]。

5. 在下一個頁面上，驗證**銷售**的位址、**收取**的資訊及專案的**訂單**。 如果您需要進行任何變更，請選取 [適用] 區段旁的 [**變更**]。

6. 完成後，請選取 [**接受合約 & 下訂單**]。

::: moniker-end

::: moniker range="o365-germany"

1. 在系統管理中心中，前往 [帳單]**** \> [訂閱]<a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank"></a> 頁面。

2. 在 [**訂閱**] 頁面上，為您要購買的試用訂閱選取 [**立即購買**] 按鈕。

3. 查看定價資訊，然後選取 **[下一步]**。

4. 提供您的付款資訊，然後選取 [**下單**]。

5. 在 [確認] 頁面上，選取 [**移至系統管理首頁**]。

::: moniker-end

::: moniker range="o365-21vianet"

1. 在系統管理中心中，前往 [帳單]**** \> [訂閱]<a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank"></a> 頁面。

2. 在 [**訂閱**] 頁面上，為您要購買的試用訂閱選取 [**立即購買**] 按鈕。

3. 查看定價資訊，然後選取 **[下一步]**。

4. 提供您的付款資訊，然後選取 [**下單**]。

5. 在 [確認] 頁面上，選取 [**移至系統管理首頁**]。

::: moniker-end

## <a name="buy-a-different-plan-than-your-trial"></a>購買與試用版不同的方案

若購買與試用版不同的訂閱類型，您必須將您的使用者授權從您的試用方案重新指派給新方案（在您的試用計畫到期後的90天寬限期之前），以便保留您的資料、帳戶和設定。

::: moniker range="o365-worldwide"


1. 在系統管理中心中，移至 [**帳單** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">購買服務</a>] 頁面。

2. 在 [**購買服務**] 頁面上，選取您要購買的訂閱，然後選取 [**購買**]。

3. 輸入您需要的授權數目，並選擇要按月付款或按年付款。 選擇是否要自動將授權指派給目前未獲授權的所有使用者。 然後選取 [**立即查看**]。

4. 購物車隨即開啟。 如果螢幕上沒有您的購物車，請選取 [**購物車**] 將其重新開啟、查看順序，然後選取 **[取出]。**

5. 查看定價資訊，然後選取 **[下一步]**。

6. 提供您的付款資訊，然後選取 [**下訂單** \> ]**移至**[系統管理] 首頁。

::: moniker-end

::: moniker range="o365-germany"

1. 在系統管理中心中，前往 [帳單]**** \> [訂閱]<a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank"></a> 頁面。

2. 在 [**訂閱**] 頁面上，選取 [**新增訂閱**]。

    您將會看到您的組織可使用的訂閱。 您已購買的任何訂閱將會以**試用版**或已**購買**的方式識別。
  
3. 找到您要購買的訂閱，然後選取 [**立即購買**]。
  
4. 輸入您需要的使用者授權數目，並選擇要按月付款或按年付款，然後選取 [**新增至購物車**]。

5. 購物車隨即開啟。 如果螢幕上沒有您的購物車，請選取 [**購物車**] 將其重新開啟、查看順序，然後選取 **[取出]。**

6. 查看定價資訊，然後選取 **[下一步]**。

7. 提供您的付款資訊，然後選取 [**下單**]。

8. 在 [確認] 頁面上，選取 [**移至系統管理首頁**]。

::: moniker-end

1. 在系統管理中心中，前往 [帳單]**** \> [訂閱]<a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank"></a> 頁面。

2. 在 [**訂閱**] 頁面上，選取 [**新增訂閱**]。

    您將會看到您的組織可使用的訂閱。 您已購買的任何訂閱將會以**試用版**或已**購買**的方式識別。
