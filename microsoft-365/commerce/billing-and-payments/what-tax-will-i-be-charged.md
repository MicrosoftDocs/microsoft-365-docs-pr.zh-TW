# <a name="what-tax-will-i-be-charged-for-office-365"></a>我要為 Office 365 支付哪些稅？

查看下列地區收取的稅金。您也可以提供適當的文件給支援人員以[申請免稅資格](apply-for-tax-exempt-status.md)。
  
## <a name="europe-the-middle-east-and-africa-emea"></a>歐洲、中東及非洲 (EMEA)

當您在歐盟地區購買 Office 365 服務時，這項購買會被課徵加值稅 (VAT)。
  
- 如果您位於歐盟會員國，並且未提供您的有效當地 VAT 識別碼，Microsoft Ireland Operations Ltd. 就會根據您帳戶設定的帳單國家/地區套用目前的當地 VAT 稅率。

- 如果您位於瑞士或列支敦斯登，則不論您是否提供 VAT 識別碼，都會套用瑞士 VAT 稅率。

- 一般而言，如果您是位於歐盟及瑞士/列支敦斯登以外的 EMEA 國家/地區，Microsoft Ireland Operations Ltd. 就不會課徵 VAT。

- 您的 VAT ID 必須先能夠在 VAT Information Exchange System (VIES) 中驗證，我們才能驗證它。如果您的 VAT ID 無法驗證，請連絡您當地的稅務機關。

您可能符合零 VAT 稅率的資格：
  
- **如果您正在愛爾蘭以外的歐盟成員狀態：** 您可以提供您有效本機 VAT id。 這會贊成 VAT 至 Microsoft 愛爾蘭作業 Ltd.zero-rate 交易。 不過，您可能需要 accounting 義務，因此如果您有任何問題，請洽詢您的稅務 advisers 本機 VAT。 如需相關指示，請參閱 「 新增您的 VAT ID （僅限歐盟國家） 」 下方。

- **如果您是在愛爾蘭，且必須相關有效 VAT 豁免憑證：** Microsoft 愛爾蘭作業 Ltd.可能會有權去除交易從增值稅。 如果您沒有 Microsoft 愛爾蘭作業 Ltd.會套用目前的愛爾蘭 VAT 稅率，您是否提供 VAT ID 與否。

### <a name="add-your-vat-id-eu-countries-only"></a>新增您的 VAT 識別碼 (僅限歐盟國家/地區)

::: moniker range="o365-worldwide"
  
> [!NOTE]
> 如果您使用的不是新的 Microsoft 365 系統管理中心，您可以選取位於首頁頂端的 **[試用新的系統管理中心] **開關將它開啟。

1. 在系統管理中心，移至 [帳單]**** \> [產品與服務]<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank"></a> 頁面。

2. 尋找您想要更新的訂閱訂閱卡。

3. 在 [**設定 & 動作**] 區段中，選取 [**編輯服務使用地址**]。

4. 在 [**編輯服務使用地址**] 頁面上，在 [ **VAT 編號**] 方塊中，輸入您的 VAT 識別碼，然後選取 [**儲存**。

::: moniker-end

::: moniker range="o365-germany"
  
1. 在 [系統管理中心中，移至**帳單** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">訂閱</a>] 頁面。

2. 選取訂閱名稱，然後再選取 [**更多動作** \> **編輯 VAT 編號**。
  
3. 在 [**編輯服務使用地址**] 頁面上，在 [ **VAT 編號**] 方塊中，輸入您的 VAT 識別碼，然後選取**送出**。

::: moniker-end

## <a name="asia-pacific-countries-apac"></a>亞太地區的國家/地區 (APAC)

如果是由 Microsoft 地區銷售辦公室向您收費，您的發票上就會顯示「Microsoft 地區銷售」，而且通常不會向您徵收消費稅或加值稅 (VAT)，除非是國內銷售。
  
如果是從另一個地點向您收費，則會套用目前的當地稅率。您將會[在發票上](view-your-bill-or-invoice.md)看到下列明細：
  
- 新加坡對新加坡

- 韓國對韓國

- 日本對日本

- 台灣對台灣

## <a name="north-central-and-south-america"></a>北、中、南美洲

在美國及加拿大，會根據您所在地點適用各種稅率。在波多黎各，則採用當地加值稅 (VAT) 費率。
  
如果是由 Microsoft Corporation ([請查看您的發票](view-your-bill-or-invoice.md)) 開立您的發票，而您是在美國、加拿大及波多黎各以外的地區註冊 Office 365，則通常不需課稅。
