# <a name="apply-for-tax-exempt-status-for-office-365-for-business"></a>申請商務用 Office 365 的免稅資格

如果您符合免稅資格位於您國家或地區，請[連絡客戶支援](../../admin/contact-support-for-business-products.md)] 以建立服務要求，以建立免稅資格訂用帳戶。 您必須提供下列文件，當您提交要求： 
  
|**位置**|**文件**|
|:-----|:-----|
|美國和加拿大  <br/> |銷售使用稅務憑證  <br/> |
|波多黎各  <br/> |Certificado de Compras Exentas  <br/> |
|愛爾蘭  <br/> |13B/56A 免稅證明  <br/> |
|國際組織保留免稅證明  <br/> |相關的當地稅率法律文件索引的憑證/letter 確認  <br/> |
