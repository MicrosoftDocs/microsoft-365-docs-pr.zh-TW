# <a name="understand-your-invoice-for-office-365-for-business"></a>了解商務用 Office 365 發票

每個月，您會收到一封電子郵件，告訴您在系統管理中心的新帳單明細可提供。 [了解如何尋找及檢視您的帳單或發票付款](view-your-bill-or-invoice.md)。
  
您的帳單包含兩頁。 第 1 頁是帳單摘要，且包含有關帳單、 訂單、 應付金額、 付款，以及如何連絡技術支援的一般資訊。
  
![第 1 頁的帳單。](../../admin/media/b2f2cf85-a005-4f8a-a7b7-bea231372025.png)
  
第 2 頁包含每個訂閱的帳單活動詳細資料。
  
![第 2 頁的帳單。](../../admin/media/808195cb-b1d2-4492-a748-29671ccecdd2.png)
  
如需包含在您的發票的字詞與欄位的說明，請參閱本文稍後的[帳單欄位詞彙](#invoice-field-glossary)。
  
## <a name="understand-billing-concepts"></a>了解帳單概念

檢視帳單之前，很有幫助了解幾個重要的帳單概念。
  
### <a name="invoice-balance-vs-order-balance"></a>帳單餘額和計費比較

 **帳單餘額**是顯示在您的發票金額，而是只量針對該特定計費期間到期日。 您的總**訂單餘額**是帳單頻率的總和。 您可以在系統管理中心的 [**計費**] 區段中檢視訂單餘額。
  
### <a name="billing-frequency-vs-invoice-frequency"></a>計費頻率與帳單頻率的比較

 **計費頻率**表示多久收到帳單。 訂單每月或每年，哪一個選項視您選擇額度。 **帳單週期**] 表示多久收到帳單。 如果您選擇每年計費，您只會收到一個發票每年，除非訂閱活動需要費用或信用。
  
如果您有一份以上訂單，您會收到每個訂單的帳單。

## <a name="change-your-purchase-order-number"></a>變更您購買的訂單編號

如果您以發票支付時，您可以新增或變更您訂閱的購買訂單 (PO) 編號。
  
> [!NOTE]
> 您無法將 PO 編號新增至現有的發票。 PO 編號會顯示在所有未來的發票。

::: moniker range="o365-worldwide"

### <a name="use-the-new-admin-center-to-change-your-purchase-order-number"></a>使用新版系統管理中心來變更您購買的訂單編號

1. 在系統管理中心，移至 [帳單]**** \> [產品與服務]<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank"></a> 頁面。

2. 如果您是在**資料表**檢視中，選取**卡**切換檢視。

3. 尋找您想要變更的訂閱。

4. 選取 [**計費**] 區段的 [**發票**] 旁的 [**編輯**]。

5. 在 [**編輯付款明細**] 窗格底部，輸入您的 PO 編號，然後選取 [**儲存**]。

### <a name="use-the-old-admin-center-to-change-your-purchase-order-number"></a>使用舊版系統管理中心來變更您購買的訂單編號

1. 在 [系統管理中心中，移至**帳單** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">訂閱</a>] 頁面。

2. 選取您要變更的訂閱，然後在 [**付款方式**] 區段中，選取 [**更新付款明細**。

3. 在 [**更新付款明細**] 窗格中，選取 [**沒有感謝您**。

4. 在 [**更新付款明細**] 窗格底部，輸入您的 PO 編號，選取 [**提交**]，然後**關閉**。

::: moniker-end

2. 選取您要變更的訂閱，然後在 [**付款方式**] 區段中，選取 [**變更付款明細**。
