# <a name="manage-payment-methods"></a>管理付款方式

當您從 Microsoft 購買商務版產品或服務時，您可以使用現有的付款方式，或新增一個新。 您可以使用信用卡或轉帳卡或銀行帳戶支付您購買的事項。 但您只可以管理您新增付款方法。

> [!NOTE]
> 在某些國家/地區或區域並不提供使用銀行帳戶付款選項。
>
> 您必須使用發出為您的租用戶相同國家/地區的付款方式。

## <a name="update-payment-method-details"></a>更新付款方法詳細資料

您可以變更的信用卡或金融卡、 帳單地址或現有的付款方式的到期日的名稱。 不過，您無法變更的卡片或帳戶號碼。 如果帳戶號碼已經變更，[取代它具有不同的付款方式](#replace-a-payment-method)，然後[刪除舊](#delete-a-payment-method)。

1. 在系統管理中心中，移至**帳單** > **帳單 & 定額** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">付款方式</a>] 頁面。
2. 選取要更新的付款方法的列。 在右窗格中，選取 [**編輯**]。
3. 更新您的付款方法資訊 （信用卡或金融卡、 帳單地址或到期日期上名稱），然後選取 [**儲存**]。

## <a name="replace-a-payment-method"></a>取代付款方式

當您取代付款方式時，您的所有訂閱和計費的設定檔，使用相同的付款方式取代它。 取代付款方式，不會刪除現有的付款方式。 它是仍可讓您選取並使用其他訂閱和計費的設定檔。

若要變更單一訂閱的付款方式，請參閱[變更單一訂閱的付款方式](#change-a-payment-method-for-a-single-subscription)。

1. 在系統管理中心中，移至**帳單** > **帳單 & 定額** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">付款方式</a>] 頁面。
2. 選取要取代的付款方法的列。 右窗格會列出所有帳單的設定檔及使用所選的付款方式的個別訂閱。
3. 在右窗格中，選取 [**取代所有項目的的付款方式**。
4. 若要使用現有的付款方式，選擇 [從下拉式清單，然後選取 [**取代**。
    > [!NOTE]
    > 如果您有訂閱的帳單的設定檔相關聯，您可以僅使用信用卡或金融卡支付它們。 如果您有在 [**付款方式**] 頁面上所列的銀行帳戶，他們無法使用下拉式清單中選取。
5. 若要新增新的付款方式，請選取 [**新增付款方法**。
6. 在 [**新增付款方式**] 窗格中，輸入帳戶的資訊，然後選取 [**儲存**]。 您必須從您的租用戶為相同國家/地區使用付款方式。
7. 已在下拉式清單中選取新的付款方式。 選取 [取代]****。

## <a name="change-a-payment-method-for-a-single-subscription"></a>變更單一訂閱的付款方式

您可以變更用來支付單一訂閱的付款方式。

1. 在 [系統管理中心中，移至**帳單** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">產品 & 服務</a>] 頁面。
2. 在 [**訂閱**] 索引標籤中，選取您想要使用的替代的付款方式支付的訂閱。 
3. [**帳單**] 旁的付款方式，選取 [**編輯**]。
4. 旁您現有的付款方式，選取 [**變更**]。
5. 從下拉式清單中，選擇 [替代的付款方式，或選擇 [新增付款方法。
6. 如果您新增付款方法，請輸入卡片或帳戶的詳細資訊，然後選取 [**儲存**]。
7. 確認選取的付款方式是正確，然後選取 [**儲存]**。

## <a name="delete-a-payment-method"></a>刪除付款方式

您只能刪除並不連接到訂閱或帳單的設定檔的付款方式。 這適用於所有的訂閱，無論其狀態。

### <a name="delete-a-payment-method-with-no-subscriptions-or-billing-profiles-attached"></a>刪除任何訂閱或帳單附加的設定檔的付款方式

如果付款方式未與任何訂閱或帳單的設定檔相關聯，您可以立即刪除它。

1. 在系統管理中心中，移至**帳單** > **帳單 & 定額** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">付款方式</a>] 頁面。
2. 尋找要刪除、 選取三個點，然後選取 [**刪除**的付款方式。
3. 在右窗格的底部，選取 [**刪除**]。

### <a name="delete-a-payment-method-with-subscriptions-or-billing-profiles-attached"></a>刪除與帳單附加的設定檔或訂閱的付款方式

如果付款方式附加至任何訂閱或帳單的設定檔，先取代現有的付款方式，或新增一個新然後再刪除舊的付款方式。

1. 在系統管理中心中，移至**帳單** > **帳單 & 定額** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">付款方式</a>] 頁面。
2. 選取要刪除的付款方法的資料列。 右窗格會列出使用該付款方式的現有訂閱。
3. 在右窗格中，選取 [**刪除**]。
4. 若要使用現有的付款方式，從下拉式清單中選取一個選取 [**下一步**，，然後選取 [**刪除**]。
    > [!NOTE]
    > 如果您有訂閱的帳單的設定檔相關聯，您可以僅使用信用卡支付它們。 如果您有在 [**付款方式**] 頁面上所列的銀行帳戶，它們都無法提供選擇下拉式清單中。
5. 若要新增新的付款方式，請選取 [**新增付款方法**。
6. 選擇您想要新增，輸入帳戶的資訊，，然後選取 [**儲存**的付款方式的類型。
7. 已在下拉式清單中選取新的付款方式。 選取 **[下一步]**。
8. 選取 [**刪除**]。

## <a name="troubleshoot-payment-methods"></a>疑難排解付款方式

|**問題**|**疑難排解步驟**|
|:----------|:-----|
|**我收到錯誤訊息，指出、 「 瀏覽器目前設定為封鎖 cookie 」。** |將您的瀏覽器設為允許協力廠商 Cookie，然後再試一次。 |
|**我的信用卡或金融卡遭到拒絕。** |如果您是以信用卡或金融卡支付您的卡片遭到拒絕，您會收到電子郵件通知 Microsoft 無法處理付款。 請仔細檢查，卡詳細資料&mdash;證號碼、 到期日，在卡片上的名稱及地址，包括縣/市、 狀態，以及郵遞區號 — 完全一樣在卡片上及您的陳述式會出現。 您可以更新您的卡片資訊並立即提交使用**支付餘額**連結在訂閱詳細資料] 頁面的 [**計費**] 區段中的 [付款。 如需詳細資訊，請參閱[如果我的信用卡被拒絕且款項逾期是？](pay-for-your-subscription.md#what-if-my-credit-card-was-declined-and-my-payment-is-past-due)  <br/><br/>  如果還是繼續看到 [拒絕] 訊息，請連絡您的銀行。 很可能您卡不是使用中。 如果您最近會收到與更新的到期日期郵件中的卡片，請確定已啟動。 您的銀行也可以告訴您是否您卡未獲核准進行線上、 國際或週期性交易。 |
|**我想要更新的卡片或銀行帳戶號碼。** |您無法變更現有的付款方式卡片或帳戶數。 如果您的卡片或帳戶號碼已經變更，[取代它具有不同的付款方式](#replace-a-payment-method)，這將作用中的所有訂閱的付款方式從都移至新一個，然後[刪除舊的付款方式](#delete-a-payment-method-with-no-subscriptions-or-billing-profiles-attached)。 |
|**我只能有一個卡片或銀行帳戶在 [我的帳戶] 上，我想要將它移除。** |如果您只有一個付款方式，您必須[取代它與新的付款方式](#replace-a-payment-method)之前可以將它刪除。 |
|**我無法新增我的卡片或銀行帳戶。**  |您必須使用發出為您的租用戶相同國家/地區的付款方式。 如果您無法輸入您的卡片或銀行帳戶資訊，您可以[連絡客戶支援](../../admin/contact-support-for-business-products.md)。 |
