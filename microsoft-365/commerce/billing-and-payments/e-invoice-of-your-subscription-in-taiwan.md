# <a name="understand-your-e-invoice-for-microsoft-365-for-business-taiwan"></a>瞭解 Microsoft 365 for business 的電子發票（臺灣）

電子發票是買方進行購買動作，並根據台灣國稅局對電子發票的需求，由賣方所開立的電子發票。為保留記錄，相關資訊將傳送到台灣國稅局的電子發票雲端。有關台灣電子發票的資訊，請參閱此處︰<a href="https://www.einvoice.nat.gov.tw/" target="_blank">財政部電子發票整合服務平台</a>

## <a name="what-is-my-tax-rate"></a>我的稅率為何？

做為商業購買用途，除了適用的稅金外，根據訂閱的報價，台灣國稅局有另外指定的稅金。任何有關稅務的問題或計劃，請洽詢您的稅務顧問。
  
## <a name="when-will-e-invoice-start-for-my-microsoft-365-services"></a>Microsoft 365 服務何時會啟動 e 發票？

Microsoft 會在**9 月15日**將目前的電腦化發票取代為電子發票。 9 月 15 日之前的帳單會直接寄送紙本稅務發票複本給客戶。 在9月15日之後，電子發票將會提供給您的 Microsoft 365 系統管理中心，以供您進行查看、下載和列印，而且不會再直接郵寄給您。 
  
## <a name="where-can-i-find-my-e-invoice"></a>我可以在哪裡找到電子發票？

您可以在您的帳單準備好後，透過您的月帳單來查看、下載和列印您的電子發票。 [查看您的帳單](view-your-bill-or-invoice.md)。
  
## <a name="how-do-i-update-my-vat-id"></a>如何更新 VAT ID？

您可以在上架時更新您的 加值稅 ID，或在您上架後透過系統管理中心更新。
  
在建立帳戶時，在**步驟 1**的 [歡迎] 中，讓我們知道您，**第2頁**，您將在哪裡使用？，在 [**服務收件者位址**資訊] 下，您可以新增8位數的**加值稅 ID**。 如果您沒有 VAT ID，請輸入「00000000」。
  
建立帳戶之後，您可以遵循下列步驟，透過系統管理中心更新 加值稅 ID：
  
1. 在系統管理中心中，移至 [**帳單** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">產品</a>] 頁面。
    
2. 選取訂閱，然後選取 [**其他動作**]。
    
3. 選取 [**更新服務的收件者位址**]，然後更新**稅務識別碼**資訊。 
    
## <a name="how-do-i-request-an-e-invoice-paper-copy"></a>如何要求紙本電子發票？

如果您需要贏得幸運抽獎的收據 (僅提供給沒有 VAT ID 的客戶)，我們會以掛號的方式直接寄送複本到檔案上的地址。
  
否則，如果您需要電子發票的官方紙張副本，請[與支援人員聯繫](../../admin/contact-support-for-business-products.md)。 我們收到您的要求後，會將電子發票紙本寄送到檔案上的地址。
