# <a name="what-happens-if-i-cancel-a-subscription"></a>如果我取消訂閱，會發生什麼情況？

如果您取消您的字詞結尾之前的訂閱，訂閱會直接進入停用狀態。 對大部分國家和地區的多數訂閱來說，停用狀態會持續 90 天。 當訂閱處於停用狀態時，系統管理員仍能存取和備份組織資料，不過我們建議系統管理員在取消訂閱前就先[備份資料](back-up-data-before-switching-plans.md)，尤其那是您唯一訂閱的情況下。 您所留下的任何資料在 90 天後可能會遭到刪除，並且會在取消後的 180 天內全部刪除。 
  
## <a name="what-to-expect-for-you-and-your-users-if-you-cancel-a-subscription"></a>取消訂閱後，您和您的使用者預期會面臨的狀況
  
- **系統管理存取權**系統管理員仍可以登入並存取系統管理中心中，並視需要購買其他訂閱。 全域或計費系統管理員，您有 90 天的時間可以[重新啟動此訂閱](reactivate-your-subscription.md)的所有資料不變。 
    
- **使用者存取**您的使用者將無法使用 OneDrive 等服務商務，或是存取客戶資料，例如電子郵件或小組網站上的文件。 Word 和 Excel 等 Office 應用程式最後都會進入唯讀、精簡功能模式，並顯示[未授權產品通知](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380.aspx)。
    
 **重要** 如果您想要在一般的停用期結束之前刪除您的訂閱資料，可以要求快速撤銷。 當您要求快速撤銷時，您的訂閱資料會在取消後的 3 天內刪除。 若要使用快速撤銷，[連絡支援人員](../../admin/contact-support-for-business-products.md)。
  
若要深入了解，請參閱[我的訂閱結束時，我的資料與存取權會發生什麼事？](what-if-my-subscription-expires.md)
