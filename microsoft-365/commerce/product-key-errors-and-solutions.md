# <a name="problems-with-your-microsoft-365-for-business-product-key"></a>您的商務用 Microsoft 365 產品金鑰有問題？

如果您嘗試兌換商務用 **Microsoft 365** 的產品金鑰時出現錯誤訊息，請嘗試這些常見的解決方案。 
  
 **需要立即協助嗎？** [請打電話給 Microsoft 支援服務](../admin/contact-support-for-business-products.md)。 
  
 **有關 Microsoft 365** 家用版或 Microsoft 365 個人版產品金鑰的協助，請參閱在 Office 使用 [產品金鑰](https://support.microsoft.com/office/12a5763a-d45c-4685-8c95-a44500213759.aspx)。
  
## <a name="product-key-error-help-with-microsoft-365-for-business"></a>商務用 Microsoft 365 的產品金鑰錯誤說明

| 輸入產品金鑰時出現錯誤訊息... | 試試看這個... |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 「很抱歉，這是不正確產品金鑰。 請嘗試再次輸入。 如果您的產品金鑰是 Microsoft 365 個人版或 Microsoft 365 家用版，請于 office.com/setup」。 <br/><br/>如果您使用的是日本版 Office 365 Solo：「很抱歉，這是不正確產品金鑰。 請嘗試再次輸入。 如果您的產品金鑰是 Office 365 Solo 版，請于 office.com/setup 兌換。」 | 如果您要設定 [Microsoft 365](https://support.microsoft.com/office/28cbc8cf-1332-4f04-9123-9b660abb629e.aspx)家用版或個人版，您必須在 [https://www.office.com/setup](https://www.office.com/setup) . 否則，針對商務客戶，請仔細檢查您輸入的數位和字元。 |
| 「您輸入的產品金鑰無效。 再次嘗試輸入」。 或「此產品金鑰無效。 請輸入其他產品金鑰。」 | 請仔細檢查您輸入的數字與字元。特別注意 0 和 O、 5 和 S 以及 l 和 I 等錯誤。如果問題持續發生，請連絡您購得產品金鑰的經銷商。 |
| 「您輸入此產品金鑰了。 輸入其他金鑰」。 | 檢查您已輸入的產品金鑰，確認這組金鑰是否已新增過。如果您要使用產品金鑰續約訂閱，就必須使用全新未使用的產品金鑰。  <br/><br/>需要購買新的金鑰嗎？請造訪 [Microsoft 網上商店](https://go.microsoft.com/fwlink/p/?LinkId=529160)或協力廠商經銷商；或如果您與合作夥伴一同工作，請與您的合作夥伴連絡。  <br/><br/>如果問題持續發生，請 [聯絡客戶支援](../admin/contact-support-for-business-products.md)。 如果您與合作夥伴一同工作，請與您的合作夥伴聯繫，以要求產品金鑰支援。 |
| 「您輸入的產品金鑰已過期。 輸入其他金鑰」。 | 您必須使用全新且未使用的產品金鑰來為目前的訂閱續約，或啟用全新的訂閱。<br/><br/>需要購買新的金鑰嗎？請造訪 [Microsoft 網上商店](https://go.microsoft.com/fwlink/p/?LinkId=529160)或協力廠商經銷商；或如果您與合作夥伴一同工作，請與您的合作夥伴連絡。   |
| 「此產品金鑰已使用。 請輸入其他產品金鑰。」 | 請確認您或貴組織成員尚未使用這組金鑰。如果這組金鑰還未用過，請與您的合作夥伴或是您購買這組產品金鑰的經銷商連絡。 |
| 「很抱歉，我們目前無法處理您的要求。 請稍候幾分鐘，然後再試一次。」 | 如果後續嘗試造成相同錯誤訊息超過 15 分鐘，請 [聯絡客戶支援](../admin/contact-support-for-business-products.md)。 |
| 「未提供要求訂閱。 原因可能是下列其中一個：優惠無法提供 - 這項服務不適用於您的國家/地區 - 同一個試用版無法使用/選取兩次。 如果問題持續發生，請聯絡 Microsoft 支援服務。」 | 請 [聯絡支援服務](../admin/contact-support-for-business-products.md)。 如果您與合作夥伴一同工作，請與您的合作夥伴聯繫以提供支援。 |
| 「您新增的使用者授權數量超過此優惠所允許的數量。 上限是 \<x\> 使用者授權。 移除此產品金鑰，然後輸入新增較少使用者授權的產品金鑰。」 | 請連絡您的經銷商或合作夥伴。 您購買超過此 Microsoft 365 訂閱可用的授權數量。 |
| 「您必須是全域系統管理員或帳單系統管理員才能兌換產品金鑰。」 | 請確定您的許可權已設定為計費或全域系統管理員。若要確認這一點，請以您的公司或學校帳戶註冊  [Microsoft 365，](https://support.microsoft.com/office/e9eb7d51-5430-4929-91ab-6157c5a050b4) 然後前往系統管理中心。 <br/><br/>In the admin center, select **Users** \> **Active users**. In **Filters**, choose **Global admins** or **Billing admins**.  <br/><br/>確認您的名字列在這些結果中。 |
   
> [!NOTE]
> 當您透過合作夥伴購買 Microsoft 365 時，Microsoft 會傳送一封包含指示的電子郵件，在大量授權服務中心取得您的產品金鑰並啟用 [您的訂閱](https://go.microsoft.com/fwlink/p/?LinkID=282016)。 若要深入了解大量授權，請參閱[適用於現有大量授權客戶的說明](https://go.microsoft.com/fwlink/p/?LinkId=534992)。
