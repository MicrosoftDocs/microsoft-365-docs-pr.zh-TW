# <a name="problems-with-your-office-365-for-business-product-key"></a>使用您的商務用 Office 365 產品金鑰時發生問題？

如果當您嘗試兌換商務用**Office 365 適用**的產品金鑰時，沒有錯誤訊息，請嘗試這些常見的解決方案。 
  
 **立即需要協助嗎？** [連絡 Microsoft 支援服務](../admin/contact-support-for-business-products.md)。 
  
 **如需 Office 365 家用版，或個人的產品金鑰的協助**，請參閱[使用 office 產品金鑰](https://support.office.com/article/12a5763a-d45c-4685-8c95-a44500213759.aspx)。
  
## <a name="product-key-error-help-with-office-365-for-business"></a>產品金鑰錯誤的說明與 Office 365 商務版

| 當您輸入您的產品金鑰時，出現錯誤訊息...] | 試試下列方法... |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 「 很抱歉，這是無效的產品金鑰。 請重新輸入。 如果您的產品金鑰是 Office 365 個人或 Office 365 家用版、 兌換它在 office.com/setup。 」 <br/><br/>如果您使用 Office 365 Solo 日本: 「 很抱歉，這是無效的產品金鑰。 請重新輸入。 如果您的產品金鑰是 Office 365 Solo，兌換它在 office.com/setup。 」 | 如果您要設定[為家用 Office 365](https://support.office.com/article/28cbc8cf-1332-4f04-9123-9b660abb629e.aspx)產品，您需要兌換您的產品金鑰，在[https://www.office.com/setup](https://www.office.com/setup)。 否則，用於企業客戶，請仔細檢查數字與您正在輸入的字元。 |
| 「 您輸入的產品金鑰不正確。 請重新輸入。 」 或者 「 此產品金鑰不正確。 請輸入不同的產品金鑰。 」 | 請仔細檢查您輸入的數字與字元。特別注意 0 和 O、 5 和 S 以及 l 和 I 等錯誤。如果問題持續發生，請連絡您購得產品金鑰的經銷商。 |
| 「 您已輸入此產品金鑰。 輸入不同的索引鍵。 」 | 檢查您已輸入的產品金鑰，確認這組金鑰是否已新增過。如果您要使用產品金鑰續約訂閱，就必須使用全新未使用的產品金鑰。  <br/><br/>需要購買新的金鑰嗎？ 請造訪[Microsoft 網上商店](https://go.microsoft.com/fwlink/p/?LinkId=529160)或協力廠商經銷商; 或-如果您正在與合作夥伴-您的合作夥伴連絡。 <br/><br/>如果問題仍然存在，請[連絡客戶支援](../admin/contact-support-for-business-products.md)。 如果您正在與合作夥伴合作，請連絡您的合作夥伴取得產品金鑰支援。 |
| 「 您輸入的產品金鑰已過期。 輸入不同的索引鍵。 」 | 您必須使用全新且未使用的產品金鑰來為目前的訂閱續約，或啟用全新的訂閱。<br/><br/>需要購買新的金鑰嗎？ 請造訪[Microsoft 網上商店](https://go.microsoft.com/fwlink/p/?LinkId=529160)或協力廠商經銷商; 或-如果您正在與合作夥伴-您的合作夥伴連絡。 |
| 「 此產品金鑰已經使用。 請輸入不同的產品金鑰。 」 | 確認該金鑰還未用過由您或貴組織的成員。 如果金鑰還未用過，請連絡您的合作夥伴或是您購得產品金鑰的經銷商。 |
| 「 很抱歉，我們現在無法處理您的要求。 請稍候幾分鐘然後再試一次。 」 | 如果後續嘗試仍收到相同的錯誤訊息中超過 15 分鐘，請[連絡客戶支援](../admin/contact-support-for-business-products.md)。 |
| 「 要求的訂閱是不提供。 下列原因之一，可能造成這： 優惠是無法使用-服務不適用於您的國家/地區-很難使用/選取相同的試用版兩次。 如果問題仍然存在，請連絡 Microsoft 支援服務。 」 | 請[連絡客戶支援](../admin/contact-support-for-business-products.md)。 如果您正在與合作夥伴合作，您的合作夥伴連絡支援人員。 |
| 「 您已新增更多使用者授權超過允許此提供的功能。 最大值是\<x\>使用者授權。 移除此產品金鑰與輸入一個新增較少的使用者授權 >。 | 請連絡您的經銷商或合作夥伴。 您購買的授權已超過此 Office 365 訂閱可用的上限。 |
| 「您必須是全域系統管理員或帳單系統管理員才能兌換產品金鑰。」 | 確定您的權限會被設成 [帳單] 或 [全域系統管理員。若要確認此，[登入 Office 365](https://support.office.com/article/e9eb7d51-5430-4929-91ab-6157c5a050b4)與您的公司或學校帳戶移至 [系統管理中心。 <br/><br/>In the admin center, select **Users** \> **Active users**. In **Filters**, choose **Global admins** or **Billing admins**.  <br/><br/>確認您的名字列在這些結果中。 |
   
> [!NOTE]
> 當您透過合作夥伴購買 Office 365 時，Microsoft 會傳送含有指示的電子郵件給您，方便您了解要如何在[大量授權服務中心](https://go.microsoft.com/fwlink/p/?LinkID=282016)中取得產品金鑰並啟用訂閱。若要深入了解大量授權，請參閱[適用於現有大量授權客戶的說明](https://go.microsoft.com/fwlink/p/?LinkId=534992)。
