# <a name="add-licenses-to-a-subscription-purchased-through-the-volume-licensing-service-center"></a>透過大量授權服務中心新增授權到購買的訂閱

如果您為商務用 Microsoft 365，或透過協力廠商合作夥伴購買 Office 365 企業版計畫，您必須透過該夥伴購買額外的授權。 如果是向 Microsoft 已知的合作夥伴購買訂閱，您可以使用下列程序了解如何連絡您的合作夥伴。
  
::: moniker range="o365-worldwide"

1. 在系統管理中心，移至 **[帳單]** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">[您的產品]</a> 頁面。
::: moniker-end

::: moniker range="o365-germany"

1. 在系統管理中心，移至 **[帳單]** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">[您的產品]</a> 頁面。

::: moniker-end

::: moniker range="o365-21vianet"

1. 在系統管理中心，移至 **[帳單]** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=850626" target="_blank">[您的產品]</a> 頁面。
::: moniker-end

2. 選取您要新增授權的訂閱，然後選取 **大量授權服務中心 (VLSC)** link。

3. 按照 VLSC 中的步驟完成購買。
