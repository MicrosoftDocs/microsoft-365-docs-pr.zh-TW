# <a name="manage-licenses-for-devices"></a>管理裝置的授權

如果您有 Office 365 專業增強版的教育版 （裝置），您可以將授權指派給裝置，使用 Azure AD 群組。 當裝置有授權時，使用該裝置的任何人可以使用 Office 365。 例如，假設您有 20 膝上型電腦和平板電腦以供您組織中的人員。 當您將授權指派給每個裝置時，每個登入到其中一個裝置的人會使用 Office 365，而不需要為他們自己的授權。

> [!IMPORTANT]
> Office 365 專業增強版的教育版 （裝置） 才可教育版 A3 和 A5 大量授權客戶。

若要開始，您會在 Azure Active Directory 系統管理中心中，建立群組，然後再指派給群組的 [裝置。 若要深入了解授權裝置，包括裝置需求，哪些類型的群組，您可以使用，以及如何設定 Office 365 專業增強版要使用的裝置授權，請參閱[裝置授權 Office 365 專業增強版的教育版客戶](https://go.microsoft.com/fwlink/p/?linkid=2094216)。

> [!IMPORTANT]
> 您必須是全域系統管理員，才能完成本文中的工作。

## <a name="assign-licenses-to-devices"></a>將授權指派給裝置

當您將授權指派給群組時，您會將授權指派給群組內的所有裝置中。 您只能指定一個訂用帳戶至任何單一群組。

1. 在 Microsoft 365 系統管理中心，移至**帳單** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842264" target="_blank">授權</a>] 頁面。
2. 在 [**授權**] 頁面上，選擇 [ **Office 365 專業增強版的教育版 （裝置）**。
3. 在**Office 365 專業增強版的教育版 （裝置）** 頁面上，選擇訂閱，然後選擇 [**指派授權**。
4. 在**指派給群組的授權**] 窗格中，開始輸入群組名稱，，然後從結果，以將它新增至清單中選擇。
6. 選擇 [**指派**，然後選擇 [**關閉**]。

## <a name="unassign-licenses-from-devices"></a>取消授權指派從裝置

當您取消授權指派從群組時時，請從群組中的所有裝置中移除授權。 所有應用程式和其相關聯的資料就唯讀屬性。

1. 在 [系統管理中心中，移至**帳單** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842264" target="_blank">授權</a>] 頁面。
2. 在 [**授權**] 頁面上，選擇 [ **Office 365 專業增強版的教育版 （裝置）**。
3. 在**Office 365 專業增強版的教育版 （裝置）** 頁面上，選擇 [訂用帳戶，選擇 [**更多動作**，然後選擇 [**解除指派授權**。
5. 在 [**解除指派授權**] 對話方塊中，選擇 [**解除指派]**。
