# <a name="add-licenses-to-or-extend-a-subscription-paid-for-using-a-product-key"></a>將授權指派給新增或擴充使用產品金鑰支付訂閱費用

預付的授權是以 25 個字元的英數字元代碼發送給您。 您已購買所需的授權之後，您可以將其新增至您的訂閱。 您也可以使用產品金鑰來擴充您的訂閱到期日。

> [!NOTE]
> 如果您不想購買新的產品金鑰，您可以隨時將信用卡或銀行帳戶新增至您的訂閱支付更多授權，或擴充您的訂閱到期日期。 如需詳細資訊，請參閱[更新 Office 365 企業版](../subscriptions/renew-your-subscription.md)。
  
## <a name="add-licenses-to-your-subscription"></a>將授權新增至您的訂閱

::: moniker range="o365-worldwide"

> [!NOTE]
> 如果您使用的不是新的 Microsoft 365 系統管理中心，您可以選取位於首頁頂端的 **[試用新的系統管理中心] **開關將它開啟。

1. 在系統管理中心，移至 [帳單]**** \> [產品與服務]<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank"></a> 頁面。

2. 尋找您想要新增授權的訂閱。 在 [**授權**] 區段中，選取 [**新增更多授權**]。

3. 在**新增更多授權] 窗格中**，選取 [**使用全新且未使用的產品金鑰**，然後選取 [**下一步**。

4. 輸入產品金鑰，然後選取 [**下一步**]。

    > [!NOTE]
    > 如果您有一個以上的產品金鑰，您可以選取來輸入他們的 [**新增其他產品金鑰**。
  
5. 檢閱訂單的詳細資料，然後選取 [**兌換**。

::: moniker-end

::: moniker range="o365-germany"

1. 在系統管理中心，移至 [帳單]**** \> [產品與服務]<a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank"></a> 頁面。

2. 尋找您想要新增授權的訂閱。 選取 [**更多動作**] 下拉式清單中，，然後選擇 [**續約或新增授權權杖**。

3. 在**續約或新增使用者授權**] 頁面上，選取 [**使用全新且未使用的產品金鑰**，然後選取 [**下一步**。

4. 輸入產品金鑰，然後選取 [**下一步**]。

    > [!NOTE]
    > 如果您有一個以上的產品金鑰，您可以選取來輸入他們的 [**新增其他產品金鑰**。

5. 選擇 [新增使用者至訂閱，然後選取 [**下一步**]。

6. 檢閱訂單的詳細資料，然後選取 [**兌換**。

::: moniker-end
