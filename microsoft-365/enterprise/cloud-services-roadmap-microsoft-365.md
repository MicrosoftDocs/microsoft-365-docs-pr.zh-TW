# <a name="cloud-services-roadmap-for-microsoft-365"></a>Microsoft 365 的雲端服務藍圖

若要取得 enterprise Microsoft 365 的創意、團隊合作和生產力優點，請部署最適合組織需求的雲端服務。

如果您有 Exchange、SharePoint 或商務用 Skype 的現有伺服器部署，且想要將整個組織遷移至 Microsoft 365 for enterprise，請參閱[用戶端和伺服器軟體藍圖](client-server-software-roadmap-microsoft-365.md)。 此藍圖包括 Microsoft Office 用戶端產品、內部部署 Office 伺服器產品及 Microsoft Windows 型 &ndash; 裝置。

- [將內部部署資料移轉至 Microsoft 365](migrate-data-to-office-365.md)
- 為您的使用者設定您的雲端服務
  - [Exchange Online](/Exchange/exchange-online)
  - [SharePoint Online](/sharepoint/sharepoint-online)
  - [商務用 Skype](/SkypeForBusiness/skype-for-business-online)
  - [Microsoft Teams](/MicrosoftTeams/Teams-overview)
  - [Yammer](https://support.office.com/article/e1464355-1f97-49ac-b2aa-dd320b179dbe)
- [訓練您的使用者](/office365/admin/admin-overview/get-started-with-office-365#training-resources-for-your-users)

## <a name="how-microsoft-does-cloud-services-for-microsoft-365"></a>Microsoft 如何 Microsoft 365 雲端服務

如需 Microsoft IT 如何部署或管理 Microsoft 365 雲端服務的相關資訊：

1. 移至 [ [MICROSOFT IT] 展示](https://www.microsoft.com/itshowcase)。
2. 選取 [ **搜尋內容**]。
3. 在 [**調整結果**] 底下的 [**物件**] 底下 **Pro** ，然後在 [**產品**] 底下選取雲端服務。
