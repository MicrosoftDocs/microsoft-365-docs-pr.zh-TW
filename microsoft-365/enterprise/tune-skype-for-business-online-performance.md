# <a name="tune-skype-for-business-online-performance"></a>調整商務用 Skype Online 效能

本文提供系統管理員的連結，這些系統管理員想要規劃網路頻寬需求，或需要改善商務用 Skype 線上效能的人員。 
  
## <a name="fine-tuning-skype-for-business-online-performance"></a>商務用 Skype 線上效能調整微調

- **若要瞭解維護組織之通話品質的設定和度量**，請務必閱讀 [商務用 Skype Online 中的媒體質量和網路連線效能](/skypeforbusiness/optimizing-your-network/media-quality-and-network-connectivity-performance)。 如果您想要使用 ExpressRoute （如您將需要的部署類型及其他資訊），本文會提供您的資訊。
    
- **針對內部部署和混合部署**，您可以看到商務用 Skype Server 電話撥入式會議的頻寬需求，請參閱 [商務用 Skype Server 2015 中的電話撥入式會議計畫](/skypeforbusiness/plan-your-deployment/conferencing/dial-in-conferencing)。
    
## <a name="more-ways-to-improve-skype-for-business-online-performance"></a>提高商務用 Skype 線上效能的更多方式

- [商務用 Skype 線上中 ExpressRoute 和 QoS](/skypeforbusiness/optimizing-your-network/expressroute-and-qos-in-skype-for-business-online)：連線使用 Azure Office 365 使用 Azure ExpressRoute，並 Office 365 線上商務用 Skype 專用網路連接。 
    
- [開啟和使用商務用 Skype Online 中的通話品質儀表板](/SkypeForBusiness/using-call-quality-in-your-organization/turning-on-and-using-call-quality-dashboard)：通話品質儀表板 (CQD) 可讓您監視並深入瞭解使用商務用 Skype 服務的通話品質。
