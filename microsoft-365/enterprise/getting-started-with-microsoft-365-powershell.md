# <a name="get-started-with-powershell-for-microsoft-365"></a>開始使用 Microsoft 365 的 PowerShell

*本文適用於 Microsoft 365 企業版和 Office 365 企業版。*

您可以在 Microsoft 365 的 PowerShell 中使用命令和腳本，以管理 Microsoft 365 和簡化您的日常工作。 瞭解 Microsoft 365 的 PowerShell 為何對管理 Microsoft 365 很重要，如何連線至您的 Microsoft 365 訂閱和建立報告，以及從 Microsoft 365 社區取得詳細資訊及協助的位置。
  
請從下列主題中選擇：
  
- [**為什麼您需要使用適用於 Microsoft 365 的 PowerShell**](why-you-need-to-use-microsoft-365-powershell.md)
    
    如果您是 Microsoft 365 PowerShell 的新增，請從這裡開始。 瞭解為何要使用 Microsoft 365 的 PowerShell。
    
- [**使用 PowerShell 連線至 Microsoft 365**](connect-to-microsoft-365-powershell.md)
    
    從這裡開始，使用 Microsoft 365 的 PowerShell 連線到您的 Microsoft 365 訂閱，並從命令列執行系統管理工作。
    
- [**在單一 PowerShell 視窗中連線至所有 Microsoft 365 服務**](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)
    
    您可以在個別 windows 中管理商務用 Skype Online、SharePoint 線上、Microsoft Exchange Online 和 Microsoft 365 帳戶和授權的 Microsoft 365。 或者，您可以從單一視窗進行管理。 本文說明如何進行。
    
- [**在 Microsoft 365 中使用 PowerShell 建立報告**](use-windows-powershell-to-create-reports-in-microsoft-365.md)
    
    如果您已安裝 Microsoft 365 模組的 PowerShell，並想要瞭解如何使用自動化命令快速建立報表，請從這裡開始。
    
- [**Microsoft 365 服務的 Cmdlet 參考資料**](cmdlet-references-for-microsoft-365-services.md)
    
    瞭解 Microsoft 365 模組的 PowerShell Cmdlet。
    
- [**適用於 PowerShell 的 Microsoft 365 社群資源**](microsoft-365-powershell-community-resources.md)
    
    從這裡開始，以連線至 PowerShell 社區，取得使用 PowerShell Microsoft 365 的說明或詳細資訊。
