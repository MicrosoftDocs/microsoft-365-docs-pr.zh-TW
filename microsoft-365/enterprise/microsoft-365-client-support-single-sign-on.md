# <a name="microsoft-365-client-app-support-single-sign-on"></a>Microsoft 365用戶端應用程式支援：單一 Sign-On

*本文適用於 Microsoft 365 企業版和 Office 365 企業版。*

單一登入 (SSO) 會在使用者登入 Azure Active Directory 中的應用程式時，新增安全性和便利性。 使用單一登入時，使用者可以使用一個帳戶登入一次，以存取內部部署 Active Directory 網域服務 (AD DS) 加入網域的裝置、軟體為服務 (SaaS) 應用程式和 web 應用程式。

深入瞭解 [單一登入](/azure/active-directory/manage-apps/what-is-single-sign-on)。

## <a name="supported-clients--platforms"></a>支援的用戶端 & 平臺

下列用戶端和平臺的最新版本支援單一登入。 如需 Microsoft 365 中平臺支援的詳細資訊，請參閱[Microsoft 365 的系統需求](/microsoft-365/microsoft-365-and-office-resources)。
