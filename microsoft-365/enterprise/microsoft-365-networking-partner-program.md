# <a name="microsoft-365-networking-partner-program"></a>Microsoft 365 網路合作夥伴計劃

網路連線對使用者的能力有直接的影響，可讓您快速、有效地共同作業，並使用 Microsoft 365 簡化商務程式。 針對其數位轉換的任何階段中的客戶，網路設計是一個重要的層面，必須先主動解決，才會影響使用者體驗。

當客戶採用 Microsoft 365 的商務生產力時，Microsoft 會發現網路效能及產生的使用者共同作業經驗直接受到使用者與 Microsoft 365 間路徑中的網路解決方案影響的常見趨勢。 為了協助協力廠商設計最佳網路解決方案，以及協助客戶針對這類解決方案做出明智決策，我們建立了 Microsoft 365 網路合作夥伴計畫。

Microsoft 365 網路合作夥伴計畫 deepens 我們與網路合作夥伴共同作業，並找出 Microsoft 365 遵循網路需求、建議及最佳作法的重要產品及解決方案。 Microsoft 365 網路合作夥伴計畫的目標是協助客戶透過輕鬆探索已驗證的協力廠商解決方案，改善其 Microsoft 365 體驗，以取得客戶部署中最佳 Microsoft 365 連線的最佳方式。

若要將商業網路現代化，以取得 Microsoft 365 的良好連線能力，客戶常常依靠網路解決方案提供者、內部部署或雲端式安全性服務，以及系統整合者來規劃、設計及實施雲端服務的網路連線能力。 客戶通常會詢問 Microsoft 其網路架構和解決方案是否與 Microsoft 365 搭配使用，以及是否與 microsoft 的[Microsoft 365 的網路連接原則](./microsoft-365-network-connectivity-principles.md)相符。

Microsoft 365 網路合作夥伴計畫協助您回答這些問題，並示範 Microsoft 對於協助我們的客戶建立及優化其網路架構以取得最佳 Microsoft 365 體驗的承諾。 Microsoft 365 小組與許多網路行業合作夥伴合作，以協助確保在其網路產品及解決方案中內部內置的最佳連線能力重要原則。

Microsoft 365 網路合作夥伴計畫會將合格的網路解決方案指定為「搭配 Microsoft 365 運作」。 僅適用于符合 Microsoft 嚴格測試需求的裝置或解決方案，可順利體驗安裝程式，並示範使用 Microsoft 365 時所顯示的高品質網路體驗，將會收到「使用 Microsoft 365」。 這些已驗證的協力廠商解決方案會列于 [ [Microsoft 365 網路合作夥伴計畫] 頁面](https://www.microsoft.com/microsoft-365/partners/O365networkingpartners)上，以及可深入瞭解之解決方案的詳細資訊和連結。

如果您是 Microsoft 365 客戶，可透過此頁面讓您的網路解決方案提供者知道此程式 [https://aka.ms/Office365NPP]() 。

若要查詢 Microsoft 365 網路合作夥伴計畫的網路解決方案提供者，請填寫表單 [https://aka.ms/NPPsignup](https://aka.ms/NPPsignup) 。 謝謝您的利益，我們期待取得可愛的合作關係。
