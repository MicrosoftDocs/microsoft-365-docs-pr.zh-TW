# <a name="step-4-simplify-password-resets"></a>步驟 4：簡化密碼重設

*此為選用步驟，且同時適用於 Microsoft 365 企業版 E3 和 E5 版本*

![](./media/deploy-foundation-infrastructure/identity_icon-small.png)

在本步驟中，您會啟用自助密碼重設 (SSPR)，讓使用者可重設或將其密碼或帳戶解除鎖定。若要在誤用或濫用時收到警訊，您可以使用在使用者存取系統時會進行追蹤的詳細報告以及通知。

請參閱[啟用密碼重設的指示](https://docs.microsoft.com/azure/active-directory/active-directory-passwords-best-practices)。
