# <a name="step-4-monitor-device-health-and-compliance"></a>步驟 4：監控裝置健全狀況和合規性

![](./media/deploy-foundation-infrastructure/win10enterprise_icon-small.png)

Windows 分析使用診斷資料提供豐富、 可採取的資訊可協助您獲取深層洞察力到的操作效率和您環境中的 Windows 10 裝置的狀況。

Windows 分析裝置狀況功能提供主動式觀點來協助偵測並修復問題影響的使用者。此新服務使用診斷資料提供這類前瞻沒有其他基礎結構需求。主動補救相關使用者問題可讓您降低支援成本並提高效率。尋找推出釋的其他功能、 其增強功能與此新服務的值。

請參閱了解更多、 快速入門及使用裝置的健全狀況[監視的裝置與裝置健全狀況](https://docs.microsoft.com/windows/deployment/update/device-health-monitor)。

作為過渡期的檢查點，您可以看到此步驟的[允出準則](windows10-exit-criteria.md#crit-windows10-step4)。
